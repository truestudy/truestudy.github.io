10. Suatu perusahaan elektronik menghasilkan dua Macam lampu bohlam,yaitu lampu bohlam biasa dan lampu bohlam hemat,masing-masing memerlukan dua tahap pemrosesan,lampu bohlam biasa memerlukan 15 menit proses pertama dan 10 menit proes kedua.Lampu bohlam hemat energi memerlukan 18 menit proses pertama dan 12 menit proses kedua.Keuntungan bagi setiap lampu bohlam adalah 1.500.00 untuk lampu bohlam biasa dan 3.500.00 bagi lampu bohlam hemat energi.Jika terdapat 15 jam kerja dipemrosesan pertama dan 10 jam dalam pemrosesan kedua maka hitunglah masing-masng jenis lampu bohlam yang akan memaksimalkan keuntungan?


Jawaban:

PROGRAM LINEAR

Sebuah perusahaan bola lampu menggunakan 2 jenis mesin. untuk membuat bola lampu jenis A memerlukan waktu 3 menit pada mesin I dan 5 menit pada mesin II. bola lampu jenis B memerlukan waktu 2 menit pada mesin I dan 7 menit pada mesin II. jika mesin I bekerja 1820 menit dan mesin II bekerja 4060 menit, model matematikadari permasalahan tersebut adalah ...

Jawaban

Pendahuluan

Program linear adalah salah satu bagian dari matematika terapan yang digunakan untuk memecahkan masalah pengoptimalan (memaksimalkan atau meminimalkan suatu tujuan).

Pembahasan

Diketahui :

Bola lampu jenis A memerlukan waktu 3 menit mesin I dan 5 menit mesin II.

Bola lampu jenis B memerlukan waktu 2 menit pada mesin I dan 7 menit mesin II.

jika mesin I bekerja 1820 menit dan mesin II bekerja 4060 menit

Ditanya :

Model matematika dari permasahan

Jawab :

Misalkan : x = bola lampu jenis A

                 y = bola lampu jenis B

Untuk menyusun model matematikanya dapat dibuat tabel seperti berikut.

Mesin   lampu A   lampu B    waktu

  I              3x            2y          1820

 II              5x            7y          4060

Pertidaksamaan untuk mesin I :

3x + 2y ≤ 1820

Pertidaksamaan untuk mesin II :

5x + 7y ≤ 4060

Banyak bola lampu tidak mungkin negatif sehingga

x ≥ 0  dan  y ≥ 0

Kesimpulan

Jadi model matematika dari permasalahan tersebut adalah 3x + 2y ≤ 1820  ;   5x + 7y ≤ 4060  ;  x ≥ 0  ;  y ≥ 0

12. Sebuah kardus berisi 30 bohlam lampu.terdapat 4 bohlam lampu yang mati.jika dikeluarkan dua bohlam lampu satu demi satu tanpa pengembalian,peluang kedua duanya terambil bohlam lampu mati adalah​


Peluang kedua duanya terambil bohlam lampu mati adalah​ [tex]\frac{12}{870}[/tex]

Pembahasan

Peluang adalah harapan terjadinya suatu kejadian dengan dikuantitatifkan

Rumus peluang ↓

P(A) = [tex]\frac{n(A)}{n(S)}[/tex]

Frekuensi harapan adalah hasil kali antara peluang dengan banyaknya percobaan

Rumus Frekuensi Harapan ↓

FH = P(A) x banyak percobaan

14. sebuah perusahaan elektronik menghasilkan 2 macam lampu bohlam,yaitu lampu bohlam biasa dan lampu bohlam hemat energi. masing masing lampu memerlukan 2 tahapan pemrosesan. lampu bohlam biasa memerlukan 60 menit proses pertama dan 30 menit proses kedua. lampu bohlam hemat energi memerlukan 40 menit proses pertama dan 60 menit proses kedua. keuntungan setiap unit lampu adalah 5500 untuk lampu bohlam biasa dan 9500 untuk lampu bohlam hemat energi. jika terdapat 90 jam kerja proses pertama dan 160 jam kerja di proses kedua, tentukanlah keuntungan maksimum yang diperoleh perusahaan tersebut dan banyaknya masing masing jenis lampu yang terjual. dan gambarkan daerah himpunan penyelesaiannya​


Dimisalkan :

x = lampu bohlam biasa

y = lampu bohlam hemat energi

Proses pertama :

(60 menit).(x) + (40 menit).(y) ≤ 90 jam

(1 jam).(x) + (⅔ jam).(y) ≤ 90 jam

« kedua ruas dikali 3 »

(3 jam).(x) + (2 jam).(y) ≤ 270 jam

3x + 2y ≤ 270 ... ( garis i )

Proses kedua :

(30 menit).(x) + (60 menit).(y) ≤ 160 jam

(½ jam).(x) + (1 jam).(y) ≤ 160 jam

« kedua ruas dikali 2 »

(1 jam).(x) + (2 jam).(y) ≤ 320 jam

x + 2y ≤ 320 ... ( garis ii )

Didapatkan sistem pertidaksamaan :

3x + 2y ≤ 270

x + 2y ≤ 320

x ≥ 0

y ≥ 0

Daerah Hasil Penyelesaian (DHP) terlampir di gambar.

Dari gambar, didapatkan titik selidik :

( 90 , 0 ) dan ( 0 , 135 )

Fungsi tujuan : f(x,y) = 5500x + 9500y

Menghitung nilai maksimum :

» Untuk titik ( 90 , 0 ) :

f(x,y) = 5500x + 9500y

f(90,0) = 5500.(90) + 9500.(0)

f(90,0) = 495000 + 0

f(90,0) = 495.000

» Untuk titik ( 0 , 135 ) :

f(x,y) = 5500x + 9500y

f(0,135) = 5500.(0) + 9500.(135)

f(0,135) = 0 + 1282500

f(0,135) = 1.282.500

Maka nilai maksimum yang dicapai : 1.282.500, dan tercapai pada titik ( 0 , 135 )

Kesimpulan :

Agar perusahaan mencapai keuntungan maksimum, perusahaan harus memproduksi hanya lampu bohlam hemat energi saja sebanyak 135 unit. Dan keuntungan yang akan diperoleh adalah Rp 1.282.500,00
