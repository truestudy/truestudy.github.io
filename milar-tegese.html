17. Volume milar gas yang terdapat dalam 400 gram Br kuadarat (STP) jika di ketahui Ar Br = 80 adalah


m = 400 gram
Mr = 2. Ar = 2.80 = 160
Dit = volume dalam keadaan STP

Langkah pertama cari mol (n) nya dulu
n = m/Mr
= 400 / 160 = 2,5 mol

V stp = n x 22,4
= 2,5 x 22,4
= 56

20. apa fungsi: kapasitor milar kapasitor polyester kapasitor elektrolit kumparan osilator kumparan RFC transformator input transformator stavol transformator UPS


1. Semua kapasitor fungsinya sama, yaitu untuk mengimpan energi yang berlebih.
2. Fungsi kumparan adalah untuk menghasilkan frekuensi. Sering ditemukan pada komponen radio
3. Transformator input berfungsi untuk impedansi masukan dan keluaran ada rangkaian penguat frekuansi audio amplifier,Transformator input juga berfungsi sebagai pembelah fasa sinyal AC (alternating current) informasi di bagian keluarannya(output)
4. Transormator stavol digunakan untuk menstabilkan tegangan
5. Transformator UPS fungsinya sama dengan stavol tetapi bisa untuk menyimpan energi
