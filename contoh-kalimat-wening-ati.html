22. jumlah uang wening di tambah 3 kali uang noni adalah rp 35.500 sedangkan 2 kali uang wening di tambah 4 kali uang noni adalah 50.000 tentukan besar uang wening dan noni


uang wening = x
uang noni = y

x + 3y = 35.500   →persamaan I
2x + 4y = 50.000 →persamaan II

eliminasi variable x

x + 3y = 35.500    → dikali 2 →   2x + 6y = 71.500
2x + 4y = 50.000  → dikali 1 →   2x + 4y = 50.000  -
                                                           2y = 21.000
                                                             y = 21.000
                                                                        2
                                                             y = 10.500

substitusi nilai y ke persamaan I

x + 3y = 35.500
x + 3(10.500) = 35.500
x + 31.500 = 35.500
              x  = 35.500 - 31.500
              x  = 4.000

jadi :
uang Wening = Rp4.000,-
uang Noni = Rp10.500,-
