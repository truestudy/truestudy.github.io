2. Tampilan slide untuk menjalankan presentasi adalah


Tampilan slide untuk menjalankan presentasi adalah slide show. Untuk menampilkan presentasi dapat melalui tombol From Beginning. Presentasi juga dapat dilampilkan dengan menekan tombol F5 pada keyboard. Presntasi juga dapat ditampilkan melalui icon slide show.

Pembahasan

Presentasi dapat dibuat dan ditampilkan melalui program aplikasi pengolah prensentasi, yang umum digunakan adalah microsoft powerpoint.

Pada microsoft powerpoint, tempat untuk membuat presentasi disebut dengan slide. Slide presentasi, selain di tampilkan juga dapat dicetak.

Tool slide terdapat pada Ribbon Tab Home. Pada group slide terdapat tombol add slide, layout, reset dan delete. Fungsi dari tool group ini untuk menambah slide, membuat layout dari slide dan menghapus slide.

Presentasi yang telah dibuat, dapat ditampilkan melalui slide show. Ribbon tab slide show terdiri dari tiga tool group yaitu:

Start Slide Show, untuk menentukan dari mana slide akan dijalankan.Set Up, untuk menyembunyikan slide, merekam narasi , dan menentukan urutan slide yang akan ditampilkan.Monitors untuk mengatur resolusi slide presentasi.

4. Sebutan 4 jenis tampilan slide presentasi


Empat jenis tampilan slide presentasi adalah:

Normal viewSlide Sorter ViewReading ViewSlide Show

Tampilan-tampilan tersebut disesuaikan dengan kebutuhan dari presentator dan audience.

Pembahasan

Slide presentasi seringkali kita buat pada Microsoft Power Point. Berikut adalah beberapa jenis tampilan slide presentasi pada Microsoft Power Point:

1. Normal View

Merupakan tampilan dasar pada Microsoft Power Point. Kita bisa membuat, menghapus, mengedit, mengatur slide, serta juga memodifikasi konten.

2. Slide Sorter View

Biasanya kita gunakan untuk mengatur urutan slide karena pada tampilan ini slide ditampilkan secara urutan dari kiri ke kanan dalam bentuk thumbnail slide.

3. Reading View

Slide ditampilkan dengan ukuran besar dan terdapat tombol kontrol untuk berpindah slide. Sehingga kita dapat mengoreksi file presentasi meskipun tidak bisa langsung mengedit pada Reading View. Untuk mengedit isi slide kita harus beralih ke Normal View.

4. Slide Show

Tampilan ini digunakan untuk menampilkan hasil presentasi yang telah kita buat dengan full screen. Kita juga dapat dengan leluasa memindahkan kepada slide sebelumnya atau berikutnya menggunakan pointer.

Penjelasan:

Kalau memakai keyboard, tombol yg perlu ditekan adalah:

Page down: menggulung tampilan slide ke bawah

Page up: menggulung tampilan slide ke atas.

Kalau memakai mouse, tampilan slide bisa dinaik-turunkan dengan Vertical Scrollbar. (bukan tombol)

Vertical Scrollbar disebut juga baris penggulung tegak vertikal. Berguna untuk menggulung tampilan slide secara vertikal pada tampilan presentasi Microsoft Powerpoint​.

Dalam hal ini berarti, tampilan bisa digulung dari atas ke bawah maupun dari bawah ke atas.


20. Sebutan 4 jenis tampilan slide presentasi ... *


Jawaban:

1. Ribbon Tab Home adalah menu yang paling pertama yang didalamnya terdapat 4 (empat) grup tool dengan fungsinya masing-masing yaitu:

1. Clipboard

2. Slides

3. Font

4. Paragraph

5. Drawing

5. Editing.

Grup tool ini berfungsi;

Clipboard, memungkinkan Anda menggunakan tombol copy, paste, cut, dan format painter. Slide, didalamnya terdapat tombol add slide, layout, reset dan delete. Paragraph, didalamnya terdapat tombol untuk mengatur posisi rataan (alignment), bullet and numbering, line spacing, dan beberapa tombol untuk mengatur paragraph. Drawing, didalamnya terdapat tombol Text box, Austoshape, Arrange, Quick Styles, Shape Fill, Shape Outline, dan Shape Effects. Editing, didalamnya terdiri dari tombol Find, Replace, dan Select.

2. Ribbon Tab Insert 

Ribbon tab Insert bertujuan untuk memasukkan sesuatu obyek dalam slide presentasi. Tab Insert terdiri dari beberapa group tool, antara lain:

1. Tables, perintah untuk menambahkan table pada tampilan slide Anda.

2. Images, untuk memasukan gambar, clipt art, screenshot dan photoAlbum

Illustrations, didalamnya terdapat tombol yang bisa Anda gunakan untuk menyisipkan gambar, clipart, photo album, shapes, smartart, dan chart (grafik).

3. Links, meliputi tombol yang dapat digunakan untuk membuat link pada slide Text, memasukan teks box, header dan footer, WordArt, Tanggal dan Jam, Penomoran slide, dan objek dari luar aplikasi.

4. Symbol, memasukan simbol dan menulis rumus Media Clips, untuk menambahkan file sound (suara) atau movie (film) dalam slide.

3. Ribbon Design

Ribbon tab Design, berkaitan dengan desiain slide pada presentasi. Tab Design terdiri dari beberapa tool group, antara lain;

1. Page Setup,

2. Themes,

3. Background,

berfungsi untuk mendesain slide Anda. Berikut adalah penjelasannya : 

- Page Setup,didalamnya terdapat tombol untuk mengatur model dari slide, ukuran slide dan apakah model portrait atau landscape.

- Themes, berkaitan desain tema yang sudah disiapkan oleh Ms. PowerPoint 2010.

- Background, untuk mengatur latar belakang agar slide tampak lebih hidup dan indah, kita juga bisa mengatur, memasukan background secara mandiri disini.

4. Ribbon Tab Animations 

Pada Ribbon Tab Animations mengatur animasi dalam presentasi. Melalui tab ini Anda bisa menambahkan berbagai macam bentuk animasi pada slide Anda. 

Terdapat beberapa tool group yang bisa Anda gunakan, antara lain ;

1. Preview, tombol ini biasa juga disebut pratayang atau melihat sementara hasil dari animasi yang Anda berikan untuk slide Anda.

2. Animations, Anda dapat memilih animasi bagi objek yang ada pada slide, terdiri dari animate dan custom animations.

3. Advanced animation, tambahan animasi lebih lanjut Timing, untuk mengatur durasi slide dan urutan.

5. Ribbon Slide Show

Ribbon Tab Slide Show mengatur bagaimana slide ditampilkan dalam presentasi, termasuk didalamnya misalnya dari slide berapa yang akan pertama ditampilkan. Menu Slide Show terdiri dari beberapa tool group, antara lain;

1. Start Slide Show, untuk menentukan dari mana slide Anda dijalankan, apakah dari awal (from beginning) atau dari slide yang sedang aktif (from current slide show) atau pilihan Anda sendiri (custom slide show).

2. Set Up, pada tool group ini terdapat tombol yang bisa digunakan untuk menyembunyikan slide (hide slide), merekam narasi (record narration) dan menentukan urutan slide (rehearse timings) yang akan ditampilkan. 

3. Monitors, Anda bisa mengatur resolusi dari slide presentasi Anda pada tool group ini.

6. Ribbon Review

Sesuai dengan namanya, maka Review digunakan untuk mereview slide presentasi sebelum disimpan, dicetak atau dipresentasikan. Terdapat 3 tool group pada Ribbon Tab Review ini, antara lain;

1. Proofing, digunakan untuk melakukan pengecekan pada tata tulis yang Anda buat di slide.

2. Language, Comments, Anda bisa memberikan catatan pada slide yang Anda buat

3. Compare, untuk pengaturan comment.
