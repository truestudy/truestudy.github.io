= 3√2

= 2√3 + 3√3 - √9×2

= (2√3 + 3√3) - 3√2

= 5√3 - 3√2

3. 3√5 + √20 - ⅓√45

= 3√5 + √4×5 - ⅓√9×5

= 3√5 + 2√5 - ⅓.3√5

= 3√5 + 2√5 - 1√5

= 4√5

4. √3 + √27 - √48

= √3 + √9×3 - √16×3

= √3 + 3√3 - 4√3

= 0

5. 3√24 - √150 + 2√54

= 3√4×6 - √25×6 + 2√9×6

= 3.2√6 - 5√6 + 2.3√6

= 6√6 - 5√6 + 6√6

= 7√6
