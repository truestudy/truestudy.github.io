19. Jika sin t=5/12 maka sec t adalahA.13/√119 B.12/√119 C.5/√119D.√119/5 E.13/12


Jawab : sec t = 12 / √119 Penjelasan dengan langkah-langkah:

Pendahuluan :  

Misalkan diberikan sebuah segitiga ABC dengan sudut lancip di A

sin A = sisi depan / sisi miring

cos A = sisi samping / sisi miring

tan A = sisi depan / sisi samping

cotan A = 1 / tan A

sec A = 1 / cos A

cosec = 1 / sec A

Diketahui dari soal :

Misalkan Sin t = 5 / 12

Ditanya :  

Sec t

Jawab :  

Sin t didapat dalam sebuah segitiga siku - siku

Perhatikan gambar terlampir  !!!

Misalkan terdapat segitiga TUV siku - siku di U

Maka,

Karena sin T = 5 / 12 dan sit T = UV / TV, maka UV = 5 dan TV = 12

TU = √( TV² - UV² ) = √( 12² - 5² ) = √( 144 - 25 ) = √119

Maka,

cos T = TU / TV = √119 / 12

Tan T = sin T / cos T = ( 5 / 12 ) / (√119 / 12) = (5 / 12) x (12 / √119) = 5 / √119

Sec T = 1 / cos T = 1 / ( √119 / 12 ) = 1 x ( 12 / √119 ) = 12 / √119

Kesimpulan :

Jadi, sec t = 12 / √119
