Jawaban:

Penjelasan dengan langkah-langkah:

[tex] \sum_{n = 3}^{11} [3n + 2] = \sum_{n = 1}^9[3(n+2)+2] [/tex]

[tex] \sum_{n = 3}^{11} [3n + 2] = \sum_{n = 1}^9[3n+8] [/tex]

[tex] \sum_{n = 3}^{11} [3n + 2] = 3 \cdot \sum_{n = 1}^9 n + 72 [/tex]

[tex] \sum_{n = 3}^{11} [3n + 2] = 3 \cdot \frac{9(9+1)}{2} - 72 [/tex]

[tex] \sum_{n = 3}^{11} [3n + 2] = 135 + 72 [/tex]

[tex] \sum_{n = 3}^{11} [3n + 2] = 207 [/tex]
