Jawaban:

1. Basa Ngoko : Bapak adus jam loro.

Basa Kromo : Bapak adus jam kalih.

Basa Kromo Inggil : Bapak siram jam kalih.

2. Basa Ngoko : Fara nggawa jeruk.

Basa Kromo : Fara mbektha jeram.

Basa Kromo Inggil : Fara ngasta jeram.

3. Basa Ngoko : Adit kirim sendhok.

Basa Kromo : Adit kintun sendhok.

Basa Kromo Inggil : Adit kintun lantaran.

ma'af klo slh:)

›› Gawe ukara:

No 1.

- Ngoko = Mas Bagas adus ing kali.

- Krama = Mas Bagas adus ten lepen.

- Krama inggil = Mas Bagas siram wonten lepen.

No 2.

- Ngoko = Bapak mulih nggawa jajan.

- Krama = Bapak wangsul mbektha jajan.

- Krama inggil = Bapak kondur ngasta jajan.

No 3.

- Ngoko = Juleha kirim dhuwit kanggo ibune.

- Krama = Juleha kintun yatra kangge ibune.

- Krama inggil = Juleha kintun arta kagem ibunipun.

Pembahasan

Unggah-ungguhing basa umume kaperang dadi loro. Yaiku basa ngoko lan basa krama.

1. BASA NGOKO

Ngoko lugu → Yaiku basa sing tembung tembunge nggawe basa ngoko kabeh.Ngoko alus → Yaiku basa sing tembung tembunge kecampuran ngoko lan krama alus.

2. BASA KRAMA

Krama lugu → Yaiku basa sing tembung tembunge nggawe basa krama lugu kabeh.Krama alus → Yaiku basa sing tembung tembunge nggawe krama alus kabeh lan basane luwih ngajeni.

Tuladhane:

Basa ngoko → adus, nggawa, kirim, turu, tuku, lan sapanunggalane.

Basa krama lugu → adus, mbektha, kintun, tilem, tumbas lan sapanunggalane.

Basa krama alus → siram, ngasta, kintun, sare, mundhut lan sapanunggalane.
