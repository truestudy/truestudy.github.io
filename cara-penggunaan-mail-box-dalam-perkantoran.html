23. cara mencetak semua undangan yang telah di buat dengan menggunakan mail merge adalah


Pada Merge To print terdapat dialog ALL dimana dialog tersebut digunakan untuk mencetak semua undangan yang telah di buat dengan menggunakan mail merge. Selain itu dialog All juga berguna untuk menggabungkan semua record yang telah ada.

Pembahasan :

    Penggabungan email (Mail Merge) adalah fitur yang memungkinkan pengguna untuk membuat surat, undangan, format amplop, sertifikat, dokumen, dan lainnya secara ringkas.  Mail Merge memungkinkan pengguna untuk bekerja lebih cepat pada dokumen yang berisi format yang sama bersama-sama dan memiliki ID yang berbeda. Oleh karena itu, pengguna tidak perlu mengedit dokumen dalam format yang sama satu per satu. Untuk membuat gabungan sebuah email, pengguna harus memiliki setidaknya dua file. File pertama yaitu adalah database  Microsoft Excel dan file kedua adalah format template utama pada Microsoft Word.

Langkah–langkah dalam mencetak mail merge :

Klik tab Mailings yang berada pada grup Finish, klik icon Finish & Merge.Kemudian klik pilihan Print Document. Pada fiture kotak dialog dapat terlihat beberapa tombol, seperti : All yang digunakan untuk menggabung ataupun mencetak seluruh record yang telah ada, Current Record yang digunakan untuk menggabung ataupun mencetak data secara khusus record dan data yang masih aktif dan sedang dipilih, From…To.... yang digunakan untuk menggabung ataupun mencetak data data tertentu yang mulai dari (From) sampai dengan (To).Pada bagian kotak dialog diatas, kita tentukan record atau data yang akan kita cetak, misalnya memilih pilihan All gunanya agar mencetak seluruh data ke printer.Klik tombol OK atau selesai.Pelajari Lebih Lanjut Materi tentang Mail Marge brainly.co.id/tugas/2585890 Materi tentang Kegunaan Mail Marge brainly.co.id/tugas/3475553 Materi tentang fungsi Mail Marge brainly.co.id/tugas/640413Detail Jawaban
