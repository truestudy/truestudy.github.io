5. bapak pocung cangkeme madhep mandhuwur, sabamu ing sendhang, pencokanmu lambung kering prapteng wisma, si pocung muntah kuwaya


Jawaban:

Klenthing / Wadah air

Penjelasan:

Arti Cangkriman tersebut:

Bapak pocung, mulutmu menghadap ke atas,

Tempat bermainmu di mata air,

Tempatmu di lambung sebelah kiri,

Sesampai di rumah, Si pocung mengeluarkan air.

Jadi, jawaban dari cangkriman yaitu Klenthing / Wadah air

Wangsuli pitakon

Bapak pucung, dudu watu dudu gunung, sabamu ing sendhang, pencokanmu lambung kering, prapteng wisma, si pucung muntah kuwaya.

→ bedhekane = klenthing

__________________Pembahasan

Tembang Pucung ing dhuwur iku salah saijining tembang macapat, sing nduweni watak seneng lan gembira.

Tembang Pucung nduwei paugeran :

guru gatra = 4guru wilangan = 12, 6, 8, 12guru lagu = u, a, i, a

1. bapak pucung, dudu watu dudu gunung, ( 12 )

2. sabamu ing sendhang, (6 )

3. pencokanmu lambung kering, (8 )

4. prapteng wisma, si pucung muntah kuwaya. ( 12 )

Tuladha liya ing tembang Pucung.

bapak Pucung Dudu watu Dudu gunung, sangkamu ing Sabrang, ingon ingone sang bupati, Yen lumampah si Pucung lembehan Grana....( gajah ).bapak Pucung renteng renteng kaya kalung, dawa kaya ula, pencokamu Wesi miring, sing disaba si Pucung mung turut kutha ....( sepur ).
