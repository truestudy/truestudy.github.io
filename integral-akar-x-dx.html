29. integral dari akar x dx


Integral dari akar x dx adalah ⅔ x√x + C. Integral adalah anti turunan atau lawan dari turunan. Bentuk umum integral tak tentu adalah ∫ f’(x) dx = f(x) + C. Rumus dasar Integral:

∫ axⁿ dx = [tex]\frac{a}{n+1} x^{n + 1}[/tex] + C, dengan n ≠ –1  

Jika n = –1, maka

∫ ax⁻¹ dx = a ln |x| + C


Pembahasan

∫ √x dx

= [tex]\int {x^{\frac{1}{2}}} \: dx[/tex]

= [tex]\frac{1}{\frac{1}{2}+1} x^{\frac{1}{2} + 1}[/tex] + C

= [tex]\frac{1}{\frac{3}{2}} x^{\frac{3}{2}}[/tex] + C

= [tex]\frac{2}{3} x^{\frac{3}{2}}[/tex] + C

= [tex]\frac{2}{3} x \sqrt{x}[/tex] + C
