5. dengan induksi matematika rumus deret 3+8+8+13+18+...+(5n-2) adalah​


Terdapat deret: 3+8+13+18+...+(5n-2). Deret tersebut memiliki rumus ½n(5n+1). Dengan induksi matematika, deret dan rumusnya terbukti benar. Silakan simak penjelasan di bawah.

Penjelasan dengan langkah-langkah

Umumnya, induksi matematika membuktikan pernyataan lengkap. Namun, pada soal, tidak diberikan rumus deretnya. Akan ditentukan rumus deretnya terlebih dahulu dengan rumus deret aritmatika.

Diketahui: deret: 3+8+13+18+...+(5n-2)

Ditanya: rumus deret dan bukti dengan induksi matematika

Jawab:

Rumus deret

Sn = ½n(U₁+Un) = ½n(3+5n-2) = ½n(5n+1)

Jadi, rumus deret tersebut adalah ½n(5n+1).

Pernyataan

P(n): 3+8+13+18+...+(5n-2) = ½n(5n+1)

Akan dibuktikan dengan induksi matematika.

Langkah awal

Akan dibuktikan untuk n = 1.

Ruas kiri:

5·1-2 = 5-2 = 3

Ruas kanan:

½·1(5·1+1) = ½(5+1) = ½·6 = 3

Karena 5·1-2 = ½·1(5·1+1), maka pernyataan terbukti untuk n = 1.

Langkah induksi

Asumsikan pernyataan benar untuk n = k, yaitu:

3+8+13+18+...+(5k-2) = ½k(5k+1)

Akan dibuktikan untuk n = k+1, yaitu:

3+8+13+18+...+(5k-2)+(5(k+1)-2) = ½(k+1)(5(k+1)+1)

Akan dibuktikan dari ruas kiri.

3+8+13+18+...+(5k-2)+(5(k+1)-2)

= ½k(5k+1)+(5(k+1)-2)

= ½k(5k+1)+(5(k+1)-2)

= ⁵⁄₂k²+½k+(5k+5-2)

= ⁵⁄₂k²+½k+5k+3

= ½(5k²+k+10k+6)

= ½(5k²+11k+6)

= ½(k+1)(5k+6)

= ½(k+1)(5k+5+1)

= ½(k+1)(5(k+1)+1)

Karena 3+8+13+18+...+(5k-2)+(5(k+1)-2) = ½(k+1)(5(k+1)+1), maka pernyataan terbukti untuk n = k+1. Jadi, P(n) terbukti.

15. Buktikan dengan induksi matematika bahwa :2³+4³+6³+...+ (2n)³ = 2n² (n+1)²​


Materi : Induksi MatematikaKelas : 11MAPEL : Matematika Wajib

Buktikan dengan kebenaran

2³ + 4³ + 6³ + ... + (2n)³ = 2n²(n+1)²

Untuk n = 1

2³ = 2(1)²(1+1)²

2³ = 2(2)²

8 = 8 (terbukti benar)

Asumsikan bahwa benar untuk n = k

2³ + 4³ + 6³ + ... + (2k)³ = 2k²(k+1)²

Untuk n = k + 1

= 2³ + 4³ + 6³ + ... + (2k)³ + (2(k+1))³=2(k+1)²((k+1)+1)²

= 2³ + 4³ + 6³ + ... + (2k)³ + (2k+2)³ = 2(k+1)²(k+2)²

= (2k²(k+1)²) + (2k+2)³ = 2(k+1)²(k+2)²

= (2k+2)³ = 2(k+1)²(k+2)² - 2k²(k+1)²

= (2(k+1))³ = (k+1)²(2(k+2)²-2k²)

= 8(k+1)³ = (k+1)²(2(k+2)²-2k²)

= 8(k+1) = 2(k+2)² - 2k²

= 8(k+1) = 2(k² + 4k + 4) - 2k²

= 8k + 8 = 2k² + 8k + 8 - 2k²

= 8k + 8 = 8k + 8 (terbukti benar)

Jadi, persamaan

2³ + 4³ + 6³ + ... + (2n)³ = 2n²(n+1)²

telah terbukti benar untuk semua n bilangan asli.

18. Hasil dari 4 6 8 ... (2n 2)=n2 3n dalam induksi matematika


Diketahui [tex]4+6+8+...+(2n+2)=n^2+3n[/tex]. Berdasarkan pembuktian dengan induksi matematika, rumus tersebut adalah benar untuk [tex]n\geq 1[/tex].

Penjelasan dengan langkah-langkah:

Diketahui:

[tex]4+6+8+...+(2n+2)=n^2+3n[/tex]

Ditanya:

Buktikan dengan induksi matematika!

Pembahasan:

1. Langkah dasar

Untuk n = 1

[tex]2(1)+2=(1)^2+3(1)\\2+2=1+3\\4=4[/tex]

Terbukti benar untuk n = 1

2. Langkah induksi

Asumsikan untuk n = k rumus tersebut benar

[tex]4+6+8+...+(2k+2)=k^2+3k[/tex]

Akan dibuktikan bahwa untuk n = k + 1 rumus juga benar

[tex]4+6+8+...+(2k+2)+(2(k+1)+2)=(k+1)^2+3(k+1)[/tex]

Pembuktian:

Ruas kiri

[tex]4+6+8+...+(2k+2)+(2(k+1)+2)\\\implies k^2+3k+(2(k+1)+2)\\\implies k^2+3k+2k+2+2\\\implies k^2+2k+1+3k+2+1\\\implies (k+1)^2+3k+3\\\implies (k+1)^2+3(k+1)[/tex]

Terbukti benar untuk n = k + 1

Sehingga, [tex]4+6+8+...+(2n+2)=n^2+3n[/tex] adalah benar untuk [tex]n\geq 1[/tex].

19. Diketahui 2+4+6+28+...+2n. Dengan induksi matematika, rumus deret tersebut adalah..


Diketahui 2 + 4 + 6 + 8 + ...+ 2n. Dengan induksi matematika, rumus deret tersebut adalah ...

A. nB. n + 1C. n + 2 D. n²E. n (n + 1)

Soal tersebut terdapat pada buku Matematika Wajib B. K. Noormandiri halaman 359, Erlangga.

Pendahuluan

Induksi matematika adalah proses pembuktian pernyataan kasus-kasus yang harus berlaku untuk setiap bilangan asli.

Deret aritmetika dapat dituliskan dalam notasi sigma

Sn = [tex]\sum\limits_{k~ = ~1}^n U_{k}[/tex] = U₁ + U₂ + U₃ + ... + Un = ⁿ/₂ (U₁ + Un)

atau

Sn = [tex]\sum\limits_{k~ = ~1}^n [/tex] [a + (k - 1) b]

    = a + (a + b) (a + 2b) + ... + (a + (n - 1) b)

    = ⁿ/₂ [2a + (n - 1) b]

Pelajari lebih lanjut : Notasi sigma dari : 28 + 33 + 38 + 43 + 48 + .... + 48 → brainly.co.id/tugas/24109207

Pembahasan

Deret aritmetika 2 + 4 + 6 + 8 + ... + 2n

U₁ = 2

Un = 2n

Rumus deret

Sn = ⁿ/₂ (U₁ + Un)

     = ⁿ/₂ (2 + 2n)

     = ⁿ/₂ × 2 (1 + n)

     = n (n + 1)

Jadi rumus deret tersebut adalah n (n + 1)

27. Hasil dari 4+6+8+...+(2n+2)=n2+3n dalam induksi matematika


Pembuktian 4 + 6 + 8 + ... + (2n + 2) = n² + 3n dengan menggunakan induksi matematika yaitu:

Untuk n = 1

2(1) + 2 = 1² + 3(1)

Untuk n = k

4 + 6 + 8 + … + (2k + 2) = k² + 3k

Untuk n = k + 1

4 + 6 + 8 + … + (2k + 2) + (2(k + 1) + 2) = (k + 1)² + 3(k + 1)

Penjelasan dengan langkah-langkah

Terdapat dua langkah dalam membuktikan suatu rumus dengan induksi matematika yaitu:

Buktikan bahwa untuk n = 1 benar.Dengan mengasumsikan bahwa untuk n = k benar, maka buktikan bahwa untuk n = k + 1 juga benar.

Diketahui

4 + 6 + 8 + ... + (2n + 2) = n² + 3n

Ditanyakan

Buktikan rumus deret bilangan tersebut dengan induksi matematika!

Jawab

Langkah 1

Akan dibuktikan untuk n = 1 benar.

2n + 2 = n² + 3n

2(1) + 2 = 1² + 3(1)

  2 + 2 = 1 + 3

        4 = 4

Terbukti benar

Langkah 2

Misal kita asumsikan untuk n = k rumus deret bilangan tersebut benar.

4 + 6 + 8 + … + (2k + 2) = k² + 3k

Akan dibuktikan untk n = (k + 1) juga benar.

4 + 6 + 8 + … + (2k + 2) + (2(k + 1) + 2) = (k + 1)² + 3(k + 1)

Pembuktian ruas kiri menjadi ruas kanan:

4 + 6 + 8 + … + (2k + 2) + (2(k + 1) + 2)

= [4 + 6 + 8 + … + (2k + 2)] + (2k + 2 + 2)

= [k² + 3k] + (2k + 4)

= k² + 3k + 2k + 4

= k² + 2k + 3k + 4

= k² + 2k + 3k + 3 + 1

= (k² + 2k + 1) + (3k + 3)

= (k + 1)² + 3(k + 1)

Terbukti benar
