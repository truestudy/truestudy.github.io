4. 2¹/₂ + 4¹/₄ = ⁵/₂ + ¹⁷/₄

                    = ¹⁰/₄ + ¹⁷/₄

                    = ²⁷/₄

                    = 6³/₄

5. 3²/₃ + 6³/₄ = ¹¹/₃ + ²⁷/₄

                     = ¹²⁵/₁₂
