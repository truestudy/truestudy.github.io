Tono (o) : Tina (a) = 2 : 3
Tono (o) : Toni (i) = 3 : 2
Jumlah uang mereka 456.000
Tina sedekah 25.000, sisa uang Tina?

o : a = 2 : 3 -> x3 -> 6 : 9
o : i = 3 : 2 -> x2 -> 6 : 4
Maka o : a : i = 6 : 9 : 4

Kenapa dikali 3 dan 2? Karena utk mencari KPK dari rasio uang Tono (o) 2 dan 3, yaitu 6.

o/a = 6/9
o = 6/9 a

a/i = 9/4
i = 4/9 a

o + a + i = 456.000
6/9 a + a + 4/9 a = 456.000
6/9 a + 9/9 a + 4/9 a = 456.000
19/9 a = 456.000
a = 456.000 x 9/19
a = 24.000 x 9
a = 216.000 -> uang Tina
