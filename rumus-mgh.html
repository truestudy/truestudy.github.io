7. tentukan bilangan oksidasi dalam senyawa berikut..NaOH =ClO₄⁻ =MgH₂ =HPO₄²⁻ =S₂O₃²⁻ =


Bilangan oksidasi NaOH adalah 0. Bilangan oksidasi [tex](ClO_{4})^-[/tex] adalah -1. Bilangan oksidasi [tex]MgH_{2}[/tex] adalah 0. Bilangan oksidasi [tex](HPO_{4})^-^2[/tex] adalah -2. Biloks [tex](S_{2}O_{3})^-^2[/tex] adalah -2.

Pembahasan

Biloks total seyawa NaOH adalah 0

∵Biloks Na adalah +1 ( aturan biloks no 5)

∵ Biloks O adalah -2 (aturan biloks no 9)

∵ Biloks H adalah +1 ( aturan biloks no 8)

Biloks total seyawa [tex](ClO_{4})^-[/tex] adalah -1

∵ Biloks O adalah -2 (aturan biloks no 9)

∵ Biloks Cl adalah +7

Diperoleh dari subtitusi biloks yang sudah diketahui pada  [tex](ClO_{4})^-[/tex] untuk memperoleh biloks Cl

∵Biloks total [tex](ClO_{4})^-[/tex] adalah -1 ( aturan biloks 4), dan Biloks O adalah -2 (aturan biloks ke 9)

   Biloks [tex]5 KClO_{3}[/tex] = -1

  (jumlah atom Cl x biloks atom Cl) + (jumlah atom O x biloks atom O) = -1

   (1 x biloks Cl) + ( 4 x (-2)) = -1

  Biloks Cl - 8 = -1

  Biloks Cl= 8 - 1

  Biloks Cl= +7

Biloks total seyawa [tex]MgH_{2}[/tex] adalah 0

∵Biloks Mg adalah +1 ( aturan biloks no 6)

∵ Biloks H adalah +1 ( aturan biloks no 8)

Biloks total seyawa [tex](HPO_{4})^-^2[/tex] adalah -2

∵ Biloks O adalah -2 (aturan biloks no 9)

∵ Biloks H adalah +1 ( aturan biloks no 8)

∵Biloks P adalah +5

Diperoleh dari subtitusi biloks yang sudah diketahui pada  [tex](HPO_{4})^-^2[/tex] untuk memperoleh biloks P

∵Biloks total [tex](HPO_{4})^-^2[/tex] adalah -2 ( aturan biloks 4), biloks H adalah +1  ( aturan biloks no 8) dan Biloks O adalah -2 (aturan biloks ke 9)

   Biloks [tex](HPO_{4})^-^2[/tex]  = -2

  (jumlah atom H x biloks atom H) + (jumlah atom P x biloks atom P) +

  (jumlah atom O x biloks atom O) = -2

  (1 x 1) + (1 x biloks P) + ( 4 x (-2)) = -2

  1 + Biloks P - 8 = -2

  Biloks P= 8- 2 - 1

  Biloks P= +5

Biloks total seyawa [tex](S_{2}O_{3})^-^2[/tex] adalah -2

∵ Biloks O adalah -2 (aturan biloks no 9)

∵ Biloks S adalah +4

Diperoleh dari subtitusi biloks yang sudah diketahui pada [tex](S_{2}O_{3})^-^2[/tex] untuk memperoleh biloks S

∵Biloks total [tex](S_{2}O_{3})^-^2[/tex] adalah -2 ( aturan biloks 4), dan Biloks O adalah -2 (aturan biloks ke 9)

   Biloks [tex](S_{2}O_{3})^-^2[/tex] = -2

  (jumlah atom S x biloks atom S) + (jumlah atom O x biloks atom O) = -1

   (1 x biloks S) + ( 3 x (-2)) = -2

  Biloks S - 6 = -2

  Biloks S= 6 - 2

  Biloks Cl= +4

11. normalitas 40 ml asam sulfat yang dapat dinetralkan oleh 0,29 gram Mgh dalam 100 ml adalah


Diketahui :
V larutan H2SO4 = 40 ml = 0,04 L
m Mg(OH)2 = 0,29 gram
V larutan Mg(OH)2 = 100 ml = 0,1 L

Ditanya : Nilai Normalitas H2SO4 ?

Jawab :

n(mol) Mg(OH)2 :
= m / Mr
= 0,29 gram / 58 gram/mol
= 0,005 mol

Molaritas Mg(OH)2 :
= n / V
= 0,005 mol / 0,1 L
= 0,05 mol/L
= 0,05 M

Mencari Molaritas H2SO4 :
M.V.a = M.V.b
M.0,04 L.2 = 0,05 M.0,1 L.2
M = 0,05 M .0,1 L .2 / 0,04 M.2
M = 0,125 M

Molaritas H2SO4 :
= 0,125 M
= 0,125 mol / 1 L
= 0,125 mol / 1000 ml

Massa H2SO4 :
= n . Mr
= 0,125 mol . 98 gram/mol
= 12,25 gram

BE (Berat ekuivalen) :
= Mr / Jumlah ion H+
= 98 gram/mol / 2
= 49 gram/mol

N (Normalitas) :
= (gr / BE )× (1000 / ml Larutan)
= (12,25 gram / 49 gram/mol) × ( 1000/ 1000 ml)
= 0,25 N

14. Energi potensial mangga jatuh dari ketinggian 3 meter dinyatakan dengan persamaan ep=mgh. dimensi dari energi potensial adalah​


Jawaban:

Dimensi dari energi potensial adalah [M][L]²[T]⁻².

Penjelasan:

Dimensi besaran pokok, yaitu:

Panjang = [L]

Massa =  [M]

Waktu = [T]

Suhu = [θ]

Diketahui:

Energi potensial mangga jatuh dari ketinggian 3 meter dinyatakan dengan persamaan Ep=mgh.

Ditanya:

Dimensi dari energi potensial.

Penyelesaian:

[tex]E_P = mgh\\m : massa\,(kg)[M]\\g: percepatan\ gravitasi (9,8m/s^2)([L]/[T]^2)\\h:tinggi\, (m)[L]\\\\E_P = [M]\,.\,\frac{[L]}{[T]^2}\,.\,[L]\\E_P=[M]\,.\,\frac{[L]^2}{[T]^2}\\\\E_P=\bold{[M][L]^2[T]^{-2}}[/tex]

Kesimpulan:

Dimensi dari energi potensial adalah [M][L]²[T]⁻².
