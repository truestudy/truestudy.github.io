1. Andi ngunduh layanganUkara ing dhuwur duweni pola...A. wasesa - jejer - lesanB. lesan - jejer - wasesa C. wasesa - lesan - jejer D. jejer - wasesa - lesanTolong jawab ​


Ukara ing dhuwur duweni pola :

D. Jejer - Wasesa - Lesan

～ Pembahasan ～

Menyusun Kalimat dalam Bahasa Jawa itu sama dengan menyusun kalimat Bahasa Indonesia, hanya berbeda istilah.

Jejer. Kita mengenal Jejer dengan Subjek. Seperti Nama Orang, Kata Ganti. Sebagai Contoh: Saya, Aku, dan lain sebagainyaWasesa. Kita mengenal Wasesa dengan Predikat, ditandai dengan aktivitas yang dilakukan oleh Subjek. Sebagai contoh: Berlari, Belajar, dan lain-lain.Lesan. Kita mengenal Lesan dengan Objek, ditandai dengan Kata Benda. Sebagai contoh: Buku, Pensil, dan lain sebagainya.Katrangan. Kita mengenal Katrangan dengan Keterangan, baik itu keterangan tempat maupun Keterangan Waktu. Sebagai contoh: Hari ini, Dapur, dan lain-lain.

Contoh Kalimat.

Andi Ngundhuh Layangan.

( Andi Membeli Layang-layang ).

Pola Kalimat:

Jejer/Subjeknya yaitu Andi.Wasesa/Predikatnya yaitu Ngundhuh.Lesan/Objeknya yaitu Layangan.

20. masing masing arti dari jejer , wasesa, lesan, katerangan (lengkap)


Arti jejer, wasesa, lesan, lan katrangan :

>> Jejer yaiku tembung kang biasane manggon ana ngarepe wasesa, biasane ngenani pelaku sing nglakoni pegawean ( Pelaku yang melakukan pekerjaan )

>> Wasesa yaiku pakaryan utawa penggawean kang dilakoni ( Pekerjaan yang dilakukan )

>> Lesan yaiku tembung kang biasane ana burine wasesa, kang kena pegawean ( yang terkena pekerjaan )

>> Katerangan yaiku katerangan ngenani wayah, papan, kahanan, lan liya liyane ( Keterangan mengenai tempat, waktu, keadaan, dll )

Pembahasan :

Ukara kang ana jejer, wasesa, lesan, lan katrangan iku diarani ukara sampurna

Tuladha ukara sampurna :

Ayu mangan roti mau bengi

Jejer = AyuWasesa = manganLesan = rotiKatrangan wektu = mau bengi

Simbah ngunjuk kopi wonten teras

Jejer = SimbahWasesa = ngunjukLesan = kopiKatrangan papan = wonten teras

25. apa yg dimaksut jejer,wasesa,lesan


jejer ( subjek ) : pelaku ( yang melakukan suatu pekerjaan
wasesa ( predikat ) : aktivitas yang dikerjakan
lesan ( objek ) : yang dikenai pekerjaan
semoga membantukalau di bahasa indonesia, sebagai berikut:
1) jejer = subjek
2) wasesa = predikat
3) lesan. = obyek

jadi, ketiga unsur tsb merupakan unsur kalimat. contoh :

Ibu tumbas gendhis.
Ibu = jejer
tumbas = wasesa
gendhis = gula
