4. ubah kesatuan volume, 2,8m3+800dm3=......L, 2dL+300dL=.....L, 0,7m3+200dL=...L, 30m3+400L=....dm3, tolong dibantu yang ini


1 m3 = 1.000 L
1 m3 = 1.000 dm3
1 dm3 = 1 L
1 dL = 0,1 L

maka,
2,8 m3 = 2,8 x 1000 = 2800 L
800 dm3 = 800 L
2800 L + 800 L = 3600 L

0,7 m3 = 0,7 x 1000 = 700 L
200 dL = 200 x 0,1 = 20 L
700 L + 20 L = 720 L

30 m3 = 30 x 1000 = 30.000 dm3
400 L = 400 dm3
30.000 dm3 + 400 dm3 = 30.400 dm3
