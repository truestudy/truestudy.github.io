PEMBAHASAN

Untuk menentukan bilangan berurutan dengan jumlah tertentu bisa diselesaikan dengan cara berikut. Dikarenakan yang akan dicari adalah tiga bilangan genap berurutan maka artinya setiap bilangan sesudah bilangan sebelumnya selalu ditambah dengan 2.

Misalkan tiga bilangan genap berurutan :

x, x + 2, dan x + 4

x + ( x + 2) + (x + 4) = 300

x + x + 2 + x + 4 = 300

x + x + x + 2 + 4 = 300

3x + 6 = 300

3x = 300 - 6

3x = 294

x = 294 / 3

x = 98

Dengan diperolehnya x = 98, maka kita hanya perlu menentukan dua bilangan genap sesudahnya yaitu 100 dan 102 atau bisa dicari dengan :

Bilangan genap kedua

= x + 2

= 98 + 2

= 100

Bilangan genap ketiga

= x + 4

= 98 + 4

= 102

Buktikan!

300 = 98 + 100 + 102

300 = 300 ✔️

Jadi, tiga bilangan genap berurutan dengan jumlah 300 adalah 98, 100, dan 102.
