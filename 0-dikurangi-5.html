9. cos x = 4/5 , 0 derajat kurang dari = kurang dari = 360 derajat​


Himpunan penyelesaian cos x = 4/5, 0° ≤ x ≤ 360° adalah {37°, 323°}. Nilai tersebut diperoleh dari perhitungan penyelesaian persamaan trigonometri. SImak pembahasan berikut.

PembahasanRumus penyelesaian persamaan trigonometriUntuk sinus

sin x = sin α

maka

x = α + k.360°

x = (180° - α) + k.360°

Untuk kosinus

cos x = cos α

maka

x = α + k.360°

x = - α + k.360°

Untuk tangen

tan x = tan α

maka

x = α + k.180°

dengan k = 0, 1, 2, 3, ....

Himpunan penyelesaian cos x = 4/5, 0° ≤ x ≤ 360°

jawab:

cos x = 4/5

cos x = cos 37°, 0° ≤ x ≤ 360°

cara 1:

cos x = cos 37°

x = 37° + k.360°

untuk k = 0

x = 37° + k.360°

x = 37° + 0.360°

x = 37° + 0°

x = 37°

untuk k = 1

x = 37° + k.360°

x = 37° + 1.360°

x = 37° + 360°

x = 397° (tidak memenuhi)

cara 2:

cos x = cos 37°

x = -37° + k.360°

untuk k = 0

x = -37° + k.360°

x = -37° + 0.360°

x = -37° + 0°

x = -37° (tidak memenuhi)

untuk k = 1

x = -37° + k.360°

x = -37° + 1.360°

x = -37° + 360°

x = 323°

untuk k = 2

x = -37° + k.360°

x = -37° + 2.360°

x = -37° + 720°

x = 683° (Tidak memenuhi)

Jadi himpunan penyelesaian cos x = 4/5, 0° ≤ x ≤ 360° adalah {37°, 323°}.

10. selesaian dari sistem persamaan 3 x tambah 2 Y kurang 4 = 0 dan X kurang 3 Y kurang 5 = 0 adalah


3x + 2y - 4 = 0 ... (i)

x - 3y - 5 = 0
x = 3y + 5 ... (ii)

Substitusi pers. (ii) ke pers. (i)
3x + 2y - 4 = 0
3(3y + 5) + 2y - 4 = 0
9y + 15 + 2y - 4 = 0
11y + 11 = 0
11y = -11
y = -1

Substitusi nilai y ke persamaan (ii)
x = 3y + 5
x = 3(-1) + 5
x = -3 + 5
x = 2

Jadi, penyelesaiannya adalah (2, -1)
