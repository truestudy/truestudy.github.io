26. U7, dan U10, dari barisan 1, 3, 6, 10, ...adalah ....a. 28 dan 52b. 28 dan 55c. 30 dan 42d. 36 dan 45​


U7 dan U10 dari barisan 1, 3, 6, 10 adalah 28 dan 55 (Opsi B)

[tex] \: [/tex]

Pendahuluan

Pengertian dari pola bilangan yaitu susunan bilangan yang membentuk suatu pola tertentu.

Pada pola bilangan ada beberapa pola, salah satunya yaitu pola bilangan segitiga.

[tex] \: [/tex]

Pola bilangan segitiga adalah barisan bilangan yang memiliki susunan pola berbentuk seperti segitiga.

[tex] \: [/tex]

Contoh pola bilangan segitiga :

1, 3, 6, 10, 15, ...

[tex] \: [/tex]

Berikut ini adalah rumus pola bilangan segitiga :

[tex] \: [/tex]

Rumus mencari suku ke-n

[tex]\tt {\boxed{U_n = \frac{n(n + 1)}{2}}} [/tex]

[tex] \: [/tex]

Rumus jumlah n suku

[tex] \tt {\boxed{Sn = \frac{1}{6} n(n + 1)(n + 2)}}[/tex]

[tex] \: [/tex]

PembahasanDiketahui :

Suatu barisan 1, 3, 6, 10, ...

[tex] \: [/tex]

Ditanya :

U7 dan U10?

[tex] \: [/tex]

Jawab :

Suku ke-7

[tex] \tt U_n = \frac{n(n + 1)}{2}[/tex]

[tex] \tt U_7 = \frac{7(7+ 1)}{2}[/tex]

[tex] \tt U_7 = \frac{7(8)}{2}[/tex]

[tex] \tt U_7 = \frac{56}{2}[/tex]

[tex] \tt {\bold{U_7 = 28}}[/tex]

[tex] \: [/tex]

Suku ke-10

[tex] \tt U_n = \frac{n(n + 1)}{2}[/tex]

[tex] \tt U_{10} = \frac{10(10 + 1)}{2}[/tex]

[tex] \tt U_{10} = \frac{10(11)}{2}[/tex]

[tex] \tt U_{10} = \frac{110}{2}[/tex]

[tex] \tt {\bold{U_{10} = 55}}[/tex]

[tex] \: [/tex]

Kesimpulan

U7 dan U10 dari barisan 1, 3, 6, 10 adalah 28 dan 55 (Opsi B)

28. U7 dan U10 dari barisan 1,3,6,10,... adalah​


Jawaban

U7 dan U10 dari barisan 1,3,6,10... adalah 55

Pembahasan

Jika terdapat suatu pola (aturan) tertentu antara suku-suku pada barisan yaitu selisih antara dua suku yang berurutan selalu tetap (konstan), maka barisan bilangan tersebut disebut barisan aritmatika.

=> Barisan Aritmatika

Rumus umum menentukan suku ke-n barisan aritmatika :

[tex]Un = a + (n - 1)b[/tex]

a = U1 adalah suku pertama barisan aritmatika

b = adalah beda barisan aritmatika

n = adalah jumlah suku

Un adalah jumlah suku ke-n

Kita dapat contohkan sebagai berikut.

1. Tentukan suku ke-100 barisan aritmatika : 5, 8, 11, .... !

Jawab:

a = 3, b = 3, n = 100

Un = a + (n - 1)b

U100 = 5 + (100 - 1).3 = 5 + 99.3 = 5 + 297 = 302

--> Deret Aritmatika

Kita dapat contohkan sebagai berikut.

Jumlah n suku pertama deret aritmatika dapat ditentukan dengan

[tex]Sn = \frac{1}{2} =n(2a + (n - 1)b) \: atau \: Sn = \frac{1}{2}n(a + un)[/tex]

Misalnya

Diketahui deret aritmatika : 1 + 3 + 5 + 7 + 9 + ...

Tentukan

Rumus suku ke-nRumus jumlah n suku pertama Jumlah 50 suku pertama

Jawab

1 + 3 + 5 + 7 + 9 ...

a = 1, b = 2

Un = a + (n - 1)b

Un = 1 + (n - 1)2

Un = 2n - 2

Un = 2n - 1 b

[tex]Sn = \frac{1}{2} n(2a + (n - 1)b)[/tex]

[tex]Sn = \frac{n}{2} (2(1) + (n - 1)2)[/tex]

[tex]Sn = \frac{n}{2} (2 + 2n - 2)[/tex]

[tex]Sn = \frac{n}{2} (2n)[/tex]

[tex]Sn = {n}^{2} [/tex]

[tex]Sn = n2[/tex]

[tex]S50 = (50)2 = 2.500[/tex]
