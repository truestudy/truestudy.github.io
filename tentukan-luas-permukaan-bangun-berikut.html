4. tentukan luas permukaan pada bangun berikut ini ! ​


Jawab:

a. 96 [tex]cm^{2}[/tex]

b. 1.540 [tex]cm^{2}[/tex]

c. Skip

d. 1.496 [tex]cm^{2}[/tex]

e. 2.464 [tex]cm^{2}[/tex]

Penjelasan dengan langkah-langkah:

a. Luas permukaan prisma segitiga

⇒ Luas permukaan prisma segitiga = (2 x Luas alas) + (Keliling alas x tinggi)

⇒ Luas permukaan prisma segitiga = (2 x [tex]\frac{1}{2}[/tex] x s1 x s2) + ((s1 + s2 + s3) x tinggi)

⇒ Luas permukaan prisma segitiga = ((2 x [tex]\frac{1}{2}[/tex]) x 4 x 3) + ((4 + 3 + 5) x 7)

⇒ Luas permukaan prisma segitiga = (1 x 12) + (12 x 7)

⇒ Luas permukaan prisma segitiga = 12 + 84 ⇔ 96 [tex]cm^{2}[/tex]

b.  Luas permukaan tabung

⇒ Luas permukaan prisma segitiga = (2 x Luas alas) + Luas selimut

⇒ Luas permukaan prisma segitiga = (2 [tex]\pi r^{2}[/tex]) + (2 [tex]\pi rt[/tex])

⇒ Luas permukaan prisma segitiga = 2 [tex]\pi r[/tex] (r + t)

⇒ Luas permukaan prisma segitiga = 2 x ([tex]\frac{22}{7}[/tex] x 7) (7 + 28)

⇒ Luas permukaan prisma segitiga = 2 x 22 (35)

⇒ Luas permukaan prisma segitiga = 44 x 35 ⇔ 1.540 [tex]cm^{2}[/tex]

c. Luas permukaan limas segiempat ⇔ itu gambarnya terpotong bagian alasnya

d. Luas permukaan kerucut

⇒ Luas permukaan kerucut = Luas alas + Luas selimut

⇒ Luas permukaan kerucut = [tex]\pi r^{2}[/tex] + [tex]\pi rs[/tex]

⇒ Luas permukaan kerucut = [tex]\pi r[/tex] (r + s)

⇒ Luas permukaan kerucut = ([tex]\frac{22}{7}[/tex] x 14) (14 + 20)

⇒ Luas permukaan kerucut = (22 x 2) (34)

⇒ Luas permukaan kerucut = 44 x 34 ⇔ 1.496 [tex]cm^{2}[/tex]

e. Luas permukaan bola

⇒ Luas permukaan bola = 4[tex]\pi r^{2}[/tex]

⇒ Luas permukaan bola = 4 x ([tex]\frac{22}{7}[/tex] x 14) x 14

⇒ Luas permukaan bola = (22 x 2) x (4 x 14)

⇒ Luas permukaan bola = 44 x 56 ⇔ 2.464 [tex]cm^{2}[/tex]

20. Tentukan luas permukaan bangun berikut


Jawab:

Luas permukaan benda tersebut adalah 4.555 cm².

Penjelasan dengan langkah-langkah:

Untuk menghitung luas permukaan bangun tersebut maka kita hitung luas permukaan 7 sisi sepeti yang terlampir pada gambar terlebih dahulu, kemudian dijumlahkan.

- Luas A

Permukaan A merupakan persegi yang terpotong seperempat bagian lingkaran berjari-jari 7 cm.

LA = Lpersegi – L1/4lingkaran  

LA = s² – ¼. π. r²

LA = (14 cm)² – ¼. 22/7. (7 cm)²

LA = 196 cm² – 22/28. 49 cm²

LA = 196 cm² – 38,5 cm²

LA = 157,5 cm²

- Luas B

Permukaan B sama dengan permukaan A, sehingga luasnya  

LB = LA

LB = 157,5 cm²

- Luas C

Permukaan C merupakan bagian yang menjadi alas bangun tersebut. Bentuknya persegi panjang.

LC = p x l

LC = 80 cm x 14 cm

LC = 1.120 cm²

- Luas D

Permukaan D sama dengan permukaan C, sehingga luasnya  

LD = LC

LD = 1.120 cm²

- Luas E

Permukaan E merupakan persegi panjang di bagian atas.

LE = p x l

LE = 80 cm x 7 cm

LE = 560 cm²

- Luas F  

Permukaan F merupakan bagian yang melengkung, yakni seperempat selimut tabung berjari-jari 7 cm.

LF = ¼. Kt. p, Kt adalah keliling tabung  

LF = ¼. 2. π. r. p

LF = ¼. 2. 22/7. 7 cm. 80 cm

LF = 880 cm²

- Luas G

Permukaan G sama dengan permukaan E, sehingga luasnya  

LG = LE

LG = 560 cm²

- Luas permukaan total

Lt = LA + LB + LC + LD + LE + LF + LG

Lt = 157,5 cm² + 157,5 cm² + 1.120 cm² + 1.120 cm² + 560 cm² + 880 cm² + 560 cm²

Lt = 4.555 cm²
