15. 9. Pasangan panyigeg wanda gunane kanggo ....A. nulis aksara matiB. nggawe aksaraC. nggabung aksaraD. misah aksara​


A. Nulis aksara mati

Pembahasan :

➡️Sandhangan panyigeg wanda asale saka tembung sigeg, sing tegese mati ( huruf).

➡️ Sandhangan panyigeg Wanda yaiku sandhangan sing digunakake kanggo ngganteni aksara sigeg.

➡️ Aksara sigeg sing diganteni ana 3, yaiku h, r lan Ng.

➡️ Sandhangan panyigeg Wanda ana 4 :

wignyan = ꦃ = kanggo paten Hlayar = ꦂ = kanggo paten Rcecak = ꦁ = kanggo paten Ngpangkon = ꧀ = saliyane paten H, R lan Ng.

29. tulisen wujude sandhangan panyigeg nganggo aksara Jawa​


Wangsuli pitakon

Wujude sandhangan panyigeg nganggo aksara Jawa :

wignyan = ꦃlayar = ꦂcecak = ꦁpangkon = ꧀__________________Pembahasan

Sandhangan panyigeg Wanda

yaiku sandhangan kang digunakake kanggo mateni aksara sigeg, kajaba aksara sigeg mau aksara H, R, lan Ng, amarga aksara sigeg mau wis duwe tanda dhewe.

Sandhangan panyigeg Wanda ana 4 :

1. wignyan = paten H = ꦃ

tuladha :

gajah = ꦒꦗꦃrubuh = ꦫꦸꦧꦸꦃ

2. layar = paten R = ꦂ

tuladha :

pasar = ꦲꦱꦂsubur = ꦱꦸꦧꦸꦂ

3. cecak = paten ng = ꦁ

tuladha :

cacing = ꦕꦕꦶꦁpayung = ꦥꦪꦸꦁ

4. pangkon = saliyane paten H, R lan Ng = ꧀

tuladha :

nanas = ꦤꦤꦱ꧀sepet = ꦱꦼꦥꦼꦠ꧀

wignyan = paten H = ꦃlayar = paten R = ꦂcecak = paten ng = ꦁ pangku = saliyane h, r, lan ng = ꧀

Sandhangan Aksara Jawa ana 3 :

1. sandhangan swara ana 5 :

wulu = unine i ➡️ ꦶsuku = unine u ➡️ ꦸpepet = unine e ➡️ ꦼtaling = unine é ➡️ ꦺtaling tarung = unine o ➡️ ꦴꦺ
