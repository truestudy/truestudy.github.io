Pembahasan

Karakteristik grafik berdasarkan nilai determinan

Jika D > 0 maka persamaan kuadrat memiliki dua akar real berbeda sehingga grafik akan memotong sumbu x di dua titikJika D = 0 maka persamaan kudrat memiliki dua akar real kembar atau grafik menyinggung sumbu xJika D < 0 maka persamaan kuadrat memiliki akar yang imaginer atau tidak memotong sumbu x

Karakteristik grafik berdasarkan nilai a,

Jika a > 0 maka grafik terbuka keatasJika a < 0 maka grafik terbuka kebawaJika a = 0 bukan persamaan kuadrat

13. gambar kan grafik ( x2+y2 - 1 ) x2 y3= 0


Grafik dari persamaan [tex](x^2+y^2-1) x^2 y^3=0[/tex] terlampir pada jawaban ini. Dapat dilihat bahwa grafi tersebut membentuk suatu lingkaran dengan pusat pada titik (0, 0) dan garis pada sumbu x dan sumbu y.

Penjelasan dengan langkah-langkah

Diketahui:

( x² + y² - 1 ) x² y² = 0

Ditanya:

nilai x dan y yang dapat mengisi persamaan tersebut.

Jawab:

Persamaan ( x² + y² - 1 ) x² y² = 0 diselesaikan dengan menggunakan prinsip faktor nol. Apabila ab = 0, berarti a = 0 atau b = 0. Ketika prinsip tersebut diterapkan pada persamaan  ( x² + y² - 1 ) x² y² = 0, akan didapatkan sebagai berikut:

  ( x² + y² - 1 ) = 0 atau x² y² = 0

( x² + y² - 1 ) = 0 merupakan persamaan yang menghasilkan lingkaran di tengah.

x = 0

(0 + y² - 1 ) = 0. maka y = 1  ---> (0, 1)

x = 1 dan -1

(1 + y² - 1 ) = 0, maka y = 0  ---> (1, 0), (-1, 0)

y = 0

(x² + 0 - 1 ) = 0, maka x = 1  ---> (1, 0)

y = 1 dan -1

(x² + 1 - 1 ) = 0, maka x = 0  ---> (0, 1), (0, -1)

dari persamaan tersebut, didapatkan lingkaran berjari-jari 1 di tengah.

x² y² = 0 merupakan persamaan yang menghasilkan garis pada sumbu x dan y. Garis tersebut mengikuti sumbu karena berapapun nilai salah satunya (x atau y), ketika nilai satunya merupakan 0, hasilnya memenuhi persamaan tersebut (0).

Contoh: x² y² = 0

             5² y² = 0, maka y = 0

22. X2+(Y-3√x2)2=1 Cara mencari grafiknya gimana ya


Penyederhanaan dari persamaan x² + (y - 3√x²)² = 1 adalah menjadi 10x² + y² - 6xy = 1

Grafik yang dihasilkan dari persamaan tersebut memiliki bentuk elips. Gambaran dari grafik tersebut terlampir pada jawaban ini

Penjelasan dengan langkah-langkah

Diketahui :

DIberikan sebuah fungsi kuadrat x² + (y - 3√x²)² = 1

Ditanya:

Solusi atau penyederhanaan dari soal tersebut

Jawab:

Untuk menyederhanakan persamaan tersebut, pertama-tama harus diselesaikan terlebih dahulu yang di dalam kurung.

(y - 3√x²)² = y² - 2.y.3√x² + (3√x²)²

                 = y² - 6xy + (3x)²

                 = y² - 6xy + 9x²                  

Persamaan tersebut kemudian disatukan dengan persamaan awal x² menjadi berikut:

x² + y² - 6xy + 9x² = 1

Kemudian jumlahkan suku-suku yang sejenis, sehingga menjadi

10x² + y² - 6xy = 1
