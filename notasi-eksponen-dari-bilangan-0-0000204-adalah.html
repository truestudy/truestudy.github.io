25. Tuliskan dalam bentuk notasi eksponen untuk masing masing bilangan berikut! (30) A. 4.096 B. 1024 C. 512​


Jawab:

SOAL A

4096

= 2^12

= (2^2)^6 = 4^6

= (2^3)^4 = 8^4

= (4^2)^3 = 16^3

jadi 4096 = 2^12 = 4^6 = 8^4 = 16^3

SOAL B

1024

= 2^10

= (2^2)^5 = 4^5

jadi 1024 = 2^10 = 4^5

SOAL C

512

= 2^9

= (2^3)^3 = 8^3

jadi 512 = 2^9 = 8^3
