5. Matriks A berukuran 2x3, sedangkan Matriks B berukuran 3x2, berapakah hasil dari Matriks C? di mana C = A x B


Misal matriks A berukuran 2x3 =

[tex]\left[\begin{array}{ccc}a&b\\c&d\\e&f\end{array}\right][/tex]

Sedangkan matriks B berukuran 3x2 =

[tex]\left[\begin{array}{ccc}g&h&i\\j&k&l\\\end{array}\right][/tex]

Hasil dari C = A x B =

[tex]\left[\begin{array}{ccc}a&b\\c&d\\e&f\end{array}\right] \times \left[\begin{array}{ccc}g&h&i\\j&k&l\\\end{array}\right][/tex]

= [tex]\left[\begin{array}{ccc}(a*g)+(b*j)&(a*h)+(b*k)&(a*i)+(b*l)\\(c*g)+(d*j)&(c*h)+(d*k)&(c*i)+(d*l)\\(e*g)+(f*j)&(e*h)+(f*k)&(e*i)+(f*l)\end{array}\right][/tex]

Maka dapat disimpulkan hasil dari A x B = C menghasilkan matriks berukuran 3 x 3.
