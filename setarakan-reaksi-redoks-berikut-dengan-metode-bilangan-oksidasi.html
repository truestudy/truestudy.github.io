15. Setarakan reaksi redoks berikut dengan metode setengah reaksi dan bilangan oksidasi Bi2O3+ClO_- Bi_3+Cl_(suasana basa)


Penyetaraan reaksi redoks menggunakan cara :

a. Setengah Reaksi

Bi₂O₃ + ClO⁻  ⇒ BiO₃⁻ + Cl⁻

LANGKAH PERTAMA (I)

Tentukan zat yang mengalami oksidasi dan reduksi dengan mengunakan cara berikut :

       Oksidasi

Ι-----------------------Ι

Bi₂O₃ + ClO⁻  ⇒ BiO₃⁻ + Cl⁻

+3 -2    +1 -2       +5 -3      -1

             Ι-------------------------Ι

                     Reduksi

LANGKAH KEDUA (II)

Setarakan reaksi dengan menggunakan cara setengah reaksi sebagai berikut :

Langkah 1 : Untuk mempermudah maka buatlah zat yang mengalami oksidasi dan reduksi sebagai berikut :

Oksidasi     : Bi₂O₃   ⇒   BiO₃⁻

Reduksi      : ClO⁻  ⇒   Cl⁻

Langkah 2 : Setarakan jumlah koefisien  selain atom O dan H

Oksidasi     : Bi₂O₃   ⇒   2BiO₃⁻

Reduksi      : ClO⁻  ⇒   Cl⁻

Langkah 3 : Menambah H₂O diruas yang kelebihan O sebanyak kelebihannya.

Oksidasi     : Bi₂O₃   ⇒   2BiO₃⁻  + 3H₂O

Reduksi      : ClO⁻ + H₂O    ⇒   Cl⁻

Langkah 4 : Menambah OH⁻ diruas yang kekurangan H sebanyak kekurangannya.

Oksidasi     : Bi₂O₃ +  6OH⁻  ⇒   2BiO₃⁻  + 3H₂O

Reduksi      : ClO⁻ + H₂O    ⇒   Cl⁻  + 2OH⁻

Langkah 5 : Setarakan jumlah ion dengan menambah elektron.

Oksidasi     : Bi₂O₃ +  6OH⁻  ⇒   2BiO₃⁻  + 3H₂O  + 4e         ║ × 1

Reduksi      : ClO⁻ + H₂O  + 2e  ⇒   Cl⁻  + 2OH⁻                     ║ × 2

Langkah 6 : Setarakan jumlah muatan pada kedua reaksi kemudian jumlah kedua reaksi.

Oksidasi     : Bi₂O₃ +  6OH⁻  ⇒   2BiO₃⁻  + 3H₂O  + 4e

Reduksi      : 2ClO⁻ + 2H₂O  + 4e  ⇒   2Cl⁻  + 4OH⁻

                  ________________________________ +

                   Bi₂O₃ +  2OH⁻ + 2ClO⁻ ⇒   2Cl⁻  +  2BiO₃⁻  + H₂O

b. Bilangan Oksidasi

Bi₂O₃ + ClO⁻  ⇒ BiO₃⁻ + Cl⁻

Langkah 1 : Tentukan bilangan oksidasi masing masing unsur.

       Oksidasi

Ι-----------------------Ι

Bi₂O₃ + ClO⁻  ⇒ BiO₃⁻ + Cl⁻

+3 -2    +1 -2       +5 -3      -1

             Ι-------------------------Ι

                     Reduksi

Langkah 2 : Setarakan atom atom yang berubah bilangan oksidasinya.

Bi₂O₃ + ClO⁻  ⇒ 2BiO₃⁻ + Cl⁻

Langkah 3 : Tentukan jumlah kenaikan dan penurunan bilangan oksidasi dengan mengalikannya dengan indeks.

       Naik 4

Ι-----------------------Ι

Bi₂O₃ + ClO⁻  ⇒ 2BiO₃⁻ + Cl⁻

+6 -2    +1 -2       +10 -3       -1

             Ι-------------------------Ι

                     Turun 2

Langkah 4 : Setarakan Jumlah kenaikan dan jumlah penurunan.

2Bi₂O₃ + 4ClO⁻  ⇒ 4BiO₃⁻ + 4Cl⁻

Langkah 5 : Setarakan jumlah O dengan cara menambahkan H₂O di ruas kelebihan O

2Bi₂O₃ + 4ClO⁻  ⇒ 4BiO₃⁻ + 4Cl⁻ + 2H₂O

Langkah 6 : Setarakan Jumlah H dengan menambahkan OH⁻ diruas kelebihan H sebanyak kelebihannya.

2Bi₂O₃ + 4ClO⁻  + 4OH⁻ ⇒ 4BiO₃⁻ + 4Cl⁻ + 2H₂O

Langkah 7 : Karena koefisien dalam bentuk bilangan genap, maka dapat disederhanakan lagi menjadi :

Bi₂O₃ + 2ClO⁻  + 2OH⁻ ⇒ 2BiO₃⁻ + 2Cl⁻ + H₂O

Pembahasan :

Syarat suatu reaksi dikatakan setara yaitu :

Jumlah atom diruas kiri dan ruas kanan samaJumlah muatan diruas kiri dan ruas kanan sama

Untuk memastikan reaksi sudah setara  atau belum dapat dihitung dengan cara :

Jumlah atom diruas kiri dan ruas kanan sama

Bi₂O₃ + 2ClO⁻  + 2OH⁻ ⇒ 2BiO₃⁻ + 2Cl⁻ + H₂O

Jumlah atom Bi diruas kanan dan kiri = 2

Jumlah atom O diruas kanan dan kiri = 7

Jumlah atom Cl diruas kanan dan kiri = 2

Jumlah atom H diruas kanan dan kiri = 2

Jumlah muatan diruas kiri dan ruas kanan sama

Jumlah muatan diruas kanan dan kiri = 4

16. setarakan reaksi redoks berikut dengan metode bilangan Oksidasi suasana basa: MnO4- + C2O4- --> MnO2 + CO2


Setarakan reaksi redoks berikut dengan metode bilangan Oksidasi suasana basa:

MnO₄⁻ + C₂O₄⁻   ==>  MnO₂ + CO₂

muatan  C₂O₄ seharusnya adalah -2 bukan -1, sehingga reaksi yang benar adalah:

MnO₄⁻ + C₂O₄⁻²   ==>  MnO₂ + CO₂

Jawab:

Sebelum kita mulai melakukan penyetaraan reaksi, kita hitung biloks dari masing-masing unsur terlebih dahulu, yaitu:

Biloks MnO₄⁻ = -1

Biloks Mn + 4 x Biloks O = -1

Biloks Mn + 4 x (-2) = -1

Biloks Mn + (-8) = -1

Biloks Mn = +7

Biloks C₂O₄⁻² = -2

2 x Biloks C + 4 x Biloks O = -2

2 x Biloks C + 4 x (-2) = -2

2 x Biloks C + (-8) = -2

2 x Biloks C = +6

Biloks C = +3

Biloks MnO₂ = 0

Biloks Mn + 2 x Biloks O = 0

Biloks Mn + 2 x (-2) = 0

Biloks Mn = +4

Biloks CO₂ = 0

Biloks C + 2 x Biloks O = 0

Biloks C + 2 x (-2) = 0

Biloks C = +4

Setelah semua bilangan oksidasi setiap unsur diketahui, maka kita bisa mulai melakukan penyetaraan reaksi dengan cara Bilangan Oksidasi dengan langkah-langkah berikut:

Langkah 1 Menuliskan unsur yang mengalami perubahan bilangan Oksidasi

Reaksi:

MnO₄⁻ + C₂O₄⁻²   ==>  MnO₂ + CO₂

Berdasarkan reaksi di atas diketahui:

Reaksi Reduksi :

MnO₄⁻ ==>  MnO₂

Biloks Mn dari +7 menjadi +4

Reaksi Oksidasi :

C₂O₄⁻²   ==>  CO₂

Biloks C dari +3 menjadi +4

Langkah 2 Menyetarakan unsur yang mengalami perubahan Bilangan Oksidasi

Reaksi Reduksi :

MnO₄⁻ ==>  MnO₂ (atom Mn sudah setara)

Reaksi Oksidasi :

C₂O₄⁻²   ==>  CO₂ menjadi  C₂O₄⁻²   ==> 2 CO₂ (Atom C sudah setara)

Langkah 3 Menentukan Jumlah Kenaikan dan Penurunan Bilangan Oksidasi dari unsur yang mengalami Oksidasi dan Reduksi

Reaksi Reduksi :  

MnO₄⁻ ==>  MnO₂

Perubahan biloks Mn +7 menjadi +4

Terjadi penurunan biloks sebesar 3 angka

Reaksi Oksidasi :

C₂O₄⁻²   ==>  2 CO₂

Perubahan biloks C dari +3 menjadi +4

Terjadi perubahan biloks sebesar 1 angka, namun karena melibatkan 2 atom C, maka perubahan biloks menjadi 1 x 2 = 2 angka

Langkah 4 Menyamakan jumlah perubahan Bilangan Oksidasi

Untuk menyamakan jumlah perubahan bilangan oksidasi pada reaksi reduksi dan oksidasi, maka koefisien reaksi pada reaksi reduksi dikalikan dengan nilai kenaikan bilangan oksidasi dari unsur pada reaksi oksidasi.

Sebaliknya koefisien reaksi Oksidasi dikalikan dengan nilai penurunan bilangan oksidasi dari unsur pada reaksi reduksi.

Sehingga koefisien reaksi pada reaksi redoks menjadi:

 2 MnO₄⁻ + 3 C₂O₄⁻²   ==> 2 MnO₂ + 6 CO₂

Langkah 5 Menyamakan Muatan (Karena reaksi berupa Ion)

selanjutnya kita mulai menyetarakan muatan yang terdapat dalam reaksi:

 2 MnO₄⁻ + 3 C₂O₄⁻²   ==> 2 MnO₂ + 6 CO₂

Muatan di sebelah kiri (Reaktan) = -2 + 3 . (-2) = -8

Muatan di sebelah kanan (produk) = 0

Karena reaksi berlangsung dalam kondisi basa, maka penyetaraan muatan dilakukan dengan penambahan ion OH⁻ yaitu di sisi kanan sebanyak 8 ion menjadi:

         2 MnO₄⁻ + 3 C₂O₄⁻²   ==> 2 MnO₂ + 6 CO₂ + 8 OH⁻

Setelah muatan kiri dan kanan setara maka kita lanjutkan pada langkah terakhir yaitu langkah 6.

Langkah 6  Menyetarakan atom H

Jumlah atom H di kiri (Pereaksi) = 0

Jumlah atom H di kanan (Produk) = 8

maka kita lakukan penyetaraan jumlah atom H di kiri dengan penambahan 4 molekul H₂O, menjadi:

         2 MnO₄⁻ + 3 C₂O₄⁻² + 4 H₂O   ==> 2 MnO₂ + 6 CO₂ + 8 OH⁻ (SETARA)

Sisanya kita lakukan pengecekan ulang dengan menghitung ulang jumlah atom-atom di sisi kiri dan kanan.

Jika langkah-langkah kita sudah benar maka jumlah atom Oksigen di kiri dan kanan akan otomatis sama, jika tidak sama maka perlu dicek ulang per langkah kerja.

19. Setarakan reaksi redoks dengan metode bilangan oksidasi NH3+ZNO2^2 menghasilkan N2O3+ZN


Setarakan reaksi redoks dengan metode bilangan oksidasi

NH₃ + ZnO₂⁻² ==> N₂O₃ + Zn

Jawab:

Sebelum kita mulai melakukan penyetaraan reaksi, kita hitung biloks dari masing-masing unsur terlebih dahulu, yaitu:

Biloks NH₃ = 0

Biloks N + 3 x Biloks H = 0

Biloks N + 3 x (+1) = 0

Biloks N = -3

Biloks ZnO₂⁻² = -2

Biloks Zn + 2 x Biloks O = -2

Biloks Zn + 2 x (-2) = -2

Biloks Zn + (-4) = -2

Biloks Zn = +2

Biloks N₂O₃ = 0

2 x Biloks N + 3 x Biloks O = 0

2 x Biloks N + 3 x (-2) = 0

2 x Biloks N + (-6) = 0

2 x Biloks N = + 6

Biloks N = +3

Biloks Zn = 0

Setelah semua bilangan oksidasi setiap unsur diketahui, maka kita bisa mulai melakukan penyetaraan reaksi dengan cara Bilangan Oksidasi dengan langkah-langkah berikut:

Langkah 1 Menuliskan unsur yang mengalami perubahan bilangan Oksidasi

Reaksi:

NH₃ + ZnO₂⁻² ==> N₂O₃ + Zn

Berdasarkan reaksi di atas diketahui:

Reaksi Reduksi :

ZnO₂⁻² ==> Zn

Biloks Zn dari +2 menjadi 0

Reaksi Oksidasi :

NH₃  ==> N₂O₃

Biloks N dari -3 menjadi +3

Langkah 2 Menyetarakan unsur yang mengalami perubahan Bilangan Oksidasi

Reaksi Reduksi :

ZnO₂⁻² ==> Zn

Atom Zn sudah setara

Reaksi Oksidasi :

NH₃  ==> N₂O₃  menjadi  2 NH₃  ==> N₂O₃

Atom N sudah setara

Langkah 3 Menentukan Jumlah Kenaikan dan Penurunan Bilangan Oksidasi dari unsur yang mengalami Oksidasi dan Reduksi

Reaksi Reduksi :

ZnO₂⁻² ==> Zn

Perubahan biloks Zn dari + 2 menjadi 0

Terjadi penurunan biloks sebesar 2 angka

Reaksi Oksidasi :

2 NH₃  ==> N₂O₃

Perubahan biloks N dari -3 menjadi +3

Terjadi penurunan biloks sebesar 6 angka, tetapi karena atom N yang terlibat sebanyak 2, maka perubahan biloks juga dikalikan 2, sehingga menjadi 12 angka.

Langkah 4 Menyamakan jumlah perubahan Bilangan Oksidasi

Untuk menyamakan jumlah perubahan bilangan oksidasi pada reaksi reduksi dan oksidasi, maka koefisien reaksi pada reaksi reduksi dikalikan dengan nilai kenaikan bilangan oksidasi dari unsur pada reaksi oksidasi.

Sebaliknya koefisien reaksi Oksidasi dikalikan dengan nilai penurunan bilangan oksidasi dari unsur pada reaksi reduksi.

Sehingga koefisien reaksi pada reaksi redoks menjadi:

4 NH₃ + 12 ZnO₂⁻² ==> 2 N₂O₃ + 12 Zn

Langkah 5 Menyamakan Muatan (Karena reaksi berupa Ion)

Jika misalkan reaksi ini berjalan dalam suasan asam, maka penyetaraan dilakukan dengan penambahan ion H⁺, jika suasana basa penyetaraan dengan ion OH⁻.

Jumlah muatan di kiri = 12 x -2 = - 24

Jumlah muatan di kanan = 0

Suasana Asam

Jika reaksi berjalan dalam suasana asam maka kita tambahkan ion H⁺ disisi kiri sebanyak 24 buah, menjadi:

4 NH₃ + 12 ZnO₂⁻² + 24 H⁺ ==> 2 N₂O₃ + 12 Zn

Suasana Basa

Jika reaksi berjalan dalam suasana basa maka kita tambahkan ion OH⁻ disisi kanan sebanyak 24 buah, menjadi:

4 NH₃ + 12 ZnO₂⁻² ==> 2 N₂O₃ + 12 Zn + 24 OH⁻

Langkah 6  Menyetarakan atom H

Untuk menyamakan jumlah atom H maka kita hitung jumlah di sisi kanan dan kiri kemudian disetarakan dengan penambahan molekul H₂O.

Suasana Asam

4 NH₃ + 12 ZnO₂⁻² + 24 H⁺ ==> 2 N₂O₃ + 12 Zn

Atom H di kiri = 12 + 24 = 36

Atom H di kanan = 0

Sehingga di sisi kanan kita tambahkan 18 molekul H₂O, karena dalam 1 molekul H₂O mengandung 2 atom H. Sehingga menjadi:

4 NH₃ + 12 ZnO₂⁻² + 24 H⁺ ==> 2 N₂O₃ + 12 Zn + 18 H₂O (SETARA)

Suasana Basa

4 NH₃ + 12 ZnO₂⁻² ==> 2 N₂O₃ + 12 Zn + 24 OH⁻

Atom H di kiri = 12

Atom H di kanan = 24

Sehingga di sisi kiri kita tambahkan 6 molekul H₂O, karena dalam 1 molekul H₂O mengandung 2 atom H. Sehingga menjadi:

4 NH₃ + 12 ZnO₂⁻² + 6 H₂O ==> 2 N₂O₃ + 12 Zn + 24 OH⁻ (SETARA)

Sisanya kita lakukan pengecekan ulang dengan menghitung ulang jumlah atom-atom di sisi kiri dan kanan.

Jika langkah-langkah kita sudah benar maka jumlah atom Oksigen di kiri dan kanan akan otomatis sama, jika tidak sama maka perlu dicek ulang per langkah kerja.

26. Lengkapi dan setarakan reaksi redoks berikut dengan metode bilangan oksidasi Zn(s)+NO3–(aq)+NH3(g)


Pertanyaan yang diajukan adalah Setarakan persamaan reaksi redoks  

Zn + NO₃⁻ + → Zn²⁺ + NH₃  

Jawaban  

Langkah pertama yang dilakukan adalah mengetahui bilangan oksidasi masing – masing unsur  

Zn = 0 , karena dalam bentuk unsur bebas  

NO₃⁻, Unsur oksigen dalam keadaan berikatan memiliki bilangan oksidasi -2, dan senyawa NO₃⁻ memiliki muatan total = -1 ,  

NO₃ = -1  

N + 3.(-2) = -1  

N= +5  

Zn²⁺ = +2  

NH₃ , N³⁻ 3.H⁺ bilangan oksidasi N dalam senyawa adalah -3, sedangkan bilangan oksidasi H dalam senyawa = +1

Berdasarkan nilai bilangan oksidasi yang telah dihitung, maka unsur Zn mengalami oksidasi karena terjadi peningkatan bilangan oksidasi dari 0 menjadi +2, sedangkan unsur N mengalami reduksi karena mengalami penurunan bilangan oksidasi dari +5 menjadi -3.  

Reaksi oksidasi : Zn → Zn²⁺ + 2e⁻  

Reaksi Reduksi : NO₃⁻ + 9H⁺ + 8e⁻ → NH₃ + 3H₂O  

Selanjutnya dilakukan penyetaraan reaksi, dengan menyamakan jumlah elektron yang terlibat dalam reaksi oksidasi dan reduksi, persamaan reaksi yang terbentuk adalah  

4Zn + NO₃⁻ + 9H⁺ → 4Zn²⁺ + NH₃ + 3H₂O  

Pembahasan :  

Redoks merupakan singkatan dari reduksi dan oksidasi yang menjelaskan mengenai perubahan bilangan oksidasi pada suatu reaksi. Ada dua pengertian dasar yang perlu dipahami mengenai oksidasi dan reduksi  

Reaksi Reduksi : Reduksi adalah reaksi yang melibatkan pelepasan oksigen, pengikatan hidrogen, pengikatan elektron,atau penurunan bilangan oksidasi.  Reaksi oksidasi : Oksidasi didasarkan atas penggabungan unsur atau senyawa dengan oksigen untuk membentuk zat lain. Oksidasi adalah reaksi yang melibatkan pengikatan oksigen, pelepasan hydrogen, pelepasan electron, atau kenaikan bilangan oksidasi.  

Bilangan Oksidasi atau yang biasa disingkat biloks didefinisikan sebagai jumlah muatan negatif dan positif dalam atom, yang secara tidak langsung menandakan jumlah elektron yang telah diterima atau diserahkan.  

ATURAN PENENTUAN BILANGAN OKSIDASI  

Dalam memudahkan penentuan bilangan oksidasi, harus dipahami terlebih dahulu mengenai aturan penentuan bilangan oksidasi yang dijabarkan sebagai berikut :  

1. Bilangan oksidasi untuk unsur - unsur bebas (dalam bentuk atom, atau molekul unsur) adalah 0 (nol).  

Unsur bebas berbentuk atom contohnya Ca = 0, Na = 0, Al = 0  Unsur bebas dalam bentuk molekul contohnya H₂ = 0; O₂ = 0 ; Cl₂ = 0  

2. Bilangan oksidasi logam dalam suatu senyawa bernilai positif. Nilainya bergantung pada jenis logam.  

Contohnya bilangan oksidasi Na pada NaCl, NaOH, Na₂SO₄ = +1  

3. Bilangan oksidasi unsur golongan VIA (O, S, Se, Te, Po) yang berikatan pada senyawa biner adalah -2, dan unsur golongan VIIA (F, Cl, Br, I, At) yang berikatan dalam senyawa adalah -1.’  

Bilangan oksidasi unsur Cl pada KCl, BaCl₂ adalah -1.  Bilangan oksidasi unsur S pada H₂S dan BaS adalah -2.  

4. Bilangan oksidasi unsur H pada bentuk senyawanya adalah +1.  

5. Bilangan oksidasi unsur O pada bentuk senyawanya adalah -2, kecuali  

Dalam senyawa yang berikatan dengan F, bilangan oksidasinya adalah +2.  Dalam senyawa peroksida, seperti H₂O₂, Na₂O₂ dan BaO₂ , bilangan oksidasinya adalah -1.  Dalam senyawa superoksida, seperti KO₂ dan NaO₂, bilangan oksidasinya adalah -½.  

6. Untuk atom lain yang memiliki deret bilangan oksidasi yang berbeda dapat ditentukan melalui perhitungan dari data aturan bilangan oksidasi yang sudah diketahui.
