7. 3. Buatlah persamaaan regresi untuk data di bawah ini! X 93 76 35 83 73 74 93 76 35 83 92 93 76 35 83 73 74 74 80 80 Y 83 86 81 73 68 72 83 86 81 73 93 83 86 81 73 68 72 74 93 92​


Jawaban:

Penjelasan:

Untuk membuat persamaan regresi dari data di atas, kita dapat menggunakan rumus persamaan regresi:

y = a + bx

Dimana:

y adalah nilai yang ingin diprediksi

x adalah variabel independen

a dan b adalah koefisien regresi

Untuk menghitung koefisien regresi, kita dapat menggunakan rumus:

a = (Σxy - (Σx * Σy) / n) / (Σx^2 - (Σx^2) / n)

b = (Σy - a * Σx) / n

Dimana:

a dan b adalah koefisien regresi

x dan y adalah variabel independen dan dependen

Σxy adalah hasil kali antara x dan y untuk setiap pasangan data

Σx dan Σy adalah jumlah dari masing-masing variabel

n adalah jumlah pasangan data

Untuk data di atas, kita dapat menghitung koefisien regresi dengan menggunakan rumus di atas. Misalnya, untuk menghitung a, kita dapat menggunakan rumus:

a = (Σxy - (Σx * Σy) / n) / (Σx^2 - (Σx^2) / n)

= (4671 - (1757 * 2335) / 18) / (3514 - (1757^2) / 18)

= -16.90

Sekarang kita dapat menghitung b dengan menggunakan rumus:

b = (Σy - a * Σx) / n

= (2335 - (-16.90 * 1757)) / 18

= 1.24

Dengan demikian, persamaan regresi untuk data di atas adalah:

y = -16.90 + 1.24 * x

Artinya, nilai y dapat diprediksi berdasarkan nilai x dengan menggunakan persamaan di atas.
