13. sebuah pusat perbelanjaan terdiri atas 3 lantai. setiap lantai memiliki jumlah kios yang berbeda-beda. Jumlah kios seluruhnya adalah 55. jumlah kios pada lantai pertama dan kedua adalah 30. jumlah kios pada lantak kedua dan ketiga adalah 43. berapa jumlah kios tiap lantainya?


L1 = jumlah kios lantai 1
L2 = jumlah kios lantai 2
L3 = jumlah kios lantai 3

maka,

L1 + L2 = 30
L2 = 30 - L1

eliminasikan L2 agar terbentuk hubungan antara L3 dan L1

L3 + L2 = 43
L1 + L2 = 30
------------------ -
L3 - L1 = 13
L3 = 13 + L1

substitusikan pers. L2 dan pers. L3 agar hanya tersisa variabel L1

L1 + L2 + L3 = 55
L1 + (30-L1) + (13+L1) = 55
L1 + 43 = 55
L1 = 55-43
= 12 kios

L2 = 30-12
= 18 kios

L3 = 13 + 12
= 25 kiosMisalkan
Lantai pertama = a
Lantai kedua = b
Lantai ketiga = c

Model Matematika
1. a + b + c = 55
2. a + b = 30
3. b + c = 43

Eliminasi Pers. 1 dan 2
a + b + c = 55
a + b = 30
___________-
c = 25 kios (4)

Substitusi Pers. 4 ke Pers 3
b + c = 43
b = 43 - 25
= 18 kios (5)

Substitusi Pers. 5 ke Pers 2
a + b = 30
a = 30 - 18
= 12 kios

Jadi di lantai pertama ada 12 kios, lantai kedua ada 18 kios dan lantai ketiga ada 25 kios

Ya, karena pemilik sekaligus pengelola kios majalah di hotel tersebut bekerja dalam usahanya sendiri, di mana usaha tersebut masih dalam tahap pengembangan atau bisa di katakan sebagai usaha kecil-kecilan.
