PEMBAHASAN

Barisan aritmatika adalah suatu barisan yang memiliki bentuk nilai selisih yang mencakup dua suku yang berurutan dan selalu tetap.

Selisih antara dua suku yang berurutan tersebut dikenal sebagai nilai beda, disimbolkan dengan b.

Contoh:

[tex]203, 253, 303, 353, 403, 453, 503, 553, 603, ...[/tex]

Sehingga, barisan bilangan tersebut mempunyai nilai beda yaitu 50 antara satu suku dengan suku selanjutnya.

Pada umumnya, kita dapat menuliskan suatu barisan aritmatika dengan :

[tex]{{{{{{{{a, a+b, a+2b, a+3b, ... }}}}}}}a,a+b,a+2b,a+3b,...}[/tex]

Keterangan :

a : suku pertama,

b : nilai beda.

Rumus-Rumus Barisan Aritmatika

1. Untuk mencari Suku ke-n :

[tex]{\boxed{\large{{\blue{{Un = a + (n - 1)b}}}}}}[/tex]

Keterangan :

Un : suku ke-n

a : suku pertama

b : nilai beda

n : banyak suku

2. Untuk mencari nilai beda :

[tex]{\boxed{\large{{\blue{{b = Un-U(n-1)}}}}}}[/tex]

Keterangan :

b : nilai beda

Un : suku ke-n

3. Untuk mencari Suku Tengah :

Untuk mencari suku tengah yang mempunyai n suku ganjil (terdapat banyak suku ganjil) yang mana diketahui suku pertama dan suku terakhir, maka digunakan rumus :

[tex]{\boxed{\large{{\blue{{Ut = a + Un2}}}}}}[/tex]

Keterangan :

Ut : suku tengah

a : suku pertama

Un : suku ke-n (sebagai suku terakhir)

Dengan permasalahan lainnya, apabila kita menemukan ketika mencari suku tengah yang mana hanya diketahui suku pertama, banyaknya n suku dan nilai beda, maka digunakan rumus:

[tex]{\boxed{\large{{\blue{{Ut = a + (n-1)b2}}}}}}[/tex]

Keterangan :

Ut : suku tengah

a : suku pertama

n : banyaknya suku

b : nilai beda

[tex]kotak \: 1 = 4[/tex]

[tex]kotak \: 2 = 4 + { \green{2}} = 6[/tex]

[tex]kotak \: 3 = 6 + { \green{2}} = 8[/tex]

[tex]kotak \: 4 = 8 + { \green{2}} = 10[/tex]

[tex]{\boxed{\large{{\blue{{Un = a + (n - 1)b}}}}}}[/tex]

[tex]U_{4} = 4 + (4-1).2[/tex]

[tex]U_{4} = 4 + (3).2[/tex]

[tex]U_{4} = 4 + 6[/tex]

[tex]U_{4}=10[/tex]

[tex]{\red{\boxed{\bf{4, \: 6, \: 8, \: 10}}}}[/tex]
