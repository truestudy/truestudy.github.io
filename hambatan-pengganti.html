2. Apakah pengertian dari hambatan pengganti? dan bagaimana cara menghitung hambatan pengganti?


hambatan pengganti adalah besar hambatan yang mengalir sama besar pada setiap resistor rangkaian seri maupun paralel.

* rangkaian seri :
Rp :  R1+R2+......+Rn

* rangkaian paralel :
Rp :  1/R1 + 1/R2 +.....+ 1/Rn


3. dua hambatan jika dirangkai seri hambatan penggantinya 100ohm dan jika dirangkai paralel, hambatan penggantinya 16ohm , hambatan hambatan tersebut besarnya


Misal hambatsn itu adalah R₁ dan R₂

Dirangaki seri :
R₁ + R₂ = 100 Ω
⇔ R₁ = 100 - R₂

Dirangkai paralel
1/R₁ + 1/R₂ = 16 Ω
1/R₁ + 1/R₂ = 1/Rp
1/R₁ + 1/R₂ = 1/16
(R₂ + R₁)/R₁R₂ = 1/16
R₁R₂/(R₂ + R₁) = 16/1
R₁R₂/100 = 16
R₁R₂ = 1600

R₁R₂ = 1600
(100 - R₂) R₂ = 1600
100R₂ - R₂² = 1600
R₂² - 100R₂ + 1600 = 0
(R₂ - 20) (R₂ - 80) = 0
R₂ = 20 Ω
R₂ = 80 Ω

Jika R₂ = 20 Ω, maka R₁ = 80 Ω
Jika R₂ = 80 Ω, maka R₁ = 20 Ω

Jadi, hambatan tersebut masing masing besarnya 80 Ω dan 20 Ω

7. Berapakah hambatan penggantinya?


Diketahui :

R1 = 2Ω

R2 = 4Ω

R3 = 6Ω

Ditanyakan : Rp

Jawaban :

■ R seri :

R seri = R1 + R2 = 2Ω + 4Ω = 6Ω

■ R paralel :

[tex] \frac{1}{Rparalel} = \frac{1}{Rseri} + \frac{1}{R3}[/tex]

[tex] \frac{1}{Rparalel} = \frac{1}{6Ω} + \frac{1}{6Ω}[/tex]

[tex] \frac{1}{Rparalel} = \frac{2Ω}{6Ω}[/tex]

[tex] Rparalel = \frac{6Ω}{2Ω} [/tex]

[tex]Rparalel = 3Ω[/tex]

Jadi, hambatan pengganti yang dibutuhkan adalah 3Ω.

28. hambatan pengganti dari penghambat penghambat yang terangkai pada gambar berikut adalah


Hambatan pengganti dari penghambat penghambat yang terangkai pada gambar berikut adalah A. 90,5 Ω.

Pembahasan

Rangkaian hambatan tersebut merupakan gabungan dari rangkaian seri dan paralel. Untuk menghitung nilai hambatan penggantinya, pertama kita hitung nilai hambatan pengganti untuk hambatan paralel. Untuk rangkaian hambatan paralel, rumus hambatan penggantinya adalah

1/Rp = 1/R1 + 1/R2 + ...

1/Rp = 1 / 25Ω + 1 / 100Ω, samakan penyebutnya (100 Ω)

1/Rp = 4 / 100Ω + 1 / 100Ω

1/Rp = 5 / 100Ω  

Rp = 100Ω / 5

Rp = 20 Ω

Selanjutnya kita hitung nilai hambatan pengganti total secara seri. Untuk rangkaian hambatan seri, rumus hambatan penggantinya adalah

Rs = R1 + R2 + ...

Rs = 70 Ω + 20 Ω + 0,5 Ω

Rs = 90,5 Ω

Jadi, nilai hambatan pengganti untuk rangkaian hambatan tersebut adalah 90,5 Ω.
