15. diketahui Download lagu ay sarwendah-kesakitanku Pasto-aku tak sempurnapersamaan fungsi kuadrat f(x) = x²-px+3


Diketahui persamaan fungsi kuadrat f(x) = x² – px + 3 memiliki sumbu simetri x = 4 maka nilai p adalah 8.

Bentuk umum fungsi kuadrat adalah

f(x) = ax² + bx + c dengan a ≠ 0.  

Menentukan koordinat titik puncak/titik balik yaitu (xp, yp)

xp = [tex]-\frac{b}{2a}[/tex] ⇒ biasanya disebut sumbu simetri yp = [tex]-\frac{D}{4a}[/tex], dengan D = b² – 4ac (D = diskriminan)  

yp = f(xp) ⇒ biasanya disebut nilai balik maksimum atau minimum

Pembahasan    

Diketahui

f(x) = x² – px + 3 sumbu simetrinya: x = 4

Ditanyakan

Nilai p = … ?

Jawab

f(x) = x² – px + 3

a = 1 b = –p c = 3

Persamaan sumbu simetri

xp = 4

[tex]-\frac{b}{2a}[/tex] = 4

–b = 4(2a)

–b = 8a

–(–p) = 8(1)

p = 8
