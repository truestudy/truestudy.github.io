1. 16. Uang Ella : Uang Frita, 2 : 5 jika selisih uang Ella dan Frita Rp. 45.000,00, maka jumlah uang mereka adalah​


Jawaban singkat:

Rp. 105.000,00

Penjelasan tambahan:

Uang Ella = [tex]\frac{Rasio.Ella}{(Selisih.Rasio)}[/tex] x selisih uang mereka

Uang Ella = [tex]\frac{2}{(5-2)}[/tex] x Rp. 45.000,00

Uang Ella = [tex]\frac{2}{3}[/tex] x Rp. 45.000,00

Uang Ella = Rp. 30.000,00

Uang Frita = [tex]\frac{Rasio.Frita}{(Selisih.Rasio)}[/tex] x selisih uang mereka

Uang Frita = [tex]\frac{5}{(5-2)}[/tex] x Rp. 45.000,00

Uang Frita = [tex]\frac{5}{3}[/tex] x Rp. 45.000,00

Uang Frita = Rp. 75.000,00

Jumlah uang mereka = Rp. 30.000,00 + Rp. 75.000,00

Jumlah uang mereka = Rp. 105.000,00

Jadi jumlah uang mereka adalah Rp. 105.000,00

PERBANDINGAN UANG ELLA DAN FRITA :

2 : 5

P.ELLA DAN FRITA = 5 - 2 = 3

MAKA :

UANG ELLA :

= 2/3(45.000)

= 2(45.000/3)

= 2(15.000)

= 30.000

UANG FRITA :

= 5/3(45.000)

= 5(45.000/3)

= 5(15.000)

= 75.000

TOTAL :

30.000 + 75.000

= 105.000
