Pembahasan

Tabung merupakan gabungan dari 2 bangun datar yaitu persegi panjang dan dua buah lingkaran.

Lingkaran 2 untuk tutup tabung , dan

Jaring jaring tabung merupakan gabungan dari lingkaran dan persegi panjang.

Jadi, Tabung merupakan gabungan dari 2 bangun datar yaitu persegi panjang dan dua buah lingkaran.

Luas = luas ¼ lingkaran besar + luas ½ lingkaran kecil

Luas ¼ lingkaran besar = ¼ π r²

[tex] \frac{1}{4} \times \frac{22}{7} \times 21 \times 21 [/tex]

= 346,5 cm²

Luas ½ lingkaran kecil = ½ π r²

[tex] = \frac{1}{2} \times \frac{22}{7} \times \frac{21}{2} \times \frac{21}{2} \\ = 173.25[/tex]

= 173,25 cm²

Luas gabungan = 346,5 + 173,25 = 519,75 cm²

19. 5 contoh bentuk bangun datar beserta gambar sketsa,lokasi ditemukan , dan manfaatnya


1. Persegi panjang
Contoh benda : Papan tulis
Dapat ditemukan di : Depan tiap kelas
Kegunaan : Menulis pelajaran ; media pembelajaran
2. Persegi
Contoh benda : Ubin lantai
Dapat ditemukan di : Lantai bangunan
Kegunaan : Melapisi lantai agar permukaannya rata dan tidak kasar
3. Segitiga
Contoh benda : Papan penunjuk arah
Dapat ditemukan di : Pinggir jalan raya
Kegunaan : Menunjukkan tempat atau lokasi suatu bangunan
4. Lingkaran
Contoh benda : Cermin
Dapat ditemukan di : Kamar mandi, ruang kelas, wastafel
Kegunaan : Memantulkan benda yang ada di depanya ; untuk berkaca
5. Belah ketupat
Contoh benda : Papan peringatan (misalnya peringatan ada konstruksi jalan dsb)
Dapat ditemukan di : Pinggir jalan, di jalan raya
Kegunaan : Memperingatkan para pengguna jalan raya agar lebih berhati-hati agar tidak celaka karena ada konstruksi jalan dengan alat-alat berat serta instalasi listrik dengan voltase yang tinggi

Hitunglah luas bangun datar tersebut!​


B.)

Jari-jari(r) = Diameter / 2

jari-jari lingkaran besar = 21 / 2 = 10,5

Luas lingkaran = [tex]\pi[/tex] x r x r

dikarenakan cuma setengah, maka

L [tex]\frac{1}{2}[/tex] lingkaran ROS (lingkaran besar) = [tex]\frac{1}{2}[/tex] x 3,14 x 10,5 x 10,5

= [tex]\frac{1}{2}[/tex] x 3,14 x 110,25

= [tex]\frac{1}{2}[/tex] x 346,185

= 173,0925 cm²

Jari-jari lingkaran kecil = 10,5 / 2 = 5,25

L [tex]\frac{1}{2}[/tex] lingkaran SO (lingkaran kecil) = [tex]\frac{1}{2}[/tex] x 3,14 x 5,25 x 5,25

= [tex]\frac{1}{2}[/tex] x 3,14 x 27,5625

= [tex]\frac{1}{2}[/tex] x 86,54625

= 43,273125 cm²


Luas total = 173,0925 + 43,273125 = 216,365625 cm²
