15. Tentukan hasil penjumlahan dan pengurangan sudut berikut a. 40°32'35"+26°34'58"=....b.28°19'32"- 42°39'47"=.... C. 89°42'36"- 38°36'24"+ 27°43'57"=​


Tentukan hasil penjumlahan dan pengurangan sudut berikut:

a. 40° 32' 35" + 26° 34' 58" = 67° 7' 33"

b. 28° 19' 32" - 42° 39' 47" = - 14° 20' 15"

c. 89° 42' 36" - 38° 36' 24" + 27° 43' 57" =​ 78° 50' 9"

Penjelasan dengan langkah-langkah:

Untuk menyelesaikan soal di atas caranya adalah dengan menjumlahkan besarnya derajat dengan derajat, menit dengan menit dan detik dengan detik. Setelah itu gunakan besaran derajat menit dan detik di bawah ini untuk menyederhanakannya:

1° = 60' = 60 menit

1 menit = 60" = 60 detik

1° = 3600" = 3600 detik

Pada soal a akan ditambahkan 40° 32' 35" dengan 26° 34' 58". Langkah pertama adalah jumlahkan 40° dengan 26°, kemudian 32' dengan 34' dan terakhit 35" dengan 58"

a. 40° 32' 35" + 26° 34' 58" = ( 40° + 26° ) ( 32' + 34' ) ( 35" + 58" )

40° 32' 35" + 26° 34' 58" = 66° 66' 93"

Setelah itu sederhanakan mulai dari belakang 93" sama dengan 1' 33 detik, jadi jumlahkan angka 1' ke 66'

40° 32' 35" + 26° 34' 58" = 66° 66' + 1' 33"

40° 32' 35" + 26° 34' 58" = 66° 67' 33"

Kemudian lakukan langkah yang sama untuk satuan menit

40° 32' 35" + 26° 34' 58" = 66° + 1° 7' 33"

40° 32' 35" + 26° 34' 58" = 67° 7' 33"

Untuk soal b karena bilangan yang dikurangi lebih kecil dari pengurangnya, maka untuk mempermudahnya pengurangan dikerjakan dengan mengubah satuan derajat dan menit menjadi detik

b. 28° 19' 32" - 42° 39' 47" = ( ( 28 x 3.600" ) + ( 19 x 60" ) + 32" ) - ( ( 42 x 3.600" ) + ( 39 x 60" ) + 47" )

28° 19' 32" - 42° 39' 47" = ( 100.800" + 1.140" + 32" ) - ( 151.200" + 2.340" + 47" )

28° 19' 32" - 42° 39' 47" = 101.972" - 153.587"

28° 19' 32" - 42° 39' 47" = -51.615"

Setelah itu ubah kembali menjadi derajat menit dan detik dengan membaginya dengan 3.600 dan 60

28° 19' 32" - 42° 39' 47" = - 14° 1.215"

28° 19' 32" - 42° 39' 47" = - 14° 20' 15"

Pada soal c pengurangan dapat dikerjakan langsung tanpa mengubah menjadi detik karena angka yang dikurangi lebih kecil dari pengurangnya.

c. 89° 42' 36" - 38° 36' 24" + 27° 43' 57" =​ ( 89° - 38° ) ( 42' - 36' ) ( 36" - 24" ) + 27° 43' 57"

89° 42' 36" - 38° 36' 24" + 27° 43' 57" = 51° 6' 12" + 27° 43' 57"

89° 42' 36" - 38° 36' 24" + 27° 43' 57" = ( 51° + 27° ) ( 6' + 43' ) ( 12" + 57" )

89° 42' 36" - 38° 36' 24" + 27° 43' 57" = 78° 49' 69"

89° 42' 36" - 38° 36' 24" + 27° 43' 57" = 78° 50' 9"
