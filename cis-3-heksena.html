15. senyawa 3,4 dimetil pentana berisomer dengan??? a.normal heksena b.2-metil heksena c.2,5-dimetil heksena d.3-etil heksena e.n-heksena

Pelajaran   : Kimia
Kategori     : Hidrokarbon
Kata Kunci : alkana, ikatan tunggal, isomer

Pembahasan

Struktur dari 3,4-dimetil-pentana

CH₃ - CH₂ - CH - CH - CH₃
                     |       |
                   CH₃  CH₃

Penamaan ini belum tepat, kita luruskan namanya menjadi 2,3-dimetil-pentana

Isomer-isomernya adalah
CH₃ - CH₂ - CH₂ - CH₂ - CH₂ - CH₂ - CH₃ ....n-heptana (normal heksana)
CH₃ - CH₂ - CH₂ - CH₂ - CH - CH₃
                                        |
                                      CH₃ .....(2-metil-heksana)
CH₃ - CH₂ - CH₂ - CH - CH₂ - CH₃
                               |
                            CH₃ .....(3-metil-heksana)
CH₃ - CH - CH₂ - CH - CH₃
           |                  |
         CH₃             CH₃ ......(2,4-dimetil-pentana)

                           CH₃
                              |
CH₃ - CH₂ - CH₂ - CH - CH₃
                              |
                            CH₃ ......(2,2-dimetil-heksana)

                  CH₃
                    |
CH₃ - CH₂ - C - CH₂ - CH₃
                    |
                  CH₃ .....(3,3-dimetil-heksana)

                 CH₃
                   |
CH₃ - CH - C - CH₃
            |      |
        CH₃   CH₃......(2,2,3-trimetil-butana)

CH₃ - CH₂ - C - CH₂ - CH₃
                    |
                C₂H₅.....(3-etil-pentana)

29. Berapakah 3 cis 120• + 4 cis 270•=


Bilangan Kompleks.

cis θ = cos θ + i sin θ

3 cis 120° + 4 cis 270°
= 3(cos 120° + i sin 120°) + 4(cos 270° + i sin 270°)
= 3(-1/2 + 1/2 i√3) + 4[0 + i (-1)]
= -3/2 + 3/2 i√3 - 4i
= -3/2 - 5/2 i√3
