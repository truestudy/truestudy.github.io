9. Sebutkan 5 tools pada software coreldraw jelaskan fungsinya?


Ada banyak tools tersedia pada toolbox di software Coreldraw yang terletak di sisi kiri tampilan. Berikut ini adalah 5 tools Coreldraw berikut dengan fungsinya:

Pick tool: untuk memilih, mengatur ukuran dan memutar objek gambar.

Crop tool: menghilangkan bagian yang tidak diinginkan dalam sebuah objek.

Erase tool: menghapus area pada sebuah gambar.

Freehand tool: menggambar ruas atau lengkungan dalam bentuk satu garis.

Smart Fill tool: untuk membuat objek dan mengisinya dengan warna atau tekstur.

10. Sebutkan 5 tools pada software coreldraw dan jelaskan fungsinya


1. Shape Tool (F10)

= Berfungsi untuk mengedit objek kurva atau garis.

2. Smooth Tool

= Berfungsi untuk memperlembut objek kurva.

3. Smear Tool

= Berfungsi untuk menarik kurva menjadi tertarik keluar atau ke dalam maupun ke atas atau ke bawah, ke kanan atau kesamping. Dengan terbentuknya seperti teluk atau tanjung.

4. Twirl Tool

= Berfungsi untuk membentuk objek seperti pusaran air, dengan bentuk putaran putaran

5. Attract Tool

= Berfungsi mengubah garis luar (outline) menuju arah pointer, menjadi seperti magnet

17. corelDraw adalah jenis software​


Corel Draw adalah jenis software aplikasi yang dipergunakan untuk desain grafis atau pengolah gambar. Corel Draw merupakan suatu perangkat lunak komputer yang dikembangkan oleh Corel Corporation. Corel Corporation merupakan suatu perusahaan perangkat lunak yang berpusat di Ottawa, Kanada.

Penjelasan:

Software atau perangkat lunak merupakan suatu program di komputer yang bertugas untuk melakukan pengolahan data menjalankan fungsi tertentu.

Jenis-jenis Software

Secara garis besar, software dapat dibedakan menjadi 2, yaitu:

1. Software Sistem Operasi

Software sistem operasi merupakan jenis software yang bertugas untuk menjalankan dan mengendalikan semua komponen yang ada pada komputer. Contoh software sistem operasi: windows dan linux.

2. Software Aplikasi

Software aplikasi merupakan jenis software yang ditulis dengan bahasa pemrograman tertentu untuk menjalankan fungsi tertentu. Ada berbagai macam software aplikasi, berikut beberapa diantaranya:

- Software aplikasi pengolah kata, digunakan untuk kebutuhan menulis. Contoh, Microsoft office word dan star office writer.

- Software aplikasi pengolah angka, digunakan untuk mengerjakan segala hal yang berhubungan dengan angka. Contoh: Microsoft office excel dan lotus 123.

- Software aplikasi pengolah gambar, digunakan untuk mengolah data grafis/ gambar. Contoh Adobe Photoshop dan Corel Draw.

- Software aplikasi browser, digunakan untuk mencari informasi di internet. Contoh: Google Chrome dan Mozilla Firefox.

- Software aplikasi multimedia, digunakan untuk menyajikan atau mengedit berbagai media komunikasi seperti gambar, suara, dan tulisan. Contoh, windows media player dan windows movie maker.

-  Software aplikasi antivirus, digunakan untuk mencegah virus masuk ke dalam komputer. Contoh: Avast dan Smadav.
