Pembahasan

[tex]\boxed{\boxed{\bold{Tata~Nama~Senyawa}}}[/tex]

Untuk mengetahui nama suatu senyawa terlebih dahulu kita harus memahami tentang lambang unsur dan nama unsur kimia. Hal tersebut merupakan ilmu dasar yang harus diketahui untuk memperdalam tentang tata nama senyawa. Penamaan senyawa diatur oleh IUPAC (International Union Pure and Applied Chemistry). berdasarkan kesepakatan oleh para ahli ilmuan didunia.

[tex]\mathbf{Tata~Nama~Senyawa~Anorganik}[/tex]

[tex]\mathbf{1.~Senyawa~Biner~dari~Logam~dan~Non~Logam}[/tex]

a. Logam yang memiliki bilangan oksidasi 1 jenis (Golongan IA, IIA, IIIA)

1) Menyebutkan nama logam nya

2) Menyebutkan nama non logam nya + akhiran –ida

Contoh:

NaCl = natrium klorida

MgF2 = magnesium florida

CaCl2 = kalsium klorida

Na2O = natrium oksida

b. Logam yang memiliki bilangan oksidasi lebih dari 1 jenis (logam transisi)

1) Menyebutkan nama logam nya diikuti angka romawi sesuai dengan bilangan oksidasi unsur logam

2) Menyebutkan nama non logam nya + akhiran –ida  

Contoh :

FeO = besi (II) oksida

Cu2O = tembaga (II) oksida

PbI2 = timbal (II) iodida

[tex]\mathbf{2.~Senyawa~Biner~dari~Non~Logam~dan~Non~Logam}[/tex]

1) Menyebutkan nama Yunani jumlah atom non logam ke-1 diikuti dg nama unsur non logam ke-1

2) Menyebutkan nama Yunani jumlah atom non logam ke-2 diikuti dg nama unsur non logam ke-2 + akhiran –ida

Jumlah Atom dalam Bahasa Yunani

1 Mono

2 Di

3 Tri

4 Tetra

5 Penta

6 Heksa

7 Hepta

8 Okta

9 Nona

10 Deka

NB: untuk awala mono tidak digunakan untuk unsur non logam ke-1

[tex]\mathbf{3.~Senyawa~yang~Mengandung~Ion~Poliatom}[/tex]

a. Logam yang memiliki bilangan oksidasi 1 jenis (Golongan IA, IIA, IIIA)

1) Menyebutkan nama kation logamnya

2) Menyebutkan nama anion poliatom

b. Logam yang memiliki bilangan oksidasi lebih dari 1 jenis (logam transisi)

1) Menyebutkan nama kation logam nya diikuti angka romawi sesuai dengan bilangan oksidasi logam

2) Menyebutkan nama anion poliatom

Beberapa Ion Poliatom

NH4+ Amonium

OH- Hidroksida  

CO32- Karbonat  

NO3- Nitrat  

CH3COOH- Asetat  

PO43- Fosfat

SO42- Sulfat  

[tex]\mathbf{4.~Senyawa~Asam~(H^+)}[/tex]

1) Didahului dengan kata asam (H+ ) / kation

2) Menyebutkan nama anion nya

Contoh :

HCl = asam klorida

CH3COOH = asam asetat

H2SO4 = asam sulfat

HNO3 = asam nitrat

[tex]\mathbf{5.~Senyawa~Basa~(OH^-) }[/tex]

1) Menyebutkan nama kationnya

2) Menyebutkan nama anionnya (hidroksida = OH-)

Contoh :

NaOH = natrium hidroksida

Fe(OH)3 = besi (III) hidroksida

Ca(OH)2 = kalsium hidroksida

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]

Unsur Cl = golongan VIIA = non logam

Unsur O = golongan VIA = non logam


[tex]\mathbf{Senyawa~Biner~dari~Non~Logam~dan~Non~Logam}[/tex]

1) Menyebutkan nama Yunani jumlah atom non logam ke-1 diikuti dg nama unsur non logam ke-1

2) Menyebutkan nama Yunani jumlah atom non logam ke-2 diikuti dg nama unsur non logam ke-2 + akhiran –ida

Cl₂O₅

Jumlah atom Cl adalah 2 dalam bahasa Yunani = di

Jumlah atom O adalah 5 dalam bahasa Yunani = penta

Oksigen + -ida = oksida

maka,

Dikloro Pentaoksida


[tex]\boxed{\boxed{\bold{Kesimpulan}}}[/tex]

Dikloro Pentaoksida

Nama Lazim

Jawab:

Menentukan nama senyawa SiCl₄.

SiCl₄

Nama non logam Si = silikonJumlah atom Si = 1 (awalan mono tidak ditulis)Nama non logam Cl + -ida = kloridaJumlah atom Cl = 4 (tetra)

SiCl₄ = Silikon tetraklorida

Menentukan nama senyawa Cl₂O₅.

Cl₂O₅

Nama non logam Cl = kloroJumlah atom Cl = 2 (di)Nama non logam O + -ida = oksidaJumlah atom O = 5 (penta)

Cl₂O₅ = dikloro pentaoksida

Menentukan nama senyawa CS₂.

CS₂

Nama non logam C = karbonJumlah atom C = 1 (awalan mono tidak ditulis)Nama non logam S + -ida = sulfidaJumlah atom S = 2 (di)

CS₂ = Karbon disulfida

Berdasarkan perhitungan diatas, kesimpulan yang kita dapatkan adalah

SiCl₄ = Silikon tetraklorida

Cl₂O₅ = dikloro pentaoksida

CS₂ = Karbon disulfida
