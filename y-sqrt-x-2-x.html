4. (y- sqrt x )^2+x^ ini apa y​


Untuk penulisan fungsi kurang tepat, seharusnya (y - √|x|)² + x² = 1 agar rumus fungsinya berbentuk hati.

Pembahasan

Sederhanakan bentuk fungsinya

(y - √|x|)² + x² = 1

(y - √|x|)² = 1 - x²

y - √|x| = [tex]\sqrt{1 - x^{2} }[/tex]

       y = [tex]\sqrt{1 - x^{2} }[/tex] + √|x|

Menentukan titik koordinat

y = [tex]\sqrt{1 - x^{2} }[/tex] + √|x|

Untuk x = -1

y = [tex]\sqrt{1 - (-1)^{2} }[/tex] + √|-1|

 = [tex]\sqrt{1 - 1 }[/tex] + √1

 = 0 + 1

 = 1       titik (-1 , 1)

Untuk x = -0,75

y = [tex]\sqrt{1 - (-0,75)^{2} }[/tex] + √|-0,75|

 = [tex]\sqrt{1 - 0,5625}[/tex] + √0,75

 = √0,4375 + √0,75

 = 0,66 + 0,86

 = 1,52       titik (-0,75 , 1,52)

Untuk x = -0,5

y = [tex]\sqrt{1 - (-0,5)^{2} }[/tex] + √|-0,5|

 = [tex]\sqrt{1 - 0,25}[/tex] + √0,5

 = √0,75 + √0,5

 = 0,86 + 0,7

 = 1,56       titik (-0,75 , 1,56)

Untuk x = -0,25

y = [tex]\sqrt{1 - (-0,25)^{2} }[/tex] + √|-0,25|

 = [tex]\sqrt{1 - 0,0625}[/tex] + √0,25

 = √0,94 + √0,25

 = 0,97 + 0,5

 = 1,47       titik (-0,25 , 1,47)

Untuk x = 0

y = [tex]\sqrt{1 - (0)^{2} }[/tex] + √|0|

 = [tex]\sqrt{1 - 0}[/tex] + √0

 = 1 + 0

 = 1       titik (0 , 1)

Untuk x = 0,25

y = [tex]\sqrt{1 - 0,25^{2} }[/tex] + √|0,25|

 = [tex]\sqrt{1 - 0,0625}[/tex] + √0,25

 = √0,94 + √0,25

 = 0,97 + 0,5

 = 1,47       titik (0,25 , 1,47)

Untuk x = -0,5

y = [tex]\sqrt{1 - 0,5^{2} }[/tex] + √|0,5|

 = [tex]\sqrt{1 - 0,25}[/tex] + √0,5

 = √0,75 + √0,5

 = 0,86 + 0,7

 = 1,56       titik (0,75 , 1,56)

Untuk x = 0,75

y = [tex]\sqrt{1 - 0,75^{2} }[/tex] + √|0,75|

 = [tex]\sqrt{1 - 0,5625}[/tex] + √0,75

 = √0,4375 + √0,75

 = 0,66 + 0,86

 = 1,52       titik (0,75 , 1,52)

Untuk x = 1

y = [tex]\sqrt{1 - 1^{2} }[/tex] + √|1|

 = [tex]\sqrt{1 - 1 }[/tex] + √1

 = 0 + 1

 = 1       titik (1 , 1)

Kita hubungkan titik-titik tersebut sehingga fungsi tersebut berbentuk hati yang bisa dilihat pada lampiran.

16. Hasil dari (y-\sqrt{|x|})^2+x^)


Untuk penulisan fungsi kurang tepat, seharusnya (y - √|x|)² + x² = 1 agar rumus fungsinya berbentuk hati.

Pembahasan

Kita sederhanakan bentuk fungsinya

(y - √|x|)² + x² = 1

(y - √|x|)² = 1 - x²

y - √|x| = [tex]\sqrt{1 - x^{2} }[/tex]

[tex]\sqrt{x}-\sqrt{y}=\sqrt{x+y-2\sqrt{xy}}[/tex]
berlaku jika x ≥ y, dengan syarat x, y ≥ 0.
 

Penjelasan dengan langkah-langkah:

Persamaan [tex]\sqrt{x}-\sqrt{y}=\sqrt{x+y-2\sqrt{xy}}[/tex] terdefinisi pada himpunan bilangan real jika:

[tex]\sqrt{x}-\sqrt{y}\ \ge \ 0\ \land\ x,y \ge 0[/tex]

yang menyebabkan

[tex]\sqrt{x}\, \ge \,\sqrt{y} \implies x \ge y \ \land\ x,y \ge 0[/tex].

Jika kedua ruas kita kuadratkan, akan diperoleh:

[tex]\begin{aligned}\left(\sqrt{x}-\sqrt{y}\right)^2&=\left(\sqrt{x+y-2\sqrt{xy}}\right)^2\\x+y-2\sqrt{xy}&=x+y-2\sqrt{xy}\end{aligned}[/tex]

Ruas kiri sama dengan ruas kanan, sehingga persamaan berlaku untuk semua nilai [tex]x[/tex] dan [tex]y[/tex] yang memenuhi syarat:

[tex]x \ge y\ \land\ x,y \ge 0[/tex].

Catatan:

Jika himpunan penyelesaian untuk [tex]x[/tex] dan [tex]y[/tex] dipersempit menjadi himpunan bilangan bulat, maka hanya terdapat solusi tunggal, yaitu [tex]x = y = 0[/tex].

Kita juga dapat menyatakan solusinya, dengan [tex]x, y \in \mathbb{R}[/tex], dalam bentuk:

[tex]\begin{cases}y=0,\!\!\!&{\sf untuk\ }x=0\\0 \le y \le x,\!\!\!&{\sf untuk\ }x > 0\\\end{cases}[/tex]
