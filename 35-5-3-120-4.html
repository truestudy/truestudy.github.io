Mapel : Matematika

Kelas : 5

Materi : Bab 2 - Operasi Bil. Bulat

Kata Kunci : Bilangan

Kode Kategorisasi : 5.2.2 ( Matematika Kelas 5 Bab 2 - Operasi Bil. Bulat )

————————————————————

35 + 5 × ( - 3 ) + 120 : 4

= 35 + ( - 15 ) + 30

= 35 - 15 + 30

= 20 + 30

= 50 ..... ✔️

Bentuk Umum Operasi Bilangan Bulat.

Pengurangan :

a + ( - b ) = a - b

a - ( - b ) = a + b

Penjumlahan :

- a + b = b - a

- a + ( - b ) = - ( a + b )

Perkalian :

a × ( - b ) = - ( a × b )

- a × - b = a × b

Pembagian :

a : a = 1

- a : - a = a : a

- a : - b = a : b

Perpangkatan :

a² = a × a

a³ = a × a × a

aⁿ × aⁿ = a^(n + n)

aⁿ : aⁿ = a^(n–n)
