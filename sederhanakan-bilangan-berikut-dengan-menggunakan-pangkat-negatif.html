----------------------------Pendahuluan: Bilangan berpangkat (eksponen)Bilangan berpangkat merupakan bilangan hasil dari perkalian berulang yang memiliki bilangan sama.

Sifat-Sifat Pada Bilangan Berpangkat :

[tex] a^m \times  a^n = a^{(m+n)} [/tex][tex] a^m \div a^n = a^{(m-n)} [/tex][tex] (a^m)^n = a^{m\times n} [/tex][tex] (ab)^n =a^nb^n [/tex][tex] (\frac{a}{b})^n = \frac{a^n}{b^n} [/tex][tex] a^{-n} = \frac{1}{a^n} [/tex][tex] \sqrt[n]{ {a}^{m} } = {a}^{ \frac{m}{n} } [/tex][tex] a^0 = 1 [/tex]

[tex] = \frac{1}{ {9}^{3} } [/tex]

[tex] = \frac{1}{ {3}^{(2 \times 3)} } [/tex]

[tex] = \frac{1}{ {3}^{6} } [/tex]

[tex] = {3}^{ - 6} [/tex]

[tex] \: [/tex]

b.

[tex] = \frac{4}{12} [/tex]

[tex] = \frac{1}{3} [/tex]

[tex] = {3}^{ - 1} [/tex]

[tex] \: [/tex]

c.

[tex] = \frac{8}{625} [/tex]

[tex] = \frac{ {2}^{3} }{ {5}^{4} } [/tex]

[tex] = \frac{ {5}^{ - 4} }{ {2}^{ - 3} } [/tex]

d.

[tex] = \frac{4x}{ {(ab)}^{5} } [/tex]

27. sederhanakanlah bilangan berikut dengan menggunakan pangkat negatif8 per 625


aᵇ → a = bilangan basis dan b = pangkat dari bilangan basis

Terdapat beberapa sifat dari bilangan berpangkat

1. Jika terdapat beberapa bilangan berpangkat yang dikalikan tetapi bilangan pokoknya sama, maka pangkatnya ditambah

2. Jika terdapat 2 bilangan berpangkat yang dibagi tetapi bilangan pokoknya sama, maka pangkatnya dikurang

3. Jika terdapat bilangan berpangkat yang dipangkatkan maka pangkatnya dikalikan

4. Jika terdapat bilangan berpangkat yang pangkatnya negatif, untuk mengubah menjadi berpangkat positif, maka dijadikan 1/bilangan itu

5. Jika terdapat bilangan berpangkat yang pangkatnya positif, untuk mengubah menjadi berpangkat negatif, maka dijadikan 1/bilangan itu

6. Jika terdapatbilangan berpangkat yang pangkatnya pecahan misalnya a/b, maka menjadi ᵇ√bilanganᵃ

7. 1 pangkat berapapun hasilnya adalah 1 dan suatu bilangan pangkat 0 hasilnya adalah 1 di mana bilangan tersebut ≠ 0
