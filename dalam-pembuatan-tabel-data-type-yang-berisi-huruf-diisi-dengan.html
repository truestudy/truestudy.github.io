16. Jelaskan Fungsi tipe data text,type data number,type data date/time,type data autonumber

Tipe data number : untuk menyimpan data numeric / angka yang akan digunakan untuk proses perhitungan matematis.

26. bagaimana langkah-langkah untuk membuat grafik (chart) type balok dari sebuah data pada tabel Microsoft Excel?​


Jawaban:

Langkah-langkah untuk membuat grafik (chart) type balok dari sebuah data pada tabel Microsoft Excel:

1. Buat data terlebih dahulu, lalu blok range data.

2. klik ribbon Insert pilih Recommended Charts, sehingga jendela Insert Chart terbuka.

3. Pilih Tab All Charts, kemudian klik Column.

4. Pilih jenis Column yang berbentuk balok, kemudian klik OK.

5. Geser grafik yang dibuat, lalu tambahkan nama grafik dengan melakukan klik tulisan Chart Title.

6. Grafik telah berhasil dibuat.
