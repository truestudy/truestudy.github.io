1 bulan = 1/12 tahun

54 x 1/12 = 4,5 tahun = 4 tahun 6 bulan


2. 126 hari=.....bulan.....hari

1 hari = 1/30 bulan

126 x 1/30 = 4,2 bulan = 4 bulan 6 hari


3. 567 hari=.....bulan.....hari

1 hari = 1/30 bulan

567 x 1/30 = 18,9 bulan = 18 bulan 27 hari


4. 46 minggu=.....bulan.....minggu

1 minggu = 1/4 bulan

46 x 1/4 = 11,5 bulan = 11 bulan 2 minggu


5. 560 jam=.....hari.....jam

1 jam = 1/24 hari

560 x 1/24 = 23,33 hari = 23 hari 8 jam


6. 20 jam=.....hari.....jam

1 jam = 1/24 hari

20 x 1/24 = 0,83 hari = 0 hari 20 jam


7. 400 hari=.....tahun.....hari

1 hari = 1/365 tahun

400 x 1/365 = 1,096 tahun = 1 tahun 35 hari


8. 578 jam=..... hari

578 x 1/24 = 24,083 hari = 5 1/12 hari


9. 350 jam =.....minggu

= 20 bulan = 1 tahun 6 bulan
= 1,6 + 46
= 47 tahun 6 bulan
Dijadikan ke bulan
= 20 + ( 46 x 12 )
= 20 + 552
= 572 bulan
Dijadikan ke minggu
= ( 20 x 4 ) + ( 46 x 52 )
= 80 + 2395
= 2472 minggu
Dijadikan ke hari
= ( 20 x 30 ) + ( 46 x 365 )
= 600 + 16790
= 17390 hari
