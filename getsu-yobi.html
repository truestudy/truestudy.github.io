1. nichi yobi , getsu yobi , ka yobi , sui yobi ,moku yobi ​


Jawaban:

nichi youbi=日曜日=minggu

getsu youbi=月曜日=senin

ka youbi=火曜日=selasa

sui youbi=水曜日=rabu

moku youbi=木曜日=kamis

Nama hari =
日曜日 ( にちようび ) Nichiyōbi : Minggu
Cara baca : nichiyoubi

月曜日 ( げつようび ) Getsuyōbi : Senin
Cara baca : geysuyoubi

火曜日 ( かようび ) Kayōbi : Selasa
Cara baca : kayoubi

水曜日 ( すいようび ) Suiyōbi : Rabu
Cara baca : suiyoubi

木曜日 ( もくようび ) Mokuyōbi : Kamis
Cara baca : mokuyoubi

金曜日 ( きんようび ) Kin'yōbi : Jumat
Cara baca : kinyoubi

土曜日 ( どようび ) Doyōbi : Sabtu
Cara baca : doyoubi

20. Tolong bantuan kakak. Dalam waktu 30 menit yobi dapat berlari jarak sejauh 10km. Jika yobi berlari selama 11/2jam maka jarak yang ditempuh yobi adalah... Km


30 menit=10 km

11/2 jam =330 menit

jawab

30 m=10 km

330 m=x km

30.x=330.10

30x=3.300

x=3.300:30

x=110km

jadi dalam waktu 11/2 jam =110 km

30. nihon go jugyo wa nan yobi desu ka


Nihon go wa kantan desu wa omoshiroi desu.Nihongo
N5

Nihon go jugyou wa nan youbi desu ka
