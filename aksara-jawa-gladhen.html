_______________________________________

Penjelasan:......1. ꦠꦸꦏꦸ ꦏꦺꦴꦥꦾꦃ = tuku kopyahaksara ta + suku, ka + suku = tukuaksara ka + taling tarung, aksara pa + pengkal + wignyan = kopyah

2. ꦲꦩ꧀ꦥꦾꦁ ꦒꦺꦴꦫꦺꦁ = ampyang gorengaksara ha, aksara ma (sigeg), pasangan pa + pengkal + cecek = ampyangaksara ga + taling tarung, aksara ta + taling + cecek = goreng

3. ꦱꦼꦠꦾ ꦭꦤ꧀ꦩꦶꦠꦸꦲꦸ = setya lan mituhuaksara sa + pepet, aksara ta + pengkal = setyaaksara la, aksara na (sigeg) = lanpasangan ma + wulu, aksara ta + suku, aksara ha + suku = mituhu

4. ꦮꦢꦾ ꦧꦭ ꦥꦿꦗꦸꦫꦶꦠ꧀ = wadya bala prajuritaksara wa, aksara da + pengkal = wadyaaksara ba lan la = balaaksara pa + cakra, ja + suku, ra + wulu, ta + pangkon = prajurit

5. ꦏꦼꦫꦺꦠ ꦏꦾ ꦲꦶ ꦗꦭꦢꦫ = kereta kyai jaladaraaksara ka + pepet, Ra + taling, aksara ta = keretaaksara ka + pengkal, aksara ha + wulu = kyaiaksara ja, aksara la, aksara da, aksara ra = Jaladara
