Jawaban:

2 kg

Penjelasan:

c = 2200 J/kg°C

delta t = 47°C

Q = 206.800 J

Q = m . c . delta t

206800 = m . 2200 . 47

m = 206800 : 2200 : 47

m = 2 kg


10. dua liter minyak deg p(massa jenis): 0,8gr/cm3 dipanaskan dr 0'c sampai 80'c. bila kalor jenis minyak 4.600j/kg K. berapa kalor yang dibutuhkan??


Diketahui :
V = 1 liter
ρ = 0,8 gr/cm3
To = 0 C
Tt = 80 C
c = 4.600 J/Kgk
Ditanyakan : Q ?

Dijawab :
V = 0,001 m³
ρ = 0,8 = 800 kg/m³

Massa :
m = v x ρ
    = 0,001 x 800
    = 0,8 kg

Kalor yang dibutuhkan :
Q = m x c x (Tt-To)
    = 0,8  x 4600 x (80-0)
    = 0,8 x 4600 x 80
    = 294.400 J

m = massa = 500 gr = 0,5 kg
c = kalor jenis
ΔT = perubahan suhu = 10°C

11.000 = 0,5 x c x 10
11.000 = 5 x c
c = 11.000 : 5 = 2.200 Joule/kg°C
