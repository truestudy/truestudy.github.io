Untuk mengetahui berapa skor yang didapatkan, sebaiknya dicari terlebih dahulu jumlah soal yang tidak terjawab, berikut penyelesaiannya:

Soal yang tidak dijawab = Keseluruhan soal olimpiade - Total soal yang dijawab

= 100 - 83

= 17 soal

Rumus perhitungan nilai soal

Setelah itu dapat dihitung skor yang didapatkan Nisa, berikut penyelesainnya:

Skor yang didapatkan Nisa = Soal dengan jawaban benar + Soal dengan jawaban salah + Soal yang tidak dijawab

= (80 x 4) + (3 x (-1)) + (17 x 0)

= 320 - 3 + 0

= 317

Jawab:

DIJAWAB BENAR = 28

DIJAWAB SALAH = 31-28 = 3

TIDAK DIJAWAB = 40 - 31 = 9

Penjelasan dengan langkah-langkah:

skor :

B = 3

S = -1

T = 0

40 SOAL

Nilai 81

81 = 3B + (-1 x (31-B))

81 = 3B + (-31 + B)

81 = 3B -31 + B

112 = 4 B

Dalam kompetisi matematika jawaban benar diberi skor 3, jawaban salah diberi skor -1, dan jika tidak menjawab diberi skor 0. dari 40 soal, dedi menjawab 31 soal, skor yang diperoleh 81. Banyak soal yang dijawab benar oleh Dedi adalah 28 butir soal

Diketahui :

Skor benar      = 3

Skor salah       = -1

Tidak dijawab = 0

Banyak soal dijawab = 31

Skor didapat = 81

Ditanyakan :

Banyak soal yang dijawab benar = . . . .   butir soal

Jawab :

Misalkan :

Banyak soal dijawab benar = [tex]x[/tex]

Maka banyak soal dijawab salah = [tex](31 - x)[/tex]

Soal yang tidak dijawab sebanyak = [tex]40 - 31 = 9[/tex]

Nilai = (3 x soal benar) + (-1 x soal salah) + (0 x tidak dijawab)

  81  = ( [tex]3x[/tex] ) + ( [tex]-1 (31 - x)[/tex] ) + ([tex]0 \times 9[/tex] )

  81  = ( [tex]3x[/tex] ) + ( [tex]-31 + x[/tex] ) + ([tex]0[/tex] )

  81  = [tex]3x[/tex]  [tex]-31 + x[/tex]

  81  = [tex]4x[/tex]  [tex]-31[/tex]

  [tex]4x[/tex] = [tex]81 + 31[/tex]

  [tex]4x[/tex] = [tex]112[/tex]

    [tex]x[/tex] = 28 butir soal

∴ Jadi banyak soal yang dijawab benar oleh Dedi adalah 28 butir soal
