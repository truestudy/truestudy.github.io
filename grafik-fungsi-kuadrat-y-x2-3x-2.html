7. Sumbu simetri dari grafik fungsi kuadrat y = x2 3x 2 adalah


Jika diketahui fungsi kuadrat [tex] \rm y = x^2+3x+2[/tex]. Maka sumbu simterinya adalah [tex] \bf (-\frac{3}{2} , -\frac{1}{4})[/tex]

Penjelasan dengan langkah-langkah

Diketahui :

Fungsi kuadrat [tex] \rm y = x^2+3x+2[/tex]

Ditanya :

Sumbu simteri?

Jawab :

Dari fungsi diketabui :

a = 1b = 3c = 2

Sumbu simetri (x,y). Cari titik x :

[tex] \rm x = \frac{-b}{2a}[/tex]

[tex] \rm x = \frac{-3}{2(1)}[/tex]

[tex] \rm x = -\frac{3}{2}[/tex]

Cari titik y :

[tex] \rm y = \frac{b^2-4ac}{-4a}[/tex]

[tex] \rm y = \frac{3^2-4(1)(2)}{-4(1)}[/tex]

[tex] \rm y = \frac{9-8}{-4}[/tex]

[tex] \rm y =- \frac{1}{4}[/tex]

Jadi, sumbu simteri grafik fungsi tersebut adalah [tex] \bf (-\frac{3}{2} , -\frac{1}{4})[/tex].

Pembahasan

Bentuk umum fungsi kuadrat adalah :

f(x) = y = ax² + bx + c

a ≠ 0

a dan b adalah koefisien, c adalah konstanta.

Grafik akan memotong sumbu X jika f(x) = 0

Grafik akan memotong sumbu Y jika x = 0

1.

Diketahui :

A(-1,0)

B(-4,-3)

C(-6,5)

Ditanya :

Persamaan kuadrat yang melalui ketiga titik tersebut

Jawab :

Masukkan ketiga titik di atas ke dalam bentuk umum persamaan kuadrat!

A(-1,0) => x = -1 dan y = 0

y = ax² + bx + c

0 =  a(-1)² + b(-1) + c

a - b + c = 0

c = b - a

B(-4,-3) => x = -4 dan y = -3

f(x) = ax² + bx + c

-3 = a(-4)² + b(-4) + c

16a - 4b + c = -3

B(-6,5) => x = -6 dan y = 5

f(x) = ax² + bx + c

5 = a(-6)² + b(-6) + c

36a - 6b + c = 5

16a - 4b + c = -3

16a - 4b + (b - a) = -3

16a - 4b + b - a = -3

15a - 3b = -3

5a -b = -1

-b = -1 - 5a

b = 5a + 1

36a - 6b + c = 5

36a - 6b + (b - a) = 5

36a - 6b + b - a = 5

35a - 5b = 5

7a - b = 1

7a - (5a + 1) = 1

7a - 5a - 1 = 1

2a = 1 + 1

2a = 2

a = 1

b = 5a + 1

b = 5(1) + 1

b = 5 + 1

b = 6

c = b -a

c = 6 - 1

c = 5

Masukkan nilai a, b, dan c ke dalam bentuk umum fungsi kuadrat :

y = ax² + bx + c

y = (1)x² + 6x + 5

y = x² + 6x + c

Jadi pilihan jawaban yang benar adalah c. y = x² + 6x + 5

2.)  Pembahasan :

Suatu fungsi dari himpunan A ke himpunan B adalah suatu relasi khusus sedemikian hingga setiap anggota himpunan A dipasangkan dengan tepat satu anggota himpunan B.

Himpunan A dinamakan domain atau daerah asal dan himpunan B dinamakan kodomain atau daerah kawan.

Jika fungsi f memetakan setiap x ∈ A dengan tepat ke satu anggota y ∈ B, maka f : x → y.

Peta dari x ∈ A oleh fungsi f sering dinyatakan sebagai f(x) dan bentuk f(x) dinamakan rumus fungsi f.

Himpunan y ∈ B yang merupakan peta dari x ∈ A dinamakan range atau daerah hasil.

Misalkan fungsi f dinyatakan dengan f : x → ax + b dengan a dan b merupakan konstanta dan x merupakan variabel, maka rumus fungsinya adalah f(x) = ax + b.

Jika nilai variabel x = m, maka nilai f(m) = am + b.

Dengan demikian, kita dapat menentukan bentuk fungsi f, bila diketahui nilai-nilai fungsinya.

Selanjutnya, nilai konstanta a dan b ditentukan berdasarkan nilai-nilai fungsi yang diketahui.
