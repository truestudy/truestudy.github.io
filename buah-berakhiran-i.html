Dua buah modal mempunyai nilai awal yang sama.
Misalkan modal awal I = modal awal II = M.
Modal I dibungakan dengan bunga majemuk 10% per tahun.
Setelah n tahun, modal I akan menjadi:
Tabungan tahun ke-n = (100% + 10%)ⁿ × M = (1,1)ⁿ × M
Modal II dibungakan dengan bunga tunggal 12% per tahun.
Setelah n tahun, modal II akan menjadi:
Tabungan tahun ke-n = Modal awal + (n × besar bunga × Modal awal)
                                  = M + 0,12n × M = (1 + 0,12n)M
Akan ditentukan n di mana nilai akhir modal I lebih besar dari nilai akhir modal II.

nilai akhir modal I > nilai akhir modal II
(1,1)ⁿ × M > (1 + 0,12n)M
↔ (1,1)ⁿ > 1 + 0,12n
untuk n = 1, (1,1)¹ = 1,1 < 1,12 = 1 + 0,12(1)
untuk n = 2, (1,1)² = 1,21 < 1,24 = 1 + 0,12(2)
untuk n = 3, (1,1)³ = 1,331 < 1,36 = 1 + 0,12(3)
untuk n = 4, (1,1)⁴ = 1,4641 < 1,48 = 1 + 0,12(4)
untuk n = 5, (1,1)⁵ = 1,61051 > 1,6 = 1 + 0,12(5)

Jadi, nilai akhir modal I akan lebih besar dari nilai akhir modal II pada akhir tahun ke-5.
