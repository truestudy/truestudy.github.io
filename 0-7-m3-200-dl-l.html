Pembahasan  

30 dam³  

= (30 x 1.000) m³  

= 30.000 m³


82 m³  

= (82 x 1.000) dm³  

= 82.000 dm³


0,076 m³

= (0,076 x 1.000.000) cm³  

= 76.000 cm³


70 L  

= (70 x 10) dL  

= 700 dL


500 L  

= 500 dm³  

= (500 : 1.000) m³  

= 0,5 m³


25.000 dm³ + 45 dam³  

= (25.000 : 1.000) m³ + (45 x 1.000) m³

= 25 m³ + 45.000 m³

= 45.025 m³


2,8 m³ + 800 dm³  

= (2,8 x 1.000) dm³ + 800 dm³

= 2.800 dm³ + 800 dm³

= 3.600 dm³

= 3.600 L


2 daL + 300 dL

= (2 x 10) L + (300 : 10) L

= 20 L + 30 L

= 50 L


0,7 m³ + 200 dL

= (0,7 x 1.000) dm³ + (200 : 10) L

= 700 dm³ + 20 L

= 700 L + 20 L

= 720 L


20 m³ + 400 L  

= (20 x 1.000) dm³ + 400 dm³

= 20.000 dm³ + 400 dm³

= 20.400 dm³
