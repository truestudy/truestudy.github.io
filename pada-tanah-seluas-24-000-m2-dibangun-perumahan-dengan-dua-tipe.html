7. Pada tanah seluas 24.000 m2 dibangun perumahan dengan dua tipe. Tipe A dengan luas 150m2 dan tipe B dengan luas 100 m2. Jumlah rumah yang dibangun tidak lebih dari 200 unit. Jika laba untuk setiap rumah tipe A Rp4.000.000,00 dan tiap rumah tipe B Rp3.000.000,00, maka laba maksimum yang dapat diperoleh adalah … Select one:


Jawaban:

ada dibawah kak jangan lupa lope sama rating 5nya hehew

Penjelasan:

Rumah      | Tipe A (x)   | Tipe B (y)    |

Luas          |    150x        |  100y          | 24.000

Kapasitas  |       x           |     y             | 200

Laba          | 4.000.000x | 3.000.000y | ... ? 

1) 150x + 100y ≤ 24.000 => kedua ruas bagi 50

=> 3x + 2y ≤ 480

Jika x = 0 => y = 240 ==> (0, 240)

Jika y = 0 => x = 160 ==> (160, 0)  

2) x + y ≤ 200

Jika x = 0 => y = 200 ==> (0, 200)

Jika y = 0 => x = 200 ==> (200, 0)  

Fungsi sasaran :

f(x, y) = 4.000.000x + 3.000.000y  

Titik potong kedua garis

3x + 2y = 480 |x1|

x + y = 200 |x2|  

3x + 2y = 480

2x + 2y = 400

------------------ -

x = 80  

x + y = 200

80 + y = 200

y = 120  

Titik potong kedua garis (80, 120)  

Setelah kita gambar sketsa grafiknya (bisa dilihat pada lampiran)

Diperoleh titik – titik sudutnya yaitu (160, 0), (0, 200) dan (80, 120)

Substitusikan ke fungsi sasaran

f(x, y) = 4.000.000x + 3.000.000y

f(160, 0) = 4.000.000(160) + 3.000.000(0)

= 640.000.000 + 0

= 640.000.000

f(0, 200) = 4.000.000(0) + 3.000.000(200)

= 0 + 600.000.000

= 600.000.000

f(80, 120) = 4.000.000(80) + 3.000.000(120)

= 320.000.000 + 360.000.000

= 680.000.000  

Jadi keuntungan maksimum yang diperoleh adalah Rp680.000.000,00

(dengan menjual 80 rumah tipe A dan 120 rumah tipe B) .

Pada daerah tanah seluas 24.000 m² dibangun perumahan dengan dua tipe.Tipe A dengan luas 150 m² dan tipe B dengan luas 100 m². Jumlah rumah yang dibangun tidak lebih dari 200 unit. Jika laba untuk setiap rumah tipe A Rp4.000.000,00 dan setiap rumah tipe B Rp3.000.000,00 maka laba maksimum yang dapat diperoleh adalah…  

Pembahasan :  

Untuk memudahkan membuat model matematikanya, kita susun dalam tabel berikut

Rumah      | Tipe A (x)   | Tipe B (y)    |
Luas          |    150x        |  100y          | 24.000
Kapasitas  |       x           |     y             | 200
Laba          | 4.000.000x | 3.000.000y | ... ? 

1) 150x + 100y ≤ 24.000 => kedua ruas bagi 50
=> 3x + 2y ≤ 480
Jika x = 0 => y = 240 ==> (0, 240)
Jika y = 0 => x = 160 ==> (160, 0)  

2) x + y ≤ 200
Jika x = 0 => y = 200 ==> (0, 200)
Jika y = 0 => x = 200 ==> (200, 0)  

Fungsi sasaran :
f(x, y) = 4.000.000x + 3.000.000y  

Titik potong kedua garis
3x + 2y = 480 |x1|
x + y = 200 |x2|  

3x + 2y = 480
2x + 2y = 400
------------------ -
x = 80  

x + y = 200
80 + y = 200
y = 120  

Titik potong kedua garis (80, 120)  

Setelah kita gambar sketsa grafiknya (bisa dilihat pada lampiran)
Diperoleh titik – titik sudutnya yaitu (160, 0), (0, 200) dan (80, 120)
Substitusikan ke fungsi sasaran
f(x, y) = 4.000.000x + 3.000.000y

f(160, 0) = 4.000.000(160) + 3.000.000(0)
= 640.000.000 + 0
= 640.000.000

f(0, 200) = 4.000.000(0) + 3.000.000(200)
= 0 + 600.000.000
= 600.000.000

f(80, 120) = 4.000.000(80) + 3.000.000(120)
= 320.000.000 + 360.000.000
= 680.000.000  

Jadi keuntungan maksimum yang diperoleh adalah Rp680.000.000,00
(dengan menjual 80 rumah tipe A dan 120 rumah tipe B)

Jika laba untuk setiap rumah tipe A Rp 10.000.000 dan tiap rumah tipe B Rp 5.000.000, maka laba maksimum yang dapat diperoleh adalah Rp 1.400.000.000.

Penjelasan dengan langkah-langkah:

Soal di atas dapat diselesaikan dengan membuat persamaan linear dua variabelnya. Dengan rumah tipe A sebagai variabel x dan rumah tipe B sebagai variabel y.

Diketahui luas tanah yang tersedia adalah 21.000 m² dan setiap rumah tipe A membutuhkan lahan seluas 150 m² dan setiap rumah tipe B membutuhkan lahan seluas 100 m². Maka persamaannya adalah:

150 x + 100 y = 21.000 disederhanakan menjadi

3x + 2y = 420

Jumlah rumah yang dibangun tidak lebih dari 150 buah, maka persamaannya adalah:

x + y = 150

Setelah dibuat model matematikanya kemudian dibuat grafik dari model matematika tersebut dan dicari titik potong grafik tersebut dengan sumbu x dan sumbu y. Untuk persamaan pertama titik potong dengan sumbu x dan sumbu y adalah:

3x + 2y = 420 → x = 0 → 0 + y = 420 → y = 210

Titik potong = ( 0 , 210 )

3x + 2y = 420 → y = 0 → x + 0 = 420 → x = 140

Titik potong = ( 140 , 0 )

Untuk persamaan kedua titik potong dengan sumbu x dan sumbu y adalah:

x + y = 150 → x = 0 → 0 + y = 150 → y = 150

Titik potong = ( 0 , 150 )

x + y = 150 → y = 0 → x + 0 = 150 → x = 150

Titik potong = ( 150 , 0 )

Daerah penyelesaian dari soal diatas adalah daerah antara titik ( 0 , 0 ), ( 140 , 0 ), ( 0 , 150 ) dan titik potong kedua garis. Untuk mencari titik potong kedua garis digunakan cara eliminasi dan substitusi dari nilai x dan y dari kedua persamaan diatas. Langkah pertama eliminasi variabel x dari kedua persamaan di atas.

Eliminasi variabel y dari kedua persamaan untuk menentukan nilai x

3x + 2y = 420    | x1

x + y = 150         | x2

3x + 2y = 420

2x + 2y = 300     -

x = 120

Substitusikan nilai x ke persamaan x + y = 150

120 + y = 150

y = 150 - 120

y = 30

Jadi titik potong dari kedua garis tersebut adalah ( 120 , 30 ). Setelah itu dilakukan pengujian dari ketiga titik potong tersebut dengan memasukkan nilai keuntungan dari setiap tipe rumah

( 140 , 0 ) → 140 x 10.000.000 → 1.400.000.000

( 0 , 150 ) → 150 x 5.000.000 → 750.000.000

( 120 , 30 ) → 120 x 10.000.000 + 30 x 5.000.000 → 1.200.000.000 + 150.000.000 → 1.350.000.000

Keuntungan terbesar ada pada titik potong garis dengan sumbu x. Keuntungan maksimum yang bisa didapatkan dari pembangunan rumah tersebut adalah Rp 1.400.000.000.
