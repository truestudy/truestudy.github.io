4. pak joko mengendarai sepeda motor dengan kecepatan 50000 m/jam.Jika kecepatan pak joko di hitung dalam km/jam maka hasilnya adalah ​


Jika kecepatan Pak Joko dihitung dalam km/jam, maka hasilnya adalah 50 km/jam.

Satuan kecepatan dapat dinyatakan dalam berbagai macam satuan, salah satunya adalah meter per jam (m/jam) dan kilometer per jam (km/jam). Satuan meter per jam menunjukkan berapa banyak meter yang dapat ditempuh oleh benda per satuan waktu, sedangkan satuan kilometer per jam menunjukkan berapa banyak kilometer yang dapat ditempuh oleh benda per satuan waktu.

Dalam kasus ini, Pak Joko mengendarai sepeda motor dengan kecepatan 50000 m/jam. Jika kecepatan tersebut dihitung dalam km/jam, maka kita dapat menggunakan rumus konversi sebagai berikut:

V(km/jam) = V(m/jam) / 1000

Dimana V(km/jam) adalah kecepatan dalam km/jam, dan V(m/jam) adalah kecepatan dalam m/jam.

Dengan menggunakan rumus tersebut, kecepatan Pak Joko dapat dihitung sebagai berikut:

V(km/jam) = 50000 m/jam / 1000

= 50 km/jam

Jadi, jika kecepatan Pak Joko dihitung dalam km/jam, maka hasilnya adalah 50 km/jam.

20. Buatlah sebuah program untuk menghitung gaji harian,menggunakan c++ gaji perjam = 50000, bila jumlah jam kerja lebih dari 7 jam, maka lebihnya dianggap lembur dan gaji perjam lembur = 15 x gaji perjam


Buatlah sebuah program dengan input bilangan bulat :

>> N (jam)

Dan dengan output bilangan bulat gaji yang didapat setelah N (jam) bekerja :

>> gaji

Batasan :

0 <= N <= 24

Contoh program :

#include <iostream>  

using namespace std;

int N;

int gaji;  

int jamlembur, gajilembur;

bool lembur;

int main()

{

   cin >> N;

   lembur = false;

 

   if  ((N >= 0) && (N <= 7)) {

gaji = N*50000;

    } else if ((N > 7) && (N <= 24)) {

jamlembur = N-7;

gajilembur = (7*50000) + (jamlembur*15*50000);

lembur = true;

     } else {

cout << "Tidak ada jam yang bernilai negatif atau lebih dari 24 !" <<   endl;  

     }

    if (lembur) {

cout << gajilembur << endl;

    } else if (!lembur) {

    cout << gaji << endl;

    }  

return 0;

}

Catatan :

Sebenarnya program di atas tidak perlu menggunakan tipe data boolean (bool) dengan variabelnya yang dideklarasikannya yaitu 'lembur'.

Cobalah hapus tipe data dan variabel boleannya dan ubah program tersebut supaya tidak menggunakan boolean !

Petunjuk :

1) Yang pasti hapus dulu bool lembur; dan sintaks lainnya yang mengandung boolean.

2) Ubahlah bagian if(.......) di bawah ini jika boolean lembur tidak ada.

   if (lembur) {

cout << gajilembur << endl;

    } else if (!lembur) {

    cout << gaji << endl;

    }
