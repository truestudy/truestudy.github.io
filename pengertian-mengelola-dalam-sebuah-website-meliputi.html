3. pengertian website statis dan website dinamis


Singkatnya:
website statis adalah website yang isi kontennya tidak dapat diubah oleh user (client). Karena website ini hanya menampilkan informasi yang ditulis sesuai dengan script dan biasanya tidak memiliki database. Biasanya, script yang digunakan hanya menggunakan pemrograman html. Misalnya: website profil perusahaan. Informasi yang ditampilkan tidak akan berubah kecuali admin mengubah script yang telah ditulisnya.

untuk website dinamis, adalah kebalikannya. Konten/tampilan/informasi yang ditampilkan bisa diubah oleh user (client). Menggunakan database dan biasanya ada tambahan pemrograman php didalamnya. Misalnya: Website jejaring sosial atau forum. Disitu kita bisa mengisi biodata atau dengan kata lain bisa mengubah isi konten dari web itu sendiri.

18. 18 melalui cPanel anda dapat mengelola berbagai fasilitas hosting, kecuali. a. Mengelola e-mail b. Mengupload website c. Membuat dan mengupload database d. Melihat statistik website e. Mengamankan jaringan.


Jawaban:

Jawaban yang tepat adalah: e. Mengamankan jaringan.

cPanel adalah aplikasi yang digunakan untuk mengelola fasilitas hosting pada sebuah server. Melalui cPanel, pengguna dapat mengelola berbagai fasilitas hosting seperti mengelola email, mengupload website, membuat dan mengupload database, dan melihat statistik website. Namun, cPanel tidak memiliki fitur untuk mengamankan jaringan. Fitur tersebut biasanya disediakan oleh perangkat lunak keamanan jaringan atau firewall yang terpisah.

23. pengertian define website


Pada bahasa pemrograman PHP ada sebuah fungsi yang bernama “define”. Yap, dari namanya saja sudah bisa ditebak guna dari fungsi tersebut. Fungsi “define” adalah untuk mendefinisikan. Misalnya kita akan mendefinisikan bahwa DODOL adalah “makanan khas Garut”.  Format fungsi define adalah sebagai berikut :

define(name,value,case_insensitive)

Jadi ada tiga parameter dalam fungsi define, yaitu name, value dan case_insensitive.  Kita akan bahas satu persatu biar jelas.  name adalah parameter nama yang akan didefinisikan (“DODOL”)  value adala parameter isi/arti dari sesuatu yang didefinisikan (“makanan khas Garut”)  case_insensitive adalah nilai boolean untuk penulisan definisi (TRUE/FALSE). Jika bernilai TRUE, maka pada pemanggilan definisi tidak akan case sensitive.  Catatan, untuk parameter ketiga boleh tidak diisi.
