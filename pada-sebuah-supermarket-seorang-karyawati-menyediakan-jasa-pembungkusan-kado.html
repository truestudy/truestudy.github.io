jika ingin mendapat upah maksimal maka dimaksimalkan jumlah bahannya (dihabiskan)
kertas pembungkus → 2x + 2y = 50
pita → 2x + y = 40

2x + 2y = 50
2x + y = 40
——————— -
y = 10
2x + 10 = 40
2x = 30
x = 15

upah maks = 5000x + 4000y
= 5000*15 + 4000*10
= 75000 + 40000
= Rp 115000,-

[tex]\boxed{Pembahasan:}[/tex]Fungsi Linear

Fungsi linear digunakan menyelesaikan masalah dengan model matematika yang terdiri atas pertidaksamaan linear yang mempunyai banyak kemungkinan penyelesaian. Dari semua hasil kemungkinan satu akan memberikan hasil yang optimal.

[tex]\boxed{Diketahui:}[/tex]Kado jenis A membutuhkan 2 lembar kertas dan 2 m pitaKado jenis B membutuhkan 2 lembar kertas dan 1 m pitaTersedia 50 lembar kertasTersedia 40 m pitaUpah untuk Kado jenis A Rp. 5.000Upah Untuk Kado jenis B Rp. 4.000

[tex]\boxed{Ditanyakan:}[/tex]Upah maksimal yang dapat diperoleh = ...

[tex]\boxed{Penyelesaian:}[/tex]

Dari Data Diatas kita buat tabel Untuk memudahkan :

   Model  ||  Kertas  ||  Pita  ||  Upah

   A          ||      2       ||    2    ||   5.000

   B          ||      2       ||     1    ||   4.000

Tersedia  ||     50     ||    40  ||   Optimum ?

Dari Tabel kita buat modelmatematika :

dengan Kertas sebagai x dan pita sebagai y  :

2x + 2y < 50   ⇒ dibagi 2

x + y < 25

2x + y < 40

x > 0

y > 0

Cari Titik Koordinatnya :

(1) Dari x + y < 25

x = 0 ⇒ y = 25 (0,25)

y = 0 ⇒ x = 25 (25,0)

(2) Dari 2x + y < 40

x = 0 ⇒ y = 40 (0,40)

y = 0 ⇒ x = 20 (20,0)

(3) Titik potong antara dua garis :

Kita eliminasi y :

x  +  y  = 25

2x + y  = 40

x + y ≤ 20  

• pita  2x + y ≤ 30  

• x ≥ 0, y ≥ 0  

• Upah => fungsi sasaran
f(x, y) = 2.500x + 2.000y  

Kemudian kita gambar grafiknya, yaitu dengan cara :  

1) x + y ≤ 20 => diarsir ke bawah
x = 0 => y = 20 ... (0, 20)
y = 0 => x = 20 ... (20, 0)
tarik garis yang melalui titik (0, 20) dan (20, 0)  

2) 2x + y ≤ 30 => diarsir ke bawah
x = 0 => y = 30 ... (0, 30)
y = 0 => x = 15 ... (15, 0)
tarik garis yang melalui titik (0, 30) dan (15, 0)

2x + y = 30
x + y = 20
----------------- -
x = 10  

x + y = 20
10 + y = 20
y = 10  
(10, 10)

(0, 0) => 2.500(0) + 2.000(0) = 0

(10, 10) => 2.500(10) + 2.000(10) = 45.000
(15, 0) => 2.500(15) + 2.000(0) = 37.500

Penyelesaian Soal :

Langkah pertama untuk mengerjakan soal di atas adalah dengan melakukan permisalan untuk kedua variabel bebasnya, yaitu jenis kado dan bahan yang digunakan :

misalkan kado jenis A = x

               Kado jenis B = y

Oleh karena banyaknya lembar kertas kado yang diperlukan sebanyak 3 lembar dan 2 lembar dengan persediaan 60 lembar, maka 3x +2 y ≤ 60 .....  (persamaan I). Program Linear

Program linear merupakan suatu metode penentuan nilai optimum dari suatu persoalan linear. Nilai optimum (maksimal atau minimum) diperoleh dari nilai dalam suatu himpunan penyelesaiaan persoalan linear. Di dalam persoalan linear terdapat fungsi linear yang bisa disebut sebagai fungsi objektif. Persyaratan, batasan, dan kendala dalam persoalan linear merupakan sistem pertidaksamaan linear.

Model Matematika Program Linear

Persoalan dalam program linear yang masih dinyatakan dalam kalimat-kalimat pernyataan umum, kemudian diubah kedalam model matematika. Model matematika merupakan suatu cara merubah permasalahan sehari-hari ke dalam bahasa matematika dalam bentuk persamaan, pertidaksamaan, dan fungsi.
