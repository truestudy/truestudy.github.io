13. Format gambar yang sesuai untuk file ePub adalah…​

File EPub

EPUB adalah format file standar terbuka untuk buku digital yang dikembangkan oleh International Digital Publishing Forum (IDPF). Salah satu manfaat utama format EPUB untuk pembaca adalah, format ini memungkinkan teks buku otomatis menyesuaikan atau "mengalir ulang" ke berbagai ukuran layar, misalnya untuk ponsel cerdas, netbook, dan perangkat eReader. EPUB juga mendukung konten tata letak tetap. Pelajari lebih lanjut tentang format EPUB di EPUBZone IDPF.

Terdapat sejumlah alat dan layanan untuk membantu Anda membuat file EPUB. Meskipun kami tidak menyarankan salah satunya secara khusus karena banyaknya opsi yang dimiliki mitra kami, namun perangkat lunak yang pada umumnya digunakan meliputi Adobe InDesign CS dan Apple Pages for Mac OS X. 22. tuliskan macam macam format file gambar ​


Jawaban:

JPG, PNG, TIFF, BMP, GIF

Penjelasan:

TIFF (Tagged Image Format)

File gambar berformat TIF merupakan file gambar yang tak dikompres. Kompres yaitu memadatkan file sehingga ukuran file menjadi lebih kecil. Namun, karena dikompresi, gambar akan menjadi kurang mendetail. Karena tak dikompres, file gambar berformat TIF berukuran besar. Nah, karena file tersebut berukuran besar, file gambar berformat TIFF sebenarnya lebih mendetail. File gambar berformat TIFF diakhiri dengan ekstensi file *.tif. MIsalnya, gambar.tif. File gambar TIFF menjadi salah satu format file yang biasa digunakan dalam bidang fotografi digital. Bila kita menginginkan tampilan gambar wallpaper yang tajam, menggunakan wallpaper berformat TIFF dapat menjadi alternatif

JPG (Joint Photographic Expert Group)

File gambar berformat JPG yaitu file gambar yang dikompresi. Karena dikompresi, bila dibandingkan dengan file gambar berformat TIFF, gambar JPG sebenarnya terlihat kurang mendetail. Ditinjau dari ukuran file, file gambar JPG relatif berukuran kecil. Tentu saja, karena telah dikompresi. Karena berukuran kecil, gambar JPG menjadi gambar yang biasa dilampirkan pada situs internet untuk menghias tampilan web.

GIF (Graphic Interchange Format)

File gambar GIF tak dikompresi seperti halnya file gambar JPG. Namun, file gambar ini hanya mampu menampilkan sedikit warna saja. Kita pun tak bisa mengubahnya menjadi format gambar JPG. Ukuran file ini pun relatif kecil. Karenanya, biasa ditampilkan pada situs web untuk menghias tampilan web. File gambar ini pun dapat juga dibuat animasi dengan menggunakan beberapa gambar GIF. Gambar GIF pun biasanya digunakan untuk menghias tampilan web.

PNG (Portable Network Graphics)

File gambar berformat PNG, pada dasarnya, diciptakan untuk menggantikan file gambar GIF. Sebabnya, file gambar GIF telah dipatenkan. Siapapun yang ingin menggunakan format gambar tersebut untuk tujuan komersial tentunya mesti membayar lisensi. Nah, file gambar berformat PNG mampu menampilkan lebih banyak warna. Namun, kurang cocok untuk bidang fotografi. Sebabnya, ukurannya relatif besar. Akan tetapi, bila terdapat teks pada gambar, cocok menggunakan format PNG. Sebabnya, teks akan terlihat lebih tajam.

BMP (Bitmap)

File gambar berformat BMP, secara mendasar, yaitu kumpulan jutaan titik-titik warna. Karena, gambar BMP sebenarnya terlihat begitu realistis. Namun, ukuran file relatif besar. Selain itu, bila gambar BMP kita perbesar ataupun perkecil, gambar cenderung akan terlihat kurang mendetail.
