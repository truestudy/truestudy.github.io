Jawaban:

The correct statement based on the following text is A. We must respect our friends who are working seriously.

Penjelasan:

The text states that "We must respect our friends who are working seriously." This implies that making noise in the class would disturb those who are working seriously, and therefore, we should not be noisy. Option A correctly summarizes this idea. Option B contradicts the text, and options C and D misinterpret the text.
