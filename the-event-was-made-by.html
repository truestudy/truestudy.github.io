29. Write the active sentenceThis apple pie was madeby Grandma.​


Active Sentence:

Grandma made this apple pie

PEMBAHASAN Perintah Soal

Write the active sentence “This apple pie was made by Grandma.“

Structure kalimat di atas, bisa disimpulkan bahwa kalimat tsb merupakan Passive dari Past Tense karena memiliki pola Be + V3. Berikut Berdasarkan Rumus:

Passive-Past Tense

(+) O + Be (was/were) + V3 + By S

=> This apple pie was made by Grandma.

Keterangan:

Object => This apple pieBe Past => wasVerb-3 => madeBy Subject => by grandma

Maka kalimat Aktifnya:

Active-Past Tense

(+) S + V2 + O

=> Grandma made this apple pie

Keterangan:

Subject => GrandmaVerb-2 => madeObject => this apple pie
