3. Zat penyedap masakan (MSG) merupakan senyawa yang mengandung nitrogen dengan kadar 9 persen. Jika masa molar MSG = 155 gram/mol, maka jumlah atom nitrogen dalam setiap molekul MSG adalah


Massa nitrogen = 9/100 × 155 = 13.95 g (boleh dibulatkan ke 14 g).
Ar N = 14 g/mol
mol N = 13.95 / 14 = 1 mol

Jumlah atom N = mol × NA (tetapan Avogadro)
Jumlah atom N = 1 × 6.02 × 10^23
Jumlah atom N = 6.02 × 10^23 atom

21. Monosidium glutamat (MSG) adalah zat penyedap rasa yg lebih dikenal dengan nama vetsin. Senyawa dengan massa molar sekitar 169 g/mol tsb mengandung 35,51% karbon; 4,77% hidrogen; 37,85% oksigen; 8,29% nitrogen; dan 13,60% natrium. Tentukan rumus kimia senyawa tsb!


C₅H₈NO₄Na

Pembahasan

Rumus kimia senyawa adalah rumus molekul yang diolah dari rumus empiris, namun terkadang ditemui pula rumus molekul yang tepat sama dengan rumus empiris. Rumus empiris tersusun dari perbandingan banyaknya atom unsur. Rumus molekul adalah kelipatan dari rumus empiris yang dihubungkan dengan massa molar.

Karena diketahui persentase komponen unsur, pada umumnya kita misalkan senyawa tersebut mempunyai massa 100 gram. Berikut massa dari tiap komponen unsur penyusun MSG:

[tex]\boxed{ \ C = 35,51 \ gram \ }[/tex][tex]\boxed{ \ H = 4,77 \ gram \ }[/tex] [tex]\boxed{ \ O = 37,85 \ gram \ }[/tex] [tex]\boxed{ \ N = 8,29 \ gram \ }[/tex] [tex]\boxed{ \ Na = 13,60 \ gram \ }[/tex]

Massa atom relatif (Ar) dari C = 12, H = 1, O = 16, N = 14, Na = 23.

Step-1: rumus empiris

Disiapkan perbandingan mol tiap unsur untuk menyusun rumus empiris Ingat bahwa [tex]\boxed{ \ mol = \frac{massa}{Ar} \ }[/tex]

[tex]\boxed{ \ C : H : N : Na : O = \frac{35,51}{12} : \frac{4,77}{1} : \frac{8,29}{14} : \frac{13,60}{23} : \frac{37,85}{16} \ }[/tex]

[tex]\boxed{ \ C : H : N : Na : O = 2,9 : 4,77 : 0,6 : 0,6 : 2,36 \ }[/tex]

Semua angka banding dibagi 0,6 supaya lebih sederhana.

[tex]\boxed{ \ C : H : N : Na : O = 4,83 : 7,95 : 1 : 1 : 3,94 \ }[/tex]

Dibulatkan.

[tex]\boxed{ \ C : H : N : O : Na = 5 : 8 : 1 : 4 : 1 \ }[/tex]

Dengan demikian rumus empirisnya adalah [tex]\boxed{ \ C_5H_8NO_4Na \ }[/tex]

Step-2: rumus molekul

Rumus molekulnya adalah [tex]\boxed{ \ (C_5H_8NO_4Na)_n \ }[/tex]

Kita hitung nilai n sebagai angka pengali.

[tex]\boxed{ \ (C_5H_8NO_4Na)_n = massa \ molar \ MSG \ }[/tex]

[tex]\boxed{ \ [(5 \times 12) + (8 \times 1) + (1 \times 14) + (4 \times 16) + (1 \times 23)] \times n = 169 \ }[/tex]

[tex]\boxed{ \ (60 + 8 + 14 + 64 + 23) \times n = 169 \ }[/tex]

[tex]\boxed{ \ 169 \times n = 169 \ }[/tex]

[tex]\boxed{ \ n = 1 \ }[/tex]

Substitusikan kembali nilai n.

[tex]\boxed{ \ (C_5H_8NO_4Na)_1 \rightarrow C_5H_8NO_4Na \ }[/tex]

Jadi, rumus kimia senyawa monosodium glutamat (MSG) adalah [tex]\boxed{\boxed{ \ C_5H_8NO_4Na \ }}[/tex] yang juga tepat sama dengan rumus empirisnya.

- - - - - - - - - -

Catatan:

Monosodium glutamat (MSG) adalah zat penyedap rasa yang lebih dikenal dengan nama vetsin. Penggunaannya sangat luas pada industri makanan khususnya di rumah-rumah makan karena dapat memberikan rasa gurih pada makanan. MSG yang diproduksi di Indonesia berasal dari sari tetes tebu yang merupakan hasil sampingan penggilingan gula. Konsumsi MSG secara berlebihan ditengarai dapat mempengaruhi kerja sistem syaraf.
