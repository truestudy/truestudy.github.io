2. Suku banyak berderajat 3 jika dibagi (x2 + 2x – 3) bersisa (3x – 4) dan jika dibagi (x2 – x – 2) bersisa (2x + 3). Suku banyak tersebut adalah ….


Materi: Polinomial

Penjelasan dengan langkah-langkah:

Karena suku banyak berderajat 3, maka kita misalkan:

[tex]p(x)=ax^3+bx^2+cx+d[/tex]

Diketahui pada soal:

+ p(x) dibagi x^2+2x-3 bersisa 3x-4, maka bisa kita tuliskan:

[tex]p(x)=q(x)r(x)+s(x)\\p(x)=(x^2+2x-3)r(x)+(3x-4)\\p(x)=(x+3)(x-1)r(x)+(3x-4)...(1)[/tex]

+ p(x) dibagi x^2-x-2 bersisa 2x+3, maka bisa kita tuliskan:

[tex]p(x)=(x^2-x-2)r(x)+(2x+3)\\p(x)=(x-2)(x+1)r(x)+(2x+3)...(2)[/tex]

Dengan mengambil persamaan (1) dan (2), akan diperoleh data sebagai berikut:

s(x)=3x-4

s(-3)=3(-3)-4=-13

s(1)=3(1)-4=-1

s(x)=2x+3

s(2)=2(2)+3=7

s(-1)=2(-1)+3=1

Substitusikan semua data tersebut ke polinomialnya, sehingga diperoleh 4 buah persamaan linear.

[tex]f(-1)=s(-1)\\1=a(-1)^3+b(-1)^2+c(-1)+d\\-a+b-c+d=1\,(\text{kali}\,-1)\\a-b+c-d=-1...(1)\\f(1)=s(1)\\-1=a(1)^3+b(1)^2+c(1)+d\\a+b+c+d=-1...(2)\\f(-3)=s(-3)\\-13=a(-3)^3+b(-3)^2+c(-3)+d\\-27a+9b-3c+d=-13\,(\text{kali}\,-1)\\27a-9b+3c-d=13...(3)\\f(2)=s(2)\\7=a(2)^3+b(2)^2+c(2)+d\\8a+4b+2c+d=7...(4)[/tex]

Eliminasi d dari persamaan 4 dan 2, diperoleh:

7a+3b+c=8...(5)

Eliminasi d dari persamaan 3 dan 1, diperoleh:

26a-8b+2c=14...(6)

Eliminasi d pada persamaan 2 dan 1, diperoleh:

2a+2c=-2 (bagi 2)

a+c=-1...(7)

Eliminasi b pada persamaan 6 dan 5, diperoleh:

134a+14c=106 (bagi 2)

67a+7c=53...(8)

Eliminasi c dari persamaan 8 dan 7, diperoleh:

60a=60

a=1

Karena a=1, maka c adalah:

a+c=-1

1+c=-1

c=-2

Substitusi a=1 dan c=-2 ke persamaan 5, maka:

7a+3b+c=8

7(1)+3b-2=8

3b+5=8

3b=3

b=1

Substitusi a=b=1 dan c=-2 ke persamaan 2, maka:

a+b+c+d=-1

1+1-2+d=-1

d=-1

Karena a,b,c, dan d telah kita dapatkan maka akan diperoleh fungsi polinomialnya, yaitu:

[tex]p(x)=x^3+x^2-2x-1[/tex]

Semoga jawaban ini membantu.

16. Suku banyak berderajat 3, jika dibagi (x2 − x − 6) bersisa (5x − 2), jika dibagi (x2 − 2x − 3) bersisa (3x + 4). Suku banyak tersebut adalah….


Suku banyak berderajat 3
jika dibagi (x² - x - 6) bersisa (5x - 2),
jika dibagi (x² - 2x - 3) bersisa (3x + 4) maka suku banyaknya adalah ...

Pembahasan :

f(x) : (x² - x - 6) bersisa (5x - 2)
(x² - x - 6) = (x - 3)(x + 2)
x = 3 atau x = -2
substitusikan ke (5x - 2)
f(3) = 5(3) - 2 = 13
f(-2) = 5(-2) - 2 = -12

f(x) : (x² - 2x - 3) bersisa (3x + 4)
(x² - 2x - 3) = (x - 3)(x + 1)
x = 3 atau x = -1
substitusikan ke (3x + 4)
f(3) = 3(3) + 4 = 13
f(-1) = 3(-1) + 4 = 1


Karena ada x yang sama yaitu x = 3 maka koefisien x³ sama dengan 1

f(x) = p(x) . h(x) + s(x)
Keterangan :
f(x) = suku banyak yang dibagi
p(x) = pembagi
h(x) = hasil bagi
s(x) = sisa

Karena f(x) berderajat 3 dibagi dengan p(x) berderajat 2 maka hasil baginya akan berderajat 1

jika p(x) = (x² - 2x - 3),
s(x) = (3x + 4) dan kita misalkan
h(x) = (x + a)
maka kita pilih x = -2 sehingga f(-2) = -12
p(-2) = (-2)² - 2(-2) - 3 = 5
s(-2) = 3(-2) + 4 = -2
h(-2) = -2 + a

f(x) = p(x) . h(x) + s(x)
f(-2) = p(-2) . h(-2) + s(-2)
-12 = 5 (-2 + a) + (-2)
-12 = -10 + 5a - 2
-12 = -12 + 5a
0 = 5a
a = 0

Jadi
f(x) = p(x) . h(x) + s(x)
f(x) = (x² - 2x - 3) (x + a) + (3x + 4)
f(x) = (x² - 2x - 3) (x + 0) + (3x + 4)
f(x) = x³ - 2x² - 3x + (3x + 4)
f(x) = x³ - 2x² + 4

Kita coba cek apakah benar f(3) = 13, f(-2) = -12 dan f(-1) = 1

f(x) = x³ - 2x² + 4
f(3) = 27 - 18 + 4 = 13
f(-2) = -8 - 8 + 4 = -12
f(-1) = -1 - 2 + 4 = 1

Jadi benar bahwa suku banyak tersebut adalah :
f(x) = x³ - 2x² + 4
