10. Ruli menabung di bank wijaya dengan bunga 9% pertahun jika tabungan ruli Rp.3.000.000,00 dan jumlah tabungan ruli sekarang Rp.3.405.000,00 maka lama ruli menanbung adalah...bulan


M = Rp 3.000.000,00
J. tabungan = Rp 3.405.000,00
B = 9%
B = Rp 3.405.000,00 - Rp 3.000.000,00 = Rp 405.000,00
B = a% X n/12 X M
405.000 = 9/100 X n/12 X 3.000.000
405.000 = 22.500 X n
405.000/ 22.500 = n
18 bln = n

Jadi, Ruli menabung selama 18 bulan
