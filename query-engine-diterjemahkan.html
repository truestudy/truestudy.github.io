11. Hal apa yang tidak bisa dilakukan dalam query


Hal yang tidak bisa dilakukan dalam query adalah menyimpan data sebagaimana table karena sumber data dari query adalah dari table.

Pembahasan

Query adalah fungsi untuk menampilkan suatu data dari database dimana mengambil dari tabel-tabel yang ada di database.Manfaat Query adalah :Untuk mengolah data dari beberapa table menjadi satu informasi yang diinginkan.Untuk melihat, memodifikasi dan menganalisa data, serta sebagai sumber data untuk object form, report dan page.Untuk menampilkan data yang didapat dari menggabungkan beberapa table menjadi satu tampilan datasheet.Dapat menggunakan rumus yang terdapat didalam Access.Didalam Microsoft Access, terdapat lima jenis Query yang dapat dibuat, yaitu :Select Queries, yaitu query yang dibuat dengan cara menggabungkan beberapa table sehingga didapatkan sekelompok data yang ditampilkan dalam format datasheet.Parameter Queries, yaitu query yang bekerja berdasarkan data-data dan parameter lainnya yang dimasukkan terlebih dahulu kedalamnya.Crosstab Query, yaitu digunakan untuk menghitung dan restrukturisasi data agar analisis data lebih mudah.Action Query,  yaitu digunakan untuk membuat  update pada beberapa record hanya dalam satu operasi.SQL (Structured Query Language ) Query, yaitu digunakan untuk memperbaharui dan mengelola database relasional seperti Access.

25. apa yang dimaksud dengan query?


Query adalah suatu bentuk pemanggilan data yang ada pada database. Pemanggilan data menggunakan query harus memenuhi format dan aturan yang sesuai dengan ketentuan Structure Query Language/SQL agar database dapat membaca pemanggilan secara tepat. Penggunaan query dapat digunakan untuk melakukan menambahkan data, mengupdate data dan melakukan penghapusan data.

Pembahasan

Query merupakan suatu hal yang identik dengan database karena keduanya memiliki hubungan. Tanpa query, data yang tersimpan dalam database tidak dapat dilakukan manipulasi terhadap isi didalamnya karena diibaratkan pengguna melakukan requesting terhadap database untuk melakukan tindakan terhadap data didalamnya. Contoh query adalah penggunaan SQL statement seperti SELECT [column, ...] FROM [table name]. Untuk mengakses data dalam database, ada cara lain selain menggunakan query, namun, query tetap dianggap solusi paling efektif dalam melakukan pemanggilan data di database.
