Turunan dari fungsi f(x) = √x adalah...

Jawab :
Diketahui fungsi
f(x) = [tex] \sqrt{x} [/tex]
⇔ f(x) = [tex]x^{ \frac{1}{2} }[/tex]

Turunannya
f'(x) = [tex] \frac{1}{2} [/tex] [tex]x^{ \frac{1}{2}-1 }[/tex]
⇔ f'(x) = [tex] \frac{1}{2} [/tex] [tex]x^{-\frac{1}{2}}[/tex]
⇔ f'(x) = [tex] \frac{1}{2} [/tex] [tex] \frac{1}{x^{ \frac{1}{2} } } [/tex]
⇔ f'(x) = [tex] \frac{1}{2} [/tex] [tex] \frac{1}{ \sqrt{x} } [/tex]
⇔ f'(x) = [tex] \frac{1}{2 \sqrt{x} } [/tex]

Jadi, turunan fungsi f(x) = [tex] \sqrt{x} [/tex] adalah f'(x) = [tex] \frac{1}{2 \sqrt{x} } [/tex].

11. integral dan turunan dari akar (1 - akar(x))


Integral :

∫√(1 - √(x)) dx

misal,

u = 1 - √x
u - 1 = -√x
(u - 1)² = (-√x)²
(u - 1)² = x
2(u - 1) du = dx

∫√(1 - √x)) dx = ∫√u.2(u - 1) du
= ∫√u.(2u - 2) du
= ∫2u^(3/2) - 2u^(1/2) du
= (4/5)u^(5/2) - (4/3)u^(3/2) + C
= (4/5)u²√u - (4/3)u.√u + C
= (4/5)(1 - √x)²√(1 - √x) - (4/3)(1 - √x)√(1 - √x) + C

==========

√(1 - √(x))

da/dx = - 1/(2√x)

y = √a
dy/da = 1/(2√a)

dy/dx = dy/da * da/dx
= 1/(2√a) * (-1/(2√x))
= -1/(2√a.2√x)
= -1/(4.√(1 - √x).√x)
= -1/(4√(x - x√x))[tex]\displaystyle \frac{d}{dx}\left ( \sqrt{1-\sqrt{x}} \right )=\frac{d}{dx}\left ( 1-x^{\frac{1}{2}} \right )^\frac{1}{2}[/tex]

[tex]\displaystyle =\frac{-\frac{1}{2\sqrt{x}}}{2\sqrt{1-\sqrt{x}}}[/tex]
[tex]\displaystyle =-\frac{1}{4\sqrt{x}\sqrt{1-\sqrt{x}}}[/tex]

23. nilai turunan dari g(x)=akar x- akar x+1 di x=3 adalah


Materi Turunan

Diketahui [tex] g(x) = \sqrt{x - \sqrt{x + 1}} [/tex]
Nah, yang ditanyakan adalah g'(3).
Pertama-tama, turunkan dulu g(x) dengan menggunakan Aturan Rantai.
[tex] g(x) = (x - \sqrt{x + 1})^{\frac{1}{2}} \\ g'(x) = \dfrac{1}{2} \times \left(x - \sqrt{x + 1})^{-\frac{1}{2}} \times (1 - \dfrac{1}{2} \times (x+1)^{-\frac{1}{2}}\right) [/tex]
Langsung aja masukkan x = 3, sehingga ditulis
[tex] g'(3) = \dfrac{1}{2} \times (3 - \sqrt{3 + 1})^{-\frac{1}{2}} \times (1 - \dfrac{1}{2} \times (3+1)^{-\frac{1}{2}})\\ g'(3) = \dfrac{1}{2} \times 1 \times \dfrac{3}{4} = \dfrac{3}{8} [/tex]

29. berapa turunan dari f(x)=x akar x


turunan dikenal juga dengan nama difrensial

Turunan fungsi aljabar merupakan perluasan dari materi limit fungsi. Turunan fungsi dinotasikan f'(x), dengan rumus :

f'(x) =               f(x + h) - f(x)

limx→0          ------------------

                              h 

Bentuk limit di atas disebut dengan derivatif atau turunan pertama fungsi f(x) dan ditulis f'(x). Proses mencari derivatif disebut dengan differensial.

Pembahasan

Diketahui:

f(x) = x√x

Ditanya:

turunan dari f(x) = ...

Jawab:

f(x) = x akar x

f(x) = x^(³/₂)

turunan dari f(x) = ³/₂ x^(³/₂ - 1)

 = ³/₂ x^(¹/₂)

atau  ³/₂ √x

keterangan:

^ = pangkat
