penulisan bentuk baku

bentuk baku adalah penulisan bilangan dengan mengkalikan 2 faktor dari bilangan itu, dimana faktor pertama merupakan bilangan yang lebih besar atau sama dengan 1 dan kurang dari 10, dan faktor kedua merupakan bilangan berpangkat dengan basis 10 (bilangan pokoknya 10)

contoh:

bentuk baku dari

a) 12340 = 1,234 x 10⁴

b) 0,0000123 = 1,23 x 10⁻⁵

c) 123 x 10³ = 1,23 x 10² x 10³

= 1,23 x 10²⁺³

= 1,23 x 10⁵

dari penjelasan diatas mari kita bahas soal

a. 12 × 10⁵

12 = 1,2 x 10

12 x 10⁵ = 1,2 x 10 x 10⁵

= 1,2 x 10¹⁺⁵

= 1,2 x 10⁶


b. 123 × 10⁻⁷

123 = 1,23 x 10²

123 x 10⁻⁷ = 1,23 x 10² x 10⁻⁷

= 1,23 x 10²⁻⁷

= 1,23 x 10⁻⁵


c. 4567 × 10⁶

4567 = 4,567 x 10³

4567 x 10⁶ = 4,567 x 10³ x 10⁶

= 4,567 x 10³⁺⁶

= 4,567 x 10⁹


d. 6780000

6780000 = 6,78 x 1000000

= 6,78 x 10⁶


e. 78000000000

78.000.000.000 = 7,8 x 10.000.000.000

= 7,8 x 10¹⁰


f. 0,000678

0,000678 = 6,78 x 1/10.000

= 6,78 x 10⁻⁴


g. 0,00000000078

0,00000000078 = 7,8 x 1/10000000000

= 7,8 x 10⁻¹⁰
