20. Coba cangkriman-cangkriman ing ngisor iki aranana kalebu jinise cangkriman singendi!1.putri Melik-melik sendhen kayu2.itik Pertis,ibo perebeng,ijak perlong3.apa sing di lebokne kok malah metu?4.wiwawite lesmba dhonge karwa pake.​


Wangsulan

1. Putri melik-melik sendhen kayu legi => Cangkriman Pepindhan.

2. Itik pertis, ibo perbeng, ijak perlong => Cangkriman Wancahan.

3. Apa sing dilebokne kok malah metu? => Cangkriman Pepindhan.

4. Wiwawite lesmba dhonge karwa pake => Cangkriman Wancahan.

Penjelasan

Cangkriman yaiku unen-unen sing kudu dibedhek utawa dibatang maksude. Mula saka iku, cangkriman uga diarani bedhekan utawa batangan. Cangkriman iku wujude ana 4, yaiku:

1. cangkriman sing wujude wancahan (cekakan),

2. cangkriman sing wujude pepindhan (irib-iriban),

3. cangkriman sing wujude plesetan, lan

4. cangkriman sing sinawung ing tembang.
