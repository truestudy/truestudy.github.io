11. Panjang sisi-sisi sebuah segitiga siku-siku merupakan bilangan bulat serta membentuk barisan aritmatika, maka luas yang mungkin adalah ...


Panjang sisi-sisi segitiga siku-siku membentuk barisan aritmatika, maka luas yang mungkin adalah 216 cm².

Karena tak ada keterangan lain, maka kita coba buat keliling segitiganya = 72 cm.

Penjelasan dengan langkah-langkah:

Diketahui :

Panjang sisi segitiga siku-siku membentuk barisan aritmatika.

Keliling segitiga = 72 cm

Ditanya :

Luas yang mungkin.

Penyelesaian :

Menentukan panjang alas segitiga

Karena panjang sisi segitiga membentuk barisan aritmatika, maka kita misalkan :

U₁ = tinggi = a

U₂ = alas = a + b

U₃ = sisi miring = a + 2b

Dengan demikian,

U₁ + U₂ + U₃ = keliling

a + a + b + a + 2b = 72

3a + 3b = 72  

a + b = [tex]\frac{72}{3}[/tex]

a + b = 24

Menggunakan Pythagoras pada segitiga siku-siku

U₁² + U₂² = U₃²

a² + (a + b)² = (a + 2b)²

a² + a² + 2ab + b² = a² + 4ab + 4b²

2a² - a² + 2ab - 4ab + b² - 4b² = 0

a² - 2ab - 3b² = 0    (difaktorkan)

(a - 3b) (a + b) = 0

a - 3b = 0      atau      a + b = 0

      a = 3b                        a = -b (negatif tak memenuhi)

Subtitusikan a = 3b ke dalam U₂.

a + b = 24

3b + b = 24

     4b = 24

       b = [tex]\frac{24}{4}[/tex]

       b = 6

Menentukan luas segitiga siku-siku

Panjang sisi pada segitiga siku-siku

tinggi (a) = 3b = 3 × 6 = 18 cm

alas = 18 + 6 = 24 cm

sisi miring = a + 2b = 18 + 2 × 6 = 30 cm

Luas Δ = [tex]\frac{1}{2}[/tex] × alas × tinggi

           = [tex]\frac{1}{2}[/tex] × 24 cm × 18 cm

           = 12 × 18 cm²

           = 216 cm²

Jadi luas yang mungkin adalah 216 cm².
