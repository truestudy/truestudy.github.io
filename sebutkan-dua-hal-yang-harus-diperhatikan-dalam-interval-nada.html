16. 40. Perhatikan tangga nada diatonis mayor berikut! C E F 7 5 7 7 7 7 1 4 Sebutkanlah dua interval nada terts sesuai tangga nada tersebutpliss 5 menit lagi dikumpulin​


Interval terts adalah interval (jarak) antara nada satu dengan nada ketiga yang berjarak dua langkah. Dalam tangga nada diatonis C mayor, terdapat beberapa interval terts, di antaranya adalah antara C dan E, F dan A.

Pembahasan

Interval dalam musik merupakan jarak dua nada, baik jarak ke atas maupun ke bawah. Setiap interval memiliki nama yang berbeda, tergantung dari jarak di antara kedua nada terkait.

Pada umumnya, ada delapan jenis interval dalam satu tangga nada. Delapan interval tersebut adalah :

Prime, yaitu interval untuk dua buah nada yang sama. Contoh interval prime dalam tangga nada C mayor adalah C ke C.Second, yaitu interval antara nada satu dengan nada kedua (berjarak selangkah). Contohnya dalam tangga nada C mayor adalah C ke D, F ke G.Terts, yaitu interval dari sebuah nada ke nada ketiga (berjarak dua langkah). Contohnya dalam tangga nada C mayor adalah C ke E, F ke A.Kuarts, yaitu interval dari nada satu ke nada keempat (berjarak tiga langkah). Contohnya dalam tangga nada C mayor adalah C ke F, F ke B.Kwint, yaitu interval dari nada satu ke nada kelima (berjarak empat langkah). Contohnya dalam tangga nada C mayor adalah C ke G, D ke A.Sekst, yaitu interval dari nada satu ke nada keenam (berjarak lima langkah). Contohnya dalam tangga nada C mayor adalah C ke A.Septim, yaitu interval dari nada satu ke nada ketujuh (berjarak enam langkah). Contohnya dalam tangga nada C mayor adalah C ke B.Oktaf, yaitu jarak antara nada satu dengan nada kedelapan (berjarak tujuh langkah). Contohnya dalam tangga nada C mayor adalah C ke C'.
