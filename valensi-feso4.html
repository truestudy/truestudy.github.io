9. Sebanyak 20.6 gram hidrat FeSO4 xH20 dipanaskan Dan dihasilkan 15.2 gram FeSO4 anhidrat. Rumus hidrat tersebut adalah


Jawaban:

Rumus hidrat tersebut adalah FeSO₄ · 3 H₂O.

Penjelasan:

Diketahui : Kristal Besi (II) Sulfat dengan,

⬦ Massa kristal hidrat = 20,6 gram

⬦ Massa kristal anhidrat = 15,2 gram

⬦ Ar Fe = 56 sma, S = 32 sma, O = 16 sma dan H = 1 sma

Ditanya : Rumus hidrat kristal Besi (II) Sulfat...?

Penyelesaian :

Mencari Mr FeSO₄

Mr = (Ar Fe) + (Ar S) + (Ar O × 4)

Mr = 56 + 32 + (16 × 4)

Mr = 56 + 32 + 64 = 152 gram/mol

⟡ ⟡ ⟡

Mencari Mr H₂O

Mr = (Ar H × 2) + (Ar O)

Mr = (1 × 2) + 16

Mr = 2 + 16 = 18 gram/mol

⟡ ⟡ ⟡

Mencari massa H₂O

[tex]Massa\ air = Massa\ kristal\ hidrat - Massa\ kristal\ anhidrat[/tex]

[tex]Massa\ air = 20,6\ gram - 15,2\ gram[/tex]

[tex]Massa\ air = 5,4\ gram[/tex]

⟡ ⟡ ⟡

Persamaan Reaksi

FeSO₄ · x H₂O (s)  →  FeSO₄ (s) + H₂O (g)

Perbandingan mol FeSO₄ dan H₂O:

[tex]\frac{m\ FeSO_{4}\ anhidrat}{Mr\ FeSO_{4}} :\frac{m\ H_{2}O}{Mr\ H_{2}O}[/tex]

⇒ [tex]\frac{15,2\ gram}{152\ gram/mol} :\frac{5,4\ gram}{18\ gram/mol}[/tex]

⇒ [tex]0,1\ mol : 0,3\ mol[/tex]

⇒ [tex]1:3[/tex]

⟡ ⟡ ⟡

Jadi, rumus senyawa kristal Natrium Fosfat hidrat tersebut adalah FeSO₄ · 3 H₂O. Dengan jumlah molekul air sebanyak 3 molekul.

⟡ ⟡ ⟡

11. larutan feSo4 dengan elektrode besi


Larutan Besi(ll) Sulfat (FeSO4) dengan elektrode grafit(C)
Katoda = Fe^2+(aq) + 2e^- → Fe(s) (× 2) (Fe tereduksi sendiri)
Anoda = 2H2O(l) → 4H^+(aq) + O2(g) + 4e^- (× 1) (C tdk pengaruh, dan SO4 mengandung oksigen, jadi yang teroksidasi adalah air)

Reaksi elektrolisis :
2Fe^2+(aq) + 4e^- → 2Fe(s)
2H2O(l) → 4H^+(aq) + O2(g) + 4e^-
__________________________ +
2Fe^2+(aq) + 2H2O(l) → 2Fe(s) + 4H^+(aq) + O2(g)

Larutan CuSO4 dengan katoda Ag dan anoda Fe
Katoda = Cu^2+(aq) + 2e^- → Cu(s) (Katoda tdk ada pengaruh dgn elektroda apapun, jadi Cu tetap sbg katoda)
Anoda = Fe(s) → Fe^2+(aq) + 2e^- (Selain Pt, Au, dan C(grafit), Fe ada pengaruhnya, jadi Fe menggantikan SO4)

Reaksi elektrolisis :
Cu^2+(aq) + 2e^- → Cu(s)
Fe(s) → Fe^2+(aq) + 2e^-
___________________ +
Cu^2+(aq) + Fe(s) → Cu(s) + Fe^2+(aq)
