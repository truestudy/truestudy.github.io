Pembahasan

Folder merupakan suatu program atau tempat yang berguna sebagai alat penyimpanan berbagai jenis file didalamnya. Fungsi utama folder adalah untuk menyimpan file sekaligus mengelompokkannya menurut jenis file itu sendiri. Kita memerlukan beberapa wadah untuk dalam penyimpanan file musik, dokumen dan foto.

Manajemen  file  adalah  pengelolaan  file  dan  folder  supaya  rapi  dan  mudah dalam penggunaannya. Folder adalah cara yang bagus untuk menjaga komputer tetap teratur. Dapat menggunakannya untuk menyimpan semua file penting di satu tempat.  dapat menggunakannya untuk menyimpanfile-file penting atau dapat menggunakannya untuk menyimpan kertas-kertas yang kurang penting dengan aman. Folder adalah cara yang bagus untuk menyimpan dokumen di satu tempat.

Ada banyak jenis folder yang dapat gunakan. Beberapa yang paling populer termasuk:

My Documents: Ini adalah folder yang biasanya terletak di hard drive komputer Anda. Ini adalah tempat yang bagus untuk menyimpan semua file pribadi Anda.Desktop: Desktop adalah folder lain yang terletak di hard drive Anda. Di sinilah Anda dapat menyimpan pintasan ke program yang paling sering Anda gunakan.Unduhan: Folder unduhan adalah tempat semua file yang Anda unduh dari internet disimpan.

28. jelaskan Langkah-langkah dalam: 1. Membuat folder 2. Menghapus Folder 3. Menggandakan Folder 4. Menyalin Folder 5. Memberi nama Folder


Jawaban:

Membuat folder :

1. Klik kanan mouse

2. Arahkan kursor ke pilihan new

3. Ketika pilihan lain muncul, klik folder.

Menghapus Folder :

1. Klik kanan mouse pada folder

2. Klik delete

Atau :

1. Pilih folder dengan menekan kiri mouse

Menggandakan folder :

1. Klik kanan mouse pada folder

2. Klik copy

3. Klik paste

Menyalin folder :

1. Klik kanan mouse pada folder

2. Klik copy (ctrl + c)

Memberi nama folder :

1. Buat folder baru

2. Beri nama

Kalau mau mengubah nama foldernya, lakukan dengan cara :

1. Klik foldernya dua kali di nama folder itu

2. Tulis nama folder yang baru

3. Atau gunakan rename

30. 1. Jelaskan langkah mengcopy file/folder?2. Jelaskan langkah mengcopy sebagian file/folder?3. Jelaskan langkah memindahkan/memotong file/folder?4. Jelaskan langkah memindahkan/memotong sebagian file/folder?5. Jelaskan langkah menghapus file/folder?6. Jelaskan langkah menghapus sebagian file/folder?TOLONG JAWAB YAJANGAN CUMA CARI NILAINYA


1. Pilih file/folder yang akan dicopy, lalu tekan "Ctrl+C"
2. Pilih satu file/folder yang akan dicopy, kemudian tekan "Shift" lalu pilih file/folder lain yang ingin dicopy (tanpa melepas "Shift") setelah itu tekan "Ctrl+C"
3. Pilih file/folder yang ingin dipindahkan (cut), kemudian tekan "Ctrl+X"
4. Pilih satu file/folder yang akan dipindahkan/dipotong, kemudian tekan "Shift" lalu pilih file/folder lain yang ingin dipindahkan (tanpa melepas "Shift") dan setelah itu tekan "Ctrl+X"
5. Pilih file/folder yang ingin dihapus, kemudian tekan shortcut "Delete" pada keyboard
6. Pilih file/folder yang ingin dihapus, kemudian tekan "Shift" lalu pilih file/folder lain yang ingin dihapus (tanpa melepas "Shift") dan setelah itu tekan tombol "Delete" pada keyboard
