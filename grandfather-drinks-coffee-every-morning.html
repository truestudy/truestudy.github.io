9. negative and interrogativegrandfather drinks coffee every morning​


Mapel: Bahasa Inggris

Materi: Simple Present Tense

Kelas: 7 SMP

Jawaban:

(+) Grandfather drinks coffee every morning.(-) Grandfather does not drink coffee every morning.(?) Does grandfather drink coffee every morning?

Penjelasan:

⇔ Kalimat tersebut merupakan bentuk waktu Simple Present Tense. Yaitu menyatakan kegiatan masa kini yang dilakukan secara berulang-ulang.

⇔ Rumus kalimat postif-negatif-interogatif Simple Present Tense adalah:

(+) S + V1 + (s/es) + O(-) S + does + not + V1 + O(?) Do/Does + S + V1 + O + ?

⇔ Keterangan:

S = Subject (subjek dari kalimat)V1 = Verb 1/infinitive verb (kata kerja pertama/infinitif)Do = Kata kerja aux. untuk kata ganti orang pertama, jamak dan keduaDoes = Kata kerja aux. untuk kata ganti orang ketiga tunggalO = Object (objek dari kalimat) (s/es) = Akhiran kata kerja dalam kalimat positif SPS.(s) = Akhiran kata kerja untuk kata kerja ucapan akhir huruf non-vokal (contoh: work)(es) = Akhiran kata kerja untuk kata kerja ucapan akhir huruf vokal (contoh: go + es = goes)

PENYELESAIAN SOAL:

Analisis kalimat dan terapkan sesuai rumus kalimat diatas:

(+) Grandfather drinks coffee every morning.(-) Grandfather does not drink coffee every morning.(?) Does grandfather drink coffee every morning?

10. Grandfather drinks coffee every morning. Buat menjadi kalimat negatif !


Turn the sentence into negative form!

Jawaban

Grandfather drinks coffee every morning.

( - ) Grandfather doesn't drink coffee every morning.Penjelasan

Kalimat di atas adalah kalimat Simple Present Tense dengan bentuk positif. Jika ingin mengubahnya menjadi bentuk negatif, maka harus diberi NOT. Berikut adalah pattern/pola bentuk-bentuk Simple Past Tense:

Positive form    

( + ) subject + verb 1 (-s/-es/ies) + object

( + ) subject + to be (is/am/are) + object

I teach mathematic. Rina is a magician.

Negative form

( - ) subject + do/does + NOT + verb 1 + object

( - ) subject + to be (is, am, are) + not + object

I do not cook breakfast. My parents are not Indonesians.

Interrogative form

( ? ) do/does + subject + verb 1 + object + ?

( ? ) to be (is/am/are) + subject + object + ?

Do I know you? Is she a teacher?

Penambahan akhiran -s/-es/-ies pada simple present tense hanya digunakan pada kalimat positive ( + ) ketika subjek ketiga tunggal, seperti he, she, dan it atau mother, father, friend, teacher (sebutan atau jabatan), (a) cat, (a) goose (hewan), (a) car, (a) house (benda), Dian, Mr. Risa, Joyce (nama orang), dan sebutan subjek ketiga tunggal lainnya.

PEMBAHASAN

Present Tense adalah sebuah tenses untuk menyatakan Kegiatan berulang atau kebenaran umum / FAKTA.

Adverb of time

=> Every morning ✔️

=> Every day

=> usually

=> Always

Rumus Present Tense

(+) S + V1 (-s/es) + O + Adv

(-) S + do/does + not + V1 + O + Adv

(?) Do/does + S + V1 + O + Adv?

Catatan untuk Subject "He/She/it/My father"(+) Pada kalimat Positive VERB+s (drinks)(-) dan (?) Tapi pada kalimat Negative dan Interrogative kembali VERB 1 tanpa (-s/es)

Keterangan:

a.To drink = untuk minum

b. Drinks => mungkin maksudnya “drink”

C. drinking = Continuous Tense

KESIMPULAN

✔️ maka kalimat tersebut adalah Present Tense karena Kegiatan berulang atau kebenaran umum / FAKTA.

Pembahasan

Present tense adalah bentuk kalimat yang menjelaskan peristiwa yang dilakukan secara terus menerus atau berulang-ulang

Does digunakan pada I, We, You, They

Do digunakan pada He, She, It

Kata "not" atau "n't" itu hanya kita gunakan pada kalimat negatif

Kata "Does" atau "Do" hanya digunakan di kalimat tanya

Rumus

(+) Subject + Verb 1 + Object

(-) Subject + Do/Does + Not + Verb 1 + Object

(?) Do/Does + Subject + Verb 1 + Object
