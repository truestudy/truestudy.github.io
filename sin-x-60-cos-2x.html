Pembahasan

Trigonometri adalah cabang ilmu Matematika yang mempelajari tentang hubungan sisi terhadap sudut pada segitiga.

Nilai sinus dan cosinus sudut istimewa

[tex] \boxed{ \sin( {30}^{ \circ} ) = \frac{1}{2} } \\ \boxed{ \sin( {45}^{ \circ} ) = \frac{1}{2} \sqrt{2} } \\ \boxed{ \sin( {60}^{ \circ} ) = \frac{1}{2} \sqrt{3} } \\ \boxed{ \sin( {90}^{ \circ} ) = 1 } \\ \\ \boxed{ \cos( {30}^{ \circ} ) = \frac{1}{2} \sqrt{3} } \\ \boxed{ \cos( {45}^{ \circ} ) = \frac{1}{2} \sqrt{2} } \\ \boxed{ \cos( {60}^{ \circ} ) = \frac{1}{2} } \\ \boxed{ \cos( {90}^{ \circ} ) = 0 } \\ \\ [/tex]

Penyelesaian:

[tex] \: \: \: \: 2 \times \sin( {45}^{ \circ} ) \times \cos( {60}^{ \circ} ) \\ = 2 \times \frac{1}{2} \sqrt{2} \times \frac{1}{2} \\ = \frac{1}{2} \sqrt{2} \\ [/tex]
