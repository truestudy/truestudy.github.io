2. hitunglah besar sudut sudut berikut


Nomor 1 :

Sudut Pusat BOC = 84°

Sudut Keliling BOC

= 42°

Sudut Pusat POQ

= 2 × Sudut Keliling PRQ

= 2 × 31°

= 62°

23. dua sudut saling berpelurus. jika selisihnya 25 hitunglah besar masing masing sudut berikut!!


Diketahui selisih kedua sudut berpelurus = 25.
Sudut A + Sudut B = 180, maka Sudut A = 180 - Sudut B
Sudut A - Sudut B = 25, maka Sudut A = 25 + sudut B
Kurangilah, maka:
0 = 155 - 2 Sudut B
Sudut B = 72,5
Sudut A = 25 + 72,5 = 97,5

26. perhatikan gambar berikut, kemudian hitung besar sudut APA dan sudut BOC​


Jawaban:

52,5° dan 37,5°

Penjelasan:

90° = 7x + 5x

90° = 12x

x = 7,5°

AOB = 7 x 7,5

= 52,5°

BOC = 5 x 7,5

= 37,5°

Jawaban:

Karena berbentuk siku siku maka besar sudut AOB sudah pasti 90°

Maka, AOB+BOC= 7x+5x= 12x=90°

AOB= 7x/12x×90°

= 52,5°

BOC= 5x/12x×90°

= 37,5°
