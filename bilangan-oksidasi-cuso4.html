8. tentukan bilangan oksidasi s dalam CuSO4


Kelas         : X
Pelajaran   : Kimia
Kategori     : Redoks
Kata Kunci : bilangan oksidasi (biloks), belerang, tembaga (II) sulfat

Pembahasan

Senyawa CuSO₄ memiliki nama tembaga (II) sulfat. Bilangan romawi menyatakan muatan atau bilangan oksidasi dari tembaga (Cu) yakni +2. Ingat, total bilangan oksidasi dari senyawa netral adalah nol.

Cara Pertama Menghitung Bilangan Oksidasi S

Misalkan bilangan oksidasi dari belerang atau S adalah x, maka
⇔ (+2) + x + 4(-2) = 0
⇔ 2 + x - 8 = 0
∴ x = 6

Cara Kedua Menghitung Bilangan Oksidasi S

Perhatikan ionisasi dari CuSO₄ berikut ini:

CuSO₄ → Cu²⁺ + SO₄²⁻

Senyawa ion sulfat, SO₄²⁻, memiliki muatan -2 yang merupakan total bilangan oksidasi dari S dan O.

Bilangan oksidasi dari oksigen atau O adalah -2.

Misalkan bilangan oksidasi dari belerang atau S adalah x, maka
⇔ x + 4(-2) = -2
⇔ x - 8 = -2
⇔ x = 8 - 2
∴ x = 6

Jadi, bilangan oksidasi S pada CuSO₄ adalah +6

16. pada reaksi : CuSO4+Kl→Cul +K2SO4 bilangan oksidasi Cu berubah dari


pada reaksi : CuSO₄ + Kl → Cul + K₂SO₄ bilangan oksidasi Cu berubah dari +2 menjadi +1.

Pembahasan :

CuSO₄ + Kl → Cul + K₂SO₄

Tahap 1 :  

Tentukan terlebih dahulu biloks Cu dalam CuSO₄ :

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.  

biloks CuSO₄ = 0  

biloks Cu + biloks S + (4 × biloks O) = 0  

biloks Cu + (+6) + (4 × (-2)) = 0  

biloks Cu + (+6) + (-4) = 0  

biloks Cu = +2

Tahap 2 :  

Selanjutnya tentukan biloks Cu dalam CuI :  

Biloks Cu dalam Cul

biloks I = -1 (Golongan VIIA)  

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.  

biloks CuI = 0  

biloks Cu + biloks I = 0  

biloks Cu + (-1) = 0  

biloks Cu = +1

CuSO₄ + Kl → Cul + K₂SO₄

|___________|

+2                     +1  

Unsur Cu mengalami perubahan biloks dari +2 menjadi +1.

19. mencari bilangan oksidasi dari , FeCN3, FeCO3, CuSO4 ,NaAuF4

Aturan-aturan untuk menentukan Bilangan Oksidasi

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• Fluorin, unsur yang mempunyai biloks -1

• Golongan I (logam alkali ) biloks = +1 contoh: Li, Na, K, Rb, Cs

• Golongan II (Alkali Tanah) biloks = +2 contoh = Be, Mg, Ca, Sr, Ba

• biloks suatu unsur dalam suatu ion tunggal sama dengan muatannya Contoh: biloks Fe dalam ion Fe²+ = +2 S²- = 2-

• Biloks H umumnya= +1, kecuali dalam senyawa dengan logam maka biloks = -1 Contoh: NaH BaH2 = -1

• biloks O umunya = -2 Contoh: biloks O dalam H2O, MgO = -2 kecuali
-Dalam Fe2O = +2
-Dalam Peroksida, seperti H2O2, biloks O = -1 -Dalam Senyawa superoksida, seperti KO2 biloks O= -1/2
-Dalam senyawa terner, biloks O selalu = -2
-Dalam senyawa biner, biloks O belum tentu = +2

• biloks halogen, sperti F, Cl, I, Br, At = -1

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0

• bilols unsur-unsur dalam senyawa poliatom = muatannya contoh = SO4²- = -2

• biloks dalam senyawa ion = muatan kation dan anionya.

26. dalam senyawa berikut ini, harga bilangan oksidasi belerang yang paling besar adalah. A. Na2S B. K2SO4 C. CuSO4 D. NaHSO3 E. SO2


Dalam senyawa berikut ini, harga bilangan oksidasi belerang yang paling besar adalah.

A. Na₂S

B. K₂SO₄

C. CuSO₄

D. NaHSO₃

E. SO₂

Jawab:

Sebelum menjawab saya akan coba mengingatkan kembali dengan menuliskan Aturan Penentuan Bilangan Oksidasi (Biloks).

Dengan mempertimbangkan keelektronegatifan unsur, dapat disimpulkan suatu aturan untuk menentukan bilangan oksidasi adalah sebagai berkut:

1. Unsur Bebas, atomnya memiliki bilangan oksidasi = 0

2. Flourin, unsur yang paling elektronegatif dan membutuhkan tambahan 1 elektron, atomnya mempunyai bilangan oksidasi -1 pada semua senyawanya.

3. Bilangan oksidasi atom unsur logam selalu bertanda +

   Untuk logam Alkali (Golongan I A) = +1

   Untuk logam Alkali Tanah (Golongan II A) = +2

4. Bilangan oksidasi suatu unsur dalam bentuk ion tunggal adalah sama dengan muatannya

   Contoh:  Cl⁻ = -1

5. Bilangan Oksidasi atom H umumnya = +1, kecuali dalam senyawanya logam, dimana bilangan oksidasi atom H = -1 

Contoh:

Bilangan Oksidasi atom H dalam NaH, BaH₂ = -1

6. Bilangan Oksidasi atom O umumnya = -2  kecuali pada contoh:

   F₂O = +2

   H₂O₂ = -1

   KO₂ = -1/2

7. Jumlah Bilangan Oksidasi atom unsur-unsur dalam suatu senyawa = 0

8. Jumlah Bilangan Oksidasi atom unsur-unsur dalam suatu ion poliatom = muatannya.

Berdasarkan aturan di atas kita coba kerjakan soal-soalnya:

A. Na₂S

  Biloks Na₂S ==>

  2 x Biloks Na + Biloks S = 0

  2 x (+1) + Biloks S = 0

   (+2) + Biloks S = 0

    Biloks S = -2

B. K₂SO₄

   Biloks K₂SO₄ ==>

   2 x Biloks K + Biloks S + 4 x Biloks O = 0

   2 x (+1) + Biloks S + 4 x (-2) = 0

    (+2) + Biloks S + (-8) = 0

     Biloks S + (-6) = 0

     Biloks S = +6

C. CuSO₄

    Biloks CuSO₄ ==>  Cu⁺² + SO₄⁻²

    Biloks Cu⁺² = +2

    Biloks S + 4 x Biloks O = -2

    Biloks S + 4 x (-2) = -2

    Biloks S + (-8) = -2

    Biloks S = +6

D. NaHSO₃

    Biloks NaHSO₃  ==>

    Biloks Na + Biloks H + Biloks S + 3 x Biloks O = 0

    (+1) + (+1) + Biloks S + 3 x (-2) = 0

    (+2) + Biloks S + (-6) = 0

     Biloks S + (-4) = 0

     Biloks S = +4

E. SO₂

    Biloks SO₂ ==>

    Biloks S + 2 x Biloks O = 0

    Biloks S + 2 x (-2) = 0

    Biloks S + (-4) = 0

    Biloks S = +4

Jadi, Nilai Biloks Belerang (S) paling tinggi terdapat pada senyawa K₂SO₄ dan CuSO₄ dengan nilai +6.

Aturan-aturan untuk menentukan Bilangan Oksidasi

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• Fluorin, unsur yang mempunyai biloks -1

• Golongan I (logam alkali ) biloks = +1 contoh: Li, Na, K, Rb, Cs

• Golongan II (Alkali Tanah) biloks = +2 contoh = Be, Mg, Ca, Sr, Ba

• biloks suatu unsur dalam suatu ion tunggal sama dengan muatannya Contoh: biloks Fe dalam ion Fe²+ = +2 S²- = 2-

• Biloks H umumnya= +1, kecuali dalam senyawa dengan logam maka biloks = -1 Contoh: NaH BaH2 = -1

• biloks O umunya = -2 Contoh: biloks O dalam H2O, MgO = -2 kecuali
-Dalam Fe2O = +2
-Dalam Peroksida, seperti H2O2, biloks O = -1 -Dalam Senyawa superoksida, seperti KO2 biloks O= -1/2
-Dalam senyawa terner, biloks O selalu = -2
-Dalam senyawa biner, biloks O belum tentu = +2

• biloks halogen, sperti F, Cl, I, Br, At = -1

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0

• bilols unsur-unsur dalam senyawa poliatom = muatannya contoh = SO4²- = -2

• biloks dalam senyawa ion = muatan kation dan anionya.
