27. basa kramane ibu tuku klambi ing pasar


Bahasa kramane Ibu tuku klambi ing pasar yaiku Ibu mundhut ageman wonten peken.

PEMBAHASAN :

Unggah - ungguh basa :

1. Ngoko Lugu

Ngoko lugu yaiku tembung ngoko kabeh. Ngoko lugu biasanipun digunakake kangge guneman wong tua marang wong seng luwih enom, guru maring muride, kanca seng wes raket banget lan kakang karo adhine.

2. Ngoko Alus

Ngoko alus yaiku tembung ngoko kecampur krama inggil. Tembung ngoko alus biasanipun digunakake marang wong tua marang wong sing lewih enom kangge ngajeni.

3. Krama Lugu

Krama lugu yaiku tembung krama kabeh ora kecampur ngoko lan krama inggil.

4. Krama alus

Krama alus yaiku tembung krama kecampur krama inggil utawi krama inggil kabeh. Biasanipun digunakake wong enom marang wong sing lewih tua.

Ibu

Ngoko = Ibu

Krama madya = Ibu

Krama inggil = Ibu

Tuku

Ngoko = tuku

Krama madya = tumbas

Krama inggil = mundhut

Klambi

Ngoko = klambi

Krama madya = rasukan

Krama inggil = ageman

Ing

Ngoko = ing

Krama madya = wonten

Krama inggil = wonten

Pasar

Ngoko = pasar

Krama madya = peken

Krama inggil = peken

Ibu + tuku + klambi + ing + pasar

= Ibu + mundhut + ageman + wonten + peken

= Ibu mundhut ageman wonten peken

30. Budi dikongkon tuku gula basa kramane Tembung dikongkon yaiku


ngowahi Tembung dadi basa Krama.

- dikongkon = basa Kramane dikengken.

Pembahasan

Basa Krama yaiku basane Krama nanging ora dicampur karo basa Ngoko lan Inggil.

→ panggonane :

wong sadrajat sing durung raketwong sing durung kenal

→ tuladha :

ibu taksih ten sekolah, mangke jam kaleh wangsule.bapak mpun bidal sadeyan Ten peken awit jam gangsal enjing.Kula mboten siyos tumut lomba, amargi wonten perlu.

Budi dikongkon tuku gula.

→ dikongkon = basa Ngoko lugu

→ diutus = basa Ngoko Alus

→ dikengken = basa Krama lugu

→ dipun utus = basa Krama Alus
