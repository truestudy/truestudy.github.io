24. tentukan panjang busur pada lingkaran berikut 1diketahui panjang busur ABO adalah=45° dan sudut AB 28cm tentukan panjang busur AB​


Jawaban:

Tentukan panjang busur pada lingkaran berikut:

1. Panjang busur CD = ...

∠COD = 72°

OD = 20 cm

2. Panjang busur KL = ...

∠KOL = 90°

OD = 14 cm

3. Panjang busur AB = ...

∠AOB = 45°

OD = 28 cm

PENYELESAIAN

Ingat rumus mencari panjang busur sebuah lingkaran:

Panjang Busur = (Besar Sudut Juring/360°) x Keliling Lingkaran

Panjang Busur = (Besar Sudut Juring/360°) x πd

Dengan menggunakan rumus panjang busur di atas, maka penyelesaian untuk permasalahan di atas adalah sebagai berikut:

1. Panjang busur CD

∠COD = 72°

OD = 20 cm

Panjang busur CD = (∠COD/360°) x πd

Panjang busur CD = (72°/360°) x 3.14 x 20

Panjang busur CD = 0.2 x 3.14 x 20

Panjang busur CD = 12.56 cm

2. Panjang busur KL

∠KOL = 90°

OD = 14 cm

Panjang busur KL = (∠KOL/360°) x πd

Panjang busur KL = (90°/360°) x 22/7 x 14

Panjang busur KL = 0.25 x 22/7 x 14

Panjang busur KL = 11 cm

3. Panjang busur AB

∠AOB = 45°

OD = 28 cm

Panjang busur AB = (∠AOB/360°) x πd

Panjang busur AB = (45°/360°) x 22/7 x 28

Panjang busur AB = 0.125 x 22/7 x 28

Panjang busur AB = 11 cm

26. Jika panjang busur AB = 15 cm, tentukan panjang busur CD.​


Perhatikan gambar pada soal !

Jika panjang busur AB adalah 15 cm, maka panjang busur CD adalah 9 cm

Pendahuluan

Lingkaran merupakan salah satu bangun datar dengan jarak antara titik-titik pada lingkaran tersebut terhadap titik tertentu selalu sama (tetap). Titik tertentu tersebut dinamakan titik pusat dan jarak yang tetap disebut jari-jari lingkaran.

Rumus yang berlaku pada lingkaran

1. Keliling lingkaran (K)

  [tex]\boxed {\text K = 2~\pi~ \text r}[/tex] atau  [tex]\boxed {\text K = \pi~ \text d}[/tex]

2. Luas lingkaran (L)

  [tex]\boxed {\text L = \pi~ \text r^2}[/tex]

3. Panjang busur

  [tex]\boxed {\text {Panjang Busur} = \frac{\text {Sudut Pusat}}{360^o} \times 2\pi \text r}[/tex]

4. Luas juring

  [tex]\boxed {\text {Luas Juring} = \frac{\text {Sudut Pusat}}{360^o} \times \pi \text r^2}[/tex]

Pembahasan

Diketahui :

Lingkaran dengan pusat O

A, B, C dan D pada keliling lingkaran

Panjang busur AB = 15 cm

Besar ∠AOB = a + 40°

Besar ∠COD = a

Besar ∠AOD = 80°

Besar ∠BOC = 120°

Ditanyakan :

Panjang busur CD = . . .    .

Jawab :

Menentukan nilai a

Besar sudut lingkaran adalah 360°, sehingga :

∠AOB + ∠BOC + ∠COD + ∠AOD = 360°

⇔ ∠AOB + ∠BOC + ∠COD + ∠AOD = 360°

⇔ (a + 40°) + 120° + a + 80° = 360°

⇔      a + a + 40° + 120° 80° = 360°

⇔                         2a + 240° = 360°

⇔                                     2a = 360° - 240°

⇔                                     2a = 120°

⇔                                       a = 60°

Jadi nilai a = 60°

Besar ∠AOB = a + 40° = 60° + 40° = 100°

Besar ∠COD = a = 60°

Untuk menentukan panjang CD digunakan perbandingan, yaitu :

[tex]\frac{\text {Panjang busur AOB}}{\text {Panjang busur COD}} = \frac{\text {Besar sudut AOB}}{\text {Besar sudut COD}}[/tex], sehingga :

⇔ [tex]\frac{\text {Panjang busur AB}}{\text {Panjang busur CD}} = \frac{\text {Besar sudut AOB}}{\text {Besar sudut COD}}[/tex]

⇔ [tex]\frac{\text {15}}{\text {Panjang busur CD}} = \frac{100^o}{60^o}[/tex]

⇔ [tex]\frac{\text {15}}{\text {Panjang busur CD}} = \frac{5}{3}[/tex]

⇔ [tex]\text {Panjang busur CD}[/tex] = [tex]15\times \frac{3}{5}[/tex]

⇔ [tex]\text {Panjang busur CD}[/tex] = [tex]\frac{45}{5}[/tex]

⇔ [tex]\text {Panjang busur CD}[/tex] = 9 cm

∴ Jadi panjang busur CD adalah 9 cm
