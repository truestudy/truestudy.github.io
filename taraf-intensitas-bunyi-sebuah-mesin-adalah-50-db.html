ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

TIn = ?

10. taraf intensitas bunyi sebuah mesin rata-rata 50 DB apabila 3 mesin dihidupkan bersama maka taraf intensitasnya adalah​


Jawaban dan Penjelasan:

Diketahui,

Taraf Intensitas rata-rata mesin, misal TI(1) = 50dB.

Jika ada 3 mesin dihidupkan bersama, berapa taraf intensitasnya?

Rumus:

Jika ada n mesin dinyalakan dengan taraf intensitas satu mesinnya sebesar TI(1), maka taraf intensitas n mesin seluruhnya adalah

TI(n) = TI(1) + 10 log n

Sehingga,

TI(3) = (50 + 10 log 3) dB

TI(3) = (50 + 1,47) dB

TI(3) = 51,47 dB

Maka, Taraf Intensitas 3 mesin yang dihidupkan secara bersamaan adalah 51,47 dB.

Interferensi gelombang bunyi  

Bunyi kuat ( interferensi konstruktif ) terjadi ketika perpaduan kedua gelombang bunyi pada suatu titik adalah sefase atau memiliki beda lintasan jarak tempuh yang nilainya sama dengan kelipatan bulat dari panjang gelombang bunyi.  

Bunyi kuat Δs = nλ; n = 0, 1, 2, 3, . . .                                          

Bunyi lemah ( Interferensi destruktif ) terjadi jika kedua gelombang yang bertemu pada suatu titik adalah berlawanan fase atau memiliki beda lintasan,  

Bunyi lemah Δs = (n+0,5) λ ; n = 0, 1, 2, 3, . . .

13. Taraf intensitas bunyi sebuah mesin rata-rata 50 dB. Apabila seratus mesin dihidupkan bersama, maka taraf intensitasnya adalah ...


Jawaban:

70 dB

Penjelasan:

Rumus umum :

TI₂ = TI₁ + 10 log n

TI₂ = Taraf intensitas 2 ( dB )

TI₁  = Taraf intensitas 1 ( dB )

n    = kelipatan benda

=============================

Diketahui :

TI₁ = 50 dB

n   = n₂/n₁

n   = 100 / 1

n   = 100

Ditanya : TI₂ ?

DIjawab :

TI₂ = TI₁ + 10 log n

TI₂ = 50 dB + 10 log 100

TI₂ = 50 dB + 10 (2) dB

TI₂ = 70 dB

>>Bunyi: Taraf Intensitas<<

-> Taraf Intensitas Bunyi

[tex]\boxed{\boxed{TI=10log\frac{I}{I_o} }}[/tex]

dimana:

TI = Taraf Intensitas (dB)

I = Intensitas Bunyi (watt/m²)

I₀ = Intensitas Ambang (10⁻¹² watt/m²)

-> Taraf Intensitas karena Adanya Jarak Sumber Bunyi

[tex]\boxed{\boxed{TI_2=TI_1-20log\frac{R_2}{R_1} }}[/tex]

dimana:

R = Jarak antara sumber bunyi dengan pendengar (m)

-> Taraf Intensitas Bunyi karena Banyaknya Sumber Bunyi

[tex]\boxed{\boxed{TI_2=TI_1+10log\frac{n_2}{n_1} }}[/tex]

dimana:

n = Jumlah/banyaknya sumber bunyi

Diketahui:

n₁ = 1 <-- Sebuah

TI₁ = 50 dB

n₂ = 100

TI₂ = ?

Jawab:

[tex]\boxed{\boxed{TI_2=TI_1+10log\frac{n_2}{n_1} }}\\TI_2=50+10log\frac{100}{1}\\TI_2=50+10log(10^2)\\TI_2=50+10(2)\\TI_2=50+20\\\boxed{TI_2=70dB}[/tex]
