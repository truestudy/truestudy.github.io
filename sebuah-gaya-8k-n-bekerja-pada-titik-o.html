4. Hasil dari n sigma k=1 (4k-1)(k+1). A. 1/6k (8k^2+21k+7)b. 1/3k (8k^2+21k+7)c. 1/2k (8k^2+21k+7)d. K (8k^2+21k+7)e. 2k (8k^2+21k+7).


Jawaban yang tepat adalah pilihan A yaitu 1/6k (8k^2+21k+7). Soal ini berkaitan dengan notasi sigma.

Penjelasan dengan langkah-langkah

Diketahui :

Diketahui sebuah notasi sigma sebagai berikut

[tex]\sum_{k=1}^{n}[/tex](4k-1)(k+1)

Ditanyakan :

Jumlah dari notasi sigma tersebut adalah

Jawab :

Rumus dasar notasi sigma untuk persamaan sederhana :

[tex]\sum_{k=1}^{n}a = a.n[/tex] , dengan a adalah konstanta

[tex]\sum_{k=1}^{n}k = \frac{n(n+1)}{2}[/tex], dengan k adalah variabel

[tex]\sum_{k=1}^{n}k^{2} = \frac{n(n+1)(2n+1)}{6}[/tex], dengan k adalah variabel kuadrat

Langkah 1

Soal dapat kita jabarkan menjadi

[tex]\sum_{k=1}^{n}[/tex](4k-1)(k+1) = [tex]\sum_{k=1}^{n}[/tex][tex]4k^{2} + 3k - 1[/tex] = [tex]\sum_{k=1}^{n}4k^2 + \sum_{k=1}^{n}3k - \sum_{k=1}^{n}1[/tex]

Langkah 2

Selanjutnya kita hitung satu persatu

[tex]\sum_{k=1}^{n}4k^2 = 4\sum_{k=1}^{n}k^2=4.(\frac{n(n+1)(2n+1)}{6} )[/tex][tex]\sum_{k=1}^{n}3k = 3\sum_{k=1}^{n}k=3.(\frac{n(n+1)}{2} )[/tex][tex]\sum_{k=1}^{n}1[/tex] = 1.n = n

Langkah 3

Masukkan ketiga persamaan ke persamaan utama, sehingga menjadi

[tex]\sum_{k=1}^{n}[/tex](4k-1)(k+1) [tex]= 4.(\frac{n(n+1)(2n+1)}{6} )+3.(\frac{n(n+1)}{2} ) +n[/tex]

                        [tex]= 2.(\frac{n(n+1)(2n+1)}{3} )+3.(\frac{n(n+1)}{2} ) +n[/tex]

                        [tex]= \frac{4n(2n^2 + 3n +1)+9n(n+1)-6n}{6}[/tex]

                        [tex]= \frac{n(8n^2 +12n 4+9n+9-6)}{6}[/tex]

                        [tex]= \frac{n(8n^2 +21n+7)}{6}[/tex]

ubah n menjadi k sehingga

[tex]\sum_{k=1}^{n}[/tex](4k-1)(k+1) = 1/6k (8k^2+21k+7)

10. Vektor satuan m : 4i - 5j + 6k n : j - 8k maka nilai | m + n | adalah ....


Jawaban:

Nilai variabel n dari vektor b adalah 4. Nilai tersebut diperoleh dengan perhitungan skalar dua buah vektor yang saling tegak lurus, yakni:

a . b = 0

dengan:

a = vektor a

b = vektor b

Simak pembahasan berikut.

Pembahasan

Diketahui:

vektor a = 3i + j - 2k

vektor b = 4i + nj + 8k

vektor a dan b saling tegak lurus

Ditanya: Nilai variabel n pada vektor b

Jawab:

Perkalian skalar dua buah vektor dirumuskan dengan:

a . b = |a| |b| cos α

dengan:

a = vektor a

b = vektor b

|a| = panjang vektor a

|b| = panjang vektor b

α = besar sudut yang diapit kedua vektor

Dari soal diketahui vektor a dan b saling tegak lurus. Maka sudut yang diapit kedua vektor tersebut adalah 90°. Sehingga diperoleh persamaan berikut:

a . b = |a| |b| cos α

a . b = |a| |b| cos 90°

ingat! cos 90° = 0

a . b = |a| |b| (0)

a . b = 0

Artinya apabila dua buah vektor saling tegak lurus, maka perkalian skalar kedua vektor tersebut sama dengan nol.

a . b = 0

(3i + j - 2k) . (4i + nj + 8k) = 0

(3i . 4i) + (j . nj) + (-2k . 8k) = 0

12 + n - 16 = 0

n = 16 - 12

n = 4

Jadi nilai variabel n dari vektor b adalah 4.

27. Gaya-gaya yang bekerja pada sebuah partikel P :F1 = 3i + j + 3k NF2 = 2i -2j + 7k NF3 = i - 8k NTentukan vektor dan besar resultan gaya yang bekerja pada partikel P!​


Jawaban: Vektor dan besar gaya resultan pada partikel P adalah 6i - j - 2k N dan 6,4 N.

PembahasanMateriPenjumlahan Vektor

Vektor merupakan besaran yang memiliki arah.

Vektor dalam 2D

[tex]\vec{A}= A_x\,\vec{i}+A_y\,\vec{j}[/tex]

Vektor dalam 3D

[tex]\vec{A}= A_x\,\vec{i}+A_y\,\vec{j}+A_z\,\vec{k}[/tex]

Jika terdapat dua vektor yang ingin dijumlahkan, maka:

[tex]\to\vec{A}= A_x\vec{i}+A_y\vec{j}+A_z\vec{k} \\\to \vec{B} =B_x\vec{i}+B_y \vec{j}+B_z\vec{k} \\\\\vec A +\vec B=(A_x+B_x)\vec i +(A_y+B_y)\vec j + (A_z+B_z)\vec k[/tex]

Diketahui

Gaya-gaya yang bekerja pada sebuah partikel P :

[tex]\vec{F_1} = 3\vec i + \vec j + 3\,\vec k N\\\\\vec{F_2} = 2\vec i -2\vec j + 7\vec k N\\\\\vec{F_3} = \vec i - 8\vec k N[/tex]

Ditanya

Vektor dan besar resultan gaya yang bekerja.

Penyelesaian

[tex]\vec {\Sigma F} = \vec F_1+\vec F_2+\vec F_3\\\vec {\Sigma F} = (3+2+1)\vec i +(1+(-2)+0)\vec j +(3+7+(-8))\vec k\\\vec {\Sigma F} = (3+2+1)\vec i +(1-2)\vec j +(3+7-8)\vec k\\\\\vec {\Sigma F} = \bold{\underline{\underline{6\,\vec i -\vec j +2\vec k}}}[/tex]

[tex]|\vec{\Sigma F}|=\sqrt{6^2+(-1)^2+2^2} \, N\\|\vec{\Sigma F}|=\sqrt{36+1+4}\,N\\|\vec{\Sigma F}|=\sqrt{41}\,N\\|\vec{\Sigma F}|=\bold{\underline{\underline{6,4 N}}}[/tex]

Kesimpulan

Vektor dan besar gaya resultan pada partikel P adalah 6i - j - 2k N dan 6,4 N.
