13. jelaskan langkah menambahkan konten dalam bentuk teks dengan gaya list​


Jawaban:

Menambahkan konten dalam bentuk teks dengan gaya daftar dapat dilakukan dengan beberapa cara:

1. Dengan menggunakan tag unordered list atau <ul>

2. Dengan menggunakan  tag ordered list atau <ol>

Langkahnya sbb:

Misalnya kita akan membuat daftar tiga baris yang terdiri dari kalimat kaliat berikut per baris nya:

Banda Aceh di pulau Sumatra

Balikpapan di pulau Kalimantan

Banjarnegara di pulau Jawa.

A. Langkah membuat teks bergaya daftar dengan tag <ul>

- awali dengan mengetik tag <ul> lalu tekan Enter

- lalu buat baris data pertama dan diapit tag <li> dan </li> lalu tekan Enter

- lalu lanjutkan dengan baris data kedua dan ketiga

- akhiri dengan mengetikkan tag </ul>

script lengkapnya sbb:

<ul>

  <li>Banda Aceh di pulau Sumatra</li>

  <li>Balikpapan di pulau Kalimantan</li>

  <li>Banjarnegara di pulau Jawa.</li>

</ul>

B. Langkah membuat teks bergaya daftar dengan tag <ol>

- awali dengan mengetik tag <ol> lalu tekan Enter

- lalu buat baris data pertama dan diapit tag <li> dan </li> lalu tekan Enter

- lalu lanjutkan dengan baris data kedua dan ketiga

- akhiri dengan mengetikkan tag </ol>

script lengkapnya sbb:

<ol>

  <li>Banda Aceh di pulau Sumatra</li>

  <li>Balikpapan di pulau Kalimantan</li>

  <li>Banjarnegara di pulau Jawa.</li>

</ol>

Perbedaan hasil dari kedua daftar di atas adalah baris yang menggunakan tag <ol> akan menggunakan pengurut berupa angka di awal baris, sedangkan tag <ul> akan menampilkan titik.

27. Daftar berurut (Ordered List), daftar tidak berurut (Unordered List), daftar istilah (Definition List), dan daftar bersarang mempunyai tag dan atribut masing-masing. Tag yang digunakan untuk membuat list adalah ​


Jawaban:

1. Ordered List menggunakan tag <ol></ol>

2. Unordered List menggunakan tag <ul></ul>

3. Definition List menggunakan tag <dl></dl>

Penjelasan:

Untuk membuat sebuah list dalam HTML, struktur code biasanya seperti ini

1. Unordered List

<ul>

 <li>Coffee</li>

 <li>Tea</li>

 <li>Milk</li>

</ul>

2. Ordered list

<ol>

 <li>Coffee</li>

 <li>Tea</li>

 <li>Milk</li>

</ol>

3. Definition List

<dl>

 <dt>Coffee</dt>

 <dd>- black hot drink</dd>

 <dt>Milk</dt>

 <dd>- white cold drink</dd>

</dl>

tag <dt> bertujuan untuk menjelaskan term dari item list

sedangkan <dd> untuk deskripsi dari item

Itu bentuk sederhana dari list dalam HTML
