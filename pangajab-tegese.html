24. tegese kapitunan,nyengkuyung,pangajab,bawana,wigati,ngupadi,paru-paru kutha,reboisasi,global warming,go green lan contoh ukarane


           Ing ngisor iki bakal tak jelaskae apa iku artine kapitunan, nyengkuyung, pangajab, bawana, wigati, ngupadi, paru-paru kutha, reboisasi, global warming, lan go green, ugi bakal tak gaweake tuladha ukarane ingkang migunakake tembung iku.

Pembahasan

           Ing ngisor iki teges lan tuladha ukaranipun.

Kapitunan            = Ngrugi, Kerugian, Ora Untung. Nyengkuyung      = Menunjang, Mendukung.  Pangajab             = Tujuanya Bawana               = Omah Panggonan, Jagat, Bumi Wigati                  = Sangat Penting. Ngupadi              = Mencari Paru-Paru Kutha = Taman sing akih wit-witan ing tengah kota. Reboisasi            = Nandur malih hutan ingkang sakwise ditebangi. Global Warming  = Pemanasan bumi. Go Green            = Upaya nylametake bumi.

Tuladhane Ukara :  

Kapitunan, Pemda Bantul kapitunan 100 Juta amargi kedadean longsor kala wingi. Nyengkuyung, Tetuwuhan iki pada ditanduri kanggo nyengkuyung program pemda bantul. Pangajab, tetuwuhan iki pada ditanduri miduweni pangajab kangge ngirangi potensi longsor malih. Bawana, Bawana iki kudu dijaga lan dirawat. Wigati, Tetuwuhan iki wigati kangge njaga struktur lemah. Ngupadi, Pemda Bantul ngupadi ilmuwan ingkang saged mecahake permasalahan ing Bantul. Paru-Paru Kutha, Paru-paru kutha iku miduweni fungsi kang wigati. Reboisasi, Dianakake reboisasi pangajabipun kangge ngirangi potensi longsor. Global Warming, amarga katah tetuwuhan ingkang risak, sakmeniki Bawana kita kagungan Global Warming. Go Green, Reboisasi, Ndamel Paru-paru kutha, lan migunakake listrik kanthi hemat, iku sekedhik tuladhane perilaku go green.

28. tegese tembung ngandhut; pangajab; mligi; pungkuran; nyucekake; ngadani; gotheke wong akeh; prasetya pasemon; mengku surasa;


1. Ngandhut tegese ngandung / nggegawa.
2. Pangajab tegese ndonga / kekarepan.
3. Mligi tegese khusus / ora umum.
4. Pungkuran tegese kelakon.
5. Nyucekake tegese ngresiki.
6. Ngadani tegese nyekel utowo nglakoni adzan.
7. Gotheke wong akeh tegese wakane / duwekane wong akeh.
8. Prasetya tegese kasetyan / setya / kukuh karo sing dipercoyo.
9. Pasemon tegese semu, penyemuan, sindiran sing alus.
10. Mengku surasa tegese ngandung / nggegawa maksud.
