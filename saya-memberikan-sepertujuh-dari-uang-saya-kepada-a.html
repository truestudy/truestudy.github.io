1. saya memberikan sepertujuh dari uang saya pada si A dan 3 kali sebanyak itu kepada si B.sisa uang saya sekarang Rp.1200'-.Berapa uang saya semula?


dimisalkan uang total = X
uang diberikan kepada:
A = 1/7 X
B = 3A = 3/7 X

perhitungan:
X - (1/7 X + 3/7 X) = 1200
X - (4/7 X) = 1200
(7X - 4X) = 1200 x 7
3X = 8400
X = 2800

jadi uang semula adalah 2800

Jawab:

Rp.1.400

Pembahasan:

Misalkan Jumlah uang = 1

diberikan ke A = 1/7

diberikan ke B = 3(1/7) = 3/7

sisa uang = y = Rp.600

A + B + y = 1

1/7 + 3/7 + y = 1

4/7 + y = 1

y = 1 - 4/7

y = 7/7 - 4/7

y = 3/7

Untuk mencari nilai keseluruhannya:

Nilai bagian y adalah 3 bagiannya, maka:

y/3 = Rp.600 / 3 = Rp.200

Berapa bagian A? A = Rp.200 x 1 bagian = Rp.200

Berapa bagian B? B = Rp.200 x 3 bagian = Rp.600

Total uang = Rp.200 + Rp.600 + Rp.600 = Rp.1.400

Cara lainnya:

Karena total uang adalah 7 bagian, maka 7 x Rp.200 = Rp.1.400

Jadi, uang semula adalah Rp.1.400

Penjelasan dengan langkah-langkah:

Diketahui:

Saya memberikan sepertujuh dari uang saya pada si A,

Tiga kali sebanyak itu saya berikan pada si B.

Sisa uang saya sekarag Rp.900.

Ditanya:

Berapakah uang saya semula ?

Jawab:

Misalkan uang saya semula adalah Y.

Saya berikan pada si A = [tex]\frac{1}{7} Y[/tex]

Saya berikan pada si B = [tex]\frac{3}{7} Y[/tex]

Maka:

[tex]Y-\frac{1}{7}Y-\frac{3}{7} Y=Rp900[/tex]

[tex]\frac{7}{7} Y-\frac{1}{7} Y-\frac{3}{7} Y=Rp900[/tex]

[tex]\frac{7-1-3}{7} Y=Rp900[/tex]

[tex]\frac{3}{7} Y=Rp900[/tex]

[tex]Y=Rp900*\frac{7}{3}[/tex]

[tex]Y=Rp2.100[/tex]

Uang saya mula-mula sebesar Rp 2.100
