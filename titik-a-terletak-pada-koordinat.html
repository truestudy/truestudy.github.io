A (2 , - 2√3)   ⇒ x = 2  ; y = - 2√3  di kuadran iv

r² = x² + y²
   = 2² + (- 2√3)
   = 4 + 12
   = 16
 r = √16
 r = 4

tan α = y/x
         = - 2√3 / 2
         = - √3
tan α = - tan (360 - 60)°
tan α = tan 300
      α = 300°

A (2 , - 2√3)  ⇔ A (4 , 300°)

30. Titik A terletak pada koordinat (1,-4) terhadap titik acuan (2,1). Jika titik acuannya mempunyai koordinat (3,-2), koordinat titik A adalah


Koordinat titik A jika titik acuannya mempunyai koordinat (3, -2) adalah titik (0, -1). Dengan koordinat asal titik A adalah (3, -3). Nilai tersebut diperoleh dari perhitungan koordinat terhadap titik acuan. Simak pembahasan berikut.

Pembahasan

Diketahui:

Titik A terletak pada koordinat (1, -4) terhadap titik acuan (2, 1)

Ditanya: Koordinat titik A jika titik acuannya mempunyai koordinat (3, -2)

Jawab:

Misalkan titik A memiliki kordinat (x₁, y₁) dan titik acuan A adalah B memiliki koordinat (x₂, y₂). Maka koordinat A terhadap titik acuan B adalah:

C = A - B

(x, y) = (x₁, y₁) - (x₂, y₂)

(x, y) = (x₁ - x₂, y₁ - y₂)

atau dapat ditulis sebagai berikut:

x = x₁ - x₂

y = y₁ - y₂

Titik asal A adalah koordinat titik A terhadap titik asal atau (0, 0). Maka untuk menentukan titik asal A, diperoleh

C = A - B

A = C + B

Dari soal diketahui:

titik C = (1, -4)

titik B = (2, 1)

Maka titik asal A diperoleh

A = C + B

A = (1, -4) + (2, 1)

A = (1 + 2, -4 + 1)

A = (3, -3)

Titik asal A diperoleh A(3, -3).

Sehingga koordinat titik A(3, -3) jika titik acuannya mempunyai koordinat (3, -2) adalah

(x₁, y₁) = (3, -3)

(x₂, y₂) = (3, -2)

x = x₁ - x₂

x = 3 - 3

x = 0

dan

y = y₁ - y₂

y = -3 - (-2)

y = -3 + 2

y = -1

Sehingga koordinat titik A(3, -3) jika titik acuannya mempunyai koordinat (3, -2) adalah titik (0, -1).
