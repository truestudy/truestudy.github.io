2. SISTEM PERTIDAKSAMAAN LINIER DAN KUADRAT 1. TENTUKAN DAERAH HIMPUNAN PENYELESAIAN DARI SISTEM PERTIDAKSAMAAN 2. TENTUKAN DAERAH HIMPUNAN PENYELESAIAN DARI SISTEM PERTIDAKSAMAAN 3. PERHATIKAN GAMBAR BERIKUT! TENTUKAN SISTEM PERTIDAKSAMAAN DARI DAERAH PENYELESAIAN PADA DAERAH YG DIARSIR DIATAS!


Jawaban:

Sistem pertidaksamaan dari daerah penyelesaian (daerah yang diarsir) pada grafik berikut adalah

a. 3x + y ≥ 21; x + 3y ≤ 21; y ≥ 0

b. 3x + y ≤ 3; x + y ≤ 2; x ≥ 0; y ≥ 0

Persamaan garis yang melalui titik (0, a) dan (b, 0) adalah

ax + by = ab

Jika koefisien y adalah bilangan positif, maka

ax + by ≥ ab, daerah himpunan penyelesaiannya diarsir ke atas

ax + by ≤ ab, daerah himpunan penyelesaiannya diarsir ke bawah

Pembahasan

a) Persamaan garis yang melalui titik (0, 21) dan (7, 0) adalah

21x + 7y = 21(7)

==> kedua ruas dibagi 7 <==

3x + y = 21

Karena daerah arsirnya di atas garis 3x + y = 21, maka pertidaksamaannya adalah

3x + y ≥ 21

Persamaan garis yang melalui titik (0, 7) dan (21, 0) adalah

7x + 21y = 7(21)

==> kedua ruas dibagi 7 <==

x + 3y = 21

Karena daerah arsirnya di bawah garis x + 3y = 21, maka pertidaksamaannya adalah

x + 3y ≤ 21

Daerah arsirnya di atas sumbu x maka pertidaksamaannya adalah

y ≥ 0

Jadi sistem pertidaksamaan dari grafik tersebut adalah

3x + y ≥ 21

x + 3y ≤ 21

y ≥ 0

b) Persamaan garis yang melalui titik (0, 3) dan (1, 0) adalah

3x + 1y = 3(1)

3x + y = 3

Karena daerah arsirnya di bawah garis 3x + y = 3, maka pertidaksamaannya adalah

3x + y ≤ 3

Persamaan garis yang melalui titik (0, 2) dan (2, 0) adalah

2x + 2y = 2(2)

==> kedua ruas dibagi 2 <==

x + y = 2

Karena daerah arsirnya di bawah garis x + y = 2, maka pertidaksamaannya adalah

x + y ≤ 2

Daerah arsirnya di atas sumbu x maka pertidaksamaannya adalah

y ≥ 0

Daerah arsirnya di sebelah kanan sumbu y maka pertidaksamaannya adalah

x ≥ 0

Jadi sistem pertidaksamaan dari grafik tersebut adalah

3x + y ≤ 3

x + y ≤ 2

x ≥ 0

y ≥ 0

15. tentukanlah sistem pertidaksamaan dari daerah penyelesaian pada gambar berikut ini ​


( i ) #Mencari persamaan fungsi kuadrat yang mempunyai titik balik maksimum ( 2 , 9 ) dan memotong sumbu-Y di titik ( 0 , 5 ).

Dimisalkan : fungsi kuadrat tersebut adalah :

y = ax² + bx + c

¤ Grafik fungsi memotong sumbu-Y di titik ( 0 , 5 ) :

y = ax² + bx + c [tex]\to[/tex] x = 0 dan y = 5

5 = a.(0)² + b.(0) + c

5 = 0 + 0 + c

c = 5

¤ Grafik fungsi mempunyai titik balik maksimum ( 2 , 9 ) :

Suatu grafik fungsi mencapai titik balik pada saat : y' = 0

y = ax² + bx + c [tex]\to[/tex] y' = 2ax + b

y' = 0

2ax + b = 0

b = –2ax

Substitusikan nilai absis x = 2 dari titik balik maksimum ( 2 , 9 ) :

b = –2a.(2)

b = –4a ... ( i )

¤ Grafik fungsi melalui titik ( 2 , 9 ) :

y = ax² + bx + c [tex]\to[/tex] x = 2 dan y = 9

9 = a.(2)² + b.(2) + c

9 = 4a + 2b + c

Substitusikan nilai c yang didapatkan dan nilai b dari ( i ) :

9 = 4a + 2.(–4a) + 5

9 = 4a - 8a + 5

9 = –4a + 5

4a = 5 - 9

4a = –4

a = –1

Substitusikan nilai a yang didapatkan ke ( i ) :

b = –4a

b = –4.(–1)

b = 4

Sehingga persamaan fungsi kuadrat tersebut adalah ( a = –1 ; b = 4 ; c = 5 ) : y = –x² + 4x + 5

» Mencari tanda ketidaksamaan dengan men-substitusikan nilai ( x , y ) = ( 0 , 0 ) yang pada gambar terletak pada daerah penyelesaian :

y ... –x² + 4x + 5

0 ... –(0)² + 4.(0) + 5

0 ... 0 + 0 + 5

0 ≤ 5

Jadi pertidaksamaan fungsi kuadrat yang daerah hasilnya seperti terarsir di gambar adalah : [tex]\boxed{y~\leq~-x^2~+~4x~+~5}[/tex]

( ii ) #Mencari persamaan garis lurus yang melalui titik ( 3 , 0 ) dan titik ( 0 , –2 ) :

===========================================

Rumus mencari persamaan garis lurus yang melalui titik ( a , b ) dan titik ( c , d ) :

[tex]\boxed{\frac{x~-~a}{c~-~a}~=~\frac{y~-~b}{d~-~b}}[/tex]

===========================================

Maka persamaan garis tersebut adalah :

[tex]\frac{x~-~3}{0~-~3}~=~\frac{y~-~0}{-2~-~0}[/tex]

[tex]\frac{x~-~3}{-3}~=~\frac{y}{-2}[/tex]

–2(x - 3) = –3y

–2x + 6 = –3y

Sehingga persamaan garis lurus tersebut adalah : 2x - 3y = 6.

» Mencari tanda ketidaksamaan dengan men-substitusikan nilai ( x , y ) = ( 0 , 0 ) yang pada gambar terletak pada daerah penyelesaian :

2x - 3y ... 6

2.(0) - 3.(0) ... 6

0 - 0 ... 6

0 ≤ 6

Jadi pertidaksamaan fungsi linier yang daerah hasilnya seperti terarsir di gambar adalah : [tex]\boxed{2x~-~3y~\leq~6}[/tex]

Kesimpulan :

Daerah yang diarsir adalah daerah penyelesaian dari sistem pertidaksamaan :

[tex]\boxed{\boxed{\begin{array}{ccc}y~\leq~-x^2~+~4x~+~5\\2x~-~3y~\leq~6\end{array}}}[/tex]
