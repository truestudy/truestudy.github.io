karena gradien itu miring (jatuhnya miring), artinya dari atas ke bawah, jadi dimulai dari sumbu y baru sumbu x

logikanya seperti itukarena untuk mencari kemiringan, berlaku rumus : kemiringan = perubahan panjang sisi tegak ( vertikal ) dibagi perubahan panjang sisi mendatar ( horizontal )

• Soal

▪︎ Tuliskan Rumus Menghitung Vektor Pada Sumbu x Dan sumbu Y

__________________________

✏ Penyelesaian

Ditunjukkan Sebuah vektor ( A ) yang bisa diuraikan jadi komponen vektor pada Sumbu x, yaitu Ax dan komponen

vektor pada Sumbu y, yaitu Ay

Contohnya Sudut Vektor A dengan Sumbu X yaitu θ, maka besar Ax Dan Ay Bisa kamu dari perbandingan Sinus ( Sin ) Dan Kosinus ( Cos )

_________________________

✏ Keterangan

A = Vektor A

Ax = komponen Vektor A pada Sumbu x

Ay = Komponen Vektor A Pada Sumbu y

θ = Besar sudut yang dibentuk antara vektor A dengan Sumbu x

_________________________

✏ Contoh Soal :

Sebuah Vektor yang panjang nya 20 Cm membentuk Sudut 30° Terhadap Sumbu x positif.

Penyelesaian : Antara Vektor A dengan Sumbu y pasti 90° - 30° = 60°. 90° Dapat Dari mana? Dari Sudut Siku - Siku yang terbentu antara Ax Dan Ay

_________________________

✏ Pembahasan

Komponen Pada Sumbuh X :

• Ax = A Cos θ

• Ax = ( 20 Cm ) Cos 30°

• Ax = ( 20 Cm ) ( 1/2✔30 )

• Ax = 10✔3 Cm

Komponen Pada Sumbu Y :

• Ay = A Sin θ

• Ay = ( 20 Cm ) Cos 30°

• Ay = ( 20 Cm ) ( 1 / 2 )

• Ay = 10 Cm

9. Tentukan rumus fungsi kuadrat jika grafiknya memotong sumbu x di (-1,0) dan (5,0) dan memotong sumbu y dititik (0,3)


Jawab:

f(x) = -3/5 x²+ 12/5 x + 3

Penjelasan dengan langkah-langkah:

Persamaan grafik fungsi kuadrat yang memotong sumbu x di (x1, 0) dan (x2, 0) serta melalui titik (x3, y3) adalah

f(x) = a(x - x1)(x - x2)

Diketahui:

(x1, 0) = (-1, 0)

(x2, 0) = (5, 0)

(x3, y3) = (0, 3)

》Mencari a

Substitusi (0, 3) ke f(x), diperoleh:

y3 = a(x3 - x1)(x3 - x2)

3 = a(0 + 1)(0 - 5)

3 = a(1)(-5)

3 = -5a

-3/5 = a

》Substitusi a ke f(x), diperoleh:

f(x) = a(x - x1)(x - x2)

f(x) = -3/5 (x + 1)(x -5)

f(x) = -3/5 (x² - 5x + x - 5)

f(x) = -3/5 (x² - 4x - 5)

f(x) = -3/5 x²+ 12/5 x + 3

11. bagaimana rumus untuk mendapatkan kordinat bayangan pada sumbu -y dari suatu titik yang direfleksikan terhadap garis y=h


Pencerminan/refleksi merupakan salah satu bentuk transformasi geometri. Transformasi adalah pemetaan titik dari suatu bidang ke titik lain pada bidang yang sama. Jenis transformasi selain rotasi adalah translasi, dilatasi, dan refleksi.  

Terdapat beberapa jenis pencerminan diantaranya adalah pencerminan terhadap sumbu x, sumbu y, garis y = x, garis y = -x. garis x = h, garis y = k, dan terhadap titik O(0,0)  

Pembahasan

Rumus pencerminan terhadap garis y = h adalah

 P(x,y) akan menjadi P'(x, 2h - y)  jika dicerminkan terhadap garis y  = h

Jika garis OA kita putar searah jarum jam sehingga berimpit dengan sumbu X maka diperoleh garis OA’. Titik-titik yang terletak pada garis OA’ memiliki ordinat 0, sehingga gradien garis OA’ yakni perbandingan antara komponen y dan x akan di dapat:

<=> m = komponen y/OA’

<=> m = 0/OA’

<=> m = 0

Jadi, gradien suatu garis yang sejajar sumbu X adalah nol.
