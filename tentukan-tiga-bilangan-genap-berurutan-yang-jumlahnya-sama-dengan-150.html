4. 2. Tentukan tiga bilangan genap berurutan yang jumlahnya sama dengan 150kak tolong dengan caranya​


Jawaban:

Hasil dari ketiga bilangan = 150

Misalkan :

- Bil. pertama = x

- Bil. kedua = x + 2

- Bil. ketiga = x + 4

Mengapa ditambah 2 dan 4 ? Itu dikarenakan kita akan mencari bil. genap yg berurutan.

Selanjutnya, kita masukkan semua yg kita ketahui tadi menjadi bentuk persamaan.

x + (x + 2) + (x + 4) = 150

x + x + x + 2 + 4 = 150

3x = 150 - 6

x = 144/3

x = 48 ✔

Maka,,,

bil. kedua → a + 2

= 48 + 2

= 50 ✔

bil. ketiga → a + 4

= 48 + 4

= 52 ✔

Pembuktian :

= 48 + 50 + 52

= 150 ✔

Urutan bilangan = 48, 50, 52

__-----------__-----------__-----------__

Cara simpel :

Tiga bilangan = a, b, c.

Jumlah bil. = 150

Untuk mendapatkan salah satu bilangan, kita bagi jumlah bilangan dg banyak bilangan.

a = 150 : 3

a = 50

Selanjutnya, tambah 2 pd bilangan untuk menghasilkan bil. genap

b = 50 + 2

b = 52

Secara logika, jika bil. 52 ditambah 2 lagi, maka hasilnya akan melebihi 150. Maka, kita kurangi bil. 50 dg 2.

c = 50 - 2

c = 48

Urutan bilangan = 48, 50, 52.

Jadi, tiga bilangan genap yg jumlahnya 150 adalah bilangan 48, 50, 52.
