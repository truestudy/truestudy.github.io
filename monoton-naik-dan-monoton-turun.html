2. apa yg dimaksud monoton naik dan monoton turun, jelaskan dengan contohnya


Suatu fungsi dikatakan monoton naik jika berlaku x1 > x2, maka f(x1) > f(x2) untuk selang tertentu.
Monoton turun jika x1 < x2, maka f(x1) < f(x2)
Fungsi itu Hanya Naik Atau turun pada selang tersebut.
Contohnya:
f(x) = x^2
Saat int (negatif tak hingga, 0), fungsi itu monoton turun menuju titik balik (0, 0). Lalu, fungsi itu akan monoton naik sampai (tak hingga, 0).

Catatan: Monoton = tetap/konstan

15. f(x) = 4x^3 - 4 tentukan monoton naik & monoton turun, kecekungan, titik ekstrim, dan titik belok


Tentukan beberapa fungsi turunannya:

f'(x) = 4(3x²) - 0
f'(x) = 12x²

f"(x) = 12(2x)
f"(x) = 24x

Nampak beberapa hal.

Monoton naik ketika f'(x) > 0
Monoton turun ketika f'(x) < 0
Dengan f'(x) = 12x², yang mana f'(x) tak akan negatif.

Maka, fungsi monoton naik pada x < 0 atau x > 0

Kecekungan,
Cekung ke atas apabila f"(x) > 0
Cekung ke bawah apabila f"(x) < 0
Dengan f"(x) = 24x, fungsi berikut akan cekung ke bawah untuk x < 0 dan cekung ke atas untuk x > 0

Titik ekstrim terjadi ketika f'(x) = 0, dengan f'(x) = 12x².
Terdapat titik ekstrim di x = 0, dan y = -4.
Titik ekstrimnya adalah (0,-4)

Titik belok terjadi ketika f"(x) = 0, dengan f"(x) = 24x
Terdapat titik belok di x = 0, dan y = -4
Titik belok fungsi naiknya adalah (0,-4)

27. mau tanya ini monoton naik atau turun? pelajaran kelas X(10) ipa​


Fungsi [tex]f(x)=2^{-x}[/tex] termasuk monoton turun.

PEMBAHASAN

Turunan atau Diferensial merupakan pengukuran terhadap bagaimana fungsi berubah seiring perubahan nilai input. Salah satu fungsi dari turunan adalah menentukan interval fungsi naik atau fungsi turun, dimana :

Fungsi akan naik pada saat f'(x) > 0.Fungsi akan turun pada saat f'(x) < 0.

.

Jika pada interval tertentu fungsi hanya naik atau hanya turun saja, maka disebut monoton naik atau monoton turun.

.

DIKETAHUI

[tex]f(x)=2^{-x}[/tex]

.

DITANYA

Tentukan apakah fungsi termasuk monoton naik atau monoton turun.

.

PENYELESAIAN

Kita cari terlebih dahulu turunan dari fungsi f(x).

[tex]f(x)=2^{-x}\\\\y=2^{-x}~~~~~~~...logaritma~naturalkan~kedua~ruas\\\\lny=ln\left ( 2^{-x} \right )\\\\lny=-xln2\\\\\frac{1}{y}y'=-ln2\\\\y'=-yln2\\\\y'=-2^{-x}ln2\\\\y'=-\frac{1}{2^x}ln2\\\\\\Diperoleh~f'(x)=-\frac{1}{2^x}ln2\\[/tex]

.

Perhatikan bahwa [tex]2^x[/tex] nilainya selalu positif untuk semua nilai x. Mengakibatkan[tex]-\frac{1}{2^x}ln2[/tex] nilainya akan selalu negatif untuk semua nilai x. Dapat disimpulkan bahwa : f'(x) < 0 → fungsi monoton turun.

.

KESIMPULAN

Fungsi [tex]f(x)=2^{-x}[/tex] termasuk monoton turun.

.
