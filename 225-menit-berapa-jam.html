Satuan waktu ada 3 , yaitu:

JamMenitDetik

1 jam = 3.600 detik

Penyelesaian

225 menit = 225 : 60

= 3,75 jam

= 3 jam 45 menit ✔

Kecepatan = 80 km/jam

waktu = 225 menit :60 = 3,75 jam

Kecepatan = 60 km/jam.

waktu ?..= X

Pakai perbandingan berbalik nilai :

3,75 / X = 60/80

60.X = 3,75 × 80

60. X = 300

X = 300/60

X = 5 jam

ATAU

Jarak = Kecepatan × Waktu

= 80 × 3,75

= 300 km

Waktu = Jarak / Kecepatan

= 300 / 60

= 5 jam
