2. terangkan dasar BIOS untuk hard disk


Terangkan dasar BIOS untuk hard disk, sebelum membahas lebih lanjut mengenai BIOS untuk hardisk, mari kita bahas terlebih dahulu apa itu BIOS. BIOS merupakan singkatan dari Basic Input Output System, fungsi utama bios adalah untuk mengidentifikasi perangkat keras yang terhubung dengan motherboard yang nantinya bekerja secara bersama-sama. Dalam BIOS terdapat berbagai macam menu, mulai dari pengaturan password BIOS, pengaturan urutan booting, pengaturan alokasi VGA onboard,pengaturan interface hardisk dan melakukan perubaha-perubahan angka clock processor dan masih banyak lainnya.

Salah satu fungsi BIOS adalah untuk mendeteksi hardisk yang kita pasang. Teknik dasar BIOS untuk hardisk sendiri adalah mendeteksi adanya hardisk atau tidak, selain itu BIOS juga bisa kita atur untuk melakukan booting dari hardisk. Pengaturan interface hardisk pada BIOS juga bisa kita lakukan seperti melakukan perubahan IDE Mode atau AHCI Mode untuk hardisk kita.

Demikian pembahasan mengenai teknik dasar dari BIOS, tanpa adanya BIOS kita tidak akan bisa menghidupkan perangkat komputer dengan kita sebagaimana mestinya. Untuk membaca materi lainnya mengenai BIOS dan perangkat komputer lainnya dapat dibaca pada link berikut

4.
Cara Kerja BIOS  

Seperti telah dijelaskan diatas bahwa fungsi utama BIOS adalah melakukan POST. Sedangkan urutan-urutan atau cara kerja BIOS adalah dimulai dengan proses inisialisasi, dimana dalam proses ini kita bisa melihat jumlah memory yang terinstall, jenis hardisk dan kapasitasnya dan sebagainya. 

BIOS kemudian akan mencari, menginisialisasi dan menampilkan informasi dari Graphics Card. Kemudian akan mengecek device ROM lain seperti hardisk dan kemudian melakukan pengetesan RAM yaitu memory count up test. Setelah  semua test komponen berhasil dilakukan, BIOS kemudian akan mencari lokasi booting device dan Sistem Operasi.

Mengenai cara kerja BIOS atau urutan-urutan apa saja yang dikerjakan BIOS pada saat melakukan POST ini bisa dibaca pada postingan mengenai urutan proses startup

20. jelaskan cara untuk masuk ke dalam mode konfigurasi BIOS dengan tipe Award BIOS!​


Jawaban:

Langkah pertama yang perlu dilakukan adalah, jika PC Anda sudah dalam keadaan menyala, maka restart-lah PC Anda. Atau tekan tombol power untuk menyalakannya jika PC dalam keadaan mati.

Yang kedua, amati tulisan “masuk setup” yang ada pada beberapa detik pertama setelah PC Anda menyala. Namun pesan ini akan berbeda-beda pada setiap jenis PC dan tombol mana yang harus ditekan. Di bawah ini ada beberapa cara masuk ke Bios berdasarkan jenis PC pabrikannya. Yaitu :

Bios Award. Caranya dengan menekan tombol Del atau Ctrl+Alt+Esc

Bios Ami. Caranya dengan menekan tombol Del.

Microid Research (MR Bios). Caranya dengan menekan tombol Esc.

IBM Aptivas dan think pads. Caranya dengan menekan tombol F1.

Caranya dengan menekan tombol F10.

Caranya dengan menekan tombol Alt + return atau tombol reset dua kali.

IBM PS /2s. Caranya dengan menekan tombol Insert.

Komputer Riba Toshiba. Caranya dengan menekan tombol Esc lalu F1.

Bios phoenix. Caranya dengan menekan tombol F2.

Olivetti PC Pro. Caranya dengan menekan tombol Ctrl+Shift+Alt+Del.

Caranya dengan menekan tombol Ctrl+Shift+Esc.

AST Advantage. Caranya dengan menekan tombol Ctrl+Alt+Esc.

Phonix dan Zenith. Caranya dengan menekan tombol Ctrl+Alt+S, Ctrl+Alt+Insert.

Caranya dengan menekan tombol Ctrl+Esc.

Demikianlah ulasan ini mengenai cara masuk Bios pada PC Anda. Masing-masing PC memiliki cara yang berbeda untuk masuk ke Bios, jadi perhatikan jenis PC Anda. Lakukanlah dengan hati-hati untuk merubah setting agar PC Anda tidak menjadi rusak. smoga dpt mmbantu

3. Cara memasang password pada Bios

Masuk menu bios dengan tombol F2 (sesuaikan dengan merk bios yang digunakan)

Geser ke tab Security pilih menu Power-On Password

Muncul jendela Set Power-On Password

Tulis password pada kolom Enter New Password dan Confirm New Password. Penting! penulisan password harus sama pada kolom Enter New Password dan Confirm New Password. Lalu tekan Enter

Untuk menyimpan hasil settingan tadi, tekan F10 lalu tekan Enter
