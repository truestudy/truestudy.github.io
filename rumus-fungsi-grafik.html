2. rumus fungsi grafik


garis melalui titik ( - 2 , 0) dan (0 , 6)
persamaan grs :

(y - y1) / (y2 - y1) = (x - x1) / (x2 - x1)
    (y - 0) / (6 - 0)  = (x -  - 2) / (0 - - 2)

              6 (x + 2) = 2y
               6x + 12 = 2y

4. rumus fungsi dari grafik pada gambar adalah​


Jawab:

f(x) = x - 1

Penjelasan dengan langkah-langkah:

kita beri nama setiap titik tersebut

( 0 , -1 ) >> x1 = 0 dan y1 = -1

( 1 , 0 ) >> x2 = 1 dan y2 = 0

maka kita masukin ke rumus persamaan garis

( y - y1 ) / ( x - x1 ) = ( y2 - y1 ) / ( x2 - x1 )

( y - (-1) ) / ( x - 0) = ( 0 - (-1) ) / ( 1 - 0 )

( y + 1 ) / x = 1 / 1

( y + 1 ) / x = 1

y + 1 = x

y = x - 1

jadi persamaannya adalah y = x - 1 dan Karena y dapat di tulis dengan f(x) maka f(x) = x - 1

13. rumus fungsi grafik disamping berapa?


Diketahui dua titik (2,0) dan (0,-6)
x1 = 2 ; x2 = 0 ; y1 = 0 ; dan y2 = -6
Maka:
(y-y1)/(y2-y1) = (x-x1)/(x2-x1)
(y-0)/(-6-0)     = (x-2)/(0-2)
  y/-6              = (x-2)/-2
-6(x-2)           = -2 . y
-6x + 12         = -2y
-6x + 12 + 2y = 0         masing-masing suku dibagi 2:
-3x + y + 6     = 0         (pada persamaan suku awal tidak boleh negatif)
Maka:
3x - y - 6 = 0



14. rumus fungsi dari grafik tersebut adalah


titik pertama ---> x1 = 0 dan y1 = 6

titik ke-2 ---> x2 = 6 dan y2 = 15

gradien garis lurus = m

m = ( y1 - y2 ) / ( x1 - x2 )

m = ( 6 - 15 ) / ( 0 - 6 )

m = -9 / -6

m = 3/2

persamaan garis lurus :

y - y1 = m ( x - x1 )

y - 6 = 3/2 ( x - 0 )

y - 6 = 3/2 x

y = 3/2 x + 6

rumus fungsi dari grafik tersebut adalah : f (x) = 3/2 x + 6

jawaban nya : C

25. Rumus fungsi kuadrat dari grafik tersebut adalah​


[tex]\displaystyle\rm~f(x) = {x}^{2} - x - 6[/tex]

Penjelasan dengan langkah-langkah:

Berdasarkan grafik, fungsi kuadrat memotong sumbu-x di titik (-2,0) dan (3,0) serta melalui titik (0,-6).

Maka :

[tex]\displaystyle\rm~x_1 = -2~dan~x_2 = 3[/tex]

Penyelesaian :

[tex]\colorbox{thistle}{\color{purple}{\boxed{\displaystyle\bf~{y = a(x - x_1)(x - x_2)}}}}[/tex]

[tex]\displaystyle\rm~y = a(x - x_1)(x - x_2) \\ \displaystyle\rm~ - 6 = (0 + 2)(0 - 3) \\ \displaystyle\rm~ - 6 = a( 2)( - 3) \\ \displaystyle\rm~ - 6 = a( - 6) \\ \displaystyle\rm~ - 6 = - 6a \\ \displaystyle\rm~a = 1[/tex]

[tex]\displaystyle\rm~y = a(x - x_1)(x - x_2) \\ \displaystyle\rm~y = 1(x + 2)(x - 3) \\ \displaystyle\rm~y = (x + 2)(x - 3) \\ \displaystyle\rm~y = {x}^{2} - 3x + 2x - 6 \\ \displaystyle\rm~y = {x}^{2} - x - 6[/tex]

Jadi, jawabannya c.
