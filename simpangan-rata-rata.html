9. rata rata simpangan dari data4,1,3,5,1,4 adalah


Jawab:

RS = 4/3


Penjelasan dengan langkah-langkah:

Nilai simpangan rata - rata adalah ukuran keseragaman data. Angka yang didapatkan dari simpangan rata - rata akan menjawab pertanyaan : "secara umum, seberapa dekat nilai - nilai data dengan rata - rata nya?".


Bentuk umum untuk menentukan simpangan rata - rata adalah :

RS = (1/n) × ∑ | xi - (nilai rata-rata) |


di mana

RS = Rata - Rata Simpangan

n = banyaknya data

xi = data ke-i


Maka, untuk menyelesaiakan soal di atas, kita perlu mencari nilai rata - ratanya terlebih dahulu.


Nilai rata - rata = (4 + 1 + 3 + 5 + 1 + 4) / 6

= 18 / 6

= 3


Lalu kita lanjutkan dengan mencari simpangan rata - rata.


RS = (1/n) × ∑ | xi - (nilai rata-rata) |

RS = 1/6 × (|4-3| + |1-3| + |3-3| + |5-3| + |1-3| + |4-3|)

RS = 1/6 × (1 + 2 + 0 + 2 + 2 + 1)

RS = 1/6 × 8 = 8/6

RS = 4/3

30. simpangan baku dan simpangan rata rata dari data 6,5,8,10,12,7 adalah

x = 48/6

Sr = (|6 - 8| + |5 - 8| + |8 - 8| + |10 - 8| + |12 - 8| + |7 - 8|) / 6

Sr = (|-2| + |-3| + |0| + |2| + |4| + |-1|)/6

Sr = (12)/6

Sr = 2

R = 34/6

R = 17/3

Simpangan baku :
Sb = √R
