4. unsur klorin memiliki dua unsur isotop yaitu cl -35 dan cl-37 tentukan kelimpahan kedua isotop tersebut jika di ketahui massa atom relatif cl adalah 36,5


Unsur klorin memiliki dua unsur isotop yaitu Cl -35 dan Cl-37 tentukan kelimpahan kedua isotop tersebut jika di ketahui massa atom relatif Cl adalah 36,5

Diketahui:

Cl - 35

Cl -37

Ar Cl = 36,5

Ditanya: Kelimpahan masing-masing isotop...?

Jawab:

untuk menghitung massa atom relatif suatu unsur dari isotop-isotopnya, kita bisa menggunakan rumus berikut:

Ar isotop = (% kelimpahan isotop-1 x massa isotop 1) + (% kelimpahan isotop-2 x massa isotop-2)

Jadi, jika kita diminta mencari nilai kelimpahan dari masing-masing isotop yang sudah diketahui massa relatif atomnya, maka langkah pertama kita harus melakukan pemisalan terlebih dahulu.

Misalkan:

Kelimpahan isotop Cl-35 = x, maka

Kelimpahan isotop Cl-37 = 100 - x

sehingga jika kita substitusikan kedalam rumus di atas, maka:

Ar Cl = (x% x Massa Cl-35) + ((100 - x)% x Massa Cl-37

36,5 = ((x/100) x 35) + ((100 - x)/100) x 37)

36,5 = ((35 x/100) + ((3700 - 37 x)/100)

36,5 = ((35x - 37x) + 3700)/100

3650 = -2x + 3700

3650 - 3700 = -2x

-50 = - 2x

x = 25

Jadi kelimpahan isotop Cl-35 adalah sebesar 25%, sedangkan kelimpahan isotop Cl-37 adalah sebesar 75%.

Konfigurasi Elektron

Penyusunan elektron pada subkulit atom disebut dengan konfigurasi elektron. Terdapat tiga aturan yang digunakan dalam penulisan konfigurasi elektron yaitu prinsip Aufbau, asas larangan Pauli, dan kaidah Hund.

1. Prinsip Aufbau

Berdasarkan aturan Aufbau, pengisian elektron dimulai dari sub kulit yang memiliki tingkat energi rendah menuju tingkat energi yang tinggi.  

1s < 2s < 2p < 3s < 3p < 4s < 3d < 4p < 5s < 4d < 5p < 6s < 4f < 5d < 6p < 7s < 5f < 6d < 7p

2. Kaidah Hund

Pengisisan elektron dalam orbital cenderung untuk tidak berpasangan lebih dahulu, kemudian apabila tidak ada lagi orbital yang kosong maka barulah elektron akan berpasangan.

3. Larangan Pauli

Bahwa dalam satu atom maka tidak diperbolehkan terdapat dua elektron yang memiliki empat bilangan kuantum yang sama.

13. 1. Tentukan massa atom relative dari unsur Oksigen, dimana oksigen memiliki 3buah isotope dengan kelimpahan masing masing 99,7% isotope O-16, 0,1%isotope O-17 Dan 0,2% Isotope O-18!​


Massa atom relatif (Ar) nya adalah 16

»PEMBAHASAN«

Untuk menghitung massa atom relatif (Ar) jika diketahui persen isotop dapat menggunakan rumus:

Ar Unsur = Σ(kelimpahan isotop x massa isotop)

Ar Unsur = (%isotop x massa isotop)

===================================

Diketahui:

isotop O-16 = 99,7%

isotop O-17 = 0,1%

isotop O-18 = 0,2%

Ditanya:

Massa atom relatif (Ar)?

Jawab:

Kita gunakan rumus diatas:

Ar O = Σ(kelimpahan isotop x massa isotop)

Ar O = (%isotop x massa isotop)

Ar O = (%16-O x Massa 16-O) + (%17-O x Massa 17-O) + (%18-O x Massa 18-O)

Ar O = ([tex] \frac{99,7}{100} \times 16 [/tex]) + ([tex] \frac{0,1}{100} \times 17 [/tex]) + ([tex] \frac{0,2}{100} \times 18 [/tex])

Ar O = (0,997 x 16) + (0,001 x 17) + (0,002 x 18)

Ar O = 15,95 + 0,017 + 0,036

Ar O = 16,003

Ar O = 16

Jadi massa atom relatif (Ar) O adalah 16
