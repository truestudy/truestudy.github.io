Gambarkan grafik fungsi eksponen dari persamaan y = 3ˣ

Ciri-ciri grafik dari y = aˣ adalah

selalu memotong sumbu y di titik (0, 1)

tidak pernah memtong sumbu x (sumbu x sebagai asimtot datar)

Jika 0 < a < 1, maka kurvanya turun dari kiri ke kanan

Jika a > 1, maka kurvanya naik dari kiri ke kanan

Penjelasan dengan langkah-langkah:

y = 3ˣ

Karena a > 1 yaitu a = 3, maka kurvanya naik

Memotong sumbu y di titik (0, 1)

Asimtot datarnya adalah y = 0 atau sumbu x

Kemudian kita akan mencari titik-titik yang dilalui grafik tersebut dengan mensubstitusikan  

x = –2, –1, 0, 1, 2, ….

Jika x = –2, maka

y = 3⁻² =

(–2, )

Jika x = –1, maka

y = 3⁻¹ =

(–1, )

Jika x = 0, maka

y = 3⁰ = 1

(0, 1)

Jika x = 1, maka

y = 3¹ = 3

(1, 3)

Jika x = 2, maka

y = 3² = 9

(2, 9)

Hubungkan titik-titik tersebut sehingga terbentuk grafik dari y = 3ˣ

19. Berikut yang merupakan grafik y = 3x+¹


Jawaban:

liat pada gambar

Ingat : untuk menggambar grafik y = ax+c, tentukan titik potong terhadap sumbu x (y=0) dan titik potong terhadap sumbu y (x=0). Setelah itu hubungkan kedua titik.

Sehingga,

y = 3x-1

• titik potong terhadap sumbu x → y= 0

3x-1 = 0

3x = 1

x = ⅓

Titik potong terhadap sumbu x : (⅓, 0)

• titik potong terhadap sumbu y → x= 0

y =3(0)-1

y = -1

Titik potong terhadap sumbu y : (0, -1)

Hubungkan titik (⅓, 0) dan (0, -1) dan perpanjang
