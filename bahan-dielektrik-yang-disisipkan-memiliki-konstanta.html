4. sebuah kapasitor pelat sejajar, luas setiap pelatnya 500cm² dan terpisah sejauh 0,25 cm satu sama lain. beda potensial diantara pelat 2 kV jika berisi udara dan 1 kV jika diantara pelat disiapkam bahan dielektrik. hitunglah: a). kapasitas mula mula. b). muatan q pada setiap pelat. c). kapasitas setelah penyisipan bahan dielektrik. d). konstanta dielektrik bahan. e). perwitivitas bahan dielektrik. f). kuat medan listrik mula mula diantara pelat. g). kuat medan listik E' setelah penyisipan dielektrik


a) Besar kapasitar  mula - mula adalah 17,7 x 10⁻¹¹ F.

b) Besar muatan q pada setiap plat adalah 35,4 x 10⁻⁸ C.

c) Besar kapasitas setelah disisip bahan dielektrik adalah 35,4 x 10⁻¹¹ F.

d) Besar konstanta dielektrik bahan tersebut adalah 2.

e) Besar permitivitas bahan dielektrik tersebut adalah 17,7 x 10⁻¹² C² / Nm²

f) Besar kuat medan listrik mula - mula adalah 8 x 10⁵ V/m.

g) Besar kuat medan setelah penyisipan dielektrik adalah 4 x 10⁵ V/m.

[tex]\boxed{\boxed{PEMBAHASAN}}[/tex]

[tex]\boxed{\boxed{BAB\:2.\:LISTRIK\:STATIS}}[/tex]

Gaya Coulumb merupakan salah satu besaran vektor, karena memiliki nilai dan arah.

Persamaan gaya coulumb adalah :

[tex]\boxed{F=k \frac{Q_{1} Q_{2}}{r^{2}}}[/tex]

Persamaan Besarnya Medan Listrik adalah:

[tex]\boxed{E=\frac{F}{Q} }[/tex]

atau

[tex]\boxed{E=k\frac{Q}{r^{2} }}[/tex]

Persamaan Energi potensial listriknya adalah :

[tex]\boxed{W=QV}[/tex]

[tex]\boxed{Diketahui}[/tex]

Luas Plat (A) = 500 cm² = 500 x 10⁻⁴ m²

Jarak antar plat (d) = 0,25 cm = 0,25 x 10⁻² m²

Beda potensial berisi udara (V) = 2 kV = 2 x 10³ V

Beda potensial berisi bahan (V') = 1 kV = 1 x 10³ V

[tex]\boxed{Ditanya}[/tex]

a). kapasitas mula mula.

b). muatan q pada setiap pelat.

c). kapasitas setelah penyisipan bahan dielektrik.

d). konstanta dielektrik bahan.

e). perwitivitas bahan dielektrik.

f). kuat medan listrik mula mula diantara pelat.

g). kuat medan listik E' setelah penyisipan dielektrik

[tex]\boxed{Penyelesaian}[/tex]

a). kapasitas mula mula.

Besar kapasitas mula - mula bisa dicari dengan menggunakan persamaan :

C = ε₀ A / d

dimana ε₀ = 8,85 x 10⁻¹² C²/Nm sehingga :

[tex]C=8,85x10^{-12} \frac{C^{2} }{Nm^{2} }\frac{500x10^{-4}m^{2} }{0,25x10^{-2}m^{2}  }[/tex]

C = 17,7 x 10⁻¹¹ F

Jadi, Besar kapasitar  mula - mula adalah 17,7 x 10⁻¹¹ F.

b). muatan q pada setiap pelat.

Besar muatan q pada tiap plat bisa dicari dengan menggunakan persamaan :

q = C V

masukkan nilai V pada saat berisi udara dan C mula - mula sehingga :

q = 17,7 x 10⁻¹¹ F  2 x 10³ Volt  

q = 35,4 x 10⁻⁸ Coulumb

Jadi, besar muatan q pada setiap plat adalah 35,4 x 10⁻⁸ C.

c). kapasitas setelah penyisipan bahan dielektrik.

Besar kapasitas setelah penyisipan bahan bisa di cari dengan menggunakan persamaan :

C' = q / V'

C' = 35,4 x 10⁻⁸ C / 1 x 10³ V

C' = 35,4 x 10⁻¹¹ F

Jadi, besar kapasitas setelah disisip bahan dielektrik adalah 35,4 x 10⁻¹¹ F.

d). konstanta dielektrik bahan.

Besar konstanta dielektrik bisa dicari dengan menggunakan persamaan :

K = C' / C

K = 35.4 x 10⁻¹¹ F / 17.7 x 10⁻¹¹ F

K = 2

Jadi, konstanta dielektrik bahan tersebut adalah 2.

e). permitivitas bahan dielektrik.

Besar permitivitas bahan dielektrik tersebut bisa dicari dengan persamaan :

ε = K ε₀

ε = 2 x 8,85 x 10⁻¹² C² / Nm²

ε = 17,7 x 10⁻¹² C² / Nm²

Jadi, permitivitas bahan dielektrik tersebut adalah 17,7 x 10⁻¹² C² / Nm²

f). kuat medan listrik mula mula diantara pelat.

Besar kuat medan Listrik mula - mula bisa dicari dengan menggunakan persamaan :

E = V / d

E = 2 x 10³ Volt / 0,25 x 10⁻² m

E = 8 x 10⁵ V/m

Jadi. kuat medan listrik mula - mula adalah 8 x 10⁵ V/m.

g). kuat medan listik E' setelah penyisipan dielektrik

Besar kuat medan listrik setelah di sisip bahan dielektrik bisa dicari dengan menggunakan persamaan :

E' = V' / d

E' = 1 x 10³ Volt / 0,25 x 10⁻² m

E' = 4 x 10⁵ V/m

Jadi, besar kuat medan setelah penyisipan dielektrik adalah 4 x 10⁵ V/m.

[tex]\boxed{Kesimpulan}[/tex]

a) Besar kapasitar  mula - mula adalah 17,7 x 10⁻¹¹ F.

b) Besar muatan q pada setiap plat adalah 35,4 x 10⁻⁸ C.

c) Besar kapasitas setelah disisip bahan dielektrik adalah 35,4 x 10⁻¹¹ F.

d) Besar konstanta dielektrik bahan tersebut adalah 2.

e) Besar permitivitas bahan dielektrik tersebut adalah 17,7 x 10⁻¹² C² / Nm²

f) Besar kuat medan listrik mula - mula adalah 8 x 10⁵ V/m.

g) Besar kuat medan setelah penyisipan dielektrik adalah 4 x 10⁵ V/m.

Kapasitas Kapasitor Keping Sejajar (C)

Dua keping yang sejajar apabila diberikan muatan listrik berbeda (+) dan (-) maka dapat menyimpan muatan listrik, sehingga dua keping sejajar tersebut mempunyai kapasitas kapasitor (C).

Apabila diatara keping sejajar itu terdapat udara atau ruang hampa, rumus kapasitas kapasitor (C) bisa kita tuliskan :

C = εo . A / d

Keterangan:
C = kapasitas kapasitor (F)
εo = permitivitas udara atau ruang hampa (8,85 × 10^-12 C/Nm²)
d = jarak antar keping (m)
A = luas penampang keping (m²)

Jika di antara keping sejajar terdapat bahan dielektrik, permitivitas yang dipakai adalah permitivitas zat dielektrik (ε)
ε = K.εo

Sehingga,
C = K.εo. A/d
keterangan: K = konstanta dielektrik

Diketahui :
K = 50
d = 0,1 mm = 10^-4 m
A = 5 cm² = 5 x 10^-4 m²
εo = permitivitas udara atau ruang hampa (8,85 × 10^-12 C/Nm²)

Ditanyakan :
a) C jika permitivitas dielektrik
b) C jika dielektrik di keluarkan atau kata lainnya C jika permitivitas medium udara

Jawab :
a) disisipkan dielektrik
C = K. εo A/d
C = (50) . (8,85 x 10^-12) . (5 x 10^-4) / (10^-4)
C = 2.212,5 x 10^-12
C = 2,2 x 10^-9 Farad
Atau
C = 2,2 nanoFarad
C = 2,2 nF

b) C jika dielektrik di keluarkan atau kata lainnya C jika permitivitas medium udara

C = εo . A/d
C = (8,85 x 10^-12) . (5 x 10^-4) / (10^-4)
C = 44,25 x 10^-12 Farad
atau
C = 44,25 pikoFarad
C = 44,25 pF

30. Sebuah kapasitor keping sejajar memiliki kapasitas 0,2 mF. Jika di antara keping disisipi bahan dielektrik yang memiliki konstanta dua kali semula dan jarak antar keping didekatkan sehingga menjadi setengahnya, maka kapasitas kapasitor menjadi


Sebuah kapasitor keping sejajar memiliki kapasitas 0,2 mF. Jika di antara keping disisipi bahan dielektrik yang memiliki konstanta dua kali semula dan jarak antar keping didekatkan sehingga menjadi setengahnya, maka kapasitas kapasitor menjadi ...
PEMBAHASAN

Diketahui :

C₁ = 0,2 mF

k₂ = 2k₁


Ditanya :

C₂ = ?


PENDAHULUAN

Jika terdapat dua buah pengantar yang diantaranya terdapat bahan penyekat dan menghasilkan muatan listrik lalu disimpan kedalam komponen yang disebut Kapasitor. Kapasitor dapat dirumuskan dengan


[tex] \boxed {C = k \epsilon_0 \frac {A}{d}} [/tex]


Dengan,

C = Kapasitor (Farad)

A = luas penampang penghantar (m²)

d = Jarak antar keping (m)

k = konstanta dielektrik


PENYELESAIAN

Jawaban Pendek

Jika di antara keping disisipi bahan dielektrik yang memiliki konstanta dua kali semula dan jarak antar keping didekatkan sehingga menjadi setengahnya, maka kapasitas kapasitor menjadi 0,4 mF.


Jawaban Panjang

[tex] C = k \epsilon_0 \frac {A}{d} [/tex]

C d = k εₒ A

[tex] \frac {C}{k} = \frac {\epsilon_0 A}{d} [/tex]

[tex] \frac {C}{k} = konstan [/tex]


Karena nilai εₒ, A dan d tdk berubah untuk kedua peristiwa maka konstan.


[tex] \frac {C_1}{k_1} = \frac {C_2}{k_2} [/tex]

[tex] \frac {C_1}{k_1} = \frac {C_2}{2k_1} [/tex]

[tex] \frac {C_1}{1} = \frac {C_2}{2} [/tex]

2C₁ = C₂

2(0,2) = C₂

C₂ = 0,4 mF


KESIMPULAN

Sebuah kapasitor keping sejajar memiliki kapasitas 0,2 mF. Jika di antara keping disisipi bahan dielektrik yang memiliki konstanta dua kali semula dan jarak antar keping didekatkan sehingga menjadi setengahnya, maka kapasitas kapasitor menjadi 0,4 mF.
