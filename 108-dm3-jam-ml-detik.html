7. 108 cm3/jam.........ml/detik​


Jawab:

[tex]\large {\blue {\sf \bf 108 \ cm^3/jam = \underline {388.800} \ mL/detik}}[/tex].

Penjelasan dengan langkah-langkah:

Ubah satuan waktu ke detik.

[tex]\sf \bf 108 \ cm^3/jam \to Ubah \ jam \ ke \ menit\\\sf \bf 108 \ cm^3/jam \times 60 \ menit = 6.480 \ cm^3/menit \to Ubah \ menit \ ke \ detik\\\sf \bf 6.480 \ cm^3/menit \times 60 \ detik = \underline {388.800 \ cm^3/detik}[/tex]

Ubah satuan volume ke liter. [tex]\iff \sf \bf \boxed {\sf \bf 1 \ dm^3 = 1 \ liter} \ \to \ sehingga \ \boxed {\sf \bf 1 \ cm^3 = \frac{1}{1000} \ dm^3 = \frac{1}{1000} \ liter}[/tex].

[tex]\sf \bf 388.800 \ cm^3/detik\\\sf \bf (388.800 \ cm^3/detik \times \frac{1}{1000}) \ liter\\\sf \bf 388,8 \ liter/detik \to \boxed {\sf \bf 1 \ liter = 100 \ mL}\\\sf \bf 388,8 \ liter/detik \times 100 \ mL\\\boxed {\blue {\sf \bf 388.800 \ mL/detik}}[/tex]

15. 18.000 dm3 / jam = . . . ml / detik​


Penjelasan dengan langkah-langkah:

(c)

18.000 dm³ / jam = 5.000 ml / detik

---> 18.000 dm³ = 18.000 liter = 18.000.000 ml

---> 1 jam = 3.600 detik

[tex]\sf{18.000 \: dm^{3} \: / \: jam}[/tex]

[tex]\sf{= 18.000.000 \: ml \: / \: 3.600 \: detik}[/tex]

[tex]\sf{= (\dfrac{18.000.000}{3.600}) \: ml \: / \: detik}[/tex]

[tex]\sf{= 5.000 \: ml \: / \: detik}[/tex]

(d)

125 l / detik = 750 dal / menit

---> 125 l = 12,5 dal

----> 1 detik = 1/60 menit

[tex]\sf{125 \: l \: / \: detik}[/tex]

[tex]\sf{= 12,5 \: dal \: / \: \dfrac{1}{60 } \: menit}[/tex]

[tex]\sf{= (\dfrac{12,5}{\frac{1}{60}}) \: dal \: / \: menit}[/tex]

[tex]\sf{= (12,5 \times 60) \: dal \: / \: menit}[/tex]

[tex]\sf{= 750 \: dal \: / \: menit}[/tex]
