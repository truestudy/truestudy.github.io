11. Pecepatan sebuah benda dinyatakan dengan persamaan a =4t+2. Jika saat t=2 sekon, kecepatan benda 16 m/s dan posisinya 20 m, tentukanlah a.persamaan percepatan benda b. Kecepatan awal benda (V0) c. Posisi awal benda (X0) d. Persamaan posisi benda


a) Maksudnya persamaan kecepatan benda, kalau persamaan percepatan benda sudah diketahui yaitu a=4t+2. . v = ∫a.dt = 2t² + 2t + C   saat t=2 s, v=16 m/s 16 = 2.4 +2.2 + C C = 16 - 12 = 4 v =  2t² + 2t + 4 . b) Kecepatan awal benda  t=0. v₀ = 2.0 +2.0 + 4 = 4 m/s . d) Persamaan Posisi Benda. X = ∫v.dt = ∫(2t² + 2t + 4)dt X = 2/3.t³ + t² + 4t + C saat t=2 , X = 20 m 20 = (2/3).8 + 4 + 4.2 + C C = 20 - 16/3 - 12 C = 60/3 - 16/3 - 36/3 C = 8/3 = 2,66 . Persamaan posisi X = 2/3.t³ + t² + 4t + 2,66 . c. Posisi awal benda, t=0 Xo = 2,66 m

1. V₆ = V₄ + a x t;  6 = 4 + a x 3  ; a = 2/3 m/s²
2. S = 4 x 3 + 1/2 x 2/3 x 3² = 13 + 3 = 16 m
jarak yang ditempuh dari mulai percepatan selama 3 detik adalah 16 m

S = (1/2) * (v₁ + v₂) * t
S = (1/2) * (4 + 6) * 3 = 15 meter
