5. apa yang harus di instal terlebih dahulu jika kita mengkonfigurasi FTP server?


Pada saat kita akan melakukan pengaturan FTP server, maka langkah yang harus kita lakukan terlebih dahulu adalah melakukan penginstallan FTP server terlebih dahulu. Untuk melakukan proses penginstallan FTP server pada sistem operasi windows, kita bisa melakukannya melalui control panel pada menu Program and Features, atau kita bisa menginstall software Filezilla. Sedangkan pada sistem operasi Linux, kita bisa menginstall proftp.

Pembahasan

FTP merupakan singkatan dari File Transfer Protocol, sebuah protocol yang berfungsi untuk saling bertukar file. Untuk bisa bertukar menggunakan protocol FTP, kita memerlukan FTP server dan client server. Beberapa fungsi dari FTP server adalah

Untuk bertukar file yang terhubung dalam suatu jaringan komputerSebagai tempat untuk menyimpan file dari beberapa penggunaSebagai penyedia data yang lebih aman dan bisa diakses oleh user yang sudah memiliki ijin

Dengan menggunakan FTP, proses pertukaran data akan semakin mudah dilakukan, bahkan ketika kita menggunakan software filezilla, kita bisa mengambil atau menaruh file ke FTP dengan cara klik dan drag, seperti pada saat kita menggunakan komputer seperti biasa.

16. Langkah langkah konfigurasi failover menggunakan vps ubuntu


Jawaban:

Cara Konfigurasi dan Membuat High Availability/Clustering pada OS Linux

High Availability (HA) hanya mengacu pada kualitas sistem untuk beroperasi terus menerus tanpa kegagalan untuk jangka waktu yang lama. Solusi HA dapat diimplementasikan menggunakan perangkat keras dan / atau perangkat lunak, dan salah satu solusi umum untuk menerapkan HA adalah clustering.

Dalam komputasi, sebuah cluster terdiri dari dua atau lebih komputer (umumnya dikenal sebagai node atau anggota) yang bekerja bersama untuk melakukan tugas. Dalam pengaturan seperti itu, hanya satu simpul yang menyediakan layanan dengan simpul sekunder mengambil alih jika gagal.

Cluster terbagi dalam empat tipe utama:

Storage: memungkinkan server untuk secara bersamaan membaca dan menulis ke sistem file bersama.

High Availability: menghilangkan titik kegagalan tunggal dan dengan gagal layanan dari satu node cluster ke yang lain dalam kasus node menjadi tidak beroperasi.

Load Balancing: mengirimkan permintaan layanan jaringan ke beberapa node cluster untuk menyeimbangkan beban permintaan di antara node cluster.

High Performance: melakukan pemrosesan paralel atau bersamaan, sehingga membantu meningkatkan kinerja aplikasi.

24. Carilah informasi cara menginstal software!


Mapel: TI

Materi: Instalasi Software

--Soal--

Terlampir.

--Jawaban--

Dalam melakukan instalasi terhadap software, ada beberapa hal yang harus diperhatikan dan dilakukan.

1. Biasakan baca teks README atau referensi menginstal software tersebut dari sumber resminya serta pastikan developernya terpercaya.

2. Cek apakah software tersebut kompatibel dengan perangkat kita, jika tidak segera hubungi developer software tersebut untuk panduan.

3. Pastikan ruang penyimpanan disk perangkat kita tidak penuh untuk menginstall/memasang software dan requirementsnya.

4. Setelah mengetahui software tersebut kompatibel, pasang/installlah terlebih dahulu requirements software tersebut. Seperti MS Visual Studio, Net Framework, Java Runtime, dan sebagainya sesuai referensi.

5. Setelah semua siap, install/pasang software tersebut sesuai tutorial/referensi yang tersedia. Baca End-User License & Agreement dengan saksama. Kemudian jika sesuai keinginan, pasang software tersebut.

6. Software sudah siap dipakai, jangan lupa untuk senantiasa mengecek sumber software seperti website untuk menerima update & perbaikan bug terbaru.
