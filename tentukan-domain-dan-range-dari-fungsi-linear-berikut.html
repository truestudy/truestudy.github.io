3. tentukan domain dan range fungsi linear berikut! . . tolong bantu yaa☹️☹️


Fungsi adalah pemetaan setiap anggota sebuah himpunan (yang dinamakan sebagai domain atau daerah asal) kepada anggota himpunan yang lain (yang dinamakan sebagai kodomain atau daerah kawan). Konsep fungsi adalah salah satu konsep dasar matematika.

Anggota himpunan yang dipetakan dapat berupa apa saja (kata, orang, atau objek lain), namun biasanya yang dibahas adalah besaran matematika seperti bilangan riil.

Pembahasanf(x) = 2x + 5

Fungsi f(x) = 2x + 5 tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.

f(x) = 3 - x

Fungsi f(x) = 3 - x tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.

f(x) = [tex]\frac{1}{2}[/tex]x - 4

Fungsi f(x) = [tex]\frac{1}{2}[/tex]x - 4 tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.

f(x) = 5 - 3x

Fungsi f(x) = 5 - 3x tidak mempunyai batas domain, sehingga domain fungsi tersebut -∞ < x < ∞.

Range dari fungsi tersebut adalah -∞ < x < ∞.

5. Tentukan domain dan range dari fungsi linear berikut! f(×) = 2× + 5


f(x) = 2x + 5

domain x = -1
range f(-1) = -2 + 5 = 3

domain x = 0
range f(0) = 0+5 = 5

domain x = 1
range f(1) = 7

Domain = { x | -1 ≤ x ≤ 1 }
Range = { x | 3 ≤ x ≤ 7 }
