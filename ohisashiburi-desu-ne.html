Rina : (Rina) desu ga, (Nita)-san o (onegaishimasu).

Nita : Watashi desu.

Rina : (Issho ni) eiga o (mimasenka)?

Nita : Ii desu ne, (mimashou). Itsu desuka?

Rina : Doyoubi wa (dou) desuka?

Nita : Ii desu ne, sou shimashou.

Rina : Jya, mata (doyoubi).

Nita : Mata doyoubi.

Aisatsu ini sangatlah penting dan selalu digunakan bukan hanya orang Jepang saja, tapi semua orang termasuk orang Indonesia dalam kehidupan sehari-hari. Aisatsu ada yang sopan dan ada juga yang tidak sopan, maka dari itu harus berhati-hati dalam mengucapkannya.

13. apa perbedaan sou desu ne dengan sou desu ka ? Perbedaan soudesuka dan soudesuka


そうですね (Sou desu ne) : Iya ya, Benar juga, Iya kan?, dsb.そうですか (Sou desu ka) : Iya kah?, Benar kah?, Oh begitu?, dsb.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Kedua ungkapan 'Sou desu ne' dan 'Sou desu ka' sama-sama terdapat 'Sou desu'. Hanya berbeda di akhirnya saja. 'Sou desu ne' menggunakan akhiran partikel Ne, sedangkan 'Sou desu ka' menggunakan akhiran partikel Ka. Sebelum ke pembahasan mengenai arti dari kedua ungkapan ini, akan saya bahas dulu mengenai partikel Ne dan Ka.

Fungsi Partikel ね (Ne)

Partikel ini merupakan salah satu partikel dalam kelompok Shuujoshi (partikel akhir kalimat). Fungsinya adalah untuk menyatakan kesan, serta seperti untuk menanyakan persetujuan lawan bicara. Kalau diartikan kurang lebih seperti "..ya", "..bukan?", "..kan?", dsb.

Cth: Kawaii ne (Imut ya), Kore desu ne (Ini kan?)

Fungsi Partikel か (Ka)

Partikel ini juga termasuk ke dalam kelompok Shuujoshi. Fungsinya adalah untuk membuat kalimat tanya. Kalau diartikan kurang lebih seperti "..kah?", "..nggak?", atau bisa juga hanya berperan seperti tanda tanya.

Cth: Ii desu ka (Boleh?), Kore desu ka (Ini kah?)

- - - - - - -

Setelah memahami kegunaan dari kedua partikel tersebut, mari kita lanjut ke pembahasan utama dari soal ini. Kedua ungkapan 'Sou desu ne' dan 'Sou desu ka' sama-sama terdapat 'Sou desu' di dalamnya.

Arti そうです (Sou desu)

Ungkapan ini bisa diartikan "Ya", "Benar", dsb. Merupakan ungkapan persetujuan terhadap pernyataan dari lawan bicara. Ungkapan ini merupakan ungkapan yang sopan. Untuk membuatnya menjadi ungkapan yang kasual, dapat dengan menghilangkan 'Desu'.

Contoh percakapan satu:

A : Azu-kun desu ka? (Apa kamu Azu?)B : Hai, sou desu (Ya, benar)

Contoh percakapan dua:

A : Kou? (Begini?)B : Sou sou (Ya ya)

Arti そうですね (Sou desu ne)

Ungkapan ini dapat berarti "Iya ya", "Benar juga", "Iya kan?", dsb. Tergantung dari intonasinya, ungkapan ini juga bisa menjadi ungkapan untuk menanyakan persetujuan lawan bicara. Ungkapan ini bahkan juga bisa hanya menjadi ungkapan untuk merespon pendapat lawan bicara, tanpa mengandung makna persetujuan.

Contoh percakapan satu:

A : Ano ko kawaii ne (Cewe itu imut ya?)B : Sou desu ne (Iya ya)

Contoh percakapan dua:

A : Dou omou? (Bagaimana pendapatmu?)B : Sou desu ne... (Hmm...)

Arti そうですか (Sou desu ka)

Ungkapan ini memiliki arti "Iya kah?", "Benar kah?", "Oh begitu?", dsb. Digunakan sebagai ungkapan untuk bertanya tentang kebenaran dari pernyataan lawan bicara. Selain itu juga bisa menjadi ungkapan untuk menyatakan ketidaksetujuan.

Contoh percakapan satu:

A : Ano hito wa Azu-san (Orang itu Azu)B : Sou desu ka (Oh begitu)

Contoh percakapan dua:

A : Ano ko kawaii ne (Cewe itu imut ya?)B : Sou desu ka (Ngga ah)

26. beda sou desu ne dengan sou desu ka


Sou desu ne : iya juga, oh gitu, iya ya.Sou desu ka : begitu ya?, oh gitukah?, iya kah?

Pembahasan

Sou desu ne「 そうですね 」

Frasa ini bentuk ungkapan untuk menyatakan ekspresi rasa kesetujuan mengenai pernyataan atau pertanyaan dari sang lawan bicaranya.

Penggunaannya tersebut bisa dipakai ketika formal, kasual, non-formal (cocok sehari-hari). Hanya saja, penggunaan frasa ini tidak sepenuhnya cocok dalam sehari-hari jika frasa ini selalu sering diucapkan sehingga memberikan kesan yang kurang sopan.

–› Contoh penggunaannya

= アイスクリームが美味しい (aisukuriimu ga oiishi) : eskrimnya enak.

そうですね (sou desu ne) : iya ya.

= お前が知っている、 彼が悪い人だ (omae ga shltteiru, kare ga warui hito da) : lu tahu ya, dia (lk) itu jahat.

そうですね (sou desu ne) : iya juga ya.

Sou desu ka「 そうですか 」

Frasa ini juga cocok digunakan dalam sehari-hari dan juga di berbagai situasi manapun (mau itu kasual, formal/non-formal/dll), akan tetapi penggunaannya akan memberi kesan yang kurang sopan jika digunakan kepada sang bos/atasan.

Nah ungkapan ini digunakan oleh sang pembicara yang telah memahami atau menyetujui dengan baik mengenai apa yang disampaikan oleh lawan bicara.

Akan tetapi, penggunaannya bisa sebagai bentuk penegasan kepada sang lawan bicara dan tidak hanya itu frasa ini juga bisa ketika pada saat dimana sang lawan bicara menyampaikan atau meminta pemahaman/kesetujuan kepada sang pembicara, tapi sang pembicara tidak mengerti/menyetujui (merasa ragu) apa yang disampaikan oleh lawan bicara.

–› Contoh penggunaannya

= お母さんは今日家にいます (okaasan wa kyou ka ni imasu) : ibu ada dirumah hari ini.

そうですか, 大丈夫 準備できました (sou desu ka, daijoubu junbi dekimashlta) : oh begitu kah? oke siap.

= 今日君の友達が家に来ました (kyou kimi no tomodachi ga kaette ni ikimashlta) : temanmu pergi ke rumah.

そうですか? どうでもいい (sou desu ka? dou demo ii) : iyakah? bodo amat.

-------

Frasa そうですね (sou desu ne) dan そうですか (sou desu ka) terdapat partikel です (desu), ね (ne), dan か (ka).

Untuk partikel です (desu) dalam frasa tersebut sekedar memberikan kesan sopan, sedangkan ね (ne) jenis partikel yang bertujuan menghaluskan salah satu kalimat kalau dalam bahasa Indonesia dapat diartikan ya, kan.

Kemudian partikel か (ka) sangat umum dipakai untuk kalimat tanya, ataupun penegasan.

Ungkapan ini bahkan juga bisa hanya menjadi ungkapan untuk merespon pendapat lawan bicara, tanpa mengandung makna persetujuan.

Contoh percakapan satu:

A : Ano ko kawaii ne (Cewe itu imut ya?)B : Sou desu ne (Iya ya)

Contoh percakapan dua:

A : Dou omou? (Bagaimana pendapatmu?)B : Sou desu ne... (Hmm...)

Arti そうですか (Sou desu ka)

Ungkapan ini memiliki arti "Iya kah?", "Benar kah?", "Oh begitu?", dsb. Digunakan sebagai ungkapan untuk bertanya tentang kebenaran dari pernyataan lawan bicara. Selain itu juga bisa menjadi ungkapan untuk menyatakan ketidaksetujuan.

Contoh percakapan satu:

A : Ano hito wa Azu-san (Orang itu Azu)B : Sou desu ka (Oh begitu)

Contoh percakapan dua:

A : Ano ko kawaii ne (Cewe itu imut ya?)B : Sou desu ka (Ngga ah)

28. Apa Maksud Dari Pernyataan:"Suki Ga Kirei Desu Ne~"BANTU JAWAB ;)​


Penulisannya salah, mungkin yang kamu maksud adalah "tsuki ga kirei desu ne" yang artinya adalah "bulannya indah ya."

Pembahasan

Tsuki「月・つき」

Artinya "bulan." Merujuk terhadap bulan dalam kalender atau bisa juga merujuk terhadap bulan yang kerap muncul di langit malam.

Ga「が」

Partikel dengan fungsi sebagai penghubung atau penunjuk subjek.

Kirei「きれい」

Artinya yaitu "indah," "cantik," "rapi," "bersih." Masuk ke gol. kt sifat na-keiyoushi 2.

Desu「です」

Akhiran yang kerap digunakan untuk sebagai bentuk pelembut kalimat.

Ne「ね」

Partikel akhir ini memiliki arti yang menegaskan persetujuan kepada sang lawan bicara yang terletak di akhir kalimat, artinya tersebut diantaranya seperti "ya," "kan?", "bukan?"

_____

Sehingga untuk arti kalimat dari 月がきれいでくね (tsuki ga kirei desu ne) adalah bulannya indah ya/bulan (itu) indah ya.
