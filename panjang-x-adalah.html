2. Luas ABCD = panjang .... x panjang ....= p X ....= .... Satuan luasMaka luas persegi panjang ABCDL = panjang .... x panjang ....L = .... X ...​


Apabila diketahui persegi panjang ABCD sebagai berikut, maka perlu diketahui dulu RUMUSnya.

Rumus Luas Persegi Panjang adalah :

Luas = Panjang persegi  X  Lebar persegi

Maka, apabila tidak diketahui panjang dan lebar persegi. Cara mengetahui luas nya adalah :

L =  Panjang AB  X  Panjang BC

L =  AB  X  BC

L = (hasil dari perkalian itu) Satuan luas.

NOTE :

Kalian bisa menggunakan bagian panjang dan lebar manapun.  Karena persegi panjang mempunyai 2 sisi yang sama, yaitu sisi atas dengan sisi bawah, dan sisi kiri dengan sisi kanan.

jawaban : C, K = 2 x (panjang + lebar)

penjelasan:

Misalkan, panjang sisi PQ dan RS adalah p dan panjang sisi QR dan PS adalah L. Keliling persegi panjang dapat ditentukan dengan cara berikut.

Keliling persegi panjang

= PQ + QR + RS + PS

= p + l + p + l

= (2 x p) + (2 x L)

= 2 x (p + l)

jadi, keliling persegi panjang dapat ditulis sebagai berikut.

dengan, K = Keliling persegi panjang

P = panjang persegi panjang

L = lebar persegi panjang
