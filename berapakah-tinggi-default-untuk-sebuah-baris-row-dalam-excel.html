3. Berapakah tinggi default untuk sebuah baris(frow)dalam excel


Anda dapat merubah tinggi baris dari 0 sampai dengan 409 poin, dimana satu poin kira-kira sama dengan 1/72 inci atau 0,0035 cm. Jika tinggi baris adalah 0 (zero), maka baris disembunyikan atau tidak ditampilkan.


4. ^'Apa itu row dan range dalam excel?


Range adalah kumpulan dari beberapa sel
Row (baris) adalah ruang antara dua garis mendatar pada worksheet. Rows : sel baris dari atas sampai bawah : misal A2 : A8

Range : Sekumpulan data dari sel , untuk menentukan informasi atau hasil
