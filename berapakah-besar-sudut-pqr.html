1. Pada segitiga pqr diketahui besar sudut rpq adalah setengah dari besar sudut pqr dan besar sudut prq adalah tiga per dua dari besar sudut pqr besar sudut pqr adalah


dalam segitiga
*sudut p = 1/2 q
*sudut r = 3/2 q
*sudut q = 1q

jadi ➡
180= p + q + r
180= 1/2q + q + 3/2q
180= 1/2q + 2/2q +3/2q
180=6/2q
180=3q
180/3=q
60=q
jadi besar sudut pqr adalah 60°

2. Jika besar sudut PQR = 55 ° maka besar sudut PQR =......°


Jawaban:

110°

Penjelasan dengan langkah-langkah:

POR = PQR×2 = 55×2 = 110°


3. Diketahui sudut pqr dan sudut rst saling berpenyiku, jika besar sudut pqr lima kali besar sudut rst, maka besar sudut pqr adalah.... *


Jawaban:

75°

Penjelasan dengan langkah-langkah:

< pqr = 5 × < rst

< pqr = 5 < rst

< pqr + < rst = 90° (karena berpenyiku)

5 < rst + < rst = 90°

6 < rst = 90°

< rst = 90° : 6

< rst = 15°

< pqr = 5 × < rst

< pqr = 5 × 15°

< pqr = 75°

5. Besar sudut pqr adalah


<PRQ menghadap diameter maka besar sudutnya 90°
<PQR = 180 - (90 + 20)
<PQR = 70°

Semoga membantu :)Diketahui: ∠RPO = 20°
Ditanya   : ∠PQR = ...?
Dijawab  :

∠ROQ = 2.∠RPO
∠ROQ = 2.20°
∠ROQ = 40°

180° = ∠ROP + ∠ROQ (Berpelurus)
180° = ∠ROP + 40°
180° - 40° = ∠ROP
        140°  = ∠ROP

180° = ∠ROP + ∠RPO + ∠PRO (Jumlah sudut dalam segitiga)
180° = 140° + 20° + ∠RPO
180° = 160° + ∠RPO
180° - 160° = ∠RPO
          20°    = ∠RPO

90° = ∠RPO + ∠QRO (Sudut yg menghadap diameter, besarnya 90°)
90° = 20° + ∠QRO
90° - 20° = ∠QRO
       70°   = ∠QRO

180° = ∠QRO + ∠ROQ + ∠PQR
180° = 70° + 40° + ∠PQR
180° = 110° + ∠PQR
180° - 110° = ∠PQR
           70°  = ∠PQR

Jadi jawabannya adalah C. 70°

12. Besar sudut PQR adalah


caranya, buat garis horizontal (ke samping) pada titik Q, sehingga <Q terbagi mjd 2 sudut (<Q atas dan <Q bawah)

utk sudut yg berada di dekat garis PQ, <P itu berada di dalam berseberangan dgn <Q bawah, shg <Q bawah =26°

utk sudut yg berada di dekat garis RQ, <R juga dalam berseberangan dgn <Q atas, shg <Q atas = 38°

jadi, didapat hasil :
<PQR = < Q atas + < Q bawah
<PQR = 38° + 26°
<PQR = 64° (B)
