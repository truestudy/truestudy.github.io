Penjelasan dengan langkah-langkah:

1.) RENDANG

total unsur = 7!

unsur ganda = 2!

(7x6x5x4x3x2x1) / (2x1)

5.040 / 2

= 2.520 susunan kata

2.) TAHU SUMEDANG

total unsur = 12!

unsur ganda = 2! / 2!

(12x11x10x9x8x7x6x5x4x3x2x1) / (2x1) /

(2x1)

479.001.600 / 4

= 119.750.400 susunan kata

3.) TAHU GEJROT

total unsur = 10!

unsur ganda = 2!

(10x9x8x7x6x5x4x3x2x1) / (2x1)

3.628.800 / 2

= 1.814.400 susunan kata

4.) TAHU JURALIT

total unsur = 11!

unsur ganda = 2! / 2! / 2!

(11x10x9x8x7x6x5x4x3x2x1) / (2x1) /

(2x1) / (2x1)

39.916.800 / 8

= 4.989.600 susunan kata
