20. bahasa Jepangnya Pensil =Penghapus =Papan Tulis =Kapur =​


Bahasa Jepang dari pensil 鉛筆 ( Enpitsu ) Penghapus 消しゴム ( Keshigomu ) Papan tulis 黒板 ( Kokuban ) Kapur チョーク ( Chooku )

Pembahasan

Barang-barang yang terdapat dalam kelas antara lain;

pensil 鉛筆 ( Enpitsu ) Penghapus 消しゴム ( Keshigomu ) Papan tulis 黒板 ( Kokuban ) Kapur チョーク ( Chooku ) kata serapan; Chalk

____________________________________

Barang-barang yang berada di kelas;

Buku;本 ( Hon )Pensil; 鉛筆 ( Enpitsu )Kertas; 髪 ( Kami )Kalkulator; 電卓 ( Dentaku )Rak buku; 本棚 ( Hondana )Tas; かばん ( Kaban )

Contoh kalimat;

1.私のかばん ( Watashi No Kaban ) → Tas Saya ( + )

Terjemah Per-Kata;

私 ( Watashi ) → Memiliki arti saya

の ( No ) → Partikel yang menunjukkan Kepemilikan

かばん ( Kaban ) → Memiliki arti Tas

2.私の鉛筆ではありません ( Watashi no Enpitsu dewa Arimasen ) → Bukan Pensil saya ( - )

Terjemah Per-Kata;

私 ( Watashi ) → Memiliki arti saya

の ( No ) → Partikel yang menunjukkan

鉛筆 ( Enpitsu ) → Memiliki arti Pensil

では ありません ( Dewa Arimasen ) → Memiliki arti bukan

3.何の髪( Nani no Kami?) →Kertas apa ? ( ? )

Terjemah Per-Kata;

何 ( Nani ) → Kalimat Tanya memiliki arti apa

の ( No ) → Partikel yang menunjukkan Kepemilikan

髪 ( Kami ) → Memiliki arti Kertas

4.これ本はいくらですか? ( Kore Hon wa ikura desu ka ? ) → Berapa harga Buku ini ? ( ? )

これ ( Kore ) → Memiliki arti Ini

Terjemah Per-Kata;

本 ( Hon ) → Memiliki arti Buku

は ( Wa ) → Partikel yang menunjukkan Subjek

いくら ( Ikura ) → Memiliki arti berapa bentuk lain dari Ikutsu

です ( Desu ) → Tidak memiliki arti berfungsi sebagai penghalus kalimat

か ( Ka ) → Partikel Tanya

5.緑の消しゴム ( Midori no Keshigomu ) → Penghapus warna hijau ( + )

Terjemah Per-Kata;

緑 ( Midori ) → Memiliki arti hijau

の ( No ) → Partikel yang menunjukkan Kepemilikan

消しゴム ( Keshigomu ) → Memiliki arti sepatu roda

Tambahan;

Kore; Memiliki arti "Ini" Diucapkan ketika ada benda yang ditunjukkan dalam jarak yang dekat dan jauh

Sore; Memiliki arti "Itu [ Dekat]" Diucapkan ketika ada benda yang ditunjukkan dalam jarak yang dekat

Are; Memiliki arti "Itu [Jauh]" Diucapkan ketika ada benda yang ditunjukkan dalam jarak yang jauh
