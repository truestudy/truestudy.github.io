1. Persamaan kuadrat jika diketahui akar akarnya adalah -2 dan 1/2 adalah.. A. 2x² - 5x+ 2 = 0 B. 2x² + 5x - 2 = 0 C. 2x² - 5x - 2 = 0 D. 2x² + 5x + 2 = 0


x1 = -2

x2 = 1/2

x1 + x2

= -2 + 1/2

= -4/2 + 1/2

= -3/2

x1 . x2

= (-2) x (1/2)

= -1

Pesamaan kuadrat

x² - (x1 + x2)x + (x1.x2) = 0

x² - (-3/2)x + (-1) = 0

x² + 3/2 x - 1 = 0

Kedua ruas dikali 2

2x² + 3x - 2 = 0

Jawaban:

2x² + 3x - 2 = 0

Penjelasan dengan langkah-langkah:

Misalkan x1 = -2 dan x2 = ½

(x - x1)(x - x2) = 0

(x - (-2))(x - ½) = 0

(x + 2)(x - ½) = 0

x² - ½x + 2x - 1 = 0

[tex] {x}^{2} - \frac{1}{2} x + \frac{4}{2} x - 1 = 0[/tex]

[tex] {x}^{2} + \frac{3}{2} x - 1 = 0[/tex]

[tex]2 {x}^{2} + 3x - 2 = 0[/tex]

Jawabannya tidak ada di pilihan ganda.

Jika jawaban persamaannya ada di pilihan ganda, maka akar akarnya adalah -2 dan -½, bukan -2 dan ½

Diketahui persamaan kuadrat 2x² - 3x - 1 = 0 mempunyai akar akar x1 dan x2. Persamaan kuadrat baru yang akar akarnya x1 - 2 dan x2 - 2 adalah 2x² + 5x + 3 =0

PEMBAHASAN

Diketahui :

Persamaan kuadrat 2x² - 3x - 1 = 0 mempunyai akar akar x1 dan x2

Ditanya : Persamaan kuadrat baru yang akar akarnya x1 - 2 dan x2 - 2 ?

Jawab :

ax² - bx - c = 0

2x² - 3x - 1 = 0

a = 2

b = -3

c = -1

x1 + x2

= -b / a

= 3/2

x1 . x2

= c / a

= -1/2

Akar baru :

2x1 - 2 = a

2x2 - 2 = b

Masukkan x1 + x2 = 3/2

a + b = ( x1 - 2) + (x2 - 2)

a + b = (x1 + x2) - 4

a + b = 3/2 - 4

a + b = -5/2

Masukkan x1. x2 = -1/2

a . b = ( x1 - 2) (x2 - 2)

a . b = (x1 . x2) - 2.x1 - 2.x2 + 4

a . b = -1/2 - 2(x1+x2) + 4

a . b = -1/2 - 2(3/2) + 4

a . b = -1/2 - 3 + 4

a . b = 3/2

Masukkan a + b = -5/2 dan a . b = 3/2 ke bentuk persamaan umum :

= x² - (a + b)x - a . b = 0

= x² - 5/2x - 3/2 = 0 (dikali 2)

= 2x² + 5x +3 = 0
