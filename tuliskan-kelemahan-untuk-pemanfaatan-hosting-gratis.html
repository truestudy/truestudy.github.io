6. perbedaan hosting gratis pada wordpress.com dengan hosting gratis pada 000webhostapp


Jawaban:

WordPress.com menyediakan layanan hosting gratis yang terbatas. Pada hosting gratis di WordPress.com, pengguna hanya bisa memiliki satu website dengan nama domain yang bersifat subdomain (nama website.wordpress.com). Pengguna juga tidak diizinkan untuk menggunakan plugin atau mengedit file PHP. Selain itu, ada iklan yang ditampilkan pada website yang dibuat di WordPress.com.

Sementara itu, 000webhostapp juga menyediakan layanan hosting gratis yang terbatas. Namun, pada hosting gratis di 000webhostapp, pengguna bisa memiliki lebih dari satu website dengan nama domain gratis yang bersifat subdomain (nama.000webhostapp.com). Pengguna juga diizinkan untuk menggunakan plugin dan mengedit file PHP. Namun, ada batasan penggunaan bandwith dan storage yang diberikan pada hosting gratis di 000webhostapp. Selain itu, tidak ada iklan yang ditampilkan pada website yang dibuat di 000webhostapp.

Penjelasan:

WordPress.com dan 000webhostapp merupakan layanan hosting yang menyediakan fitur hosting gratis. Namun, ada beberapa perbedaan yang terdapat pada layanan hosting gratis di kedua platform tersebut, seperti jumlah website yang bisa dibuat, nama domain yang digunakan, keizinan untuk menggunakan plugin dan mengedit file PHP, serta ada atau tidaknya iklan yang ditampilkan.


7. Keunggulan dan kelemahan Dedicated Hosting Server yaitu :


Kelebihan:
1. Akses Penuh 
Dedicated server memeberikan kebebasan kepada penyewa untuk memiliki hak akses yang dimiliki 100%. Menariknya lagi adalah Dedicated Server juga akan memudahkan dan membebaskan penggunanya untuk melakukan konfigurasi berdasarkan bentuk spesifikasi yang dibutuhkan.

2. Aman 
Kelebihan dari penggunaan Dedicated Server selanjutnya adalah apabila terjadi kerusakan pada server, Anda tidak bertanggung jawab pada kerusakan tersebut. Sehingga sangat aman bagi Anda untuk mengoperasikan Dedicated Server tersebut.

3. Efektif, Bisa Menggunakan Remote 
Kelebihan lainnya adalah penyewa bisa mengontrol server menggunakan remote saja. Anda tidak perlu melakukan pengawasan pada fisik server secara langsung. Sehingga tidak akan merepotkan atau membebani Anda selama mengelola website.

4. Tidak Terganggu Aktifitas Lain 
Ketika Anda menggunakan Dedicated Server, Anda tidak perlu merasa terganggu dengan klien yang lain. Mengapa seperti itu? Sebab hosting dengan jenis Dedicated Server hanya digunakan oleh 1 orang pengguna saja. Jadi, tidak akan mengganggu kinerja website apabila sewaktu-waktu jumlah pengunjungnya meningkat. Hosting ini juga memungkinkan penggunanya untuk mengelola banyak domain pada satu server yang dimiliki. Dimana manajemen website juga akan semakin maksimal karena Anda lah yang mempunyai kontrol dan kuasa penuh terhadap server tersebut.

Kekurangan:
1. Harganya Mahal 
Kekurangan yang dimiliki oleh hosting dengan tipe Dedicated Server adalah biaya yang dibutuhkan melakukan masuk ke dalam kategori mahal. Semakin tinggi spesifikasi server yang diinginkan maka semakin mahal pula harga yang harus dibayarkan. Berbeda dengan share hosting yang memungkinkan Anda untuk membayar sewa bersama dengan pengguna lainnya.

2. Skill yang Advanced 
Kekurangan lain dari penggunakan Dedicated Server adalah penyewa harus memahami kinerja dan cara kontrol server. Hal tersebut sangat penting agar Anda bisa mengelola dedicated server dengan maksimal. Apalagi, ada cukup banyak pengaturan di berbagai aplikasi yang harus dilakukan. Sehingga Anda harus benar-benar mempelajari pengelolaan data menggunakan dedicated server sebelum menyewanya.

8. Tuliskan beberapa layanan hosting yang ada dan berikan pengertiannya !


Pertanyaan

Tuliskan beberapa layanan hosting yang ada dan berikan pengertiannya !

Jawaban:

Beberapa layanan hosting adalah sbb:

A. Domain Hosting

B. Web/Application Hosting, umumnya terdiri dari;

a. Shared Hosting

b. VPS (Virtual Private Server) Hosting

c. Cloud Hosting

d. Dedicated Hosting

Penjelasan:

A. Domain hosting adalah layanan untuk layanan tempat untuk nama identitas seseorang atau sebuah kelompok (organisasi, komunitas, perusahaan dsb) agar dapat diakses dalam jaringan internet.

B. Web Hosting adalah layanan server online untuk menempatkan file file website/aplikasi web agar dapat melayani permintaan akses dari jaringan internet.

contoh: sebuah sekolah membuat layanan berbasis internet dengan membuat sebuah website dan aplikasi portal layanan sekolah untuk guru, siswa dan orang tua murid.

Sekolah harus menyewa layanan web/application hosting untuk dapat meletakkan file file website dan aplikasi di server online web/application hosting. (jadi fungsinya seperti rumah dalam jaringan internet)

Sekolah juga harus membuat sebuah nama yang unik (misalnya: sekolahterbaik.ac.id) dan menyewa sebuah domain hosting agar nama tersebut bisa dipublikasikan ke jaringan global. Layanan domain hosting lalu mengarahkan nama sekolahterbaik.ac.id tadi agar langsung mengakses file di server web/application hosting saat nama sekolahterbaik.ac.id tadi diakses di jaringan internet. (jadi fungsinya seperti alamat di  jaringan internet)

Jenis Layanan Web/Application Hosting;

a. Shared Hosting

Layanan termurah karena provider nya menyediakan sebuah server yang membagi satu buah server menjadi beberapa lingkungan virtual untuk kisaran 10-20 pelanggan. Harga paling murah, namun untuk kecepatan tingkatnya yang paling rendah.

b. VPS (Virtual Private Server) Hosting

Layanannya mirip dengan Shared hosting namun satu server hanya dibagi menjadi maksimal 3 lingkungan virtual untuk tiga pelanggan.  Harga sedikit lebih mahal, namaun kecepatan dan kehandalan lebih tinggi.

c. Cloud Hosting

Layanannya mirip dengan VPS, namun server yang dibagi tersebut memiliki server paralel yang sangat banyak dalam sebuah jaringan cloud  (jaringan komputer di beberapa data center yang dikelola provider web/application hosting). Jadi kemampuan komputasi server menjadi lebih tinggi.

Biayanya diantara shared hosting dan VPS. Dan ini yang sangat populer saat ini.

d. Dedicated Hosting

Layanan web/application hosting yang memperuntukkan satu server untuk satu pelanggan. Ada 2 metodenya, yaitu provider menyediakan servernya untuk pelanggan atau pelanggan yang menitipkan server fisiknya ke datacenter provider (atau biasa disebut co-location).

Layanan ini paling mahal namun paling aman dan handal karena tidak bercampur dengan pelanggan lainnya.

29. Tuliskan yang kamu ketahui tentang control panel hosting dan fungsinya !


Jawaban:

Control panel hosting adalah tool yang akan membuat Anda dapat melakukan manajemen seluruh aspek layanan hosting. Melalui hosting control panel, Anda bisa melakukan beragam pengaturan pada sistem admin yang kompleks dengan hanya melakukan beberapa klik pada interface admin.

Fungsi Control Panel Hosting

1. Manajemen Domain

2. Menginstal aplikasi dengan mudah

3. Melakukan FTP

4. Manajemen File

5. Manajemen Email

6. Manajemen Database

7. Backup Data

Penjelasan:

3. Melakukan FTP

FTP adalah proses transfer file dari komputer ke server website Anda. FTP juga bisa membantu Anda melihat dan mengatur file/folder pada website Anda. Akan tetapi, sebelum bisa melakukan FTP, Anda harus mengaturnya dahulu di control panel hosting.

4. Manajemen File

Selain melihat file/folder melalui FTP, Anda juga bisa melakukannya langsung melalui control panel. Sebab, ada beberapa hosting yang juga menyediakan fitur ini. Walaupun tidak selengkap FTP, tapi fitur ini tentu saja membantu jika Anda hanya ingin melakukan pengecekan.
