12. Jika massa rata-rata I atom silikon adalah 1.99 x 10-24 kg, berapa latom C-12 (ArSı:28) :​


Jawaban:

Jadi, massa atom silikon (Ar=28) 5.56 x 10^-21 kali lebih besar dari massa atom karbon-12 (Ar=12).

Penjelasan:

Untuk menjawab pertanyaan ini, kita perlu menggunakan perbandingan massa relatif (Ar) dan konversi massa atom ke dalam kilogram.

Diketahui:

Massa rata-rata 1 atom silikon = 1.99 x 10^-24 kg

Ar silikon = 28

Ar karbon-12 = 12

Dicari: Berapa kali massa atom silikon (Ar=28) dibandingkan dengan massa atom karbon-12 (Ar=12)

Langkah penyelesaian:

Hitung massa atom silikon dalam kg dengan membagi massa rata-rata 1 atom silikon dengan konstanta Avogadro:

1 atom Si = 1.99 x 10^-24 kg / 6.022 x 10^23 = 3.31 x 10^-47 kg

Hitung massa atom karbon-12 dalam kg dengan mengalikan massa 1 atom C-12 dengan konstanta massa 1 u (1 u = 1.66 x 10^-27 kg):

1 atom C-12 = 12 u x 1.66 x 10^-27 kg/u = 1.99 x 10^-26 kg

Hitung perbandingan massa atom silikon (Ar=28) dengan massa atom karbon-12 (Ar=12):

Perbandingan massa = (Ar Si / Ar C-12) x (massa atom C-12 / massa atom Si)

Perbandingan massa = (28 / 12) x (1.99 x 10^-26 kg / 3.31 x 10^-47 kg)

Perbandingan massa = 5.56 x 10^-21
