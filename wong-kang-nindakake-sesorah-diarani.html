Jawaban:

Sesorah pangajak

Penjelasan:

Miturut ancase sesorah, yaiku:

1) Atur pambagyaharja, yaiku sesorah kanggo nampa rawuhipun para tamu ing adicara apa wae. Tuladha: ing pahargyan (pista), sukuran, lan sapanunggalane.

2) Menehi informasi utawa pawarta, yaiku informasi bisa arupa palapuran utawa sawijining bab kanthi gamblang/cetha. Tuladha: sesorah ing rapat, promosi barang, lan sapanunggalane.

3) Atur panglipur aweh hiburan, yaiku sesorah iki nduweni ancas wong kang mirengake seneng penggalihe. Tuladha: ing acara layatan, atur saking pambelasungkawa.

4) Pangajak, ancase sesorah iki kanggo milut atine wong saengga gelem nindakake apa kang diwedharake. Tuladha: penyuluhan kesehatan, penyuluhan pendhidhikan, lan sapanunggalane.

21. Apa kang kudu digatekake nalika nindakake sesorah


Kang kudu digatekake nalika nindakake sesorah

Nyiapake underan / tema sesorah.Nyiutake tema dadi irah-irahan (judul).Neliti kahanan pamiarsa.Ndamel cengkorongan / kerangka sesorah.Nemtokake ancase / tujuane sesorah.Ngempalake bahan sesorah.Ngembangake / ngrembaakake cengkorongan.Gladhen sesorah.Penjelasan :

Sesorah(pidato) yaiku micara/matur/nglairake gagasan sarana lisan ing sangarepe wong akeh kanthi ancas tinamtu.

Artinya:

Pidato adalah berbicara secara lisan di depan orang banyak untuk maksud atau tujuan tertentu.

Kang kudu digatekake nalika nindakake sesorah

Nyiapake underan / tema sesorah. Wong sing bade sesorah biasane kudu gawe tema wiwit awal, opo kang dadi tujuan saka sesorah utawa pidato.Nyiutake tema dadi irah-irahan (judul).Neliti kahanan pamiarsa. neliti kahanan pamiarsa sing dimaksud kudu ngerti apa sing cocok karo tema sing digawe.Ndamel cengkorongan / kerangka sesorah. Tujuane ngeringkes bab-bab sing bade diutarakna maring sesorah.Nentokake ancase / tujuane sesorah. bagian paling penting sesorah yaiku kudu bisa nentokake tujuane sesorah supaya bisa diterima pamiarsa.Ngempalake bahan sesorah. Sakdurunge dadi sesorah tahap awal yaiku ngumpulake bahan lan sakwisae iku didadekna kerangka sesorah.Ngembangake / ngrembaakake cengkorongan. wong sing bade sesorah kudu bisa ngembangake bab-bab kang wis cepak dadi kerangka sesorah.Gladhen sesorah. tahap kang paling akhir yaiku gladen utawa latihan wiwit awal nganti akhir kanggo neliti apa kurange sing wis dadi sesorah.

26. 2. Sesorah supaya wong kang ngrungokake gelem nindakake apa kang dikarepakediarani ancas3. Metode pidhato kanthi nggawa cathetan kang isine bab-bab wigatine sesorahdiarani​


Wangsulan

2. Ancas atur pangajak.

3. Methode ekstemporan.

Penjelasan

2. Atur pangajak yaiku ancas sesorah sing tujuwane kanggo ngajak pamirsa supaya gelem nindakake apa sing dikarepake saka isi sesorah.

3. Methode pidhato/sesorah :

→ Methode apalan

= Yaiku teknik utawa sistem pidhato/ sesorah kang dilakoke nganggo cara ngapalake teks pidhato.

→ Methode naskah/teks

= Yaiku tektik utawa sistem pidhato / sesorah kang dilakoke nganggo cara maca teks pidhato.

→ Methode ekstemporan

= Yaiku teknik utawa sistem sesorah kanthi nganggo catetan cilik kang isine pokok-pokok pidhato kang arep diaturke marang para pamirsa.

→ Methode impromtu

= Yaiku teknik sesorah kanthi dadakan utawa durung ngrancang cengkorongane sesorah sing arep diaturake marang pamirsa.
