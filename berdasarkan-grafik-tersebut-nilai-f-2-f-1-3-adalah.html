1. Berdasarkan grafik tersebut, nilai f(2) +f-1(-3) adalah...


Perhatikan grafik fungsi kuadrat berikut. Berdasarkan grafik tersebut , nilai f(2) + f⁻¹(-3) adalah ...

Fungsi kuadrat adalah fungsi dengan variabel berpangkat tertinggi dua

Fungsi invers adalah invers (lawan) yang berbentuk fungsi.

Gunakan konsep

[tex]\boxed{~f(x) = y ~~ --->~~ f^{-1} (y) = x~}[/tex]

Pembahasan

Untuk menjawab soal diatas kita tidak perlu buat rumus fungsi kuadrat dari titik yang dilaluinya dan mengubahnya jadi funsgsi invers.

Kita cukup gunakan konsep :  f(x) = y  --->  f⁻¹(y) = x

Grfik yang melalui titik (2, 3) yaitu x = 2 dan y = 3

maka f(x) = y adalah f(2) = 3

Grafik yang dilalui (-2, -3) dan (5, -3), maka

f(x) = y  --->  f⁻¹(y) = x

f(-2) = -3  →  f⁻¹(-3) = -2

f(5) = -3  →  f⁻¹(-3) = 5

Berdasarkan grafik nilai f(2) + f⁻¹(-3), diperoleh

f(2) + f⁻¹(-3) = 3 + (-2)

                           = 1

f(2) + f⁻¹(-3) = 3 + 5

                          = 8

Jadi berdasarkan grafik tersebut , nilai f(2) + f⁻¹(-3) adalah 1 atau 8

Pembahasan:

soal ini kurang dilengkapi gambar, saya coba melengkapinya dengan memberinya gambar, jika gambarnya seperti yg saya lampirkan ini maka penyelesaiannya sebagai berikut:

dari gambar kita ambil 2 titik sembarang, dalam hal ini saya ambil titik (6,2) dan (4,3)
dari dua titik yang kita ambil itu kita tentukan persamaan fungsi garis g
f(g) = ax + b
f(6) = 2
f(4) = 3

f(g) = ax + b
f(6) = 6a + b
f(6) = 2
6a + b = 2 ....(persamaan 1)

f(g) = ax + b
f(4) = 3
f(4) = 4a + b
4a + b = 3 ....(persamaan 2)

kita eliminasi persamaan 1 dan 2
6a + b = 2
4a + b = 3
--------------- -
2a   = -1
a = -1/2

subsitusi nilai a ke persamaan 1 atau persamaan 2,
dalam hal ini saya subsitusikan ke persamaan 2
4a + b = 3
4(-1/2) + b = 3
-2 + b = 3
b = 3 + 2
b = 5

sehingga kita dapati f(g) = -1/2 x + 5

kita cari f^-1(g)
-1/2 x + 5 = y
-1/2 x = y - 5
x = -2(y-5)
x = -2y + 10
f^-1(g) = -2x + 10

setelah mengetahui rumus fungsinya 
kita tinggal subsitusi ke soal
soal:
nilai f(2) + f^-1(-3)
=  -1/2 (2) + 5 +  -2(-3) + 10
=  -1 + 5 + 6 + 10
= 20
