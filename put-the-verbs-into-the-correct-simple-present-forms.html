Penjelasan :

Untuk bagian A kebanyakan kalimat nya bentuk positif (+), dan ada beberapa yang termasuk bentuk negatif (-). Adapun rumus yang digunakan yaitu :

(+) S + V2 + O

(-) S + did + not + V1 + O

Untuk kalimat positif (+) karena kata kerja yang digunakan adalah verb-2, maka kata kerja yang di dalam kurung harus di ubah ke bentuk verb-2. Dan berikut ini adalah perubahan kata kerja di atas :

Make => Made => Made

Go => Went => Gone

Color => Colored => Colored

Add => Added => Added

Finish => Finished => Finished

Look => Looked => Looked

Like => Liked => Liked

Hope => Hoped => Hoped

Sedangkan untuk kalimat negatif (-) karena sudah ada tanda "not" di dalam kurung nya, maka tinggal menambahkan auxiliary verb "did" di depan kata "not" nya.

Untuk bagian B ini kalimat nya ada yang termasuk simple past verbal dan ada yang simple past nominal.

Simple past verbal :

(+) S + V2 + O

(-) S + Did + not + V1 + O

Hal ini telah dijelaskan pada bagian A

Simple past nominal

(+) S + To be (was/were) + Complement

Adapun yang termasuk ke bentuk ini terdapat pada kalimat di nomor 1, 4, 21 dan 31.Semua nya menggunakan to be was dikarenakan subject nya I/it ataupun menggunakan singular noun "throat".
