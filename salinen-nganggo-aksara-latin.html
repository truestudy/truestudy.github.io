Koprasi Sekolah

Anggota koprasi sekolah iku siswa lan warga sekolah. Dadi anggota koprasi iku kudu ambayar simpenan wajib kang bisa dibayar saben sawulan sepisan. Jumlahe simpenan wajib ing saben koprasi sekolah beda-beda. Kejaba nyedhiyakake piranti sekolah kanggone siswa, koprasi sekolah uga nampa manawa ana siswa kang arep nyelèngi.

Reganè piranti sekolah kang didol ing koprasi luwih murah tinimbang ing toko jaban sekolah kaya dènè sesantine kang gawe mayare warga sekolah anggone nyukupi kabutuhan sekolah. Bab iku gajumbuh karo ancase kang gawe kasejahteraane anggone.

Aksara legena lan pasangan

Aksara legena yaiku aksara sing durung oleh sandhangan/pasangan. Dene pasangan yaiku aksara sing digunakake kanggo nyambung sesigeg ing tengah tembung. Cacahe padha² ana 20, yaiku:

ꦥ꧀ꦥ꧈ ꦝ꧀ꦝ꧈ ꦗ꧀ꦗ꧈ ꦪ꧀ꦪ꧈ ꦚ꧀ꦚ

ꦩ꧀ꦩ꧈ ꦒ꧀ꦒ꧈ ꦧ꧀ꦧ꧈ ꦛ꧀ꦛ꧈ ꦔ꧀ꦔ

Sandhangan

sandhangan swara (wulu, suku, taling, pepet, lan taling tarung)sandhangan panyigeg wanda (layar, wignyan, cecek, lan pangkon)sandhangan wyanjana (Cakra, Cakra keret, lan pengkal)

Penjelasan

Aksara Jawa sing cacahe ana 20 wis diterangake menawa awujud legena. Dene menawa supaya muni swara liyane, kudu diwenehi sandhangan. Sandhangan iku ana telung warna yaiku :

sandhangan swara => Wulu, suku, pepet, taling, taling tarung.sandhangan panyigeg wanda => Cecak, layar, wignyan.sandhangan wyanjana => Cakra, keret, pengkal.

- Tanda waca Aksara Jawa :

( ꧈ )Pada lingsa : pamacane mandheg sedhela (koma).

( ꧉ ) Pada lungsi : ukarane wis rampung. (titik)

( ꧇ ) Pada pangkat : tembung-tembung ing mburine wigati banget.

( ꧋ ) Adeg-adeg (ada-ada) : kanggo miwiti ukara.

Babagan aksara puniku, paling utamane kudu mangerteni aksara legena, sandhangan, lan pasangan. Saliyane iku, wacan mawa aksara Jawa ing dhuwur uga nggunakake angka Jawa.

=====================

Aksara legena

Yaiku aksara sing durung oleh pasangan/sandhangan. Cacahe ana 20, yaiku:

ꦲ꧈ ꦤ꧈ ꦕ꧈ ꦫ꧈ ꦏꦢ꧈ ꦠ꧈ ꦱ꧈ ꦮ꧈ ꦭꦥ꧈ ꦝ꧈ ꦗ꧈ ꦪ꧈ ꦚꦩ꧈ ꦒ꧈ ꦧ꧈ ꦛ꧈ ꦔ

Pasangan aksara

Yaiku kanggo sesambung manawa ana sesigeg ing satengahing tembung. Cacahe uga 20, yaiku:

꧀ꦲ꧈ ꧀ꦤ꧈ ꧀ꦕ ꧈꧀ꦫ ꧈ ꧀ꦏ ꧀ꦢ꧈ ꧀ꦠ꧈ ꧀ꦱ꧈ ꧀ꦮ꧈ ꧀ꦭ ꧀ꦥ꧈ ꧀ꦝ꧈ ꧀ꦗ꧈ ꧀ꦪ꧈ ꧀ꦚ ꧀ꦩ꧈ ꧀ꦒ꧈ ꧀ꦧ꧈ ꧀ꦛ꧈ ꧀ꦔ

Sandhangan

Ana 3 jinise, yaiku:

sandhangan swara → wulu, suku, pepet, taling, lan taling tarungsandhangan panyigeg wanda → layar, wignyan, cecek, lan pangkonsandhangan wyanjana → cakra, keret, lan pengkal

Angka Jawa

Perlu digatekake, nalika nulis angka Jawa iku kudu diapit nganggo pada pangkat ( ꧇ ꧇ ). Angka² Jawa yaiku:

Angka 0 = ꧐Angka 1 = ꧑Angka 2 = ꧒Angka 3 = ꧓Angka 4 = ꧔Angka 5 = ꧕Angka 6 = ꧖Angka 7 = ꧗Angka 8 = ꧘Angka 9 = ꧙
