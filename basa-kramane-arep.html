3. apa basa kramane,ibu arep Maring pasar?​


Ibu arep Maring pasar?

➡️Basa krama : Ibu badhe dhateng peken.

Pembahasan :

➡️ Unggah ungguh basa ana 2 :

Basa ngokoBasa Krama

➡️Basa ngoko ana 2 :

Ngoko luguNgoko alus

➡️1. Ngoko lugu yaiku basa sing tembung-tembunge nggawe basa Ngoko lugu kabeh.

2. Ngoko alus yaiku basa sing tembung-tembunge wis kecampuran antara basa Ngoko Karo basa Krama.

➡️Basa Krama ana 2 :

Krama luguKrama alus

➡️1. Krama lugu yaiku basa sing tembung-tembunge nggawe basa Krama lugu kabeh.

2. Krama alus yaiku basa sing tembung-tembunge nggawe basa Krama Alus kabeh tumrap wong sing diajak guneman.

➡️Ibu arep Maring pasar

arep = Ngoko lugubadhe = Krama AlusMaring = Ngoko lugudhateng = Krama Aluspasar = Ngoko lugupeken = Krama Alus

15. piro tamune sing arep teko basa kramane​


pira tamune sing arep teka? basa kramane yaiku pinten tamune ingkang badhe rawuh?

Pembahasan :

Basa krama/krama alus/krama inggil yaiku basa Jawa sing paling sopan. Basa iki digawe ngomong marang wong kang luwih tuwa. Kosok baline basa krama/krama alus/krama Inggil yaiku basa ngoko/ngoko lugu. Basa ngoko/ngoko lugu biasa digawe marang kanca cedhak lan wong kang luwih enom.
