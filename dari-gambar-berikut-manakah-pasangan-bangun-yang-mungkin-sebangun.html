21. Bantu jawab kak. Dari gambar berikut, manakah pasangan bangun yang mungkin sebangun? ​


Pasangan bangun yang mungkin sebangun adalah bangun (a), (b), dan (d).

Bangun (c) tidak sebangun karena sudut yang bersesuaian tidak sama besar.

Pembahasan

SEBANGUN

Dua buah bangun datar akan dikatakan sebangun jika semua sudut yang seletak sama besar dan sisi-sisi yang bersesuaian memiliki perbandingan.

Jadi dua benda sebangun jika bentuk kedua benda sama tetapi ukurannya berbeda.

Contoh benda yang sebangun:

Sisi muka pintu dengan bagian depan buku tulisLantai keramik dengan sebuah sisi dadu

Untuk menentukan dua buah bangun datar sebangun atau tidak, periksa sudut pada bangun datarnya. Sudut yang seletak harus sama besar.

Diketahui:

Namakan semua bangun datar di sisi kiri sebagai ABCD dan yang di sisi kanan sebagai PQRS.

a. Bangun ABCD

∠ A = 71°∠ B = 109°∠ C = 71°∠ D = 109°

Bangun PQRS

∠ P = 71°∠ Q = 109°∠ R = 71°∠ S = 109°

b. Bangun ABCD

∠ A = 90°∠ B = 82°∠ C = 75°∠ D = 113°

Bangun PQRS

∠ P = 82°∠ Q = 75°∠ R = 113°∠ S = 90°

c. Bangun ABCD

∠ A = 60°∠ B = 120°∠ C = 70°∠ D = 110°

Bangun PQRS

∠ P = 120°∠ Q = 70°∠ R = 60°∠ S = 110°

d. Bangun ABCD

∠ A = 53°∠ B = 90°∠ C = 90°∠ D = 127°

Bangun PQRS

∠ P = 90°∠ Q = 90°∠ R = 127°∠ S = 53°

Ditanyakan

Mana yang sebangun?

Penjelasan

a. Lihat sudut yang bersesuaian

∠ A = ∠ P ∠ B = ∠ Q∠ C = ∠ R∠ D = ∠ S

Bangun ABCD bersesuaian dengan PQRS.

Maka ABCD sebangun dengan PQRS.

b. Lihat sudut yang bersesuaian

∠ A = ∠ S ∠ B = ∠ P∠ C = ∠ Q∠ D = ∠ R

Bangun ABCD bersesuaian dengan SPQR.

SPQR sama dengan PQRS (perhatikan urutannya).

Maka ABCD sebangun dengan PQRS.

c. Lihat sudut yang bersesuaian

∠ A = ∠ R∠ B = ∠ P∠ C = ∠ Q∠ D = ∠ S

Bangun ABCD bersesuaian dengan RPQS.

Tapi RPQS berbeda dengan PQRS (perhatikan urutannya RPQS = PQSR).

Maka bangun ABCD tidak sebangun dengan PQRS.

d. Lihat sudut yang bersesuaian

∠ A = ∠ S ∠ B = ∠ P∠ C = ∠ Q∠ D = ∠ R

Bangun ABCD bersesuaian dengan SPQR.

SPQR sama dengan PQRS.

Maka ABCD sebangun dengan PQRS.

Pasangan bangun yang mungkin sebangun adalah bangun (a), (b), dan (d).
