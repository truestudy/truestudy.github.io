9. sebutkan Langkah Langkah Setting Jaringan Lan Untuk 4 Unit Client!


Ada beberapa hardware yang harus disiapkan sebelum membuat jaringan beberapa diantaranya

1. switch/router
2. kabel UTP
3. Lan card
4. peralatan lain untuk menyambungkan co: tang crimping, lan tester, komputer, komputer client,dll...
cara2

1. ada baiknya mempelajari terlebih dahulu teknik pemasangan jaringan misalnya teknik crimping,dll
2.saat membeli router, ada baiknya jika anda memperhatikan fitur fiturnya terlebih dahulu. beberapa router kini ada yang mempunyai sistem plug n play sehingga tidak diperlukan konfigurasi
3. pemilihan router ini juga bermanfaat bagi anda yang ingin melakukan fitur tambahan misalnya untuk membagi bandwidth tiap client, memfungsikan nya sebagai firewall, hotspot, dan lainnya

saat setting

1. lakukan crimping kable utp straight. kabel utp straight berfungsi untuk menghubungkan device yang berbeda antarmukanya misal nya laptop ke router.

urutan kabel UTP straight
oranye-putih/oranye/hijau-putih/biru/biru-putih/hijau/coklat-putih/coklat

ujung lainnya samakan

2.lakukan test terlebih dahulu pada kabel menggunakan LAN tester
jika lampu pada LAN tester menyala berurutan maka anda sudah melakukan crimping dengan benar

jika tidak sesuai maka anda harus melakukan crimping ulang karena label tersebut tidak dapat tersambung ke jaringan karena ada kabel yang putus

pastikan kabel tersebut berada menempel di ujung konektor RJ 45. jika tidak kabel akan mudah lepas karena tertarik dan sebagainya

3. install network adapter di komputer client. beberapa adaptor sudah built-in pada komputer client sehingga tidak perlu meng-install adapter lain

4. lakukan setting pada router. beberapa yang umum diantaranya yaitu DHCP,Nat,WAN,dsb

jika anda ingin memakai IP statis, anda dapat mengubah setting ke dhcp statis atau mendisable dhcp

beberapa router memang berbeda konfigurasi nya. jadi perhatikan baik baik

5. setting juga computer client yang mengakses internet. yang umum diantaranya : IP address, DNS,Subnet,dll

6. lakukan test menyeluruh terhadap jaringan (transfer data,Ping, internet,dll)

semoga membantu

15. Langkah langkah setting printer


1. Lihat Komputer yang terpasang printer (USB Conect).
2. Pada komputer tersebut Printer harus sudah dapat di sharing dengan komputer lain.
3. Lihat nama net bios pada komputer tersebut dengan cara :
4. Klik Star –) My Computer –) klik kanan –)Properties
5. Maka akan keluar Menu System Properties.
6. Klik Computer Name.
7. Lihat Computer description (Contoh : Ully)
8. Maka Komputer tersebut memiliki net bios (Contoh : Ully)Penyetingan untuk komputer yang akan di Sharing :
1. Klik Windows Explorer dengan cara simpan panah mouse di start klik kanan pilih explore
2. Setelah keluar Windows Explorer ketik di addreess nama net bios komputer yang di Sharing:
( \\netbios: Contoh: \\ully ).
3. Maka akan muncul nama printer di sebelah kanan tampilan.
4. Klik printer yang akan dipakai ( Contoh: Printer2 )
5. Maka akan muncul tampilan
6. Klik Yes, maka akan keluar:
7. Set Printer sebagai Default Printer .
8. Maka Printer dapat di pakai di Komputer anda.

21. Tuliskan langkah-langkah menggunakan fitur hotspot pada smartphone, serta cara pengoperasiannya.​


Mengaktifkan hotspot =

-Buka aplikasi setelan.

-Ketuk jaringan lalu ketuk hotspot seluler.

-Aktifkan hotspot.

-Ketuk untuk melihat atau mengubah setelan hotspot, seperti sandi atau nama.

Menyambungkan hotspot

-Tekan Wi-Fi.

-Cari hotspot sesuai nama.

-Masukkan kata sandi dengan tepat

-Tersambung.

Jawaban:

# Mengaktifkan hotspot:

Geser layar dari atas ke bawah.

Ketuk Hotspot Hotspot Wi-Fi.

Jika tidak menemukan Hotspot Hotspot Wi-Fi, di kiri bawah, ketuk Edit Edit dan tarik Hotspot Hotspot Wi-Fi ke dalam Set

# menghubungkan perangkat lain ke hotspot ponsel:

Di perangkat lain, buka daftar opsi Wi-Fi perangkat tersebut.

Pilih nama hotspot ponsel Anda.

Masukkan sandi hotspot ponsel Anda.

Klik Hubungkan.

Jika Anda tidak ingin hotspot memerlukan sandi:

Geser layar dari atas ke bawah.

Sentuh lama Hotspot Hotspot Wi-Fi.

Di bagian "Keamanan", ketuk Tidak ada.

Tips: Anda dapat berbagi data seluler ponsel dengan maksimal 10 perangkat lain melalui hotspot Wi-Fi.

#Memperpanjang masa pakai baterai saat tethering : Colokkan perangkat Anda ke sumber listrik saat tethering.

Nonaktifkan sambungan tethering setelah selesai digunakan.

Jika ponsel Anda memiliki setelan untuk menonaktifkan hotspot saat tidak ada perangkat yang tersambung, aktifkan setelan tersebut.

24. langkah-langkah setting bios dalam hardisk


nyalahkan komputer ter lebih dahulu(power)
tekan tombol f2 f1 delet f10 dan seterusnya lalu kamu masuk ke bios lalu masuk ke booting tergantung bios nya
lalau pilih dvd room1. Tekan tombol power pada PC dan monitor untuk menghidupkan komputer anda. PC akan segera melakukn proses booting.
2. Segera menekan tombol (DELETE) pada keyboard untu masuk kemenu BIOS/CMOS setup.
3. Anda akan masuk BIOS, hal pertama yang harus anda lakukan adaah mensetting waktu yang akan digunakan oleh sistem komputer. Aturlah melalui menu (main.system Time) untuk mengubahnya tekan tombol (+)(-) dan tombol (TAB) pada keyboard anda.
4. Selanjutnya dengan cara yang sama, ubahlah tanggal pada sstem komputer anda, melalui system (date).
5. Setelah ensetting waktu dan tanggal dari sistem, jika mau anda juga dapat memsang password BIOS agar orang lain tidak mendapat mengubah setelan BIOS yang anda buat, caranya pilih "SUPERVISOR PASSWORD", lalu tekan tombol (+) pada keyboard, selanjutnya masukan password anda books enter password, lalu klik (ENTER), masukan kembali password anda pada books confirm password, kemudian klik (ENTER), sekarang status opsi SUPERVISOR PASSWORD sudah menjadi enabled.
6. Bila PC anda dipakai rame- rame, anda juga mensetting user  password untuk masing - masing pengguna. Manfaatkan saja menu (USER PASSWORD). cara pengaturannya sama saja seperti pengaturan pada SUPERVISOR PASSWORD.
7. sekarang bukalah menu (Advanced) dengan menekan tombol (++) (Arah Panah Kekanan) pada keyboard anda, lalu pilih [Device Configuration] kemudian Enter.
8. Disini anda bisa menggunakan fitur ONBoard, pada mother board, anda misalnya, bila anda menggunakan kartu suara dan modem yang bukan ONBoard, ubahlah opsi ON [on - board AC97 Audio controller] dan [onboard AC97 modem controller] dengan menekan tombol (+) pada keyboard anda, selanjutnya klik (ESC) untuk kembali ke Menu (Advanced).
9. Sekarang pilih opsi (OCI Configuration) lalu tekan enter, kemudian agar slot USB andaberfungsi, set opsi (USB Function) menjadi [Enabled] jika anda belum bisa mengubahnya dengan menekan tombol (+) jika sudah kembali ke Menu [Advanced] dengan menekan tombol (ESC).
10. Selanjutnya dengan menu (BOOT), Untuk memudahkan sistem penginstalan sistem operasi yang akan dilakukan setelah ini, ubah pengaturan Boot dari PC. Kemudian set CDROOM sebagai boot device pertama, atau jika anda ingin melakukan penginstalan dari "Flashdisk" ganti CDROOM dengan nama flashdisk.

*Semoga Berhasil :)

28. langkah langkah perintah untuk setting pada bios dari awal hingga akhir​


Jawaban:

Saat sistem operasi dijalankan bios (Basic input output system) yang pertama kali dijalankan. BIOS akan mengecek dulu processor,lalu harddisk, RAM (random acess memory) dan ROM (read only memory),dan yang terakhir semua konektor yang terhubung. Jika terdengar suara "BEEP" sekali maka tidak ada masalah dan komputer siap digunakan. Jika terdengar suara "BEEP" berkali-kali maka ada konektor yang bermasalah dengan arti yang berbeda beda.
