1. Suatu larutan mengandung 50,0 g karbon tetraklorida (ccl4) dan 50,0 g kloroform (chcl3) hitunglah fraksi mol setiap komponen dalam larutan


Fraksi mol karbon tetraklorida CCl₄ adalah 0,437

Fraksi mol kloroform CHCl₃ adalah 0,563

Pembahasan

[tex]\boxed{\boxed{\bold{Stoikiometri}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus,kadar, konsentrasi, rumus empiris, rumus molekul, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

Fraksi mol (X)

Fraksi mol menyatakan perbandingan mol zat terlarut atau zat pelarut dengan jumlah mol campuran (terlarut dan pelarut).

Rumus :

[tex]\boxed{\boxed{\bold{Xt~=~\frac{nt}{nt~+~np}}}}[/tex]

atau          

[tex]\boxed{\boxed{\bold{Xp~=~\frac{np}{nt~+~np}}}}[/tex]

Xt  +  Xp  =   1

Dengan :  

Xt = fraksi mol zat terlarut                              

Xp = fraksi mol zat pelarut

nt = mol zat terlarut                                          

np = mol zat pelarut  

[tex]\boxed{\boxed{\bold{Diket}}}[/tex]  

massa CCl₄ = 50 gram

massa CHCl₃ = 50 gram

Ar C = 12

Ar H = 1

Ar Cl = 35,5

[tex]\boxed{\boxed{\bold{Ditanya}}}[/tex]  

Fraksi mol masing - masing

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]  

LANGKAH PERTAMA

Menentukan Mr kloroform (CHCl₃)

Mr CHCl₃ = 1. Ar C + 1. Ar H + 3. Ar Cl

Mr CHCl₃ = 1. 12 + 1. 1 + 3. 35,5

Mr CHCl₃ = 12 + 1 + 106,5

Mr CHCl₃ = 119,5

LANGKAH KEDUA

Menentukan jumlah mol (n) dari kloroform (CHCl₃)

Untuk menentukannya kita masih menggunakan rumus yang sama dengan yang diatas :

[tex]\boxed{\boxed{\bold{n~=~\frac{a}{Mr} }}}[/tex]

maka,

n CHCl₃ =  [tex]\frac{a}{Mr}[/tex]

n CHCl₃ =  [tex]\frac{50}{119,5}[/tex]

n CHCl₃ = 0,41841004184100418410041841004184

n CHCl₃ = 0,418 mol

LANGKAH KETIGA

Menentukan fraksi mol (X) dari kloroform (CHCl₃)

[tex]\boxed{\boxed{\bold{X~CHCl_3~=~\frac{n_{CHCl_3}}{n_{CCl_4}~+~n_{CHCl_3}}}}}[/tex]

maka,

X CHCl₃ = [tex]\frac{n_{CHCl_3}}{n_{CCl_4}~+~n_{CHCl_3}}[/tex]

X CHCl₃ =  [tex]\frac{0,418}{0,325~+~0,418}[/tex]

X CHCl₃ = [tex]\frac{0,418}{0,743}[/tex]

X CHCl₃ = 0,5625841184387617765814266487214

X CHCl₃ = 0,563

LANGKAH KEEMPAT

Menentukan Mr karbon tetraklorida (CCl₄)

Mr CCl₄ = 1. Ar C + 4. Ar Cl

Mr CCl₄ = 1. 12 + 4. 35,5

Mr CCl₄ = 12 + 142

Mr CCl₄ = 154

LANGKAH KELIMA

Menentukan jumlah mol (n) dari karbon tetraklorida (CCl₄)

n = jumlah mol (mol)  

a = massa (gram)

n CCl₄ = [tex]\frac{50}{154}[/tex]

n CCl₄ = 0,32467532467532467532467532467532

n CCl₄ = 0,325 mol

LANGKAH KEENAM

Menentukan fraksi mol (X) dari karbon tetraklorida (CCl₄)

[tex]\boxed{\boxed{\bold{X~CCl_4~=~\frac{n_{CCl_4}}{n_{CCl_4}~+~n_{CHCl_3}}}}}[/tex]

maka,

X CCl₄ = [tex]\frac{n_{CCl_4}}{n_{CCl_4}~+~n_{CHCl_3}}[/tex]

X CCl₄ = [tex]\frac{0,325}{0,325~+~0,418}[/tex]

X CCl₄ = [tex]\frac{0,325}{0,743}[/tex]

X CCl₄ = 0,4374158815612382234185733512786

X CCl₄ = 0,437

[tex]\boxed{\boxed{\bold{Kesimpulan}}}[/tex]

6. Tekanan uap campuran yang mengandung 400 g karbon tetraklorida dan 43,3 g zat yang tidak diketahui adalah 137 torr pada 300C. Tekanan uap karbon tetraklorida murni pada 300C adalah 143 torr , sedangkan untuk zat yang tidak ketahui 85 torr. Berapa kira-kira massa molekul zat tersebut ​


Jawaban:

Dikoreksi yaa kalo salah

disini ubah dulu ya torr ke mmHg, torr = mmHg

rumus yang akan dipake

P = Xp . Po

P = Tekanan uap jenuh larutan

Po = Tekanan uap jenuh murni

Xp = fraksi mol pelarut

Xp = np / n total (np+nt)

P campuran = P CCl4 + P x (disini kita misalkan zat yang tidak dik = x)

Dik :

P campuran = 137 mmHg

Po CCL4 = 143 mmHg

Po x = 85mmHg

Mr CCl4 = 154 (ini dicari di google karna tidak ada di soal)

n CCl4 = 400/154 = 2,59 mol

n x = 43,3 : Mr x

Dit : Mr x / Mm x

penyelesaian :

P campuran = P CCl4 + P x

22. kegunaan senyawa karbon tetraklorida


Kegunaan senyawa karbon tetraklorida:

Pelarut organikBahan pemadam apiBahan pendinginBahan pembesihMemunculkan tanda perangko

Pembahasan:  

Karbon tetraklorida, (juga dikenal dengan nama seperti “tetraklorometana”, “carbon tet” dalam industri pembersih, “Halon-104” dalam pemadam kebakaran, dan “Refrigerant-10” di dalam industri pendingin) adalah senyawa dengan rumus CCl₄.

Senyawa ini merupakan metana (CH₄) yang mengalami substitusi dengan 4 atom klorin (Cl). Senyawa ini bersifat tidak berwarna dengan bau agak manis. Senyawa ini tidak mudah terbakar pada suhu yang rendah.

Senyawa ini digunakan sebagai:

1. Pelarut organik

Dulunya senyawa ini merupakan pelarut yang banyak digunakan dalam kimia organik, tetapi, karena efeknya yang merugikan kesehatan, jarang digunakan saat ini.

2. Bahan pemadam api

Cairan ini disimpan dalam bentuk cair dalam tabung pemadam dan disemprotkan ke api saat terjadi kebakaran. Senyawa ini berubah menjadi gas dan bekerja dengan menghambat reaksi bahan terbakar dengan gas oksigen, sehingga memadamkan api

3. Bahan pendingin

Sejumlah besar karbon tetraklorida dulu digunakan untuk bahan pendingin (refrigeran) klorofluorokarbon R-11 (CFC) dan R-12 (dikloro-difluorometana, CCl2F2). Pendingin ini digunakan pada AC dan kulkas. Namun, pendingin ini dilarang dalam Protokol Montreal karena menyebabkan penipisan lapisan ozon.  

4. Bahan pembesih

Senyawa ini banyak digunakan untuk pembersih pada dry cleanig atau cuci tanpa air.  

5. Memunculkan tanda perangko

Salah satu penggunaan khusus karbon tetraklorida adalah dalam mengumpulkan prangko, untuk mengungkapkan tanda air (watermark) pada prangko tanpa merusaknya. Sejumlah kecil cairan senaywa ini ditempatkan di belakang perangko. Huruf atau desain tanda air kemudian dapat dilihat dengan jelas.

Paparan karbon tetraklorida konsentrasi tinggi (termasuk uapnya) dapat mempengaruhi sistem saraf pusat, merusak hati dan ginjal. Kontak yang terlalu lama bisa berakibat fatal. Karena itu penggunaan senyawa ini banyak dibatasi.
