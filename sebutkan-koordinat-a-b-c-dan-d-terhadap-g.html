Penjelasan dengan langkah-langkah:

Diketahui :

Titik A ( -7,7), titik B ( -6,4), titik C ( -3,3), titik D ( 0,2), titik E ( 2,4 ), titik F ( 5,5), titik G ( 6,2), titik H ( 4,0), titik I ( 3,-4), titik J ( 1,-5), titik K ( -2,-4), titik L ( -4,-5), titik M ( -7,-7)

Jawab :

Pada diagram kartesius, batas-batas nilai dibatasi menjadi empat bagian yaitu sebagai berikut :

Sebelah kanan atas disebut kuadran ISebelah kiri atas disebut kuadran IISebelah kiri bawah disebut kuadran IIISebelah kanan bawah disebut kuadran IV

Nilai x dan y pada tiap kuadran berbeda-beda yaitu

Kuadran I nilai x positif, nilai y negatifKuadran II nilai x negatif, nilai y positifKuadran III nilai x negatif, nilai y negatifKuadran IV nilai x positif, nilai y negati

titik A(-7,7) -> ( -13,5)titik B(-6,4) -> (-12,2)titk C(-3,3) -> (-9,1)titik D(2,0) -> (6,0)

27. Bantu aku dong!! 1) Sebutkan koordinat titik A, B , C dan D terhadap titik G 2) Sebutkan koordinat titik E, F, G dan H terhadap titik J


1) Sebutkan koordinat titik A, B, C dan D terhadap titik G
2) Sebutkan koordinat titik E, F, G dan H terhadap titik J

Pembahasan :

1) • koordinat titik A terhadap titik G
= dari titik G menuju ke titik A
= 13 satuan ke kiri, 5 satuan ke atas
= (-13, 5)

• koordinat titik B terhadap titik G
= dari titik G menuju ke titik B
= 12 satuan ke kiri, 2 satuan ke atas
= (-12, 2)

• koordinat titik C terhadap titik G
= dari titik G menuju ke titik C
= 9 satuan ke kiri, 1 satuan ke atas
= (-9, 1)

• koordinat titik D terhadap titik G
= dari titik G menuju ke titik D
= 6 satuan ke kiri, 0 satuan ke atas
= (-6, 0)

2) • koordinat titik E terhadap titik J
= dari titik J menuju ke titik E
= 1 satuan ke kanan, 9 satuan ke atas
= (1, 9)

• koordinat titik F terhadap titik J
= dari titik J menuju ke titik F
= 4 satuan ke kanan, 10 satuan ke atas
= (4, 10)

• koordinat titik G terhadap titik J
= dari titik J menuju ke titik G
= 5 satuan ke kanan, 7 satuan ke atas
= (5, 7)

• koordinat titik H terhadap titik J
= dari titik J menuju ke titik H
= 3 satuan ke kanan, 5 satuan ke atas
= (3, 5)

CARA LAIN :

sebelumnya kita tulis dulu koordinat dari masing-masing titik
A(-7, 7), B(-6, 4), C(-3, 3), D(0, 2), E(2, 4), F(5, 5), G(6, 2), H(4, 0), J(1, -5)

1) • koordinat titik A terhadap titik G
= A - G
= (-7, 7) - (6, 2)
= (-7 - 6, 7 - 2)
= (-13, 5)

• koordinat titik B terhadap titik G
= B - G
= (-6, 4) - (6, 2)
= (-6 - 6, 4 - 2)
= (-12, 2)

• koordinat titik C terhadap titik G
= C - G
= (-3, 3) - (6, 2)
= (-3 - 6, 3 - 2)
= (-9, 1)

• koordinat titik D terhadap titik G
= D - G
= (0, 2) - (6, 2)
= (0 - 6, 2 - 2)
= (-6, 0)

2) • koordinat titik E terhadap titik J
= E - J
= (2, 4) - (1, -5)
= (2 - 1, 4 - (-5))
= (1, 9)

• koordinat titik F terhadap titik J
= F - J
= (5, 5) - (1, -5)
= (5 - 1, 5 - (-5))
= (4, 10)

• koordinat titik G terhadap titik J
= G - J
= (6, 2) - (1, -5)
= (6 - 1, 2 - (-5))
= (5, 7)

• koordinat titik H terhadap titik J
= H - J
= (4, 0) - (1, -5)
= (4 - 1, 0 - (-5))
= (3, 5)
