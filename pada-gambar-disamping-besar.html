3. Perhatikan gambar di samping. Besar <ROU pada gambar di samping adalah.... ​


Penjelasan dengan langkah-langkah:

untuk menjawab soal ini teman-teman harus ingat beberapa hal berikut ini:

a. garis lurus memiliki sudut sebesar 180°b. sudut yang bertolak belakang memiliki nilai yang sama

1. langkah pertama kita harus mencari nilai x

∠PT = ∠QOR + ∠POQ + ∠SOT + ∠ROS180° = x + 2x + 3x + 4x180° = 10xx = 180° ÷ 10x = 18

2. langkah kedua kita harus mencari ∠ROU

∠ROU = ∠ROS + ∠SOT + ∠TOU∠ROU = 4x + 3x + 2x∠ROU = 9x∠ROU = 9(18)∠ROU = 162°

Jadi besar ∠ROU yaitu D. 162°

Penyelesaian :

180° = 72° + 6x°
6x° = 180° - 72°
6x° = 108°
x = 108° / 6
x = 18°

7x = 7(18°) = 126°

Sudut PRQ
= 180° - 126°
= 54°

Sudut QPR
= 180° - (72° + 54°)
= 180° - 126°
= 54°

JAWABANNYA D

Semoga Membantu :)Pada gambar,

sudut 72° berpelurus dengan sudut 6x, maka
72° + 6x = 180°
sehingga
6x = 180° - 72°
6x = 108°
x = 108° : 6 = 18°

Pada segitiga PQR, sudut 7x dan ∠PRQ berpelurus, maka
7x + ∠PRQ = 180°
7(18°) + ∠PRQ = 180°
∠PRQ = 180° - 126°
∠PRQ = 54°

Pada segitiga PQR, jumlah seluruh sudut dalam adalah 180°
∠QPR + ∠PQR + ∠PRQ = 180°
∠QPR + 72° + 54° = 180°
∠QPR = (180 - 72 - 54)°
∠QPR = 54°

23. Dari gambar di samping, besar


Jawaban:

~ Math

x + 3x + 10 = 90

4x + 10 = 90

4x = 80

x = 80 : 4

x = 20

Besar sudut AOB

= 3x + 10

= 3(20) + 10

= 60 + 10

= 70° → B

Jawaban:

B. 70°

Penjelasan dengan langkah-langkah:

Diketahui:

<AOC = 90°

<AOB = 3x+10

<BOC = x

Ditanya:

<AOB = ... °

Jawab:

<AOC = <AOB + <BOC

90 = 3x+10+x

90 = 3x+x+10

90 = 4x+10

90-10 = 4x

80 = 4x

4x = 80

x = 80/4

x = 20

<AOB = 3x+10

= 3×20+10

= 60+10

= 70
