3. suhu awal aluminium adalah 10° C. Jika kalor jenis aluminium 0,21 kalori/gram° C, massa aluminium = 50 gram dan kalor yang dibutuhkan aluminium = 6.300 Joule. Berapakah suhu akhir aluminium itu?​


Jawaban:

610 °C

Penjelasan:

t1= 10°C

c = 0,21

m = 50 gram

Q = 6.300 J

t2 = ....?

[tex]Q = m \times c \times (t2 - t1)[/tex]

[tex]6.300 = 50 \times 0.21 \times (t2 - 10)[/tex]

[tex]6.300 = 10.5 \times (t2 - 10)[/tex]

[tex]600 = t2 - 10[/tex]

[tex]610 = t2[/tex]


4. Aluminium bermassa 2kg dengan suhu 20°c menerima kalor sebesar 180 kilojoule.Jika kalor jenis Aluminium 900j/kg⁰c. Maka suhu akhir Aluminium adalah?


diketahui : m = 2 kg
                t1 = [tex] 20^{o} C[/tex]
                Q = 180 Kj = 180000 J
                 c = 900 J/kg[tex] ^{o} C[/tex]
ditanya : t2
              Q = m x c x Δt
       180000 = 2 x 900 x (t2 - t1)
       180000 = 1800 x (t2 - 20)
       t2 - 20 = [tex] \frac{180000}{1800} [/tex]
       t2 - 20 = 100
             t2 = 100 + 20
             t2 = [tex] 120^{o} C[/tex]
      [tex]\bold{Diketahui}\\ m\ = 2 \ kg\\ t_{1}\ = 20^{o}C\\ Q\ = 180\ kJ \Rightarrow 180 \times 1000 = 180.000\ Joule\\ c\ \ = 900\ J/kg^{o}C\\ \bold{Ditanya}\\ t_{2}.....?\\ \bold{Jawab}\\ \Delta t = \frac{Q}{mc}\\\\ \Delta t = \frac{180.000}{2\times900}\\\\ \Delta t = \frac{180.0\not0\not0}{18\not0\not0} \\\\ \Delta t = 100^{o}C\\\\\\ t_{2}\ = \Delta t + t_{1}\\ t_{2}\ = 100 + 20\\ \boxed{t_{2}\ = 120^{o}C}[/tex]

9. aluminium bermasa 2kg dengan suhu 20°C menerima kalor sebesar 100kj. jika kalor jenis aluminium 500 J/KG°C, maka suhu akhir aluminium tersebut adalah...​


◈ Kalor

Pembahasan

» Diketahui:

m = 2 kg

t₁ = 20°C

Q = 100 kJ = 100.000 J

c = 500 J/kg°C

» Ditanya:

tc = ?

» Jawab:

Q = m × c × ∆T

Q = m × c × (tc - t₁)

tc - t₁ = Q/(m × c)

tc = t₁ + Q/(m × c)

tc = 20 + 100.000/(2 × 500)

tc = 20 + 100.000/1.000

tc = 20 + 100

tc = 120°C

Jadi suhu akhir alumunium 120°C

— Keterangan:

Q = kalor (J)

m = massa (kg)

c = kalor jenis (J/kg°C)

∆T = perubahan suhu (°C)

t₁ = suhu awal (°C)

tc = suhu akhir (°C)
