Pengertian Bahan Bakar

Bahan Bakar adalah segala sesuatu yang terdapat api atau juga bisa terdapat energi panas yang muncul ketika berada disekitar atau didekat sumber bahan bakar tersebut menyala.

Api sangatlah banyak manfaatnya. Dengan api lah kita bisa memasak makanan, menghangatkan makanan dan sebagainya.

Manfaat api antara lain :

Untuk memasakUntuk menghangatkan makanan yang dinginUntuk pembangkit listrikUntuk sumber cahaya jika tidak ada aliran listrik disekitarnya

Selain manfaat api juga berbahaya apalagi untuk anak - anak yang suka bermain api. Api kecil mungkin adalah kawan kita akan tetapi ketika api itu membesar maka itu akan menjadi lawan bagi kita. Jadi jangan sembarangan bermain api.
