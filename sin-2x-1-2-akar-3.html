15. Himpunan penyelesaian dari persamaan sin 2x = 1/2 akar 3 untuk (0derajat < x < 360) derajat adalah


Himpunan penyelesaian dari persamaan trigonometri adalah HP = {30°, 60°, 210°, 240°}. Interval untuk himpunan penyelesaiannya antara 0° sampai 360°.

HIMPUNAN PENYELESAIAN PERSAMAAN TRIGONOMETRI

Persamaan trigonometri yang harus diingat

sin ax = sin b
Berlaku ax = b + 360° k
Berlaku ax = (180° - b) + 360° kcos ax = cos b
Berlaku ax = b + 360° k
Berlaku ax = - b + 360° ktan ax = tan b
Berlaku ax = b + 180° k

dimana k = 0, 1, 2, 3, ...

Sudut istimewa untuk trigonometri.

Untuk sudut 0°

sin 0° = 0cos 0° = 1tan 0° = 0

Untuk sudut 30°

sin 30° = [tex]\frac{1}{2}[/tex]cos 30° = [tex]\frac{1}{2} \sqrt{3}[/tex]tan 30° = [tex]\frac{1}{3} \sqrt{3}[/tex]

Untuk sudut 45°

sin 45° = [tex]\frac{1}{2} \sqrt{2}[/tex]cos 45° = [tex]\frac{1}{2} \sqrt{2}[/tex]tan 45° = 1

Untuk sudut 60°

sin 60° = [tex]\frac{1}{2} \sqrt{3}[/tex]cos 60° = [tex]\frac{1}{2}[/tex]tan 60° = [tex]\sqrt{3}[/tex]

Untuk sudut 90°

sin 90° = 1cos 90° = 0tan 90° = tak terhingga

Diketahui:

sin 2x = [tex]\frac{1}{2} \sqrt{3}[/tex]0° < x < 360°

Jawaban:

sin 2x = [tex]\frac{1}{2} \sqrt{3}[/tex]

sin 2x = sin 60°

Bentuk pertama.
[tex]2x \:=\: 60^o \:+\: 360^o k[/tex]
[tex]\frac{2x}{2} \:=\: \frac{60^o}{2} \:+\: \frac{360^o}{2} k[/tex]
[tex]x \:=\: 30^o \:+\: 180^o k[/tex]
k = 0 ⇒ x = 30°
k = 1 ⇒ x = 30° + 180° = 210°
k = 2 ⇒ x = 30° + 360° = 390° melebihi interval, tidak memenuhi.Bentuk kedua.
[tex]2x \:=\: (180^o \:-\: 60^o) \:+\: 360^o k[/tex]
[tex]2x \:=\: 120^o \:+\: 360^o k[/tex]
[tex]\frac{2x}{2} \:=\: \frac{120^o}{2} \:+\: \frac{360^o}{2} k[/tex]
[tex]x \:=\: 60^o \:+\: 180^o k[/tex]
k = 0 ⇒ x = 60°
k = 1 ⇒ x = 60° + 180° = 240°
k = 2 ⇒ x = 60° + 360° = 420° melebihi interval, tidak memenuhi.

HP = {30°, 60°, 210°, 240°}
