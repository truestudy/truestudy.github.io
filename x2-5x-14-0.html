[tex] {x}^{2} + 5x - 14 = 0 \\ {x}^{2} + 5x = 14 \\ {x}^{2} + 5x + ( \frac{1}{2} \times 5)^{2} = 14 + ( \frac{1}{2} \times 5)^{2} \\ {x}^{2} + 5x + {( \frac{5}{2}) }^{2} = 14 + { (\frac{5}{2})}^{2} \\ {(x + \frac{5}{2} )}^{2} = 14 + \frac{25}{4} \\ {(x + \frac{5}{2}) }^{2} = \frac{14 \times 4}{4} + \frac{25}{4} \\ {(x + \frac{5}{2} )}^{2} = \frac{56}{4} + \frac{25}{4} \\ {(x + \frac{5}{2} )}^{2} = \frac{81}{4} \\ x1 + \frac{5}{2} = \frac{9}{2} \\ x2 + \frac{5}{2} = - \frac{9}{2} \\ x1 = \frac{9}{2} - \frac{5}{2} = \frac{4}{2} = 2 \\ x2 = - \frac{9}{2} - \frac{5}{2} = - \frac{14}{2} = - 7[/tex]

24. x[tex]x2 + 5x - 14 \geqslant 0[/tex]


Mapel : Matematika Wajib (Kelas X SMA)
Materi : Program Linear
Indikator : Mencari HP dalam suatu pertidaksamaan linear kuadrat

Langkah Penyelesaian dan Jawaban :
x² + 5x - 14 ≥ 0

- Faktorkan terlebih dahulu :
(x - 2) (x + 7) ≥ 0,

- Sehingga diperoleh titik kritis :
x - 2 = 0
x = 2

x + 7 = 0
x = - 7

- Garis bilangan dan intervalnya :
+ + + (- 7) - - - - - - (2) + + +

- Karena diminta x ≥ 0 (positif atau sama dengan), maka :
HP = {x | x ≤ - 7 ∨ x ≥ 2, x ∈ R}
