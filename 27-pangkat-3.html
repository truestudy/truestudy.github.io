= (3^9) / (27^3)

= (3^9) / ((3^3)^3)

= (3^9) / (3^(3×3))

= (3^9) / (3^9)

= 3^(9-9)

= 3^0

= 1

26. 27 pangkat 4 : 3 pangkat 9digantikan 27=3​


Jawaban:

3³

Penjelasan dengan langkah-langkah:

27⁴ ÷ 3⁹

= 3^3^4 ÷ 3⁹

= 3^(3×4) ÷ 3⁹

= 3¹² ÷ 3⁹

= 3^(12-9)

= 3³

Hasilnya adalah 3³

PEMBAHASAN :

Rumus dasar :

[tex] {a}^{m} \div {a}^{n} = {a}^{m - n} [/tex]

PENYELESAIAN :

[tex] {27}^{4} \div {3}^{9} [/tex]

[tex] = ( {3}^{3} )^{4} \div {3}^{9} [/tex]

[tex] = {3}^{3 \times 4} \div {3}^{9} [/tex]

[tex] = {3}^{12} \div {3}^{9} [/tex]

[tex] = {3}^{12 - 9} [/tex]

[tex] = {3}^{3} [/tex]
