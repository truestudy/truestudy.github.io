Perbandingan banyak kursi pada baris ke -4 dan ke-10 adalah 2 : 5. Baris terakhir mempunyai 72 kursi. Banyak kursi yang dimiliki gedung tersebut adalah.....


MAPEL = MATEMATIKA
KELAS = 9
BAB = BARISAN & DERET

diketahui =
- beda ( b ) = 4
- U4 : U10 = 2 : 5
- Un = 72

ditanya = banyak kursi pada gedung (Sn) ??

-------- Deret Aritmatika --------

U4/U10 = 2/5
5(U4) = 2(U10)
5[ a + 3b ] = 2[ a + 9b ]
5[ a + 3(4) ] = 2[ a + 9(4) ]
5[ a + 12 ] = 2[ a + 36 ]
5a + 60 = 2a + 72
5a - 2a = 72 - 60
3a = 12
a = 12/3
a = 4

Un = 72
a + ( n - 1 )b = 72
4 + ( n - 1 )(4) = 72
4 + 4n - 4 = 72
4n = 72
n = 72/4
n = 18

S18 = n/2[ 2a + ( n - 1 )b ]
S18 = 18/2[ 2(4) + ( 18 - 1 )(4) ]
S18 = 9[ 8 + 68 ]
S18 = 9[ 76 ]
S18 = 684

jadi banyak kursi pada gedung tersebut sebanyak 684 kursi,

24. suatu  gedung pertunjukan mempunyai beberapa baris kursi setelah baris pertama , setiap baris mempunyai kursi 4 lebih banyak daripada baris sebelumnya.perbandingan banyak kursi pada baris ke 4 dan ke 10 adalah 2 : 5. baris terakhir mempunyai 72 kursi. banyakkursi yang ada di dala gedung tersebut adalah


Kelas : 9
Mapel : Matematika
Kategori : Barisan dan Deret Bilangan
Kata Kunci : Banyak Suku, Jumlah n suku
Kode : 9.2.6 (Kelas 9 Matematika Bab 6 - Barisan dan Deret Bilangan)

Pembahasan :
Untuk mencari banyaknya kursi di dalam gedung tersebut, maka digunakan rumus mencari jumlah n suku pertama barisan aritmatika.
Rumus jumlah n suku :

Sn = n/2 [2a + (n-1)b]

Untuk menggunakan rumus diatas, yang diperlukan adalah :
n = jumlah suku
a = suku pertama
b = beda barisan

Pada soal diatas, telah diketahui selisih antara baris adalah 4 lebih banyak. Pernyataan tersebut menyatakan bahwa beda barisan adalah 4. 
b = 4

Perbandingan kursi pada baris ke-4 dan baris ke-10 adalah 2 : 5. Pernyataan ini digunakan untuk mencari suku pertama.
Rumus suku ke-n :
Un = a + (n-1)b

Diketahui :
U₄ : U₁₀ = 2 : 5

U₄ = a + (4 - 1)b
U₄ = a + 3b
U₄ = a + 3(4)
U₄ = a + 12

U₁₀ = a + (10 - 1)b
U₁₀ = a + 9b
U₁₀ = a + 9(4)
U₁₀ = a + 36

Sehingga :
U₄ / U₁₀ = 2 / 5
(a + 12) / (a + 36) = 2 / 5
5(a + 12) = 2(a + 36)
5a + 60 = 2a + 72
5a - 2a = 72 - 60
3a = 12
a = 12 / 3
a = 4

Pada pernyataan "baris terakhir mempunyai 72 kursi", artinya Un = 72. Digunakan untuk mencari banyaknya suku (n).
Un = 72
a + (n - 1)b = 72
4 + (n - 1)4 = 72
4 + 4n - 4 = 72
4n = 72
n = 72 / 4
n = 18
maka gedung tersebut memiliki 18 baris kursi

Mencari banyaknya kursi :
Sn = n/2 [2a + (n-1)b]
S₁₈ = 18/2 [2(4) + (18 - 1)4]
S₁₈ = 9 [8 + (17 x 4)]
S₁₈ = 9[8 + 68]
S₁₈ = 9 x 76
S₁₈ = 684

Maka di gedung tersebut terdapat 684 kursi
