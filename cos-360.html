1. 3. Tentukan nilai dari cos 840° adalah.... ⇒cos (720° +120°) ⇒cos (360° + (360° + ⇒cos (360° + °) ⇒cos 120° ⇒cos (180° --Cos ......... ↑ ******* °))​


Jawab:

kita dapat menggunakan rumus cos (x + y) = cos x cos y - sin x sin y dan identitas sudut ganda cos 2x = cos^2 x - sin^2 x.

Penjelasan dengan langkah-langkah:

Pertama, kita dapat mengubah 840° menjadi 720° + 120° dengan cara mengurangi 840° dengan 360° sebanyak 2 kali.
cos 840° = cos (720° + 120°)Kemudian, kita dapat mengubah 720° menjadi 360° + 360° dengan cara menambahkan 360° sebanyak 2 kali.
cos (720° + 120°) = cos (360° + (360° + 120°))Kemudian, kita dapat mengubah 120° menjadi 360° - 240° dengan cara mengurangi 120° dengan 240°.
cos (360° + (360° + 120°)) = cos (360° + (360° - 240°))Selanjutnya, kita dapat mengubah 240° menjadi 180° + 60° dengan cara menambahkan 60° ke 240°.
cos (360° + (360° - 240°)) = cos (360° + (360° - (180° + 60°)))Setelah itu, kita dapat mengubah 180° menjadi cos 180° dengan cara menggantikan 180° dengan cos 180°.
cos (360° + (360° - (180° + 60°))) = cos (360° + (360° - (cos 180° + 60°)))Terakhir, kita dapat mengubah 60° menjadi cos 60° dengan cara menggantikan 60° dengan cos 60°.
cos (360° + (360° - (cos 180° + 60°))) = cos (360° + (360° - (cos 180° + cos 60°)))

cos (360° + (360° - (cos 180° + cos 60°))) = cos (360° + (360° - (-1/2 + 1/2))) = cos (360° + 360°) = cos 720° = cos (360°) = cos(0) = 1

Jadi, nilai dari cos 840° adalah 1.

Semoga penjelasan ini membantu.

7. jika cos (360-c) = p,maka cos c =


Kategori Soal : Matematika - Trigonometri
Kelas : X (1 SMA)
Pembahasan : 
Perbandingan sudut dalam trigonometri.
Dalam trigonometri, besar sudut α dibagi menjadi 4 kelompok, yaitu :
Kuadran I : 0° < α < 90° (0 < α < π/2)
sin (90° - α) = cos α
cos (90° - α) = sin α
tan (90° - α) = cotan α
cotan (90° - α) = tan α
secan (90° - α) = sec α
sec (90° - α) = cosec α

Kuadran II : 90° < α < 180° (π/2 < α < π)
sin (180° - α) = sin α
cos (180° - α) = -cos α
tan (180° - α) = -tan α
cotan (180° - α) = -cotan α
secan (180° - α) = secan α
sec (180° - α) = -sec α

Kuadran III : 180° < α < 270° (π < α < 3π/2)
sin (270° - α) = -cos α
cos (270° - α) = -sin α
tan (270° - α) = cotan α
cotan (270° - α) = tan α
secan (270° - α) = -sec α
sec (270° - α) = -cosec α

Kuadran IV : 270° < α < 360° (3π/2 < α < 2π)
sin (360° - α) = sin (-α) = -sin α
cos (360° - α) = cos (-α) = cos α
tan (360° - α) = tan (-α) = -tan α
cotan (360° - α) = cotan (-α) = -cotan α
secan (360° - α) = secan (-α) = -secan α
sec (360° - α)= sec (-α) = sec α

Mari kita lihat soal berikut.
Jika cos (360 - c)° = p, maka cos (-c)° = cos c° = p.

10. Tentukan Sin 360 dan cos 360!


Nilai dari sin 360° dan cos 360° masing-masing adalah 0 dan 1

Pembahasan

Sinus dan Cosinus adalah beberapa bentuk dari trigonometri. Trigonometri memiliki kaitan yang cukup erat mengenai segitiga. Trigonometri bisa dinyatakan dengan derajat, atau dengan panjang sisi.

Ada dua cara dalam menyelesaikan trigonometri seperti ini.

Diketahui : sin 360° dan cos 360°

Ditanya : Nilai keduanya?

Dijawab :

(270 + α ) => trigonometri berbalik.

sin 360

= sin (270 + α )

= sin ( 270 + 90 )

= cos 90°

= 0

cos 360

= cos (270 + α )

= cos ( 270 + 90 )

= sin 90°

= 1

Jadi, nilai sin 360 dan cos 360 adalah 0 dan 1
