3. sebuah kereta api yang bergerak dengan kecepatan 72 km/jam dipercepat hingga kecepatannya menjadi 90 km/jam dalam waktu 5menit. Berapakah percepatan kereta api itu?


diketahui :
v0 = 72 km/jam = 72000 m/3600 s = 20 m/s
vt = 90 km/jam = 90000 m/3600 s = 25 m/s
t = 5 menit = 5 x 60 = 300 s
ditanya : a = ?
jawab :
a = (vt - v0) / t
= (25 - 20) / 300
= 5 / 300
= 1 / 60 m/s²

4. Sebuah kereta api meninggalkan stasiun dan bergerak menuju stasiun lain dengan kecepatan 72 km/jam dalam waktu 2 jam kemudian kereta itu bergerak menuju stasiun berikutnya dengan kecepatan 53 km/jam dalam waktu 3 jam berapakah kecepatan rata- rata kereta api tersebut selama perjalanan?


Kecepatan rata- rata kereta api tersebut selama perjalanan adalah 60,6 km/jam.

Pembahasan

KECEPATAN RATA - RATA

Perpindahan benda adalah perubahan posisi benda dari posisi akhir ke posisi awalnya.

Kecepatan rata - rata pada sebuah benda adalah perbandingan perpindahan posisi benda terhadap selang waktunya.

• Apabila diketahui persamaan posisi benda r (t).

[tex]\bar{v} \:=\: \frac{r_2 \:-\: r_1}{t_2 \:-\: t_1}[/tex]

dimana

[tex]r_1[/tex] = posisi awal benda saat [tex]t_1[/tex] (m)

[tex]r_2[/tex] = posisi akhir benda saat [tex]t_2[/tex] (m)

[tex]\bar{v}[/tex] = kecepatan rata - rata (m/s)

• Apabila diketahui benda bergerak dengan tiga kecepatan yang berbeda - beda.

[tex]\bar{v} \:=\: \frac{perpindahan}{waktu}[/tex]

[tex]\bar{v} \:=\: \frac{S_1 \:+\: S_2\:+\: S_3}{t_1 \:+\: t_2 \:+\: t_3}[/tex]

[tex]\bar{v} \:=\: \frac{v_1 \times t_1 \:+\: v_2 \times t_2 \:+\: v_3 \times t_3}{t_1 \:+\: t_2 \:+\: t_3}[/tex]

dimana

v = kecepatan (m/s)

t = waktu (s)

Bila satuan waktu dalam jam, maka satuan kecepatannya dalam satuan km/jam.

Diketahui:

[tex]v_1[/tex] = 72 km/jam

[tex]t_1[/tex] = 2 jam

[tex]v_2[/tex] = 53 km/jam

[tex]t_2[/tex] = 3 jam

Ditanyakan:

[tex]\bar{v}[/tex] ?

Penjelasan:

Percepatan rata - rata diketahui dua kecepatan dan selang waktunya. Maka

[tex]\bar{v} \:=\: \frac{v_1 \times t_1 \:+\: v_2 \times t_2}{t_1 \:+\: t_2}[/tex]

[tex]\bar{v} \:=\: \frac{72 \times 2 \:+\: 53 \times 3}{2 \:+\: 3}[/tex]

[tex]\bar{v} \:=\: \frac{144 \:+\: 159}{5}[/tex]

[tex]\bar{v} \:=\: \frac{303}{5}[/tex]

[tex]\bar{v}[/tex] = 60,6 km/jam

Kecepatan rata- rata kereta api tersebut selama perjalanan adalah 60,6 km/jam.
