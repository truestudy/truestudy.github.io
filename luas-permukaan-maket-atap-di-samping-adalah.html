16. Atap sebuah rumah berbentuk limas dengan alas persegi dengan rusuk alas 24m dan tinggi atap terhadap alas 5m . Luas permukaan atap rumah itu adalah


Jawab:

624m²

Penjelasan dengan langkah-langkah:

luas permukaan limas persegi tanpa alas

= luas selimut (disini selimutnya segitiga sama kaki)

luas permukaan limas persegi tanpa alas

= (rusuk x s / 2) x 4

s = garis pelukis

cara mencari garis pelukis bisa dengan menggunakan rumus pythagoras a² + b² = c²

a = rusuk / 2

b = tinggi limas

c = garis pelukis

garis pelukis

= (24m / 2)² + 5m²

= 12m² + 5m²

= 144m + 25m

= 169 m

c = [tex]\sqrt[2]{169}[/tex]

c = 13

luas permukaan =

= (24m x 13m / 2) x 4

= 156m² x 4

= 624m²
