15. Toolbar pada internet Explorer berisi tombol yang sering digunakan untuk suffing yaitu


Toolbar pada internet explorer berisi tombol yang sering digunakan untuk suffing, mungkin maksud kamu adalah surfing? Jadi jawabannya adalah bagian - bagian di internet Explorer yang sering digunakan untuk melakukan surfing atau berselancar di inter adalah bagian Toolbar itu sendiri.

Pembahasan

Toolbar merupakan bagian yang ada di Internet Explorer, billa di jendela IE Toolbar berada di posisi atas setelah Title Bar, dan Menu bar.

Di dalam Toolbar terdapat beberapa ikon, atau tombol yang berfungsi untuk kelancaran aktivitas bersurfing internet. Ikon atau tombol tersebut berupa

Ikon Back : berfungsi untuk kembali ke halaman sebelumnya yang diakses. Ikon Forward : berfungsi untuk ke halaman sebelum menegklik tombol Back. Ikon Stop : berfungsi untuk menghentikan loading Internet Explorer yang sedang memuat suatu situs atau web. Ikon Refresh : berfungsi untuk mengulangi kembali proses loading atau memuat situs/web. Ikon Search : berfungsi untuk membuka dan mencari kata kunci yang dimasukkan oleh pengguna.Ikon Favorites : berfungsi untuk memunculkan daftar alamat internet yang pernah kita simpan menjadi favorit.Ikon Home : berfungsi untuk kembali ke halaman awal dari internet explorer yang sudah kita tentukan atau default dari IE sendiri.

29. ikon yang di internet explorer


1.Ikon Back berfungsi untuk kembali ke halaman sebelumnya yang sudah diakses.
2.Ikon Forward berfungsi untuk kembali ke halaman sebelum di Back dan untuk ke halaman berikutnya lagi.
3.Ikon Stop berfungsi untuk menghentikan browser yang sedang mengakses penampilan suatu situs web.
4.Icon Refresh berfungsi untuk mengulangi kembali proses penampilan situs web.
5.Icon Home berfungsi untuk kembali ke halaman awal dari Internet Explorer.
6.Icon Search berfungsi untuk membuka fasilitas area pencarian data atau informasi di Internet.
7.Icon Favorites berfungsi untuk menampilkan daftar alamat internet yang pernah kita simpan sehingga memudahkan kita untuk mengakses kembali.
8.Icon media berfungsi untuk menampilkan jendela WindowsMedia.com yang berisi program music, video, dan film
9.Icon History berfungsi untuk menampilkan daftar alamat yang pernah dikunjungi pada hari ini maupun pada hari sebelumnya.
10.Icon Mail berfungsi untuk mengaktifkan fasilitas Outlook Express dan Internet Explorer.
11.Icon Print berfungsi untuk mencetak halaman situs yang terbuka.
12.Icon Edit with Microsoft Programme (Ms. FontPage, Ms. Excel, Ms. Word, Ms. Wordpad) berfungsi untuk mengedit atau membuat data melalui program Microsoft.
13.Icon Discuss berfungsi untuk menampilkan Discussion Bar pada Internet Explorer (pada Windows XP).
