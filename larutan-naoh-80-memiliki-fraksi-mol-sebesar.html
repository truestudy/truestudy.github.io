5. fraksi mol NaOH dalam larutan NaOH adalah 0,1.maka molalitasnya adalah


Kelas          : XII
Pelajaran    : Kimia
Kategori      : Sifat Koligatif
Kata Kunci  : fraksi mol, molalitas NaOH, H₂O

Pembahasan

Siapkan
Massa NaOH = g
Massa air = p
Mr NaOH = 23 + 16 + 1 = 40

Fraksi mol NaOH = 0,1

Xp = [mol NaOH] / [mol NaOH + mol air]

0,1 = [g/40] / [(g/40) + (p/18)]

0,1 = [g/20] / [(g/20) + (p/9)]

0,1[(g/20) + (p/9)] = g/20

Kedua ruas kalikan 10

(g/20) + (p/9) = 10g/20

p/9 = 9g/20

g/p = ²⁰/₈₁ ⇒ substitusikan ke perhitungan di bawah ini

Ditanya molalitas

molalitas = [g/Mr NaOH] x [1000/p]

molalitas = [g/p] x [¹⁰⁰⁰/₄₀]

molalitas = [²⁰/₈₁] x 25

Jadi konsentrasinya sebesar 6,17 molal

15. larutan NaOH (Mr=40) 80% memiliki fraksi mol sebesar....


Diketahui :
% Larutan NaOH = 80%
Mr NaOH = 40 gram/mol

Ditanya : Fraksi mol ?

Jawab :

massa NaOH :
= %NaOH . Massa larutan / 100%
= 80% . 100 gram / 100 %
= 80 gram

massa air :
= 100 gram - 80 gram
= 20 gram

mol NaOH :
= m / Mr
= 80 gram / 40 gram/mol
= 2 mol

mol air :
= m / Mr
= 20 gram/ 18 gram/mol
= 1,11 mol

*Fraksi mol terlarut (Xt) :
= mol Terlarut / mol total
= mol NaOH / mol NaOH + mol air
= 2 mol / 2 mol + 1,11 mol
= 0,64

*Fraksi mol pelarut (Xp) :
= mol pelarut / mol total
= mol air / mol NaOH + mol Air
= 1,11 mol / 2 mol + 1,11 mol
= 0,36
