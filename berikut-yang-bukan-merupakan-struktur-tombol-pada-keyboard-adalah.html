1. yang tidak termasuk struktur tombol tombol pada keyboard


Hai.

Pada keyboard, selain typing keys, numeric keypad, function key, dan control key bukanlah struktur keyboard.

Typing keys meliputi huruf abjad yang ada pada keyboard hingga tombol angka di atasnya.

Numeric keypad adalah bagian paling kanan pada keyboard, meliputi nomor dan sekitarnya.

Function keys meliputi tombol Fungsi1 (F1) dan seterusnya.

Control key meliputi tombol atas, bawah, kiri, kanan, dan sekitar, juga tombol Control (Ctrl), Alternative (Alt), dan sebagainya.

Intinya, selain tombol yang disebutkan di atas adalah bukan struktur keyboard.

Penjelasannya ialah sebagai berikut:

Jawaban:

Secara umum, struktur tombol pada keyboard terbagi atas 4, yaitu:

1. Tombol Ketik (typing keys)

Tombol ketik adalah salah satu bagian dari keyboard yang berisi huruf dan angka serta tanda baca. Secara umum, ada 2 jenis susunan huruf pada keyboard, yaitu tipe QWERTY dan DVORAK. Namun, yang terbanyak digunakan sampai saat ini adalah susunan QWERTY.

2. Numeric Keypad

Numeric keypad merupakan bagian khusus dari keyboard yang berisi angka dan sangat berfungsi untuk memasukkan data berupa angka dan operasi perhitungan. Struktur angkanya disusun menyerupai kalkulator dan alat hitung lainnya.

3. Tombol Fungsi (Function Keys)

Tahun 1986, IBM menambahkan beberapa tombol fungsi pada keyboard standard. Tombol ini dapat dipergunakan sebagai perintah khusus yang disertakan pada sistem operasi maupun aplikasi.

4. Tombol kontrol (Control keys)

Tombol ini menyediakan kontrol terhadap kursor dan layar. Tombol yang termasuk dalam kategori ini adalah 4 tombol bersimbol panah di antara tombol ketik dan numeric keypad, home, end, insert, delete, page up, page down, control (ctrl), alternate (alt) dan escape (esc).

Keyboard merupakan salah satu perangkat keras dalam instalasi komputer. Keyboard berfungsi untuk menampilkan tulisan pada layar komuper. Dalam sebuah keyboard, terdapat beberapa tombol khusus, yakni

DeletePage upPage downInsertPrint screenPause breakEndPembahasan

Keyboard merupakan salah satu perangkat keras dalam instalasi komputer. Keyboard berisikan huruf, angka, dan tombol-tombol lain yang berfungsi untuk menampilkan tulisan pada layar komuper. Pada umumnya, keyboard memiliki dua jenis, yakni keuboard QWERTY dan keyboard DVORAK. Kedua keyboard tersebut merupkana keyboard yang banyak digunakan oleh pengguna komputer.

Dalam sebuah keyboard, terdapat beberapa tombol khusus, yakni

Delete. Delete digunakan untuk menghapus huruf dari depan ke belakang. Page up. Page up digunakan untuk menggerakkan layar ke atas tanpa menggunakan mouse. Page down. Page down digunakan untuk menggerakkan layar ke bawah tanpa menggunakan mouse Insert. Insert digunakan untuk mensisipkan karakter pada posisi kursor. Print screen. Print screen digunakan untuk mengcapture tampilan layar Pause break. Pause break biasanya digunakan pada sistem operasi DOS yang berungsi untuk menghentikan tampilan program berbasis text. End. End digunakan untuk memindahkan kursor pada baris paling belakang.

ALT

Ctrl

Tombol control akan berfungsi jika penggunannya dipasangkan dengan tombol lainnya seperti tombol C atau disebut juga ctrl cakan berguna untuk menyalin suatu objek yang nantinya akan disalingkan pada area kerja yang ditentukan dengan ctrl v .

Tombol F1 sampai F12

Tombol fungsi yang pemanfataanya disesuaikan dengan system operasi  atau aplikasi contoh F1= menampilkan menu help yang akan memberikan penjelasan mengenai aplikasi yang sedang berjalan .

Tombol ESC

Tombol yang kegunaannya sama dengan pilihan canstle yaitu untuk membatalkan satu tahap pekerjaan

Tombol enter

Tombol enter kegunaannya sama dengan tombol ok yaitu untuk mengatakan bahwa operasi yang dilakukan betul dan selesai. Enter juga dapat berarti menyisipkan baris kosong atau baris baru pada proses pemasukan teks .

Tombol TAB

 Menekan tombol berarti pindah ke field atau daerah isian  atau pilihan berikutnya . pada saat proses pengetikan, tab berarti tempat penghentian tab ( tab’s stop ) terdekat

Panah atas,bawah,kiri,dan kanan

Menggerakkan penunjuk karakter atau cursor sesuai arah anak panah bersangkutan .

Tombol backspace

Berguna pada saaat proses pengetikan menekan tombol backspace akan mengakibatkan karakter  atau huruf  disebelah kiri cursor  akan terhapus.

Delete

Berguna pada saat proses pengetikan menekan tombol delete akan mengakibatkan karakter atau huruf di sebelah kanan cursor akan terhapucs

HOME

Menekan tombol home akan mengakibatkan cursor akan berpindah ke awal garis dimana cursor berada.

END

Menekan tombol end akan mengakibatkan cursor akan berpindah ke akhir garis dimana cursor berada.
