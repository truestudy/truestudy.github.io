3. K2CrO7, berapa bilangan biloks Cr? CaSO4, berapa biloks S? NaNO3 , berapa biloks N ? NH4+ Berapa biloks N ? SO2, berapa biloks S?


1)  Bilangan biloks Cr dalam K₂CrO₇ adalah +12.

2) Bilangan biloks S dalam CaSO₄ adalah +6.

3) Bilangan biloks N dalam NaNO₃ adalah +5.

4) Bilangan biloks N dalam NH₄⁺ adalah -3.

5) Bilangan biloks S dalam SO₂ adalah +4.

Pembahasan : 1)  Bilangan biloks Cr dalam K₂CrO₇

biloks K = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks K₂CrO₇ = 0  

(2 × biloks K) + biloks Cr + (7 × biloks O) = 0

(2 × (+1)) + biloks Cr + (7 × (-2)) = 0

(+2) + biloks Cr + (-14) = 0

biloks Cr + (-12) = 0

biloks Cr = +12

2) Bilangan biloks S dalam CaSO₄

biloks Ca = +2 (Golongan IIA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks CaSO₄ = 0  

biloks Ca + biloks S + (4 × biloks O) = 0

(+2) + biloks S + (4 × (-2)) = 0

(+2) + biloks S + (-8) = 0

biloks S + (-6) = 0

biloks S = +6

3) Bilangan biloks N dalam NaNO₃

biloks Na = +1 (Golongan IA)

aturan biloks :  

Bilangan oksidasi unsur golongan utama (IA,IIA,IIIA,IVA,VA,VIA ) sesuai dengan golongannya. Contoh : IA = +1, IIA = +2, IIIA = +3, IVA = -4, VA = -3 , VIA = -2 , VIIA = -1 , VIIIA = 0.

biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks NaNO₃ = 0  

biloks Na + biloks N + (3 × biloks O) = 0

(+1) + biloks N + (3 × (-2)) = 0

(+1) + biloks N + (-6) = 0

biloks N + (-5) = 0

biloks N = +5

4) Bilangan biloks N dalam NH₄⁺

biloks H = +1

aturan biloks :  

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks NH₄⁺ = +1

biloks N + (4 × biloks H)= +1

biloks N + (4 × (+1)) = +1

biloks N + (+4) = +1

biloks N + (+4) = +1

biloks N = (+1) + (-4)

biloks N = -3

5) Bilangan biloks S dalam SO₂

Biloks O = -2  

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2.

biloks SO₂ = 0  

biloks S + (2 × biloks O) = 0

biloks S + (2 × (-2)) = 0

biloks S + (-4) = 0

biloks S = +4

8. biloks N dalam HNO2 adalah


Pembahasan

Soal ini dikembangkan dengan menentukan bilangan oksidasi N pada

a. HNO₂

b. [tex]NO_3^-[/tex]

c. [tex]NH_4^+[/tex]

Bilangan oksidasi dapat berupa bilangan negatif, nol, atau positif. Beberapa aturan bilangan oksidasi yang terkait soal ini meliputi:  

[1]. Bilangan oksidasi unsur bebas atau molekul bebas adalah nol. Contoh: Na, Ne, Cl₂, H₂, O₂, dan lain-lain  

[2]. Suatu senyawa netral harus mempunyai penjumlahan bilangan oksidasi unsur-unsurnya sama dengan nol. Contoh pada H₂O, NaCl, KClO₄, dan lain-lain

[3]. Suatu senyawa ion bermuatan harus memiliki penjumlahan bilangan oksidasi unsur-unsurnya sama dengan muatan ionnya. Contohnya ada pada persoalan kita kali ini.

[4]. Oksigen (O) dalam suatu senyawa mempunyai bilangan oksidasi -2, kecuali pada senyawa peroksida seperti H₂O₂, BaO₂, NaO, bilangan oksidasi O sama dengan -1. Khusus dalam OF₂, bilangan oksidasi O sama dengan +2 karena F lebih elektronegatif.

Pengerjaan

a. HNO₂

Asam nitrit adalah senyawa netral, jadi memiliki total bilangan oksidasi sama dengan nol. Mari kita tentukan bilangan oksidasi nitrogen.

(biloks H) + {biloks N) + 2[biloks O] = 0

(+1) + (biloks N) + 2(-2) = -0

Biloks N = 4 - 1

Diperoleh bilangan oksidasi N pada asam nitrit HNO₂ sebesar +3.

b. [tex]NO_3^-[/tex]

Senyawa ion nitrat memiliki total bilangan oksidasi sama dengan -1. Mari kita tentukan bilangan oksidasi nitrogen.  

(biloks N) + 3[biloks O] = -1

(biloks N) + 3(-2) = -1

Biloks N = +6  - 1

Diperoleh bilangan oksidasi N pada senyawa ion nitrat sebesar +5.

c. [tex]NH_4^+[/tex]

Senyawa ion amonia memiliki total bilangan oksidasi sama dengan +1. Mari kita tentukan bilangan oksidasi nitrogen.  

(biloks N) + 4(biloks H) = +41

(biloks N) + 4(+1) = +1

Biloks N = -3

Diperoleh bilangan oksidasi N pada senyawa ion nitrat sebesar -3.
