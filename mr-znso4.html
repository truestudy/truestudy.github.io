Pembahasan

Untuk menyelesaikan soal ini, sobat brainly bisa simak pembahasan berikut :

Menentukan Ar nya terlebih dahulu. Ar bisa kita dapatkan di tabel periodik

Mensubstitusikan nilai Ar nya

[tex] \\ [/tex]

Diketahui :

Ar Zn = 65Ar S = 32Ar O = 16

Ditanya :

Mr ZnSO4 .. ?

Dijawab :

[tex] \\ [/tex]

Substitusikan semua nilai Ar yang sudah kita dapatkan.Dari rabel periodik tadi.

[tex] \\ [/tex]

[tex] \sf \: Mr \: ZnSO_4 = Ar \: Zn + Ar \: S + 4 × Ar \: O[/tex]

[tex] \sf Mr \: ZnSO_4 = 65 + 32 + 4 × 16[/tex]

[tex]\sf Mr \: ZnSO_4 = 65 + 32 + 64[/tex]

[tex] \sf Mr \: ZnSO_4 = 65 + 96[/tex]

[tex]{\boxed{ \sf Mr \: ZnSO_4 = 161 \: gram / mol}}[/tex]

jadi, Mr ZnSO4 adalah 161 gram / mol

2. Tentukan Mr dari senyawa ZnSO4 . 7H2O


Mr ZnSO₄. 7H₂O adalah 287

Pembahasan

[tex]\boxed{\boxed{\bold{Stoikiometri}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus,kadar, konsentrasi, rumus empiris, rumus molekul, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

[tex]\boxed{\boxed{\bold{Diket}}}[/tex]  

ZnSO₄. 7H₂O

[tex]\boxed{\boxed{\bold{Ditanya}}}[/tex]  

Mr

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]  

Mr (Massa Molekul Relatif)

Massa molekul relatif  (Mr) adalah jumlah total dari massa atom relatif unsur-unsur penyusunnya dalam suatu senyawa atau molekul. Massa molekul relatif (Mr) digunakan untuk sneyawa atau molekul sedangkan massa atom relatif digunakan untuk unsur atau atom. Rumus untuk menentukan massa molekul relatif (Mr) adalah:

[tex]\boxed{\boxed{\bold{Mr~A_nB_m~=~n.~Ar~A~+~m.~Ar~B}}}[/tex]

Massa atom relatif setiap unsur penyusun dari senyawa dapat dilihat pada tabel sistem periodik unsur.

Ar H = 1

Ar N = 14

Ar O = 16

Ar C = 12

Ar S = 32

Ar Zn = 65

Mr ZnSO₄. 7H₂O = Mr ZnSO₄ + Mr 7 (H₂O)

Mr ZnSO₄. 7H₂O = { ( 1. Ar Zn ) + ( 1. Ar S ) + ( 4. Ar O ) } + 7. { ( 2. Ar H ) + ( 1. Ar O ) }

Mr ZnSO₄. 7H₂O = { ( 1. 65 ) + ( 1. 32 ) + ( 4. 16 ) } + 7. { ( 2. 1 ) + ( 1. 16 ) }

Mr ZnSO₄. 7H₂O = { 65 + 32 + 64 } + 7. { 2 + 16 }

Mr ZnSO₄. 7H₂O = { 161 } + 7. { 18 }

Mr ZnSO₄. 7H₂O = 161 + 126

Mr ZnSO₄. 7H₂O = 287

[tex]\boxed{\boxed{\bold{Kesimpulan}}}[/tex]  

Mr ZnSO₄. 7H₂O = 287

3. Sebanyak 6,5 gram zink direaksikan dengan 19,6 gram asam sulfat encer menghasilkan seng sulfat dan gas hidrogen dengan persamaan reaksi sebagai berikut.. Zn(s) + H2 SO4 (aq) → ZnSO4 (aq) + H2 (g) a. Zat manakah yang bertindak sebagai pereaksi pembatas? b. Berapa gram massa ZnSO4 yang terbentuk? c. Berapa gram massa zat pereaksi yang tersisa? (Ar Zn = 65, Mr H2 SO4 = 98, Mr ZnSO4 = 161) Dibantu akang teteh


mol Zn = [tex]\frac{6.5~g}{65~g/mol}[/tex] = 0.1 mol

mol H₂SO₄ = [tex]\frac{19.6~g}{98~g/mol}[/tex] = 0.2 mol

       Zn(s) + H₂SO₄(aq) → ZnSO₄(aq) + H₂(g)

(m) 0.1 mol ≅ 0.2 mol

(.r.) 0.1 mol ≅ 0.1 mol   ≅ 0.1 mol ≅ 0.1 mol

(.s) ---  mol ≅ 0.1 mol   ≅ 0.1 mol ≅ 0.1 mol

a. Zat manakah yang bertindak sebagai pereaksi pembatas?  Zn

b. Berapa gram massa ZnSO₄ yang terbentuk?

massa  ZnSO₄ = 0.1 mol x 161 g/mol = 16.1 g

c. Berapa gram massa zat pereaksi yang tersisa?

massa H₂SO₄ = 0.1 mol x 98 g/mol = 9.8 g

Penjelasan:

                             Mol  

Mol adalah satuan jumlah zat yang digunakan dalam perhitungan kimia. Mol dapat digunakan untuk menyatakan jumlah partikel, massa dan volume suatu zat.  

n = [tex]\frac{massa}{mr}[/tex]

Keterangan:  

     n        = mol suatu zat (mol)  

     massa = massa suatu zat (g)  

     Mr     = massa molekul relatif suatu zat (g/mol)

14. biloks zn pada znSO4


Bilangan oksidasi atom Zn dalam senyawa ZnSO₄ adalah +2. ✔

Pembahasan

bo ZnSO₄ = (bo Zn) + (bo SO₄)

0 = (bo Zn) + (-2)

bo Zn = +2 + 0

bo Zn = +2 ✔

Keterangan :

bo SO₄⁻² = -2

bo ZnSO₄ = muatan senyawa = 0

Jadi, Biloks atom Zn dalam senyawa ZnSO₄ adalah +2.

MATERI :

• Bilangan Oksidasi (biloks/bo) adalah kemampuan suatu atom memberi, melepaskan, dan menerima elektron.

• Beberapa Aturan-aturan Biloks.

Biloks atom dalam unsur bebas adalah 0.Biloks Klor (Cl), Brom (Br), Iod (I) adalah -1, kecuali jika bergabung dgn unsur yang lebih elektrononegatif.Biloks O adalah -2 , kecuali pada senyawa peroksida (-1) dan superoksida (-1/2).Biloks H pada senyawa non logam adalah +1 dan pada senyawa logam adalah -1.Biloks ion dari suatu atom (monoatom) sama dengan muatan ionnya. Contoh : Biloks S²⁻ = -2.Bilangan oksidasi fluor (F) dalam senyawa selalu -1 karena fluor merupakan unsur yg sangat elektronegatif.Jumlah total Biloks atom-atom dalam ion Poliatom sama dengan muatan ion tersebut. Contoh : bo SO₄⁻² = -2.

28. tolong kakcari biloks s dari ZnSO4​


Biloks S dari ZnSO₄ adalah +6.

Pembahasan

bo ZnSO₄ = (bo Zn) + (bo S) + 4(bo O)

0 = (+2) + (bo S) + 4(-2)

0 = bo S + (-6)

6 = bo S

bo S = +6 ✔

Keterangan :

bo Zn = muatan ion = +2

bo O = -2

Jadi, biloks S adalah +6.

MATERI :

• Bilangan Oksidasi (biloks/bo) adalah kemampuan suatu atom memberi, melepaskan, dan menerima elektron.

• Beberapa Aturan-aturan Biloks.

Biloks atom dalam unsur bebas adalah 0.Biloks Klor (Cl), Brom (Br), Iod (I) adalah -1, kecuali jika bergabung dgn unsur yang lebih elektrononegatif.Biloks O adalah -2 , kecuali pada senyawa peroksida (-1) dan superoksida (-1/2).Biloks H pada senyawa non logam adalah +1 dan pada senyawa logam adalah -1.Biloks ion dari suatu atom (monoatom) sama dengan muatan ionnya. Contoh : Biloks S²⁻ = -2.Bilangan oksidasi fluor (F) dalam senyawa selalu -1 karena fluor merupakan unsur yg sangat elektronegatif.Jumlah total Biloks atom-atom dalam ion Poliatom sama dengan muatan ion tersebut. Contoh : bo SO₄²⁻ = -2.Biloks unsur logam dalam senyawanya selalu bertanda positif yang besarnya tergantung dari elektron valensinya. Contoh : Biloks gol IA = +1, Biloks gol IIA = +2.

29. apabila 12,8 gram oksigen terdapat dalam 50 gram ZnSO4 tidak murni,maka brapakah kdar kemurnian ZnSO4 tersebut​


Apabila 12,8 gram oksigen terdapat dalam 50 gram ZnSO₄ maka kadar kemurnian ZnSO₄ sebesar 64 %

Diketahui :

massa oksigen = 12,8 gram

massa ZnSO₄   = 50 gram

Ditanya   : kadar senaywa ?

Penyelesaian :

massa ZnSO₄ yang mengandung 12, 8 gram oksigen

Massa oksigen dalam ZnSO₄ = (4 × Ar O) / Mr ZnSO₄ × massa ZnSO₄

                      12,8 gram          = (4 × 16) / 161 × massa ZnSO₄

                      12,8 gram          = 64 / 161 × massa ZnSO₄

                      12,8 gram          = 0,4 × massa ZnSO₄

              12,8 gram / 0,4         = massa ZnSO₄

                        massa ZnSO₄ = 32 gram

Kadar massa ZnSO₄ = (32 gram / 50 gram) × 100 %

                                  = 64 %
