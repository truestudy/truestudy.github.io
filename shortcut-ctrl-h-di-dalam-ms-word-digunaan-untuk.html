15. shortcut ctrl + e di dalam Microsoft Word digunakan untuk


Microsoft Word's Shortcut

ctrl + E dalam microsoft word digunakan untuk meratakan teks menjadi ke tengah

Pembahasan

Kegiatan pengetikan sudah sangat lazim dilakukan banyak orang, karena setiap organisasi dan instansi pasti memiliki berkas - berkas yang harus dicatat dan didokumentasikan. Salah satu software yang memudahkan pengetikan adalah Microsoft Word

Di samping penggunaan software Microsoft Word itu mudah, perangkat lunak buatan penyedia startup Microsoft dilengkapi dengan pintasan - pintasan yang menjadikan proses pengetikan kita menjadi lebih cepat

Di antara pintasan - pintasan itu adalah:

CTRL + A: Memilih seluruh text

CTRL + B: Membuat text  menjadi tebal (bold)

CTRL + C: Copy (Menyalin text)

CTRL + D: Membuka panel format font

CTRL + E: Membuat tulisan jadi di tengah

CTRL + F: Mencari kata  atau kalimat dari teks

CTRL + G: Menuju halaman tertentu

CTRL + H: Mengganti kata/kalimat dengan kata/kalimat lain

CTRL + I: Italic (membuat tulisan menjadi miring)

CTRL + J: Justify (membuat tulisan menjadi rata kiri dan kanan)

CTRL + K: Menambah hyperlink  (pintasan ke website atau berkas yang lain)

CTRL + L: Align left (membuat tulisan rata kiri)

CTRL + M: Membuat paragraf dari kiri

CTRL + N: Membuat file baru

CTRL + O: Membuka file

CTRL + P: Print  (mencetak dokumen)

CTRL + Q: Menghapus editing

CTRL + R: Align right (membuat tulisan rata kanan)

CTRL + S: Menyimpan

CTRL + T: Membuat paragraf menggantung seperti daftar pustaka

CTRL + U: Underline (menambah garis bawah pada text)

CTRL + V: Paste (menaruh teks yang telah disalin "copy" atau "cut")

CTRL + W: Menutup jendela Word

CTRL + X: Cut  (menghapus text sementara untuk dipindahkan ke posisi text yang lainnya)

CTRL + Y: Redo  (kembali melakukan tindakan yang dibatalkan atau telah dilakukan)

CTRL + Z: Undo (kembali ke tindakan awal)

CTRL + SHIFT + C: Copy Format

CTRL + SHIFT + D: Double Underline

CTRL + SHIFT + E: Menjelajah perubahan - perubahan

CTRL + SHIFT + F: Mengganti font

CTRL + SHIFT + H: Menyembunyikan teks

CTRL + SHIFT + K: Membuat semua huruf kapital

CTRL + SHIFT + L: Membuat list

CTRL + SHIFT + M: Menghapus paragraf dari kiri

CTRL + SHIFT + N: Membuat text menjadi style normal

CTRL + SHIFT + P: Mengganti ukuran font

CTRL + SHIFT + Q: Mengganti huruf dengan simbol

CTRL + SHIFT + S: Menerapkan style

CTRL + SHIFT + T: Mengurangi paragraf menggantung

CTRL + SHIFT + V: Paste format

CTRL + SHIFT + W: Underline tanpa spasi

CTRL + SHIFT + >: Memperbesar ukuran text

CTRL + SHIFT + <: Memperkecil ukuran text

CTRL + ]: Memperbesar ukuran text

CTRL + [: Memperkecil ukuran text

CTRL + 1: Jarak enter 1

CTRL + 2: Jarak enter 2

CTRL + 5: Jarak enter 1.5

CTRL + 0: Menghilangkan jarak antar paragraf

CTRL + Home: Menuju halaman utama

CTRL + End: Menuju halaman akhir

CTRL + Enter: Page Break

CTRL + Delete: Menghapus satu kata ke kanan

CTRL + Backspace: Menghapus satu kata ke kiri

CTRL + Tab: Tab

CTRL + Page Up: Menuju halaman sebelumnya

CTRL + Page Down: Menuju halaman selanjutnya

CTRL + Panah kiri: Pindah kata - per- kata dari kiri

CTRL + Panah kanan: Pindah kata - per - kata dari kanan

CTRL + Panah atas: Pindah ke paragraf atas

CTRL + Panah bawah: Pindah ke paragraf bawah

CTRL + ALT + 1: Membuat text menjadi style heading 1

CTRL + ALT + 2: Membuat text menjadi style heading 2

CTRL + ALT + 3: Membuat text menjadi style heading 3

CTRL + ALT + R: Simbol trademark registered ( )

CTRL + ALT + T: Simbol trademark ( )

CTRL + ALT + M: Menambah comment

CTRL + ALT + I: Mengubah tampilan menjadi print preview

CTRL + ALT + S: Membuat tampilan split

Klik Dua Kali: Memilih satu kata

Klik Tiga Kali: Memilih satu paragraf

Klik Margin Kiri: Memilih satu baris dari paragraf

Klik Margin Bawah Dua Kali: Mengatur Footer
