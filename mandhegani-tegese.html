29. Apa tegese tembung : 2. Nyander 7. Waringuten 11. Kesdik 12. Kapupu 13. Tumuli 14. Nggendring 15. Mandhegani


          Ing ngisor iki salah sawjining conto basa. Tembung-tembung ing ngisor iki bakal tak jlentrehake teges-tegese supaya dadi tembung ingkang gampang dipahami.

Pembahasan

           Teges-tegese, antarane :  

Akarya          = Nyiptakake. Antariksa       = Antarala, langit, angkasa. Bathang         = Bang-ke, Bangkai, Barang Sing Wis Bubrah. Cacahe         = Jumlahe, Jumlahnya, Banyaknya. Campuh        = Perang Digdayane     = Kakuwatane, Kemampuannya. Disudheti       = Dibedhahi Diulungake    = Diwenehake, Dikekake, Dipasrahke. Diundha        = Diumbul-umbulake Dredah          = Bertengkar, Berselisih. Gisik              = Pinggir, segara Gumun          = Heran, takjub. Kapupu         = Dipek Kebo ndanu  = Kebo Alasan Kesdik           = Waskitha, Awas Ludira            = Getih, darah. Mandhegani  = Mimpin Midhanget     = Krungu, Dengar. Murakabi       = Nyukupi. Ndasmu         = Kepalamu (Bahasa jawa kasar). Nggendring   = Ora nolah-noleh Ngugemi       = Tetep ati, Megang Janji. Nungkul         = Kalah Nyander        = Mlayu, Nguyak banter Pasulayan     = Pertengkaran, Perselisihan, pertikaian. Pesisir           = Pinggire segara, Pinggiran Pantai. Rubeda         = Alangan, rintangan, pakewuh. Sakojur          = Seluruh badan, kabeh awak. Tumuli           = Banjur Wangsit         = Wisik, Pituduh Waringuten   = Ngamuk Wigih-wigih   = Tidha-tidha, Ragu-Ragu Wilujeng        = Slamet, selamat (Selamat pagi, selamat siang).
