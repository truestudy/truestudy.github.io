2. Soal Persamaan eksponen kelas 10 .. G+H


Bagian g.
- Basis pangkat sama dengan 0
x² + 7x + 10 = 0
(x+2)(x + 5) = 0
Menyebabkan selesaian, x = -2 dan x = -5

- Basis pangkat sama dengan 1
x² + 7x + 10 = 1
x² + 7x + 9 = 0
Dengan rumus ABC akan didapat:
x = 1/2 (-7 + √13)
x = 1/2 (-7 - √13)

- Fungsi pangkat sama:
3x - 2 = 5x - 4
2x = 2
x = 1

HP = {x | 1/2 (-7 - √13), -5, 1/2 (-7 + √13), -2, 1}

bagian h.
- Basis sama dengan 0
x² - x + 1 = 0
(Tidak membentuk penyelesaian)

- Basis sama dengan 1
x² - x + 1 = 1
x² - x = 0
x (x - 1) = 0
x = 0 dan x = 1

- Fungsi pangkatnya sama.
2x² + x - 6 = x² - 2x + 2
x² + 3x - 8 = 0
Dengan rumus ABC:
x = 1/2 (-3+√41)
x = 1/2 (-3-√41)

Menyebabkan:
HP = {x | 1/2 (-3-√41), 0, 1, 1/2 (-3+√41)}

7. persamaan eksponen kelas 10​


Penjelasan dengan langkah-langkah:

h(x)^f(x) = h(x)^g(x)

Syarat - syaratnya :

• f(x) = h(x)

• h(x) = 1

• h(x) = 0 (untuk f(x), g(x) > 0)

• h(x) = -1 (untuk nilai f(x) dan g(x) keduanya genap atau keduanya ganjil)

(x - 3)ˣ⁻⁶ = (x - 3)³

h(x) = x - 3, f(x) = x - 6, g(x) = 3

• untuk f(x) = g(x)

x - 6 = 3

x = 3 + 6

x = 9

• untuk h(x) = 1

x - 3 = 1

x = 1 + 3

x = 4

• untuk h(x) = 0

x - 3 = 0

x = 3

coba kita cek :

f(3) = 3 - 6 = -3 (tidak memenuhi)

karena nilai f(x) < 0, berarti x = 3 tidak memenuhi

• untuk h(x) = -1

x - 3 = -1

x = -1 + 3

x = 2

kita cek :

f(2) = 2 - 6 = -4

g(2) = 3

karena nilai f(x) genap dan g(x) ganjil, berarti nilai x = 2 tidak memenuhi

Jadi, HP = {9, 4}

Pembahasan

Soal Nomor 4]

[tex](\frac{1}{4})^{x - 1} = \sqrt[3]{2^{3x+1}}[/tex]

Penyelesaian :

[tex](\frac{1}{4})^{x - 1} = \sqrt[3]{2^{3x+1}}[/tex]

[tex](\frac{1}{4})^{x - 1} ={2^{(\frac{3x+1}{3})}}[/tex]

[tex](\frac{1}{2^2})^{x - 1} ={2^{(\frac{3x+1}{3})}}[/tex]

[tex](2^{-2})^{(x - 1)} ={2^{(\frac{3x+1}{3})}}[/tex]

[tex](2^{-2x + 2})={2^{(\frac{3x+1}{3})}}[/tex]

Maka :

[tex]-2x + 2 = \frac{3x+1}{3}[/tex]

(-2x + 2) x 3 = 3x + 1

-6x + 6 = 3x + 1

-6x - 3x = 1 - 6

       -9x = -5

           x = [tex]\frac{5}{9} \ \ \boxed{\bold{Jawaban \ C}}[/tex]


Soal Nomor 5]

[tex]2^{2x + 2} = \frac{1}{\sqrt{8^{x+1}}}[/tex]

Penyelesaian :

[tex]2^{2x + 2} = \frac{1}{\sqrt{(2^3)^{(x+1)}}}[/tex]

[tex]2^{2x + 2} = \frac{1}{\sqrt{2^{(3x+3)}}}[/tex]

[tex]2^{2x + 2} = \frac{1}{2^{\frac{(3x+3)}{2}}}[/tex]

[tex]2^{2x + 2} = 2^{(-\frac{3x+3}{2})}[/tex]

[tex]2^{2x + 2} = 2^{(\frac{-3x-3}{2})}[/tex]

Maka :

2x + 2 = [tex]\frac{-3x-3}{2}[/tex]

(2x + 2) x 2 = -3x - 3

        4x + 4 = -3x - 3

       4x + 3x = -3 - 4

               7x = -7

                 x = -7 : 7

                 x = -1 [tex]\boxed{\bold{Jawaban \ B}}[/tex]


Soal Nomor 6]

Perhatikan persamaan kedua untuk mencari nilai x - y

[tex]2^{(x-y)}=16[/tex]

[tex]2^{(x-y)}=2^4[/tex]

Maka :

x - y = 4 [tex]\boxed{\bold{Jawaban \ D}}[/tex]


Soal Nomor 7]

[tex](\sqrt{2})^{6x - 4} = (\frac{1}{4})^{x - 9}[/tex]

Penyelesaian :

[tex](2^{\frac{1}{2}})^{6x - 4} = (\frac{1}{2^2})^{x - 9}[/tex]

[tex](2)^{3x - 2} = (2^{-2})^{x - 9}[/tex]

[tex](2)^{3x - 2} = 2^{-2x + 18}[/tex]

Maka :

3x - 2 = -2x + 18

3x + 2x = 18 + 2

       5x = 20

         x = 20 : 5

         x = 4 [tex]\boxed{\bold{Jawaban \ E}}[/tex]

20. Bantu jawab kk sekalian tentang persamaan Eksponen, Mtk peminatan kelas 10.


Pembahasan Soal

Soal Nomor 1]

[tex]\frac{1}{27^{3x - 7}} =\sqrt{3^{2 - 2x}}[/tex]

Penyelesaian :

[tex]\frac{1}{27^{3x - 7}} =\sqrt{3^{2 - 2x}}[/tex]

[tex]\frac{1}{27^{3x - 7}} = (3^{2 - 2x})^{\frac{1}{2}}[/tex]

[tex]({27^{3x - 7}})^{-1} = (3)^{\frac{2 - 2x}{2}}[/tex]

[tex]27^{-3x + 7}} = (3)^{1 - x}[/tex]

[tex](3^3)^{-3x + 7}} = (3)^{1 - x}[/tex]

[tex]3^{(-9x + 21)} = 3^{(1 - x)}[/tex]

Maka :

-9x + 21 = 1 - x

 -9x + x = 1 - 21

       -8x = -20

           x = [tex]\frac{20}{8}[/tex]

              = [tex]\frac{5}{2}[/tex] [tex]\boxed{\bold{Jawaban \ E}}[/tex]


Soal Nomor 2]

[tex]\frac{(9+\sqrt{5})(2\sqrt{5}+1)}{(\sqrt{5}+1)}[/tex]

Penyelesaian :

= [tex]\frac{(9+\sqrt{5})(2\sqrt{5}+1)}{(\sqrt{5}+1)} \times \frac{(\sqrt{5}-1)}{(\sqrt{5}-1)}[/tex]

= [tex]\frac{(18\sqrt{5}+9+10+\sqrt{5})}{(\sqrt{5}+1)} \times \frac{(\sqrt{5}-1)}{(\sqrt{5}-1)}[/tex]

= [tex]\frac{(18\sqrt{5}+19+\sqrt{5})}{(\sqrt{5}+1)} \times \frac{(\sqrt{5}-1)}{(\sqrt{5}-1)}[/tex] Lakukan Perkalian

= [tex]\frac{90-18\sqrt{5}+19\sqrt{5}-19+5-\sqrt{5}}{(5-1)}[/tex]

= [tex]\frac{76}{4}[/tex]

= 19 [tex]\boxed{\bold{Jawaban \ B}}[/tex]


Soal Nomor 3]

[tex]\frac{2^{(n+2)}\times 6^{(n - 4)}}{12^{(n-1)}}[/tex]

Penyelesaian :

= [tex]\frac{2^{(n+2)}\times 6^{(n - 4)}}{12^{(n-1)}}[/tex]

= [tex]\frac{2^{n}2^{2}\times 6^{n}6^{-4}}{12^{n}12^{-1}}[/tex] Bilangan dengan pangkat negatif ubah dalam bentuk pangkat positif

= [tex]\frac{2^{n}2^{2}\times 6^{n}12^{1}}{12^{n}6^{4}}[/tex]

= [tex]\frac{2^{n}\times4\times 6^{n}\times12}{12^{n}\times1296}[/tex]

= [tex]\frac{(2\times6)^{n}\times(4\times12)}{12^{n}\times1296}[/tex]

= [tex]\frac{12^{n}\times48}{12^{n}\times1296}[/tex]

= [tex]\frac{48}{1296}[/tex]

= [tex]\frac{1}{27}[/tex] [tex]\boxed{\bold{Jawaban \ A}}[/tex]

Penjelasan dengan langkah-langkah:

§ Eksponen §

Nomor 1

[tex] {3}^{2x - 2} = 1[/tex]

Kita ketahui bahwa :

[tex] { \alpha }^{0} = 1[/tex]

Maka nilai dari

[tex]2x - 2 = 0[/tex]

[tex]2x = 0 + 2[/tex]

[tex]2x = 2[/tex]

[tex]x = \frac{2}{2} [/tex]

[tex]{ \boxed{ \underline{x = 1}}}[/tex]

Nomor 2

[tex] {3}^{ \frac{x - 2}{3} } = 1[/tex]

Menggunakan prinsip yang sama bahwa :

[tex] { \alpha }^{0} = 1[/tex]

Maka nilai dari :

[tex] \frac{x - 2}{3} = 0 [/tex]

[tex]x - 2 = 0 \times 3[/tex]

[tex]x - 2 = 0 [/tex]

[tex]{ \boxed{ \underline{x = 2}}}[/tex]

Nomor 4

[tex]3 \sqrt{ {3}^{x - 1} } = 1[/tex]

Maka dapat diketahui bahwa

[tex] \sqrt{ {3}^{x - 1} } = 1 \div 3[/tex]

[tex] \sqrt{ {3}^{x - 1} } = \frac{1}{3} [/tex]

Untuk mengubah dari bentuk akar menjadi bentuk eksponen gunakan sifat :

[tex]{ \boxed{ \bold{ \sqrt[y]{ { \alpha }^{x} } = { \alpha }^{ \frac{x}{y} } }}}[/tex]

Maka bentuk eksponennya adalah :

[tex]{3}^{ \frac{x - 1}{2} } = \frac{1}{3} [/tex]

[tex] \frac{1}{3} [/tex] dapat kita ubah menjadi bentuk eksponen berpangkat min

[tex] {3}^{ \frac{x - 1}{2} } = {3}^{ - 1} [/tex]

Karena bil. pokok sudah sama, lihat bil. pangkatnya

[tex] \frac{x - 1}{2} = - 1[/tex]

[tex]x - 1 = ( - 1) \times 2[/tex]

[tex]x - 1 = - 2[/tex]

[tex]x = - 2 + 1[/tex]

[tex]{ \boxed{ \underline{x = - 1}}}[/tex]

Nomor 6

[tex] {3}^{ - x + {x}^{2} } = 9[/tex]

Untuk mempermudah, ubah nilai dari 9 menjadi bentuk pangkat yaitu 3²

[tex] {3}^{ - x + {x}^{2} } = {3}^{2} [/tex]

Karena bil. pokok sudah sama, lihat bil. pangkatnya

[tex] - x + {x}^{2} = 2[/tex]

Jadikan pers. kuadrat

[tex] {x}^{2} - x - 2 = 0[/tex]

[tex](x + 1)(x - 2) = 0[/tex]

[tex]x + 1 = 0[/tex]

[tex]x - 2 = 0[/tex]

[tex]{ \boxed{ \underline{x_{1} = - 1}}}[/tex]

[tex]{ \boxed{ \underline{x_{2} = 2}}}[/tex]
