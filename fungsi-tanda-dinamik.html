6. Fungsi tanda tempo,dinamik dan ekspresi


Tanda Dinamik adalah keras lembut lagu dan perubahannya.Tanda dinamik dibagi menjadi 3, yaitu sebagai berikut :

1.)Tanda dinamik lembut, misalnya

a.Piano (P) = Lembut

b.Pianissimo (PP) = Sangat lembut

2.)Tanda dinamik sedang, misalnya :

a.Mezzo Piano (MP) = Agak lembut

b.Mezzo Fotre (MF) = Agak keras

3.)Tanda dinamik keras, misalnya :

a.Forte (F) = Keras

b.Fortissimo (FF) = Sangat keras

Untuk menunjukkan perubahan tempo, dipakai istilah sebagai berikut.

1.)Cressendo (cresc) <, artinya berangsur" makin keras

2.)Decressendo (decresc) >, artinya berangsur" makin lembut

3.)Subito Forte (sf), artinya tiba" keras

4.)Subito Piano (sp), artinya tiba" lambat

Tanda tempo adalah cepat atau lambatnya sebuah lagu.Tanda tempo dibagi menjadi 3, yaitu tempo lambat, sedang dan cepat.

1.)Tanda Tempo Lambat

11. tanda dinamik adalah​


Penjelasan:

Tanda Dinamik adalah Tanda untuk menyatakan keras, lembutnya sebuah lagu yang dinyanyikan.

Contoh Tanda Dinamik :

1. f ( forte ) : keras

2. ff ( fortissimo ) : sangat keras

3. fff ( fortissimo assai ) : sekeras mungkin

4. mf ( mezzo ferto ) : setengah keras

5. fp ( forte piano ) : mulai dengan keras diikuti dengan lembut

6. p ( piano ) : lembut

7. pp ( pianissimo ) : sangat lembut

8. ppp ( pianissimo possibile ) : selembut mungkin

9. mp ( mezzo piano ) : setengah lembut
