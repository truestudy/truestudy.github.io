1. Jika panjang busur AB 15cm,AB 15cm ,maka panjang busur BC adalah


Jawab:

Besar sudut BOC = 180 - 135 = 45. sudut BOC berpelurus dengan sudut AOB.

Panjang busur BOC = ?

[tex]\frac{besar sudut AOB}{Besar sudut BOC} = \frac{Panjang busur AOB}{Panjang Busur BOC}\\ \\\frac{135}{45} = \frac{51}{x}[/tex]

[tex]\frac{3}{1} = \frac{51}{x}[/tex]

3x = 51

x = 17.

panjang busu BOC adalah 17 cm

14. Jika panjang busur AB = 32cm, berapakah panjang busur CD?


Panjang busur CD adalah 21,3 cm.

Rumus:

[tex]\frac{\text{sudut A}}{\text{sudut B}} =\frac{\text {panjang busur A}}{\text{panjang busur B}}[/tex]

Penjelasan dengan langkah-langkah:

Diketahui:

panjang busur AB = 32 cm

Ditanya:

Berapakah panjang busur CD?

Jawab:

[tex]\frac{\text{sudut A}}{\text{sudut B}} =\frac{\text {panjang busur A}}{\text{panjang busur B}}[/tex]

Sudut AOB adalah siku-siku, maka besar sudutnya adalah 90°.

Sehingga diperoleh,

[tex]\frac{60^o}{90^o} =\frac{\text {panjang busur CD}}{32}[/tex]

panjang busur CD =  [tex]\frac{60\times 32}{90}[/tex]

                               =  [tex]\frac{1920}{90}[/tex]

                               = 21,3

Jadi, panjang busur CD adalah 21,3 cm.

23. tentukan panjang busur pada lingkaran berikut 1diketahui panjang busur ABO adalah=45° dan sudut AB 28cm tentukan panjang busur AB​


Jawaban:

Tentukan panjang busur pada lingkaran berikut:

1. Panjang busur CD = ...

∠COD = 72°

OD = 20 cm

2. Panjang busur KL = ...

∠KOL = 90°

OD = 14 cm

3. Panjang busur AB = ...

∠AOB = 45°

OD = 28 cm

PENYELESAIAN

Ingat rumus mencari panjang busur sebuah lingkaran:

Panjang Busur = (Besar Sudut Juring/360°) x Keliling Lingkaran

Panjang Busur = (Besar Sudut Juring/360°) x πd

Dengan menggunakan rumus panjang busur di atas, maka penyelesaian untuk permasalahan di atas adalah sebagai berikut:

1. Panjang busur CD

∠COD = 72°

OD = 20 cm

Panjang busur CD = (∠COD/360°) x πd

Panjang busur CD = (72°/360°) x 3.14 x 20

Panjang busur CD = 0.2 x 3.14 x 20

Panjang busur CD = 12.56 cm

2. Panjang busur KL

∠KOL = 90°

OD = 14 cm

Panjang busur KL = (∠KOL/360°) x πd

Panjang busur KL = (90°/360°) x 22/7 x 14

Panjang busur KL = 0.25 x 22/7 x 14

Panjang busur KL = 11 cm

3. Panjang busur AB

∠AOB = 45°

OD = 28 cm

Panjang busur AB = (∠AOB/360°) x πd

Panjang busur AB = (45°/360°) x 22/7 x 28

Panjang busur AB = 0.125 x 22/7 x 28

Panjang busur AB = 11 cm
