20. sebutna gunane sandhangan swara​


Wangsuli pitakon

Gunane sandhangan swara kanggo menehi swara liya ( i, u, e, é lan o)

__________________Pembahasan

Aksara Jawa iku tegese aksara kang ditulis nganggo tulisan Jawa, nanging aksara Jawa uga bisa diarani aksara legena utawa aksara carakan, amarga durung kawuwuhan pasangan lan sandhangan apa apa.

tuladhane :

ha, na, ca, ra, kada, ta, sa, wa, lapa, dha, ja, ya, nyama, ga, ba, tha, nga,

Supaya bisa muni ing swara liya, aksara Jawa uga mbutuhake sandhangan swara.

Sandhangan swara cacahe ana 5 :

1. wulu unine i ➡️ꦶ

tuladha :

bibi = ꦧꦶꦧꦶnini = ꦤꦶꦤꦶ

2. suku unine u ➡️ꦸ

tuladha :

suku = ꦱꦸꦏꦸbuku = ꦧꦸꦏꦸ

3. pepet unine e ➡️ꦼ

tuladha :

sepet = ꦱꦼꦥꦼꦠ꧀cupet = ꦕꦸꦥꦼꦠ꧀

4. taling unine é ➡️ꦺ

tuladha :

sate = ꦱꦠꦺkere = ꦏꦺꦫꦺ

5. taling tarung unine o ➡️ꦴꦺ

tuladha :

coro = ꦕꦺꦴꦫꦺꦴsoto = ꦱꦺꦴꦠꦺꦴ

✓ Aksara murda ora kena dadi sesigeging wanda (aksara mati pada suku kata), dadi ora kena dipangku lan ora bisa dipasangi.
