1. fungsi linear berikut yang memotong sumbu x adalah


Kelas: 11
Mapel: Matematika
Kategori: Fungsi
Kata Kunci:fungsi, fungsi linear
Kode: 11.2.6 (Kelas 11 Matematika Bab 6-Fungsi)

Suatu fungsi akan memotong sumbu x jika y=f(x)=0.

A. f(x)=-2x+4
0=-2x+4
2x=4
x=4/2
x=2

x=2 tidak terletak pada Df={x|x≤1,x∈R}

B. f(x)=-2x-6
0=-2x-6
2x=-6
x=-6/2
x=-3

x=-3 terletak pada Df={x|x≤2,x∈R}

C. f(x)=2x-8
0=2x-8
2x=8
x=8/2
x=4

x=4 tidak terletak pada Df={x|x≤2,x∈R}

D. f(x)=3x+6
0=3x+6
3x=-6
x=-6/3
x=-2

x=-2 tidak terletak pada Df={x|x≥-1,x∈R}

E. f(x)=3x-6
0=3x-6
3x=6
x=6/3
x=2

x=2 terletak pada Df={x|x≥-1,x∈R}

Jadi, jawaban yang benar adalah B dan E

3. Fungsi linear berikut yang memotong sumbu x dan sumbu y adalah


kelas : X SMA
mapel : matematika
kategori : persamaan linier
kata kunci : persamaan linier memotong sumbu x dan y

kode : 10.2.3 [matematika SMA kelas 10 Bab 3 persamaan linier]

Pembahasan:

persamaan linier memotong sumbu x apabila y = 0, dan memotong sumbu y apabila x = 0
dari 5 fungsi yang disajikan kita lihat domainnya , jika domainnya ada 0 maka fungsi tersebut memotong sumbu sumbu y

a) domainnya {-3 ≤ x ≤ 2, x∈ R} → dari domain ini ada x = 0 , berarti fungsi pilgan a memotong sumbu y

b) domainnya {1 ≤ x ≤ 6, x ∈ R} → dari domain ini tidak ada x = 0, berarti fungsi pilgan b tidak memotong sumbu y

c) domainnya {2 ≤ x ≤ 5, x ∈ R} → dari domain ini tidak ada x = 0, berarti fungsi pilgan c tidak memotong sumbu y

d) domainnya {-1 ≤ x ≤ 4, x ∈ R} → dari domain ini ada x = 0, berarti fungsi pada pilgan d memotong sumbu y

e) domainnya {-1 ≤ x ≤ 4, x ∈ R} → dari domain ini ada x = 0, berarti fungsi pada pilgan e memotong sumbu y

karena pertanyaan memotong sumbu x dan y, 
kita cari lagi yang memotong sumbu x, namun tidak semuanya kita cek, hanya yang memenuhi, atau yang memotong sumbu y saja yang kita cek, yaitu pilgan a, d dan e

pada pilgan a
memotong sumbu x apabila y = 0
h(x) = x + 6
0 = x + 6
x = -6
-6 tidak ada dalam domain

pada pilgan d
memotong sumbu x apabila y = 0
h(x) = -2x + 4
0 = -2x + 4
2x = 4
x = 4/2
x = 2
2 ada dalam domain, jadi fungsi ini memotong sumbu x

pada pilgan e
memotong sumbu x apabila y = 0
h(x) = -2x -4
0 = -2x -4
2x = -4
x = -4/2
x = -2
-2 tidak ada pada domain, 


jadi jawaban soal ini adalah D
h(x) = -2x + 4, dengan Dh = {x I -1 ≤ x ≤ 4, x ∈ R}

Apakah memotong sumbu x?

[tex] \: \: \: \: \: \: \: \: f(x) = - 2x + 4 \\ \: \: \: \: \: \: \: \: \: 0 \: \: \: = - 2 + 4 \\ 2x = 4 \\ \: \: x = 2[/tex]

Kesimpulan

x = 2 tidak ada di Domain fungsi { x | x ≤ 1, x € R }.Tidak memotong sumbu x

[tex] \: \: \: \: \: \: \: f(x) = - 2x - 6 \\ \: \: \: \: \: \: \: \: \: \: \: \: \: 0 = - 2x - 6 \\ \: 2x = - 6 \\ \: \: \: x = - 3[/tex]

Kesimpulan:

x = -3 ada di domain fungsi { x | x ≤ - 2, x € R }.Memotong sumbu x.

x < 2 jika dibaca adalah x kurang dari 2. Maka jika dijabarkan, akan membentuk { ..., 0, 1 }.

x ≤ 2

x ≤ 2 jika dibaca adalah x kurang dari sama dengan 2. Maka jika dijabarkan, akan membentuk { ..., 0, 1, 2 }.

x > 2

x > 2 jika dibaca adalah x lebih dari 2. Maka jika dijabarkan akan membentuk { 3, 4, 5, ... }

x ≥ 2

x ≥ 2 jika dibaca adalah x kurang dari sama dengan 2. Maka jika dijabarkan akan membentuk { 2, 3, 4, 5, ... }

Kegunaan simbol-simbol pertidaksamaan

Seperti jawaban di pertanyaan ini. Simbol-simbol pertidaksamaan dapat menemukan perpotongan antara sumbu x dan letak suatu titik. Simbol-simbol pertidaksamaan ada di domain fungsi. Domain adalah daerah awal/ asal dari suatu fungsi.
