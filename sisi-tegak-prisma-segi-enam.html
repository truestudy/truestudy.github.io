11. Sebutkan sisi : rusuk : sisi tegak : titik sudut : prisma segi enam tlng jawab


Jawaban:

bangun prisma adalah bangun ruang sisi datar yang memiliki alas dan tutup yang sama bentuknya.

Pembahasan

banyak sisi, rusuk dan titik sudut pada bangun prisma

banyak sisi:

banyak sisi pada prisma segi-n dapat dirumuskan sebagai n + 2

prisma segitiga = 3 + 2 = 5

prisma segiempat = 4 + 2 = 6

prisma segilima = 5 + 2 = 7

prisma segienam = 6 + 2 = 8

prisma segi-12 = 12 + 2 = 14

prisma segi - n = n + 2

banyak rusuk:

banyak rusuk pada prisma segi-n dapat dirumuskan sebagai 3n

prisma segitiga = 3 x 3 = 9

prisma segiempat = 3 x 4 = 12

prisma segilima = 3 x 5 = 15

prisma segienam = 3 x 6 = 18

prisma segi-12 = 3 x 12 = 36

prisma segi - n = 3 x n = 3n

banyak titik sudut:

banyak titik sudut pada prisma segi-n dapat dirimuskan sebagai 2n

prisma segitiga = 2 x 3 = 6

prisma segiempat = 2 x 4 = 8

prisma segilima = 2 x 5 = 10

prisma segienam = 2 x 6 = 12

prisma segi-12 = 2 x 12 = 24

prisma segi - n = 2 x n = 2n
