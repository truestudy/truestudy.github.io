Negesi Paribasan.

1.anak polah bapa kepradhah = tingkah polahe anak dadi tanggung jawabe wong tuwane.

2.mbidhug api rowang = ethog ethog arep nulungi, nanging sejatine arep ngrusuhi.

3.rubuh-rubuh gedhang = wong sing melu Melu wae, nanging ora duwe tujuwan.

4.kadang konang = gelem ngakoni seduluran, nanging karo sing sugih wae.

5.adigang,adigung,lan adiguna = ngendhelake kekuwatane, keluhurane lan kepinterane.

6.ana catur mungkur = ora gelem ngrungokake rerasanan kang ora becik.

➡️ B. Lathi.

PenjelasanAjining dhiri gumantung saka lathi iku kalebu unen-unen jawa (Paribasan).

☞ Paribasan

» Unen-unen sing ajeg panganggone, ngemu surasa wantah.

—☆ Tuladha :

1. Kebat kliwat, gancang pincang.

✧Tegese : tumindak kanthi kesusu, asile ora bakal maremake.

2. Anak polah bapak kepradhah.

✧Tegese : tingkah polahe anak dadi tanggung jawabe wong tuwane.

3. Adigang, adigung, adiguna.

✧Tegese : ngendhalekake kekuwatane, keluhurane, lan kepinterane.

4. Yitna yuwana lena kena.

✧Tegese : sing ngati-ati bakal slamet, sing sembrana bakal cilaka.

5. Sadumuk bathuk sanyari bumi ditohi pati.

✧Tegese : prakara ngenani wanita utawa lemah, dilabuhi tekane pati.

6. Ajining dhiri gumantung saka lathi.

✧Tegese : Kapribadening manungsa ditemtokake saka omongane dhewe-dhewe.
