3. 8 pangkat 1 di tambah 8 pangkat 0​


Jawab:

9

Penjelasan dengan langkah-langkah:

[tex]8^{1} +8^{0} = 8+1 = 9[/tex]

Hasil dari [tex] \rm 8^1+8^0[/tex] adalah 9.

Pendahuluan :

Sifat-sifat Perpangkatan :

[tex]1) \: {a}^{0} = 1 [/tex]

[tex]2) \: {a}^{m} \times {a}^{n} = {a}^{m + n} [/tex]

[tex]3) \: {a}^{m} \div {a}^{n} = {a}^{m - n} [/tex]

[tex]4) \: ( {a}^{m} ) ^{n} = {a}^{m \times n} [/tex]

[tex]5) \: ({a \times b})^{m} = {a}^{m} \times {b}^{m} [/tex]

[tex]6) \: ( {a \div b})^{m} = {a}^{m} \div {b}^{m} [/tex]

[tex]7) \: {a}^{m} = \frac{1}{ {a}^{ - m} } [/tex]

[tex]8) \: {a}^{ - m} = \frac{1}{ {a}^{m} } [/tex]

[tex]9) \: {a}^{ \frac{m}{n} } = \sqrt[n]{ {a}^{m} } [/tex]

[tex] 10) \: b^{\frac{c}{a}} = \sqrt[a]{b^c}[/tex]

Pembahasan :

Diketahui :

[tex] \rm 8^1+8^0[/tex]

Ditanya :

Hasilnya?

Jawab :

Gunakan sifat perpangkatan nomor (1) untuk pangkat nol :

[tex] \rm 8^1+8^0[/tex]

[tex] \rm = 8 +1[/tex]

[tex] \bf = 9[/tex]

Kesimpulan :

Jadi, hasilnya adalah 9.
