Pembahasan

1. Pada pola bilangan persegi panjang, banyak titik pada pola ke-11 adalah ....

Jawab

Rumus suku ke n pada pola persegi panjang adalah

Un = n × (n + 1)

Jadi banyak titik pada pola ke 11 adalah

U₁₁ = 11 × (11 + 1)

U₁₁ = 11 × 12

U₁₁ = 132

Jawaban B

2. Jumlah bilangan pada baris ke-9 dari pola bilangan segitiga Pascal adalah ....

Jawab

Rumus jumlah bilangan pada baris ke n dari pola bilangan segitiga pascal adalah

Sn = 2ⁿ⁻¹

Jadi jumlah bilangan pada baris ke-9 dari pola bilangan segitiga Pascal adalah

S₉ = 2⁹⁻¹

S₉ = 2⁸

S₉ = 256

Jawaban C

3. Suku ke-25 dari barisan bilangan 2, 5, 8, 11, 14,.... adalah ....

Jawab

Barisan tersebut adalah barisan aritmatika karena selalu bertambah 3 dan barisan ini memiliki rumus suku ke n yaitu

Un = a + (n – 1)b

dengan

a = Suku pertama ⇒ a = 2

b = Beda ⇒ b = 5 – 2 = 3

Jadi suku ke 25 dari barisan tersebut adalah

Un = a + (n – 1)b

U₂₅ = 2 + (25 – 1)3

U₂₅ = 2 + (24)3

U₂₅ = 2 + 72

U₂₅ = 74

27. pada pola barisan bilangan persegi panjang tentukan banyaknya bola bola pada bilangan ke 50​


[tex]\boxed{\boxed{\bold{Pola~Bilangan}}}[/tex]

______________________________

Pola bilangan adalah susunan angka angka yang membentuk suatu pola tertentu

Beberapa pola bilangan dan rumusnya

• Pola bilangan genap

=> Un = 2n

• Pola bilangan ganjil

=> Un = 2n - 1

• Pola bilangan persegi

=> Un = n²

• Pola bilangan persegi panjang

=> Un = n(n + 1)

• Pola bilangan segitiga

=> Un =[tex]\dfrac{n(n + 1)}{2}[/tex]

______________________________

Soal :

Tentukan banyak bola pada bilangan ke 50 dari pola bilangan persegi panjang !

Penyelesaian :

Un = n ( n + 1 )

U50 = 50(50 + 1)

U50 = 50(51)

U50 = 2.550

Maka banyak bola pada bilangan ke 50 adalah 2.550
