17. untuk mengganti nama control label yang diambil dari form control pada properties, maka tempat penulisannya adalah di...​


Penjelasan:

Tempat untuk mengganti nama control label yang diambil dari form control pada properties tergantung pada platform atau aplikasi yang digunakan. Namun, umumnya penggantian nama control label dapat dilakukan pada bagian Properties atau Properti pada editor desain.

Sebagai contoh, jika Anda menggunakan Microsoft Visual Studio untuk membuat sebuah form, maka untuk mengganti nama control label, langkah-langkahnya sebagai berikut:

1. Buka project Anda di Microsoft Visual Studio.

2. Buka form yang ingin Anda ubah.

3. Klik pada control label yang ingin Anda ganti nama labelnya.

4. Buka Properties Window dengan menekan tombol F4.

5. Di bagian Properties Window, cari properti bernama Text.

6. Ubah nilai dari properti Text tersebut menjadi nama label baru yang diinginkan.

Jadi, tempat untuk mengganti nama control label yang diambil dari form control pada properties adalah pada bagian Properties atau Properti pada editor desain.

21. Syntax unload me digunakan untuk. A. Tidak melakukan unload b. Mengabaikan form c. Tidak melakukan perintah d. Menutup form​.


Jawaban:

D. Menutup form

Syntax unload me digunakan untuk menutup form pada aplikasi Visual Basic. Syntax ini berfungsi untuk menghentikan proses eksekusi pada form tersebut, sehingga form tersebut tidak ditampilkan lagi pada layar. Dengan menggunakan syntax unload me, kita dapat menutup form yang sedang aktif, sehingga aplikasi dapat memindahkan proses ke form lain yang diinginkan. Syntax ini biasanya digunakan dalam kondisi tertentu, misalnya ketika pengguna mengklik tombol keluar atau tutup pada aplikasi.

27. Bagaimana cara membuat judul form dalam header form


Pilih baris pada awal tabel yang akan dijadikan Judul tabel.
Selanjutnya Pada menu table tools, pilih ribbon Layout
Klik tombol Properties yang terletak pada bagian kiri
Langkah 2 dan 3 di atas dapat anda ganti dengan cara:
klik tombol kanan mouse pada tabel.
Pilih item Table properties
Selanjutnya anda akan diberikan sebuah dialog Table Properties.
Pada dialog tersebut pilih tab Row
Berikan tanda check pada opsi “Repeat as header row at the top of each page” untuk membuat baris yang telah Anda pilih tadi akan diulangi pada bagian atas di setiap awal halaman yang baru (selama masih ada tabelnya).
Klik OK untuk menyimpan konfigurasi tabel.
Bila tabel anda sangat panjang dan membutuhkan lebih dari satu halaman, anda akan melihat bahwa baris yang telah anda pilih tadi akan menjadi baris di setiap awal halaman.
