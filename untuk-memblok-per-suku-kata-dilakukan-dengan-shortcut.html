2. shortcut key untuk memblok semua tulisan adalah


Shortcut key yang digunakan untuk memblok semua tulisan pada sebuah halaman adalah CTRL+A. Tombol Ctrl pada keyboar ditekan bersamaan dengan tombol A.

Shortcut key merupakan cara cepat menggunakan gabungan dua tombol atau lebih pada keyboard. Untuk menggunakan shortcut key, tombol pada keyboard harus ditekan secara bersamaan. Beberapa contoh shortcut key yang cukup sering digunakan adalah

CTRL + C, untuk menyalin (copy) baik tulisan, gambar atau file.CTRL + V, untuk menempel (paste) dari tulisan, gambar atau file yang sudah disalin sebelumnya. Shortcut key ini juga dapat digunakan untuk menempel dari perintah cut.CTRL + X, untuk memotong (cut) tulisan atau file.

Pembahasan

Beberapa shortcut key yang sama, dapat digunakan pada beberapa aplikasi yang berbeda. Namun selain menggunakan kombinasi dua tombol yang ditekan bersamaan, beberapa shortcut key merupakan kombinasi dari tiga tombol pada keyboard. Berikut ini beberapa contoh shortcut key dengan tiga tombol keyboard.

CTRL+Shift+N, untuk membuat folder baru saat di File Explorer.CTRL+ALT+Tab, shorcut key ini berfungsi untuk membuka atau berpindah antar aplikasi yang sedang aktif.CTRL+ALT+DELETE, untuk membuka beberapa menu, salah satunya adala Task Manager.Tombol Windows + Shift + V, berfungsi untuk membuka notifikasi pada sistem.Tombol Windows + S, untuk membuka menu search atau pencarian.Tombol Windows + E, berfungsi untuk membuka File Explorer.
