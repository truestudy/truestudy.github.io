14. Harga 3 pensil dan 6 pulpen adalah 15.000.00 sedangkan harga 2 pensil dan 3 pulpen 8.000.00 harga 2 pensil dan 4 pulpen adalah...


Jawaban:

2x + 4y = 10000

Penjelasan dengan langkah-langkah:

Misal:

pensil = x

pulpen = y

Diketahui:

3 pensil + 6 pulpen = 15000 ~ 3x + 6y = 15000

2 pensil + 3 pulpen = 8000 ~ 2x + 3y = 8000

Ditanya:

2 pensil + 4 pulpen = ...? ~ 2x + 4y = ...?

Jawab:

Langkah 1: Eliminasi variabel agar salah satu hilang atau menghasilkan nol

Pada langkah ini akan dihilangkan variabel x agar mendapat nilai dari variabel y, dengan cara mengalikan kedua persamaan

3x + 6y = 15000 .... dikali 2

2x + 3y = 8000 .... dikali 3

hasilnya menjadi:

6x + 12y = 30000

6x + 9y = 24000

eliminasi kedua persamaan tersebut dengan menggunakan operasi pengurangan agar variabel x menjadi nol. maka,

3y = 6000

y = 6000/3

y = 2000

diperoleh, y = 2000

Langkah 2: substisusi variabel y ke salah satu persamaan awal

misal akan disubtisusi variabel y ke dalam persamaan 2x + 3y = 8000, maka

2x + 3(2000) = 8000

2x + 6000 = 8000

2x = 8000 - 6000

2x = 2000

x = 2000/2

x = 1000

telah diperoleh y = 2000, x = 1000

Ditanya: hasil dari 2x + 4y

maka masukan masing masing nilai x dan y yang telah diperoleh sebelumnya. sehingga

2(1000) + 4(2000) = 2000 + 8000 = 10000
