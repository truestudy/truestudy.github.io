Jawaban:

- kelip + (-er-) = kerelip = krelip (Lampune pating krelip.)

- jerit + (-el-) = jelerit = jlerit (Bocah-bocah padha dolalan ing latar pating jlerit.)

- kelep + (-el-) = kelelep = klelep (Agus klelep amargo ora bisa nglangi.)

- kampul + (-er-) = kerampul = krampul (Sampahe pating krampul ana ing kali.)

- keleg + (-el-) = keleleg = kleleg (Isi salak iku kleleg dening adiku.)

- cekot + (-er-) = cerekot = crekot (Sirahku mumet pating crekot.)
