22. [tex]\frac{4}{3} +\frac{2}{12}[/tex]================thank you for everything that you do for me...


♢ Pertanyaan :

Hasil dari 4/3 + 2/12 adalah ... .

♢ Penyelesaian :

Samakan Pecahan bagian bawahDan kelikan bagian atas

Maka,

= 4/3 + 2/12

= 4/3 + (2 ÷ 2/12 ÷ 2)

= 4/3 + 1/6

= (4 × 2/3 × 2) + 1/6

= 8/6 + 1/6

= 8 + 1/6

= [tex]\boxed{\green{\sf{1\frac{3}{6} }}}[/tex]

♢ Kesimpulan :

Jadi, jawabanya adalah Diatas.

Jawaban:

[tex] \frac{4}{3} + \frac{2}{12} [/tex]

[tex] \frac{9}{6} [/tex]

[tex]

a^{m} \times a^{n} = a^{m+n}\\

a^{m} \div a^{n} = a^{m-n}\\

\frac{a^{m}}{a^{n}} = a^{m-n}\\

\sqrt[n]{a^{m}} = a^{\frac{m}{n}}\\

[/tex]

[tex] \frac{ {f}^{ - \frac{1}{2} } \times {f}^{4} }{ {f}^{0} \times \sqrt{f} \div {f}^{ - 2} } \\ = \frac{ {f}^{ - \frac{1}{2} } \times {f}^{4} }{ {f}^{0} \times {f}^{ \frac{1}{2} } \div {f}^{ - 2} } \\ = \frac{ {f}^{ - \frac{1}{2} + 4}}{ {f}^{0 + \frac{1}{2} - 2} } \\ = \frac{ {f}^{ - \frac{1}{2} + \frac{8}{2} } }{ {f}^{ \frac{1}{2} - \frac{4}{2} } } \\ = \frac{ {f}^{ \frac{7}{2} } }{ {f}^{ - \frac{3}{2} } } \\ = {f}^{ \frac{7}{2} - ( - \frac{3}{2} )} \\ = {f}^{ \frac{7}{2} + \frac{3}{2} } \\ = {f}^{ \frac{10}{2} } \\ = {f}^{5} [/tex]
