Integral 25x(5x-3)^4 dx

Pembahasan

∫u dv = uv - ∫v du

∫ 25x (5x - 3)^4 dx = ...

u = 25x

du = 25 dx

dv = (5x - 3)^4 dx

v = ∫(5x - 3)^4 dx = 1/20 (5x-3)^5

∫u dv = uv - ∫v du

∫25x (5x - 3)^4 dx = (25x)(1/20 (5x - 3)^5) - ∫1/20(5x - 3)^5 (25 dx)

= (25/20)(x)(5x - 3)^5 - (25/20)(1/30)(5x - 3)^6 + C

= (5/4)(x)(5x - 3)^5 - (1/24)(5x - 3)^6 + C

16. integral x(5x^2 +3)^4 dx


Quick Tips!

⇒ ∫ aⁿ dx = aⁿ⁺¹ / n + 1

=====================

Integral dari ∫ x(5x² + 3)⁴ dx adalah ...

Misalkan :

u = 5x² + 3

du = 10x

dx = du/10x

[tex] \int\limits {x(5x^{2} + 3)^4} \, dx = \int\limits {x \cdot u^4} \, dx \\\\\
= \int\limits {x \cdot u^4} \cdot \frac{du}{10x} \\\\
= \frac{1}{10} \cdot \frac{u^{4 + 1}}{4 + 1} + c \\\\
= \frac{1}{10} \cdot \frac{1}{5}u^5 + c\\\\
= \frac{1}{50}u^5 + c\\\\
\therefore \boxed{\frac{1}{50}(5x^2 + 3)^5 + c} [/tex]

===================================

b. [tex]\int (4x^{8}+2x^{5}+3)dx=\frac{4}{9}x^{9}+\frac{1}{3}x^{6}+3x+C[/tex]
c. [tex]\int \frac{10}{x^{4}}dx=\int 10x^{-4}dx=-\frac{10}{3}x^{-3}+C=-\frac{10}{3x^{3}}+C[/tex]
d. [tex]\int \frac{(5x-4)}{x^{3}}dx[/tex]

=> [tex]\int udv=uv-\int vdu[/tex]
=> [tex]\int \frac{5x-4}{x^{3}} = -(\frac{1}{2x^{2}})(5x-4)-\int -\frac{1}{2x^{2}}(5)dx[/tex]
=> [tex]-\frac{5x-4}{2x^{2}}+\int \frac{5}{2x^{2}}dx[/tex]
=> [tex]-\frac{5x-4}{2x^{2}}-\frac{5}{2x^{2}}+C=\frac{-5x+1}{2x^{2}}+C[/tex]
e. [tex]\int x^{4}\sqrt{x}dx[/tex]

=> [tex]\int udv=uv-\int vdu[/tex]
=> [tex]\int x^{4}\sqrt{x}dx=\frac{2}{3}x^{5}\sqrt{x}-\int \frac{2}{3}x\sqrt{x}(4x^{3}})dx[/tex]
=> [tex]\frac{2}{3}x^{5}\sqrt{x}-\frac{8}{3}\int x^{4}\sqrt{x}dx[/tex]
