29. Sebuah benda berbentuk kubus denganlubang berbentuk kerucut.Berapakah volume benda tersebut?​


Jawab:

6.835,5cm³

Penjelasan dengan langkah-langkah:

Volume benda: [ Vkubus (sisi x sisi x sisi) - VKerucut (1/3 x π x r² x t)]

- π = 22/7

- r = 21 : 2 = 10,5cm

> (21cm x 21cm x 21cm) - ( 1/3 x 22/7 x 10,5cm x 10,5cm x 21cm)

> 9.261cm³ - 2.425,5cm³

> 6.835,5cm³

Jadi volume benda tersebut ialah 6.835,5cm³
