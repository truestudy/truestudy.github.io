29. Urutkan Bilangan Desimal Berikut Mulai Dari Yang Terbesar! (beserta caranya!)1.) 0, 35, 0, 65, 1, 30, 1, 65Jawab:2.) 7, 2, 6, 32, 5, 63, 5, 36Jawab:3.) 0, 5, 0, 25, 1, 05, 0, 06Jawab:Tolong:)​


Caranya adalah melihat Ada di depan koma 0 , 1 atau berapapun, jika 0 berarti lebih kecil daripada satu seperti angka biasa saja, baru lihat angka di belakang koma misalnya 0,35 Dan 0,65 jelas terlihat 0,65 lebih besar. Sehingga mengurutkan desimal hanya perlu dilihat dengan hati hati saja seperti 0,7 Dan 0,65 yang lebih besar adalah 0,7 mengapa? Karena 0,7 jika ditambah 0 menjadi 0,70 dan 0,65 jelas lebih besar 0,7 tambahkan saja 0 jika angka tersebut berbeda jumlah nominalnya.

1. 0,35 ; 0,65 ; 1,30 ; 1,65

Sehingga urutan dari yang terbesar :

1,65 ; 1,30 ; 0,65 ; 0,35

2. 7,2 ; 6,32 ; 5,63 ; 5,36

Agar kita tidak bingung melihatnya maka tambahkan satu nol pada 7,2

7,20 ; 6,32 ; 5,63 ; 5,36

Sehingga urutan dari yang terbesar :

7,2 ; 6,32 ; 5,63 ; 5,36

3. 0,5 ; 0,25 ; 1,05 ; 0,06

Tambahkan nol ( tidak juga tidak apa apa asalkan teliti )

0,50 ; 0,25 ; 1,05 ; 0,06

Sehingga urutan dari yang terbesar :

1,05 ; 0,5 ; 0,25 ; 0,06
