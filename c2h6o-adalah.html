4. Tuliskan isomer fungsi dari senyawa yang memiliki rumus molekul C2H6O


Isomer senyawa merupakan senyawa yang memiliki rumus struktur yang berbeda-beda dengan rumus molekul yang sama. Isomer ada beragam jenisnya, salah satunya adalah isomer fungsi. Isomer fungsi dari C₂H₆O adalah:

1.CH₃-CH₂-OH

Nama IUPAC: etanol

Nama trivial: etil alkohol

2.CH₃-O-CH₃

Nama IUPAC: metoksi metana

Nama trivial: dimetil eter

Pembahasan

Salah satu jenis isomer senyawa turunan alkana adalah isomer fungsi. Isomer fungsi adalah kelompok senyawa yang memiliki rumus kimia sama, tetapi memiliki gugus fungsi yang berbeda. Kelompok senyawa yang berisomer gugus fungsi adalah:

Alkohol dan eter, dengan rumus umum molekul CₙH₂ₙ₊₂OAldehid dan keton, dengan rumus umum molekul CₙH₂ₙOAsam karboksilat dan ester, dengan rumus molekul CₙH₂ₙO₂

Nah, untuk senyawa C₂H₆O memiliki rumus umum molekul CₙH₂ₙ₊₂O. Jadi, isomer fungsinya adalah alkohol (-OH) dan eter (-O-), yaitu:

1.CH₃-CH₂-OH

Nama IUPAC: etanol

Nama trivial: etil alkohol

2.CH₃-O-CH₃

Nama IUPAC: metoksi metana

Nama trivial: dimetil eter

Berikut penjelasan isomer dari senyawa turunan alkana.

Alkohol

Alkohol dengan gugus fungsi -OH, memiliki isomer posisi, isomer optis, dan isomer fungsi dengan eter. Isomer posisi adalah yaitu senyawa yang mempunyai rumus molekul dan gugus fungsi sama, tetapi posisi gugus fungsi yang berbeda.

Isomer optis terjadi pada senyawa yang mempunyai atom karbon tidak simetris, disenut sebagai senyawa kiral atom karbon C tidak simetris maksudnya adalah atom C yang mengikat empat gugus yang berbeda. Sedangkan senyawa kiral adalah senyawa yang tidak identik atau tidak dapat diimpitkan dengan bendanya sendiri.

Eter

Eter memiliki gugus fungsi -O- dan mempunyai isomer fungsi dengan alkohol.

Aldehid

Aldehid dengan gugus fungsi -CHO, tidak memiliki isomer posisi, sebab gugus fungsi dari aldehid terletak di ujung rantai. Isomer yang dimiliki aldehid adalah isomer struktur dan isomer fungsi dengan keton.

Keton

Keton memiliki gugus fungsi -CO- di antara alkil. Keton memiliki isomer posisi dan isomer fungsi dengan aldehid.

Asam karboksilat

Asam karboksilat dengan gugus fungsi  -COOH memiliki isomer struktur dan isomer fungsi dengan ester. Asam karbosilat tidak memiliki isomer posisi, karena gugus fungsinya terikat di ujung rantai.

Ester

Ester merupakan senyawa turunan alkana dengan gugus fungsi -COOR. Isomer yang dimiliki oleh ester adalah isomer posisi dan isomer fungsi dengan asam karboksilat.

19. Suatu senyawa mempunyai rumus molekul C2H6O, bereaksi dengan natrium menghasilkan gas hidrogen. Kemungkinan gugus fungsi dalam senyawa itu adalah . . . .


Jawaban :

Suatu senyawa mempunyai rumus molekul C2H6O, bereaksi dengan natrium menghasilkan gas hidrogen. Kemungkinan gugus fungsi dalam senyawa itu adalah -OH

Penjelasan :

Gugus Fungsi adalah Gugus Atom atau Kelompok atom yang paling menentukan sifat kimia suatu senyawa.

Gugus Fungsi ada Tujuh macam :

✔ Alkil Alkohol atau Alkanol

✔ Alkil alkil Eter atau Alkoksialkana

✔ Aldehida atau Alkanal

✔ Alkil alkil Keton atau Alkanon

✔ Asam Karboksilat atau Asam Alkanoat

✔ Ester atau Alkil Alkanoat

✔ Alkil Halida atau Halo Alkana

C2H6O merupakan rumus senyawa dari Alkohol dan Eter. Namun ada dua cara untuk membedakan Alkohol dengan Eter.

☑ Alkohol bereaksi dengan Logam Natrium dan membebaskan gas Hidrogen (H2) sedangkan eter tidak bereaksi.

☑ Alkohol bereaksi dengan PCl5 dan menghasilkan HCl sedangkan eter bereaksi tetapi tidak menghasilkan HCl.

Jadi, Gugus fungsi tersebut adalah - OH.

Tentukan bilangan oksidasi unsur C dalam setiap kelompok senyawa berikut:

CO = +2

[tex]\sf C_{2}H_{6}O[/tex] = -2

[tex]\sf (COOH)_{2}[/tex]  = +3

[tex]\sf Na_{2}CO_{3}[/tex] = +4

PENDAHULUAN

Sebelumnya saya jelaskan dulu apa itu Bilangan Oksidasi. Bilangan Oksidasi adalah muatan relatif dari suatu atom dibanding atom lain dalam molekul atau ion. Dalam menentukan bilangan oksidasi, terdapat ketentuan - ketentuannya, diantaranya :

1. Unsur Bebas ⇒ Biloks = 0

Unsur bebas adalah unsur yang tidak bersenyawa, yang terdiri dari 3 bentuk diantaranya :

Monoatom = Hanya terdiri dari 1 atom. Contoh : Na, Al, Fe, dllPoliatom = Yang memiliki 2 atom. Contoh : N2, H2 , O2Diatom = Lebih dari 2 atom. Contoh : O3 , P4 . S8

2. Atom dalam senyawa atau ion

a. Logam A ⇒ Biloks = + sesuai golongannya

Contoh :

Gol. IA = Ci, Na, K, Rb, Cs, Fr ⇒ Biloks = +1Gol. IIA = Be, Mg, Ca, Sr, Ba, Ra ⇒ Biloks = +2Gol. IIIA = B, Al, Ga, In, TI ⇒ Biloks = +3

b. Atom H (Biloks H = 1)

Pada umumnya atom H dalam senyawa memiliki bilangan oksidasi +1 (Biloks H = 1). Contoh : HCl. Namun, ada pengecualian. Atom H juga bisa memiliki bilangan oksidasi = -1 apabila hidrogen berada pada senyawa hidrida logam. Senyawa Hidrida Logam adalah senyawa yang terbentuk dari atom H dan atom logam. Contoh : NaH

c. Atom F ⇒ Biloks = -1. Contoh : HF , F2

d. Atom O (Biloks O = -2)

Atom O dalam senyawa umumnya memiliki biloks = -2 (Biloks O = -2). Contoh : H2OTetapi ada pengecualian, Atom O bisa memiliki bilangan oksidasi -1 (Biloks O = -1) apabila berada pada senyawa peroksida. Senyawa peroksida adalah senyawa yang kelebihan atom O. Contoh : H2O2 , Na2O2 , CaO2Biloks O = -1/2 , apabila atom O tersebut berada pada senyawa superoksida. Contoh : KO2Biloks O = Senyawa fluorida = +2 . Contoh : OF2

3. Jumlah biloks dalam senyawa = 0. Contoh : H2SO4 ⇒ Biloks = 0. Artinya : Biloks 2 atom H + Biloks S + 4 Biloks Atom O harus sama dengan 0

4. Jumlah biloks atom dalam ion = Muatannya. Contoh : NH4+ ⇒ Biloks = +1

PEMBAHASAN

Tentukan bilangan oksidasi unsur C dalam setiap kelompok senyawa berikut:

[tex]\:[/tex]

a. C dalam CO

Biloks C + Biloks O = 0

Biloks C + (-2) = 0

Biloks C = +2

[tex]\:[/tex]

b. C dalam [tex]\sf C_{2}H_{6}O[/tex]

(2 x Biloks C) + (6 x Biloks H) + Biloks O = 0

(2 x Biloks C) + (6 x 1) + (-2) = 0

(2 x Biloks C) + 6 + (-2) = 0

2 x Biloks C + 4 = 0

2 x Biloks C = -4

Biloks C = -2

[tex]\:[/tex]

C. C dalam [tex]\sf (COOH)_{2}[/tex]

(2 x Biloks C) + (4 x Biloks O) + (2 x Biloks H) = 0

(2 x Biloks C) + (4 x (-2)) + (2 x 1) = 0

2 x Biloks C + (-8) + 2 = 0

2 x Biloks C - 6 = 0

2 x Biloks C = +6

Biloks C = +3

[tex]\:[/tex]

D. Biloks C dalam [tex]\sf Na_{2}CO_{3}[/tex]

(2 x Biloks Na) + Biloks C + (3 x Biloks O) = 0

(2 x 1) + Biloks C + (3 x (-2)) = 0

2 + Biloks C + (-6) = 0

Biloks C + (-4) = 0

Biloks C = +4
