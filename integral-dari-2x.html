14. Tentukan integral 2x(2x+4)⁴dx dengan cara menggunakan integral parsial​


integral parsial

∫u dv = uv - ∫v du

___

cara 1

parsial

∫2x(2x + 4)⁴ dx

= ∫2x d(2x + 4)⁵/(2 × 5)

= 1/5 ∫x d(2x + 4)⁵

= 1/5 (x(2x + 4)⁵ - ∫(2x + 4)⁵ dx)

= 1/5 (x(2x + 4)⁵ - ∫(2x + 4)⁵ d(2x + 4)/2)

= 1/5 (x(2x + 4)⁵ - 1/2 . 1/6 (2x + 4)⁶) + C

= 1/30 (2x + 4)⁵ (5x - 2) + C

substitusi

∫2x(2x + 4)⁴ dx

= 1/60 u⁵ (5u - 24) + C
