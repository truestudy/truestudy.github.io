PEMBAHASAN

Tentukan akar persamaan kuadrat berikut dengan membentuk kuadrat sempurna terlebih dahulu.

1. x² + 10x + 24 = 0

⇒ x² + 10 = -24

⇒ x² + 10 + 25 = -24 + 25

⇒ (x + 5)² = 1

⇒ (x + 5)² =√1

⇒ (x + 5) = ±1

(i) x + 5 = 1

x = -4

(ii) x + 5 = -1

x = -6


2. x² + 5x + 3 = 0

⇒ x² + 5x = -3

⇒ x² + 5x + 6.25 = -3 + 6.25

⇒ (x + 2.5)² = 3.25

⇒ (x + 3.25) = √3.25

⇒ (x + 3.25) = 1.8

(i) x + 2.5 = 1.8

x = -0.7

(ii) x + 2.5 = -1.8

x = -4.3


3. 2x² + 7x + 3 = 0

⇒ x² + 3.5x + 1.5 = 0

⇒ x² + 3.5x = -1.5

⇒ x² + 3.5x + 3.0625 = -1.5 + 3.0625

⇒ (x + 1.75)² = 1.5625

⇒ (x + 1.75) = √1.5625

⇒ (1 + 1.75) = 1.25

(i) x + 1.75 = 1.25

x = -0.5

(ii) x + 1.75 = -1.25

x = -3
