8. 1. Jelaskan dua cara mengupload plugin ke wordpress secara offline! 2. Jelaskan bagaimana proses mengubah wordpress agar bisa menjadi sebuah toko online dengan menggunakan plugin!


Jawaban:

Cara mengupload plugin ke WordPress secara offline: a. Melalui FTP:

•Unduh plugin yang ingin Anda upload ke situs WordPress Anda

•Ekstrak file plugin tersebut

•Buka program FTP (File Transfer Protocol) seperti FileZilla

•Masukkan informasi akun FTP untuk menghubungkan dengan situs WordPress Anda

•Cari folder "wp-content/plugins" di dalam folder WordPress Anda

•Salin folder plugin yang diekstrak ke dalam folder "plugins" di WordPress

b. Melalui cPanel:

•Unduh plugin yang ingin Anda upload ke situs WordPress Anda

•Ekstrak file plugin tersebut

•Login ke akun cPanel Anda

•Cari opsi "File Manager"Buka folder "public_html" dan temukan folder "wp-content/plugins"

•Klik tombol "Upload" di bagian atas dan pilih file plugin yang sudah diekstrak

•Tunggu hingga proses upload selesai

Proses mengubah WordPress agar bisa menjadi sebuah toko online dengan menggunakan plugin: a. Instal plugin eCommerce seperti WooCommerce

•Login ke dashboard WordPress Anda

•Klik opsi "Plugins" di sidebar kiri

•Klik tombol "Add New" di bagian atas

•Cari plugin eCommerce seperti WooCommerce

•Klik tombol "Install Now" dan tunggu hingga proses selesai

•Aktifkan plugin tersebut

b. Konfigurasi plugin WooCommerce

•Setelah plugin WooCommerce diinstal, Anda akan diarahkan ke halaman konfigurasi

•Ikuti petunjuk pada halaman tersebut untuk mengkonfigurasi toko online Anda

•Atur pengaturan seperti alamat toko, jenis mata uang, dan metode pengiriman

•Atur produk dan kategori produk yang akan dijual di toko online Anda

c. Tambahkan fitur lain yang dibutuhkan untuk toko online

•Instal plugin tambahan untuk menambahkan fitur seperti pembayaran online, pengiriman internasional, atau integrasi dengan media sosial

•Konfigurasikan plugin tambahan tersebut untuk memastikan toko online Anda berjalan lancar

d. Buat halaman toko onlineBuat halaman baru di WordPress dan beri nama "Toko" atau "Belanja"

•Tambahkan shortcode WooCommerce ke halaman tersebut

•Simpan dan terbitkan halaman baru tersebut

e. Mulai menjual produk di toko online Anda

Tambahkan produk baru ke toko online Anda

Atur harga, deskripsi, dan gambar produk

•Tambahkan produk ke kategori yang sesuai

Mulai promosikan toko online Anda dan produk yang dijual ke pelanggan potensial.

i-MSCP atau kepanjangannya adalah Internet Multi Server Control Panel adalah proyek sumber terbuka yang bertujuan untuk membuat panel kontrol multi-server yang mengesankan dan kuat untuk penggunaan pribadi dan profesional. Tujuannya adalah untuk memfasilitasi pengelolaan lingkungan shared hosting  tanpa campur tangan manusia pada sistem file server itu sendiri.

i-MSCP (Internet Multi Server Control Panel) identik dengan proyek open source yang bertujuan untuk membuat panel kontrol multi-server yang berfokus pada penggunaan pribadi dan profesional. Aplikasi jenis ini sangat cepat dan mudah digunakan, tidak kalah dengan cPanel yang banyak digunakan. Tujuan utamanya adalah untuk mengelola shared hosting tanpa campur tangan manusia pada sistem file server. Aplikasi i-MSCP hanya tersedia di dua distribusi Linux: Debian (versi Wheezy dan yang lebih baru) dan Ubuntu LTS (versi 12.04 dan yang lebih baru).

Fungsi i-MSCP:

Catatan DNS khusus Manajer acara (frontend dan backend) Server surat eksternal Manajemen server email (Postfix, Courier, Dovecot, keduanya mendukung kuota) Antarmuka web untuk banyak pengguna (administrator, reseller, dan pelanggan) Dukungan MySQL MariaDB  Editor konfigurasi PHP  (per pengguna, per domain, atau per situs) API Plugin (frontend dan backend) Fungsi cetak untuk instalasi tanpa pengawasan Dukungan SQL Server Jarak Jauh Penginstal paket perangkat lunak Manajemen SQL Server (MySQL) Perlindungan folder web dengan bit FS yang tidak dapat diubah.Manajemen Server Web  (Apache2 ITK, fegid, fastCGI/PHP5-FPM)

Sistem minimal:

RAM minimal 512 MB Prosesor 2 GHz atau lebih tinggi / 32-bit (x86) atau 64-bit (x64) atau lebih tinggi direkomendasikan untuk Corei3 Hard disk 1 GB atau lebih besar Akses internet  100 Mbit/s atau lebih tinggi Aplikasi PHP harus  versi 5.3.2 atau lebih tinggi dan Perl harus  versi 5.10.1 atau lebih tinggi. Sistem operasi Linux  untuk Debian Minimal Squeeze dan Ubuntu Minimal Lucid.

17. jelaskan pengertian dari ID billing!​


Jawaban:

Berdasarkan Peraturan Direktorat Jenderal Pajak Nomor PER-205/PJ/2017 tentang Pembayaran Pajak Secara Elektronik, ID Billing atau biasa disebut kode billing merupakan kode identifikasi yang diterbitkan melalui sistem e-Billing atas suatu jenis pembayaran atau setoran yang dilakukan oleh Wajib Pajak.

Penjelasan:

Pengertian e-Billing pajak. e-Billing pajak adalah sistem pembayaran pajak secara elektronik dengan membuat kode billing pajak pada aplikasi SSE pajak online yang merupakan bagian dari sistem Penerimaan Negara. Sistem ini dikelola oleh biller Direktorat Jenderal Pajak dan menerapkan billing system.

Sebenarnya banyak cara untuk mendapatkan kode e-billing ini, seperti datang langsung ke kantor pajak, melalui teller bank, lewat kantor pos, lewat telepon, serta lewat website resmi DJP online. Untuk mendapatkan kode billing lewat website DJP online, Anda tinggal membuka websitenya dan mengisikan data diri.

Berikut cara bayar pajak dengan e-Billing DJP Online:

1. Log in ke laman djponline.pajak.go.id.

2. Masukkan NPWP, password, dan kode keamanan untuk log in ke akun Anda.

3. Selanjutnya pilih menu e-Billing System.

4. Pilih pada menu Isi SSE.

5. Kemudian Anda akan mendapat form Surat Setoran Elektronik (SSE) yang harus Anda isi.
