1. 252 km/jam = ..... m/detik

-> 1 km = 1.000 m

= 252 x 1000 m/ jam

= 25.200 m/jam

-> 1 jam = 3.600 detik

= 25.200 x [tex] { \frac {1} {3.600}}[/tex] m/detik

= [tex] { \frac {25.200} {3.600}}[/tex] m/detik

= 7 m/detik

2. 360 km/jam = ..... m/detik

-> 1 km = 1.000 m

= 360 x 1000 m/ jam

= 36.000 m/jam

-> 1 jam = 3.600 detik

= 36.000 x [tex] { \frac {1} {3.600}}[/tex] m/detik

= [tex] { \frac {36.000} {3.600}}[/tex] m/detik

= 10 m/detik

3. 35 m/detik = ....... km/jam

-> 1 km = 1.000 m  

= 35 x [tex] \frac {1} {1.000}[/tex] km/detik

= [tex] \frac {35} {1.000}[/tex] km/detik

-> 1 jam = 3.600 detik

= [tex] \frac {35} {1.000}[/tex] x 3.600 km/jam

= [tex] \frac {35} {1.000 : 200}[/tex] x ( 3.600 : 200 ) km/jam

= [tex] \frac {35} {5}[/tex] x 18 km/jam

= [tex] \frac {35 : 5} {5 : 5}[/tex] x 18 km/jam

= [tex] \frac {7} {1}[/tex] x 18 km/jam

= 126 km/jam

4. 42 km/jam = ...... m/menit

-> 1 km = 1.000 m

= 42 x 1000 m/jam

= 42.000 m/jam

-> 1 jam = 60 menit

= 42.000 x [tex] \frac {1} {60}[/tex] m/menit

= [tex] \frac {4.200} {60}[/tex] m/menit

= 70 m/menit

5. 75 m/detik = ..... hm/menit

-> 1 hm = 100 m

= 75 x [tex] \frac {1} {100}[/tex] hm/detik

= [tex] \frac {75} {100}[/tex] hm/detik

= [tex] \frac {75 : 25} {100 : 25}[/tex] hm/detik

= [tex] \frac {3} {4}[/tex] hm/detik

-> 1 menit = 60 detik

= [tex] \frac {3} {4}[/tex] x 60 hm/menit

= [tex] \frac {3} {4 : 4}[/tex] x ( 60 : 4 ) hm/menit

= [tex] \frac {3} {1}[/tex] x 15 hm/menit

= 45 hm/menit
