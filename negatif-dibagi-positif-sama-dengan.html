9. kalau negatif dibagi positif hasilnya positif atau negatif?


positif = (+)

negatif = (-)

pembagian

(+) : (+) = (+)

(-) : (-) = (+)

(+) : (-) = (-)

(-) : (+) = (-)

negatif dibagi negatif hasilnya positif

contoh

-25 : -5 = 5

-50 : -5 = 10

untuk pengetahuan lebih lengkap dapat disimak penjelasan tentang penjumlahan, pengurangan dan perkaliannya

penjumlahan

a) (+) + (+) = (+)

b) (-) + (-) = (-)

c) (+) + (-) = bisa (+) bisa juga (-)

jika (+) lebih besar maka hasilnya (+)

jika (-) lbh besar maka hasilnya (-)

contoh:

9 + (-6) = 3 → hasil positif

9 + (-16) = (-7) → hasil negatif

d) (-) + (+) = bisa (+) bisa juga (-)

jika (+) lebih besar maka hasilnya (+)

jika (-) lebih besar maka hasilnya (-)

contoh:

(-5) + 2 = (-3) → hasilnya negatif

(-5) + 9 = 4 → hasilnya positif

pengurangan

a) (+) - (+) = bisa (+) bisa juga (-)

apabila bilangan pertama lebih besar maka hasilnya positif

apabila bilangan kedua lebih besar maka hasilnya negatif

contoh:

5 - 2 = 3 → hasil positif

5 - 9 = (-4) → hasil negatif

b) (-) - (-) = bisa (+) bisa juga (-)

apabila bilangan pertama lebih besar dari bilangan kedua maka hasilnya (+)

apabila bilangan kedua yang lebih besar maka hasilnya (-)

contoh:

(-2) - (-5) = (-2) + 5 = 3 → hasilnya positif karena -2 (bilangan pertama) lbh besar dari pada -5 (bilangan kedua)

(-6) - (-2) = (-6) + 2 = (-4) → hasilnya negatif karena -6 (bilangan pertama) lebih kecil dari pada -2 (bilangan kedua)

c) (+) - (-) = (+)

d) (-) - (+) = (-)

perkalian

(+) x (+) = (+)

(-) x (-) = (+)

(+) x (-) = (-)

(-) x (+) = (-)
