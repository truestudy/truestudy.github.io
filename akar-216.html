29. berapa hasil dari akar 216


Hasil dari akar 216 adalah 6√6.

Pembahasan

Akar kuadrat adalah invers dari pangkat kuadrat.

Misalnya :

2² = 2 x 2 = 4 maka √4 = √2² = 2

5² = 5 x 5 = 25 maka √25 = √5² = 5

Akar kuadrat dapat pula dicari dengan pohon faktor.

Misal :

√36 = ...

36

/ \

2  18

     / \

   2   9

        / \

       3   3

36 = 2 x 2 x 3 x 3

36 = 2² x 3²

36 = (2 x 3)²

36 = 6²

√36 = √6²

√36 = 6

Sekarang mari kita hitung √216 !

216

/ \

2  108

     / \

    2  54

         / \

        2  27

             / \

            3   9

                 / \

                3   3

216 = 2 x 2 x 2 x 3 x 3 x 3

216 = 2² x 3² x 2 x 3

216 = (2 x 3)² x 6

216 = 6² x 6

√216 = √6² x √6

√216 = 6√6

Jadi hasil dari √216 adalah 6√6.
