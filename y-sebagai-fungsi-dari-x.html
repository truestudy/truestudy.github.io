Jawab:

y = f(x)

y = -5x + 2

Untuk x = -5, maka

y = -5(-5) + 2

y = 27

Sehingga, didapatkan koordinat titik (-5, 27)

Untuk x = 7, maka

y = -5(7) + 2

y = -33

Sehingga, didapatkan koordinat titik (7, -33)

Hubungkan kedua titik tersebut dan didapatkan sketsa grafik y = -5x + 2 untuk -5 ≤ x ≤ 7 (Lihat Gambar). Jika ingin menggambarnya sendiri di PC, kunjungi geogebra

27. Turunan fungsi y=x/1-x,adalah y '


Penjelasan dengan langkah-langkah:

y = x/(1 - x)

misalkan :

u = x

u' = 1

v = (1 - x)

v' = - 1

y = u/v

y' = (u'v - v'u)/v²

y' = [1(1- x) - (-1(x)]/(1 - x)²

y' = [1 - x) + x]/(1 - x)²

y' = 1/(1 - x)²


28. posisi grafik fungsi y=(x-5)² terhadap grafik fungsi y=x² adalah​


Grafik fungsi [tex]y=f(x-a)[/tex] diperoleh dengan cara menggeser grafik fungsi [tex]y=f(x)[/tex] ke kanan sejauh [tex]a[/tex] satuan.

Misalkan [tex]f(x)=x^2[/tex]

maka fungsi [tex]f(x-5)^2=(x-5)^2[/tex]

Hal ini berarti grafik fungsi [tex]y=(x-5)^2[/tex] merupakan hasil pergeseran grafik fungsi [tex]y=x^2[/tex] ke kanan sejauh 5 satuan.

Dengan kata lain, fungsi [tex]y=(x-5)^2[/tex] berada 5 satuan di sebelah kanan fungsi [tex]y=x^2[/tex]
