ICON SELECT

Icon Select berguna untuk menselect atau memilih gambar.

Bentuk icon Select :

1. Rectangular Select

Berguna untuk menselect gambar secara kotak

2. Freeform Select

Berguna untuk menselect gambar mengikuti bentuk gambar

Pilihan Select :

1. Select all

Untuk memilih semua gambar

2. Invert selection

Untuk memilih bukan dari yang diselect

3. Delete

Untuk menghapus gambar yang diselect

4. Transparent selection

Untuk membuat tampilan gambar transparent

Gambar yg diselect dapat di :

1. Dihapus

2. Dipindahkan

3. Digandakan/dicopy

4. Diputar atau dibalik
