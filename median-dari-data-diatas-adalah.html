= data ke. 25,5

Jawaban : C
Banyak data :
3 + 5 + 4 + 5 + 7 + 5 + 5 = 34
Karena Banyak datanya genap, maka median =
(Data ke- n/2 + Data ke- n/2 + 1) : 2
(Data ke- 34/2 + data ke- 34/2 + 1) : 2
(Data ke-17 + Data ke-18) : 2
Data ke-17 = 7
Data ke-18 = 8
MAKA MEDIANNYA :
(7 + 8) : 2 = 15 : 2 = 7,5
Jadi mediannya adalah 7,5
