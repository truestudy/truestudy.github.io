1. nada membeli kue untuk lebaran harga satu kaleng kue Nastar sama dengan 2 kali harga 1 kaleng kue keju, harga tiga kaleng kue nastar dan 2 kaleng kue keju = 480000 uang yang harus dibayar nada untuk membeli 2 kaleng kue Nastar dan 3 kaleng kue keju adalah


buat perumpamaan, misal :
kue nastar = N
kue keju    = K

Diketahui :
1N = 2K
3N + 2K = 480.000 = 6K + 2K = 480.000

Ditanyakan :
harga 2N + 3K

Jawab :
8K = 480.000
K   = 60.000
N = 2K = 2×60.000 = 120.000

Maka harga 2N + 3K = 2×120.000 + 3×60.000
                                  = 240.000 + 180.000
                                  = Rp.420.000,00.
smoga membantu ^^

y = keju
[tex]x = 2y \\ 3x + 2y = 480.000 \\ 3x + x = 480.000 \\ 4x = 480.000 \\ x = \frac{480.000}{4 } \\ x = 120.000[/tex]
x = nastar = Rp.120.000
y = kue keju = Rp.120.000 : 2 = Rp.60.000

2 nastar + 3 kue keju =
[tex]2x + 3y \\ 2.(120.000) + 3.(60.000) \\ 240.000 + 180.000 \\ 420.000[/tex]

5. Nada membeli kue untuk lebaran. Harga satu kaleng kue nastar sama dengan dua kali harga satu kaleng kue keju. Harga 3 kaleng kue nastar dan 2 kaleng kue keju Rp480, 000.00. Uang yang harus dibayarkan Nada untuk membeli 2 kaleng kue nastar dan 3 kaleng kue keju adalah... ​


Jawab:

harga 2 kaleng kue nastar dan 3 kaleng kue keju adalah rp. 420.000

Penjelasan dengan langkah-langkah:

misal : kue nastar = a

           kue keju = b

dik : a = 2b

       3a + 2b = 480.000

dit : 2a + 3b = ... ?

penyelesaian :

langkah 1.

a = 2b

3a + 2b = 480.000

3(2b) + 2b = 480.000

6b + 2b = 480.000

8b = 480.000

 b = 480.000 : 8

 b = 60.000 (harga 1 kaleng kue keju = rp. 60.000)

langkah 2.

3a + 2b = 480.000

3a + 2(60.000) = 480.000

3a + 120.000 = 480.000

3a = 480.000 - 120.000

3a = 360.000

 a = 360.000 : 3

 a = 120.000 (harga 1 kaleng kue nastar = rp. 120.000)

langkah 3.

2a + 3b = 2(120.000) + 3(60.000)

             = 240.000 + 180.000

             = 420.000

jadi, harga 2 kaleng kue nastar dan 3 kaleng kue keju adalah rp. 420.000.

harga satu kaleng kue nastar sama dengan 2 kali harga satu kaleng kue keju maka x=2y

Pertanyaan : Berapakah 2x + 3Y= .....??

Jawab : 

Langkah pertama, kita harus menentukan nilai X berapa dan nilai Y berapa terlebih dahulu. 

3x + 2y = 480.000.  ingat, nilai X = 2 kali lebih besar dari nilai Y

maka, jika Y adalah 60.000
maka, 3x + 2(60.000) = 480.000
           3x  + 120.000 = 480.000
                              3x = 480.000-120.000
                              3x = 360.000
                               x = 360.000/3 
                               x = 120.000
Benar kan? X = 2 Y. 120.000 = 2x60.000

Maka, uang yang harus dibayarkan nada untuk membeli 2 kaleng kue nastar dan 3 kaleng kue keju adalah
2x + 3Y = 2x120.000 + 3x60.000
             = 240.000 + 180.000
            = 420.000.
