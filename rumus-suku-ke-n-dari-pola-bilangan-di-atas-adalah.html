5. .1. Tentukan banyak pola ke 10 untuk bilangan bulat positif.Tentukan pula rumus suku ke-n (pada gb. Di atas )​


Jawaban:

Banyak lingkaran pada pola ke 10, ke 100,, ke n pada pola berikut, untuk sebarang n bilangan bulat positif adalah U₁₀ = 66, U₁₀₀ = 5.151 dan Un = ½ (n + 1)(n + 2). Hasil tersebut bisa menggunakan rumus jumlah n suku pertama pada deret aritmatika. Barisan aritmatika adalah barisan yang antar dua suku berdekatannya memiliki selisih yang tetap.  

Rumus suku ke n pada barisan aritmatika

Un = a + (n – 1)b

Rumus jumlah n suku pertama

Sn =  ￼ (2a + (n – 1)b)

Sn =  ￼ (a + Un)

Keterangan

a = suku pertama

b = beda ⇒ b = U₂ – U₁ = U₃ – U₂ = ....  

Pembahasan

Banyak lingkaran pada  

Pola ke 1 = 1 + 2 = 3

Pola ke 2 = 1 + 2 + 3 = 6

Pola ke 3 = 1 + 2 + 3 + 4 = 10

:

Pola ke n = 1 + 2 + 3 + ... + n + (n + 1)

Dengan menggunakan deret aritmatika

Sn =  ￼ (a + Un)

Sn =  ￼ (1 + (n + 1))

Sn =  ½ (n + 1) (n + 2)

Jadi  

Banyak lingkaran pada pola ke 10

S₁₀ = ½ (10 + 1)(10 + 2)

S₁₀ = ½ (11)(12)

S₁₀ = (11) . (6)

S₁₀ = 66

Banyak lingkaran pada pola ke 100

S₁₀₀ = ½ (100 + 1)(100 + 2)

S₁₀₀ = ½ (101) (102)

S₁₀₀ = (101) . (51)

S₁₀₀ = 5.151

Banyak lingkaran pada pola ke n

Sn = ½ (n + 1)(n + 2)

MAAF KALO SALAH

a). Rumus suku ke-n adalah Un = 6 + (n - 1)6

b). Banyak lingkaran pada pola ke-10 adalah 330 lingkaran

Untuk mengerjakan soal tersebut kita harus menggunakan rumus deret aritmatika

Penjelasan dengan langkah-langkah:

Diketahui:

a = U1 = suku pertama = 6 lingkaran

U2 = 12

U3 = 18

Ditanya:

a). Rumus suku ke-n / Barisan Aritmatika (Un)

b). Banyak lingkaran pada pola ke-10 (Sn)

Jawab:

a). Rumus suku ke-n / Barisan Aritmatika (Un)

Rumus Barisan Aritmatika adalah Un = a + (n - 1)b

b = beda antar suku

  = U2 - U1

  = 12 - 6

  = 6

Substitusikan yang diketahui ke dalam rumus

Un = 6 + (n - 1)6

b). Banyak lingkaran pada pola ke-10 (Sn)

Rumus banyak lingkaran ada pola ke-n / Deret Aritmatika adalah

[tex]Sn = \frac{n}{2} (a + Un)[/tex]

[tex]U10= 6+(10-1)6\\U10=6+54\\U10=60[/tex]

Substitusikan yang diketahui ke dalam rumus

[tex]Sn = \frac{n}{2} (6 + Un)[/tex]

Banyak lingkaran pada pola ke-10 (Sn)

[tex]S10 = \frac{10}{2} (6 + U10)\\\\S10 =\frac{10}{2} (6+60)\\\\S10 =5 (66)\\\\S10 = 330[/tex]

Jadi diperoleh banyak lingkaran pada pola ke-10 (S10) adalah 330 lingkaran

14. banyak noktah (bulatan kecil) pada gambar diatas.menunjukkan pola bilangan.a. tulislah bilangan bertingkat yang menyatakan banyak noktah untuk pola bilangan di atas sampai suku ke-5b. tentukan rumus suku ke-n pada barisan bilangan tersebut c. tentukan suku ke 9 pada barisan bilangan tersebut dengan menggunakan rumus


Banyak noktah (bulatan kecil) pada gambar diatas menunjukkan pola bilangan. Pola bilangan adalah suatu aturan dalam barisan bilangan sehingga dapat menentukan bilangan berikutnya dari barisan bilangan tersebut.  

Pembahasan

a) Tulislah bilangan bertingkat yang menyatakan banyak noktah untuk pola bilangan di atas sampai suku ke-5

Jawab

pola 1 = 3 Pola 2 = 3 + 3 = 6 Pola 3 = 3 + 3 + 4 = 10 Pola 4 = 3 + 3 + 4 + 5 = 15 Pola 5 = 3 + 3 + 4 + 5 + 6 = 21

Bilangan barisan bertingkatnya adalah 3, 6, 10, 15, 21

b. Tentukan rumus suku ke-n pada barisan bilangan tersebut

Jawab

Karena barisan yang diperoleh adalah barisan aritmatika bertingkat, maka kita misalkan suku ke n adalah

Un = an² + bn + c

3     6      10     15     21  ⇒ a + b + c = 3

 +3    +4     +5     +6      ⇒ 3a + b = 3  

     +1      +1      +1          ⇒ 2a = 1

2a = 1  

⇒ a = [tex] \frac{1}{2}[/tex]  

3a + b = 3

⇒ 3([tex]\frac{1}{2}[/tex]) + b = 3  

⇒ [tex]\frac{3}{2}[/tex] + b = 3

⇒ b = 3 – [tex]\frac{3}{2}[/tex]  

⇒ b = [tex] \frac{6}{2} - \frac{3}{2}[/tex]

⇒ b = [tex] \frac{3}{2}[/tex]

a + b + c = 3  

⇒ [tex] \frac{1}{2} + \frac{3}{2}[/tex] + c = 3  

⇒ 2 + c = 3

⇒ c = 3 – 2  

⇒ c = 1

Jadi suku ke n pada barisan bilangan tersebut adalah

Un = an² + bn + c

Un = [tex] \frac{1}{2}[/tex] n² + [tex] \frac{3}{2}[/tex] n + 1

Un = [tex] \frac{1}{2}[/tex] n² + [tex] \frac{3}{2}[/tex] n + [tex] \frac{2}{2}[/tex]

Un = [tex] \frac{1}{2}[/tex] (n² + 3n + 2)  

Un = [tex] \frac{1}{2}[/tex] (n + 1)(n + 2)

c. Tentukan suku ke 9 pada barisan bilangan tersebut dengan menggunakan rumus

Un = [tex] \frac{1}{2}[/tex] (n + 1)(n + 2)

Barisan Aritmatika adalah suatu barisan dengan selisih antara dua suku yang berurutan selalu tetap.

Rumus : \boxed{~Un=a+(n-1)b~}

Un=a+(n−1)b

Deret Aritmatika adalah jumlah suku – suku barisan aritmatika
