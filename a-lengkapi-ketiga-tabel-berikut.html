Bilangan bulat merupakan suatu kumpulan bilangan yang terdiri atas bilangan positif sampai bilangan negatif. Skor hasil ulangan harian lima siswa A, B, C, D, E baris pertama. Baris kedua menunjukkan skor A 52, B 56, C 55, D 60, E 47. Baris ketiga menunjukkan skor jika skor C dijadikan sebagai titik acuan A -3, B +1, C 0, D +5, dan E -8. Kalimat matematika yang digunakan untuk menghitung hasilnya antara lain:

[tex]A.\ 52-55=-3\\B.\ 56-55=+1\\C.\ 55-55=0\\D.\ 60-55=+5\\E.\ 47-55=-8[/tex]

Penjelasan dengan langkah-langkah:

Diketahui:

Skor hasil ulangan harian lima siswa A, B, C, D, E baris pertama. Baris kedua menunjukkan skor A 52, B 56, C 55, D 60, E 47. Baris ketiga menunjukkan skor jika skor C dijadikan sebagai titik acuan A -3, B +1, C 0, D +5, dan E -8.

Ditanyakan:

Tuliskan kalimat matematika yang digunakan untuk menghitung hasilnya!

Dijawab:

Bilangan bulat merupakan suatu kumpulan bilangan yang terdiri atas bilangan positif sampai bilangan negatif. Skor hasil ulangan harian lima siswa A, B, C, D, E baris pertama. Baris kedua menunjukkan skor A 52, B 56, C 55, D 60, E 47. Baris ketiga menunjukkan skor jika skor C dijadikan sebagai titik acuan A -3, B +1, C 0, D +5, dan E -8. Kalimat matematika yang digunakan untuk menghitung hasilnya antara lain:

Terdapat sebuah tabel pada gambar soal. Tabel tersebut menunjukkan skor hasil uji kebugaran yang dilakukan lima orang: A, B, C, D, dan E. Baris pertama tabel menunjukkan lima orang yang diuji, baris keduanya menunjukkan skor masing-masing orang, dan baris ketiganya menunjukkan skor apabila skor C sebagai titik acuan. Tabel lengkapnya dapat dicek pada penjelasan di bawah. Rata-rata skor kelima orang bernilai 54. Dengan memperoleh nilai -1 pada rata-rata nilai baris ketiga tabel, rata-rata skor uji kebugaran tersebut berbeda 1 poin di bawah skor C, yaitu 55. Ini dapat pula dinyatakan dengan: rata-rata = 55+(-1) = 55-1 = 54.

Penjelasan dengan langkah-langkah

Diketahui: tabel pada soal

Ditanya:

(1) tabel lengkap

(2) rata-rata skor dan kalimat matematika

Jawab:

Untuk nomor (1):

Metode

Tabel lengkap dapat diperoleh dengan mengisi sel-sel kosong pada baris ketiga. Nilai-nilainya diperoleh dengan menghitung selisih tiap skor terhadap skor C dengan memperhatikan nilai positif/negatif (lebih besar/kecil daripada skor C).

Selisih52-55 = -360-55 = 547-55 = -8Tabel lengkap

Untuk nomor (2):

Rata-rata nilai baris ketiga

Jumlah nilai = -3+1+0+5+(-8) = -5

Rata-rata = -5/5 = -1

Ini menunjukkan nilai rata-ratanya berada satu poin di bawah skor C.

Rata-rata sesungguhnya

Karena acuannya adalah skor C, maka:

55+(-1) = 54

Jadi, nilai rata-ratanya adalah 54.

Pembuktian

Jumlah nilai baris kedua = 52+56+55+60+47 = 270

Rata-rata = 270/5 = 54
