3. Berapa jumlah sudut segi tujuh tak beraturan berikut ini.​


Banyak sudut bangun segi tujuh tak beraturan adalah 7 (tujuh). Jadi, jumlah besar sudutnya adalah 900°.

Penjelasan dengan langkah-langkah:

Diketahui:
Bangun segi tujuh tak beraturan
Ditanyakan:
Berapa jumlah sudut segi tujuh tak beraturan?
Jawab:
Segi tujuh tak beraturan memiliki 7 sisi dan 7 sudut dengan ukuran yang tidak sama. Jumlah besar sudut pada bangun segi-n dapat diperoleh dengan rumus:
α = (n - 2) × 180°
Sehingga jumlah besar sudut segi tujuh tak beraturan (segi-7):
α = (7 - 2) × 180°
α = 5 × 180°
α = 900°
Dengan demikian, jumlah besar sudut segi tujuh tak beraturan tersebut adalah 900°.

4. berapa besar sudut putar terkecil dari segi tiga beraturan,segi empat beraturan, segi lima beraturan, segi enam beraturan, segi tujuh beraturan, segi delapan beraturan dan segi-n beraturan


Sifat istimewa dari segi-n beraturan adalah:

- memiliki n buah sisi sama panjang

- memiliki n titik sudut, dengan sudut sama besar

- memiliki n buah sumbu simetri

- memiliki n buah simetri putar

- memiliki sudut putar terkecil sebesar = 360°/n

Dari penjelasan di atas, maka:

a. Segitiga memiliki sudut putar terkecil sebesar 360°/3 = 120°.

b. Segi empat memiliki sudut putar terkecil sebesar 360°/4 = 90°.

c. Segi lima memiliki sudut putar terkecil sebesar 360°/5 = 72°.

d. Segi enam memiliki sudut putar terkecil sebesar 360°/6 = 60°.

e. Segi tujuh memiliki sudut putar terkecil sebesar 360°/7 = 51,43°.

f. Segi delapan memiliki sudut putar terkecil sebesar 360°/8 = 45°.

Sn = (n - 2) × 180°

26. Jumlah besar sudut pada segi banyak segitiga, segiempat, segilima, segi enam, segi tujuh adalah


Kelas               : VI

Pelajaran         : Matematika

Kategori           : Bangun Datar

Kata Kunci      : Jumlah sudut segi n, rumus cepat, cara mudah, 

 

Pembahasan :

Dalam menentukan jumlah sudut pada segi-n, terdapat cara lebih mudah dan lebih memangkas waktu dalam pengerjaanya. Rumus dasarnya untuk mencari jumlah sudut pada segi - n adalah : 

Segi(n) = ( n - 2 ) x [tex] 180^{o} [/tex]

Sehingga banyaknya jumlah sudut dari segitiga hingga segitujuh adalah : 

a. Segi 3 = ( 3 - 2 ) x [tex] 180^{o} [/tex] = [tex] 180^{o} [/tex]

b. Segi 4 = ( 4 - 2 ) x [tex] 180^{o} [/tex] = [tex] 360^{o} [/tex]

c. Segi 5 = ( 5 - 2 ) x [tex] 180^{o} [/tex] = [tex] 540^{o} [/tex]

d. Segi 6 = ( 6 - 2 ) x [tex] 180^{o} [/tex] = [tex] 720^{o} [/tex]

e. Segi 7 = ( 7 - 2 ) x [tex] 180^{o} [/tex] = [tex] 900^{o} [/tex]
