3. nyatakan keliling bangun datar berikut dalam bentuk aljabar


Kelas: VII (1 SMP)
Kategori Soal: Bangun Datar
Kata Kunci: persegi, persegi panjang, keliling

Pembahasan:
Persegi adalah bangun segi empat memiliki empat buah sisi sama panjang serta empat buah sudut sama besar dan siku-siku.

Keliling persegi adalah

K = 4 x s,

dengan s merupakan panjang sisi.

Luas persegi adalah

L = s x s

⇔ L = s²

dengan s merupakan panjang sisi.


Persegi panjang adalah bangun segi empat memiliki dua pasang sisi sama panjang dan sejajar serta empat buah sudut sama besar dan siku-siku.

Keliling persegi panjang adalah

K = 2 x p + 2 x l

= 2 x (p + l)

dengan p merupakan panjang dan l merupakan lebar.


Luas persegi panjang adalah

L = p x l

dengan p merupakan panjang dan l merupakan lebar.

Mari kita lihat soal tersebut.
Nyatakan keliling bangun datar pada gambar lampiran dalam bentuk aljabar!

Jawab:
a. Misalkan persegi memiliki panjang sisi s = a + a = 2a, sehingga keliling
K = 4 x s
⇔ K = 4 x 2a
⇔ K = 8a

Jadi, keliling persegi tersebut 8a.

b. Misalkan persegi panjang memiliki panjang p = a + a + a = 3a dan lebar l = b + b + b = 3b, sehingga keliling
K = 2 x (p + l)
⇔ K = 2 x (3a + 3b)
⇔ K = 6a + 6b

c. Misalkan persegi memiliki panjang sisi = a + a = 2a, sehingga
K = 2a + 2a + a + a + a + a
⇔ K = 8a

d. Misalkan persegi panjang memiliki panjang p = a dan lebar l = b, sehingga
K = a + b + c + (b - d) + (a - c) + d
⇔ K = a + a + b + b + c - c - d + d
⇔ K = 2a + 2b

22. nyatakan keliling bangun datar berikut dalam bentuk aljabar dan juga caranya...


Kelas         : V
Pelajaran   : Matematika
Kategori     : Luas & Keliling Bangun Datar
Kata Kunci : keliling, persegi, persegi panjang, bentuk aljabar

Penyelesaian

Keliling adalah jumlah panjang garis atau sisi yang membatasi daerah bangun datar

[a]
Bidang ini merupakan bentuk persegi dengan panjang sisi adalah a + a = 2a.
⇔ Keliling persegi = 4 x sisi
⇔ Keliling persegi = 4 x 2a
∴ Keliling persegi = 8a

[b]
Bidang ini merupakan bentuk persegi panjang.
Panjang = 3 x a = 3a
Lebar = 3 x b = 3b
⇔ Keliling persegi panjang = 2 x (panjang + lebar)
⇔ Keliling persegi panjang = 2 x (3a + 3b)
∴ Keliling persegi panjang = 6a + 6b

[c]
Cara Pertama
Terdapat enam buah sisi yang berukuran a, b, c, d, lalu (a - c) dan (b - d)
⇔ Keliling persegi panjang = a + b + c + d + (a - c) + (b - d)
⇔ Keliling persegi panjang = a + a + b + b + c - c + d - d
∴ Keliling persegi panjang = 2a + 2b

Cara Kedua (perhatikan gambar terlampir)
⇔ Geser dua garis agar tepat membentuk persegi panjang secara utuh.
⇔ Perhatikan hasil pergeseran yakni garis merah dan garis biru.
⇔ Panjang = a, dan lebar = b
⇔ Keliling persegi panjang = 2 x (panjang + lebar), atau 
⇔ Keliling persegi panjang = (2 x panjang) + (2 x lebar)
⇔ Keliling persegi panjang = 2 x (a + b) atau (2 x a) + (2 x b)
∴ Keliling persegi panjang = 2a + 2b

b. Misalkan persegi panjang memiliki panjang p = a + a + a = 3a dan lebar l = b + b + b = 3b, sehingga keliling
K = 2 x (p + l)
⇔ K = 2 x (3a + 3b)
⇔ K = 6a + 6b

Jadi, keliling persegi panjang tersebut adalah (6a + 6b).

K = 2a + 2a + a + a + a + a
⇔ K = 8a

Jadi, keliling persegi tersebut adalah 8a.

d. Misalkan persegi panjang memiliki panjang p = a dan lebar l = b, sehingga keliling
K = a + b + c + (b - d) + (a - c) + d
⇔ K = a + a + b + b + c - c - d + d
⇔ K = 2a + 2b

Jadi, keliling persegi panjang tersebut adalah (2a + 2b).
