Dua buah bilangan memiliki rasio 1 : 3. Jika bilangan pertama dikurang 3 dan bilangan kedua ditambah 5, maka rasionya menjadi 2 : 13. Hasil kali dua bilangan tersebut adalah 147. Rasio atau perbandingan adalah bentuk sederhana dari beberapa nilai dari besaran yang sejenis untuk membandingkan nilai-nilai besaran tersebut. Untuk menyederhanakan perbandingan tersebut yaitu dengan cara kedua bilangan dibagi atau dikali dengan bilangan yang sama.

Pembahasan

Misal  

Bilangan pertama = p Bilangan kedua = q

Dua buah bilangan memiliki rasio 1 : 3

p : q = 1 : 3

[tex]\frac{p}{q} = \frac{1}{3}[/tex]

q = 3p

Jika bilangan pertama dikurang 3 dan bilangan kedua ditambah 5, maka rasionya menjadi 2 : 13

(p – 3) : (q + 5) = 2 : 13

[tex]\frac{p - 3}{q \: + \: 5} = \frac{2}{13}[/tex]

[tex]\frac{p - 3}{3p \: + \: 5} = \frac{2}{13}[/tex]

13(p – 3) = 2(3p + 5)

13p – 39 = 6p + 10

13p – 6p = 10 + 39

7p = 49

p = [tex]\frac{49}{7}[/tex]

p = 7

q = 3p

q = 3(7)

q = 21

Jadi kedua bilangan tersebut adalah 7 dan 21, sehingga hasil kali kedua bilangan tersebut adalah:

= p × q

= 7 × 21

= 147

Penyelesaian :

Dimisalkan : Bilangan kecil = A dan bilangan besar = B

A : B = 1 : 3 ====> A = [tex]\frac {1} {3}[/tex] B

( A - 3 ) : ( B + 5 ) = 2 : 13

[tex]\frac { A - 3 } { B + 5 }[/tex] =  [tex]\frac {2} {13}[/tex] => kalikan silang

13 x ( A - 3 ) = 2 x ( B + 5 )

13A - ( 13 x 3 ) = 2B + ( 2 x 5 )

13A - 39 = 2B + 10

13A - 2B = 10 + 39

13A - 2B = 49

13 ( [tex]\frac {1} {3}[/tex] B) - 2B = 49

[tex]\frac {13 x 1} {3}[/tex] B - 2B = 49

[tex]\frac {13} {3}[/tex] B - 2B = 49

[tex] 4\frac {1} {3}[/tex] B - 2B = 49

[tex] ( 4 - 2 ) \frac {1} {3}[/tex] B = 49

[tex] 2 \frac {1} {3}[/tex] B = 49

[tex]\frac {7} {3}[/tex] B = 49

B =  49 : [tex]\frac {7} {3}[/tex]

B = 49 x [tex]\frac {3} {7}[/tex]

B = ( 49 : 7 ) x  [tex]\frac {3} {7 : 7}[/tex]

B = 7 x 3

B = 21


A =  [tex]\frac {1} {3}[/tex] B

A =  [tex]\frac {1} {3}[/tex]  x 21

A =  [tex]\frac {1} {3 : 3}[/tex]  x ( 21 : 3 )

A = 1 x 7

A = 7

Bilangan kedua = b
