y = x + 2
z = y + 2 = (x + 2) + 2 = x + 4
x + y + z = 90
x + (x + 2) + (x + 4) = 90
x + x + 2 + x + 4 = 90
x + x + x + 2 + 4 = 90
3x + 6 = 90
3x = 90 - 6
3x = 84
x = 84 : 3
x = 28
y = x + 2 = 28 + 2 = 30
z = x + 4 = 28 + 4 = 32
Jadi, 3 bilangan genap berurutan yang jumlahnya 90 adalah 28, 30, dan 32.

26. Tentukan tiga bilangan genap berurutan yang jumlahnya 90 ​


misal :

pertama = x

kedua = x + 2

ketiga = x + 4

pertama + kedua + ketiga = 90

x + ( x + 2 ) + (x + 4) = 90

3x + 6 = 90

3x = 90 - 6

3x = 84

x = 84 / 3

x = 24

pertama = x = 28

kedua = x + 2 = 28 + 2 = 30

ketiga = x + 4 = 28 + 4 = 32

Jadi, tiga bilangan genap berurutan yang jumlahnya 90 adalah 28, 30, dan 32.

3 bilangan genap berurutan yang jumlahnya 90 adalah 28, 30, dan 32

Pendahuluan

Cara menyelesaikan soal ini dengan mudah adalah memperhatikan pola bilangan genap, perhatikan barisan bilangan genap berikut

2, 4, 6, 8, 10, 12, ...

selisih suku tertentu dengan bilangan genap sebelum dan setelahnya selalu 2. kita manfaatkan hal ini untuk menyelesaikan soal ini

Pembahasan

Diketahui :

jumlah 3 bilangan genap berurutan adalah 90

Ditanyakan :

3 bilangan genap berurutan yang memenuhi

Jawaban :

Diandaikan bilangan pertama adalah a, karena selisih bilangan genap dengan bilangan genap setelahnya adalah 2, maka 3 bilangan genap berurutan tersebut adalah a, a + 2, dan a + 4

Karena jumlahnya 90, maka

a + (a + 2) + (a + 4) = 90

3a + 6 = 90

3a = 90 - 6

3a = 84

a = 84/3

a = 28

karena a = 28, maka 2 bilangan genap lainnya adalah

a + 2 = 28 + 2 = 30

a + 4 = 28 + 4 = 32

Jadi, 3 bilangan genap berurutan yang memenuhi adalah 28, 30, dan 32
