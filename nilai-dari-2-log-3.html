Logaritma adalah kebalikan (invers) dari perpangkatan, yaitu mencari pangkat dari suatu bilangan pokok sehingga hasilnya sesuai dengan yang telah diketahui.

aⁿ = b ⇔ ᵃlog b = n dengan syarat a > 0, a ≠ 1, b > 0

Sifat logaritma

ᵃlog a = 1

ᵃlog 1 = ᵃlog a⁰ = 0

ᵃlog mn = ᵃlog m + ᵃlog n

ᵃlog m/n = ᵃlog m - ᵃlog n

Untuk lebih lengkap sifat-sifat logaritma bisa dilihat pada lampiran.

PembahasanNo 1.

³log 45 + ³log 54 - ³log 30 = ³log (45 × 54 : 30)

                                            = ³log 81

                                            = ³log 3⁴

                                            = 4

Jadi nilai dari ³log 45 + ³log 54 - ³log 30 adalah 4

No 2.

jika ³log 2 = a, maka nilai ⁴log 27     (sifat no 6)

⁴log 27 = [tex]\frac{^{3}log~27}{^{3}log~4}[/tex]

            = [tex]\frac{^{3}log~ 3^{3}}{^{3}log~2^{2}}[/tex]

            = [tex]\frac{3}{2~. ^{3}log~2}[/tex]

            = [tex]\frac{3}{2a}[/tex]

Jadi nilai ⁴log 27 adalah [tex]\frac{3}{2a}[/tex]

No 3.

jika ⁵log 2 = p, maka nilai ⁵log 16    

⁵log 16 = ⁵log 2⁴

           = 4 . ⁵log 2

           = 4a

Jadi nilai ⁵log 16 adalah 4a

30. Nilai dari ²Log 2 + ²Log 6-²Log 3 ​


~Logaritma

____________

Nilai dari ²log 2 + ²log 6 - ²log 3 adalah 2

---

Pendahuluan

Logaritma adalah invers dari perpangkatan, dengan kata lain logaritma adalah kebalikan dari perpangkatan atau eksponen.

Bentuk umum logaritma adalah ᵃlog b = n ⇒ aⁿ = b

dengan syarat:

a, b, n ∈ Ra > 0a ≠ 1b > 0

Keterangan:

a → bilangan pokok alias basisb → numerusn → logaritma

Berikut beberapa sifat logaritma:

ᵃlog b + ᵃlog c = ᵃlog (b · c)ᵃlog b - ᵃlog c = ᵃlog (b ÷ c)ᵃlog a = 1ᵃlog bⁿ = n · ᵃlog bᵃlog b · ᵇlog c = ᵃlog c

---

Pembahasan

DiketahuiBentuk logaritma: ²log 2 + ²log 6 - ²log 3

Ditanya

Hasil

Jawab

Cara Pertama

= ²log 2 + ²log 6 - ²log 3

Gunakan sifat ⁿlog n = 1 pada ²log 2, dan sifat logaritma ᵃlog b - ᵃlog c = ᵃlog (b ÷ c) pada ²log 6 - ²log 3

= 1 + ²log (6 ÷ 3)

= 1 + ²log 2

Gunakan kembali sifat ⁿlog n = 1 pada ²log 2

= 1 + 1

= 2

---

Cara Kedua

= ²log 2 + ²log 6 - ²log 3

Gunakan sifat logaritma bahwa ᵃlog b - ᵃlog c = ᵃlog (b ÷ c) dan ᵃlog b + ᵃlog c = ᵃlog (b × c)

= ²log (2 × 6 ÷ 3)

= ²log (12 ÷ 3)

= ²log 4

= ²log 2²

= 2(²log 2)

= 2(1)

= 2

∴ Kesimpulan

Jadi, Nilai dari ²log 2 + ²log 6 - ²log 3 adalah 2
