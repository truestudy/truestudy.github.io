• Jawaban pendek:

Jelaskan langkah langkah menyimpan dokumen (berkas) dengan nama yang berbeda pada microsoft word dengan mengunakan menu file tab ribbon?

Untuk menyimpan dokumen dengan nama yang berbeda, harus menggunakan Save As.

Cara melakukan Save As dengan menu Tab File adalah:

1. Klik menu Tab File

2. Klik submenu Save As

3. Atur tempat penyimpanan file, nama file, serta format file dokumen yang diinginkan

4. Klik tombol Save

5. File dokumen telah disimpan dengan nama file baru yang telah diatur

8. tombol manajemen berkas dokumen pada microsoft office excel berbentuk bulat dengan logo di tengahnya disebut?


Pada soal yang ditanyakan masih terlalu umum, namun penulis memahami bahwa manajemen berkas yang dimaksud adalah bagian tab File. Sehingga yang berbentuk bulat dan logo di tengahnya adalah "Recent Workbooks".

Pembahasan

Di dalam Microsoft Excel, manajemen berkas terletak pada tab "File", dimana bagian dari File tersebut adalah:

Info: Adalah info dasar mengenai file yang sedang dibukaNew: Untuk membuat halaman baru.Open: Untuk membuka file excel, termasuk di dalamnya adalah Recent Workbooks yang bentuknya bulat dan berlogo.Save As: Untuk menyimpan file.Print: Untuk Print out document.Share: Untuk berbagi file.Export: Untuk mengekspor file ke bentuk lainnya.Close: Untuk menutup dokumen.

Microsoft excel adalah bagian dari Microsoft Office yang fungsi utamanya adalah untuk melakukan pengolahan angka dan data.

9. Apa saja jenis berkas pembuluh? Kenapa bentuk berkas pembuluh berbeda beda? Apa perbedaan fungsi antara berkas berkas pembuluh itu? Plissss kasih contoh alasan juga kenapa suatu tumbuhan punya berkas pembuluh itu


berkas pembuluh pada manusia

- berkas pembuluh arteri

berfungsi untuk mengalirkan darah keluar jantung

- berkas pembuluh vena

berfungsi untuk mengalirkan darah kembali masuk ke jantung

- berkas pembuluh getah bening / limfe

berfungsi untuk menjaga kekebalan tubuh

- berkas pembuluh saraf

berfungsi untuk menyampaikan rangsangan ke otak


berkas pembuluh pada tumbuhan :

- berkas pembuluh xilem

berfungsi untuk menyalurkan air dari akar menuju daun

- berkas pembuluh floem

berfungsi untuk menyalurkan hasil fotosintesis dari daun ke seluruh tubuh tumbuhan


jawaban dari pertanyaan :

- apa saja jenis berkas pembuluh ?

jenis berkas pembuluh manusia dengan tumbuhan terdapat pada penjelasan dan keterangan di atas

- mengapa bentuk berkas pembuluh berbeda - beda?

bentuk setiap berkas itu berbeda - beda tergantung fungsinya. contohnya pembuluh vena memiliki banyak katup yang bertujuan agar darah tidak tumpah pada organ - organ tubuh

- apa perbedaan fungsi antara berkas - berkas pembuluh itu?

perbedaan fungsi pada jenis - jenis berkas pembuluh manusia dengan tumbuhan terdapar pada penjelasan di atas

- mengapa suatu tumbuhan punya berkas pembuluh itu?

tumbuhan mempunyai berkas pembuluh xilem dan floem adalah untuk memenuhi kebutuhan si tumbuhan itu sendiri. air yang dialirkan oleh xilem akan diolah dan dicampur dengan hasil fotosintesis oleh daun, setelah selesai hasil dari fotosintesis diedarkan ke seluruh tubuh tumbuhan oleh pembuluh floem
