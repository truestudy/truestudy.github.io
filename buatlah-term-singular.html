JAWABANTHERE IS.... ✔

1. There is a pen / Is there a pen?

2. There is an erraser / Is there an erraser?

3. There is a doll / Is there a doll?

4. There is a man / Is there a man?

5. There is a cat / is there a cat?

THERE ARE.... ✔

1. There are two boys / Are there two boys?

2. There are three eggs / Are there three eggs?

3. There are two bottles / Are there two bottle?

4. There are four bags / Are there four bag?

5. There are ten balls / Are there ten balls?

PENJELASAN

Translate :

THERE IS....

1. Ada pulpen / Apakah ada pulpen?

2. Ada penghapus / Apakah ada penghapus?

3. Ada boneka / Apakah ada boneka?

4. Ada seorang pria / Apakah ada seorang pria?

5. Ada kucing / ada kucing?

THERE ARE....

1. Ada dua anak laki-laki / Apakah ada dua anak laki-laki?

2. Ada tiga telur / Apakah ada tiga telur?

3. Ada dua botol / Ada dua botol?

4. Ada empat tas / Apakah ada empat tas?

5. Ada sepuluh bola / Apakah ada sepuluh bola?

=======================================

• There is... = untuk benda yang jumlahnya hanya satu

• There are... = untuk benda lebih dari satu

Rumus Positive sentence :

There is + singular nounThere are + plural nounThere is + uncountable noun

Rumus Negative sentence :

There is + not + singular nounThere are +not+any+ plural nounThere is +not + any + uncountable noun

Rumus interrogativs sentence :

Is + there + singular noun?Are + there + any + plural noun?Is + there + any + uncountable noun?

   Contoh kata benda Uncountable Noun (tidak bisa dihitung): Water, air, Oil, pepper, sand, rice, sugar, love, assistance, coffee, cheese, fun, advice,  money, music, tea, art, salt, sadness.    singular countable thing: i have a pen. she has a blue bag. plural countable thing: our classroom has twenty chairs and ten tables. there are two blackboards on the wall.

9. buatlah kalimat simple present singular 3 dan plural 2 tentang budaya​


Jawaban:

Simple present tense, singular subject:

1. Indonesia has more than 700 regional languages.

2. Bahasa Indonesia is the official language of Indonesia.

3. Indonesia consists of hundreds of distinct native ethnic groups.

Simple present tense, plural subject:

1. Saman and Legong dances are originally from Indonesia.

2. People usually wear traditional clothes in traditional ceremonies.

Penjelasan:

Simple present tense

singular object: verb+s/es

plural object: verb not modified

30. 4. Another term for "uncountable noun"isA. plural nounB. mass nounC. countless nounD. counted nounE. singular noun​


Jawaban:

C. countless noun

Penjelasan:

Terjemahan soal : Istilah lain dari "kata benda tak terhitung" adalah...

                              A. bentuk jamak kata benda

                              B. kata benda massal

                              C. kata benda yang tak terhitung jumlahnya

                              D. kata benda terhitung

                              E. kata benda tunggal

Jadi,jawabannya C. countless noun.
